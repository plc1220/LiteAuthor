先给每轮互动补上“前提、意图、反应、结论”。**不能推动剧情或关系的互动，写得再多也只是热闹。**

## 适用情况
- 感情戏平，角色一直聊天却没火花
- 日常戏像灌水，对话来回绕圈
- 同一种互动反复出现，读起来很空
- 明明写了很多相处，关系就是不动
- 配角、主角都在交流，但情节没有任何推进

## 输入
- 参与互动的人物
- 当前关系阶段
- 当前情绪或矛盾（冷战 / 吃醋 / 试探 / 求助 / 误会等）
- 这场戏希望推进什么
- 场景类型（吃饭 / 上课 / 任务 / 争执 / 约会等）

## 步骤

### 1. 先写这轮互动的前提
没有前提，互动就容易空转。常见前提：
- 冷战
- 吃醋
- 刚动心但不肯承认
- 有求于人
- 刚被误会
- 想试探对方态度
- 必须一起完成某件事

### 2. 分开写双方意图
每个人都要有自己的目的，不要只写“他们在互动”。

示例：
- A 送早饭，意图是缓和关系
- B 拒绝，不是单纯说“不”，而是因为还在吃醋或不想轻易和好

### 3. 让反应带着当前关系状态
同一句台词，在不同前提下意义不同。

例如“不要”：
- 平时＝真的不要
- 冷战时＝别来哄得这么敷衍
- 动心时＝嘴硬，实际在等更进一步
- 吃醋时＝想到旧事，情绪被点炸

**反应必须带历史，不要像按键回话。**

### 4. 一轮互动结束后，必须有结论
结论不是“读者觉得甜”，而是剧情里真的变了什么。常见结论：
- 关系推进半步
- 冷战加深或缓和
- 一方察觉自己动心
- 新误会被埋下
- 某个信息被确认
- 下一场冲突有了入口

如果什么都没变，这轮互动就是无效互动。

### 5. 用四格表检查是不是白写
把一轮互动拆成 4 格：
1. 前提
2. 意图
3. 反应
4. 结论

四格里只要缺一格，通常就会空。

### 6. 重复互动可以，重复意义不行
同样是“送早饭”“要抱抱”“一起走”，每次都要因为前提不同而得到新结果。

变化方向可以是：
- 从试探到确认
- 从暧昧到误会
- 从冷战到和好
- 从拒绝到松动
- 从普通照顾到明确偏爱

### 7. 克制型人物优先用动作，不要全靠心理活动
尤其正经、禁欲、嘴硬角色，先写：
- 停顿
- 反常的小动作
- 改掉平时习惯
- 帮忙却不认
- 想靠近却嘴硬否认

**动作越细，互动越真。**

### 8. 配角互动可以稍轻，但信息不能无效
配角和配角的互动不一定每次都强推主线，但必须至少完成一项：
- 交代关系变化
- 提供后续冲突入口
- 传递有效信息

篇幅一长，就不能只是闲聊。

## 输出格式
1. 场景前提
2. A 的意图 / B 的意图
3. A 的反应 / B 的反应
4. 本轮结论
5. 埋下的下一步
6. 如果删掉这轮互动，会损失什么

## 快速示例
### 输入
- 关系：暧昧期
- 场景：送早饭
- 当前前提：B 在吃醋

### 输出
- 前提：B 想到 A 也给别人带过早饭，心里别扭
- A 的意图：缓和关系，进一步靠近
- B 的反应：先拒绝，再被追问时明显炸毛，不是平时那种冷淡拒绝
- 结论：A 察觉 B 在意自己，B 也暴露了占有欲
- 下一步：可以接撒娇、解释、冷战升级或反向试探
