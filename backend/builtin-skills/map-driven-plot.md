先画最小可用地图，让地点关系替剧情出题。**地图不是摆设，而是剧情生成器。地点一旦有了距离、路线、功能和气质，偶遇、冲突、回访、高光都会更自然。**

## 适用情况
- 已有城市、校园、宗门、村镇、片场、州郡等空间设定，却不知道地点怎么长出剧情
- 感情线需要更多生活气和偶遇梗
- 日常戏总在室内聊天，场景单调
- 想让同一个地点反复出现，形成时间感和情感见证
- 写古言 / 架空文时，州郡、帝都、府邸都想写，但空间关系很模糊

## 输入
- 故事地图或地点清单
- 主要角色的身份、职业、习惯、弱点、特殊能力
- 当前关系阶段
- 主线剧情推进到哪里
- 想补的内容：日常 / 暧昧 / 冲突 / 高光 / 回收

## 步骤
### 1. 先画最小可用地图
至少列出 5—10 个高频地点，并标明它们的功能。优先包含：
- 主线故事发生区
- 容易引起共鸣的常见区
- 容易激发想象的地标区
- 配合剧情出现的小功能建筑

现代文常见示例：
- 住宅区
- 工作地 / 学校 / 电视台
- 通勤路
- 吃饭地
- 公园 / 夜市 / 大学城 / 商圈
- 特殊地标
- 餐馆、咖啡店、宠物店等小建筑

### 2. 古言 / 架空文先建立“地图意识”
古言和架空不一定要先收集现成地图，但一定要知道人物能去哪里、去不了哪里。

至少先想清 2—3 张图：
- 主角 A 所在州郡
- 主角 B 所在州郡 / 封地 / 门派
- 两人共同会相逢的大舞台，例如帝都、都城、主宗门

如果某个角色活动受限，地图要细到：
- 府邸内部
- 院落、小河、庭楼、花园
- 他每天能看见的重复风景

**地图越清楚，人物的孤独、局限、阶层和生活感就越容易写出来。**

### 3. 给每个地点补一句“这里最容易发生什么”
不要只写“这里很美”。给每个地点补一句功能：
- 这里最容易偶遇谁
- 这里最容易出什么意外
- 这里最适合写暧昧、追逐、误会还是高光
- 这里最能暴露谁的人设

### 4. 写角色的生活轨迹或活动半径
分别写主角 A / B：
- 平日工作路线
- 休息日路线
- 最常去的 3 个地方
- 不想去但不得不去的地方
- 最容易暴露人设的地点

用 `起点 → 必经点 → 停留点 → 返回点` 写出来。

### 5. 从单个地点往外推“单项联想”
每个地点至少从 4 个方向发散：
- **日常**：这里最容易发生什么小事
- **关系**：两人如何自然碰上、错过、互相记住
- **职业 / 身份**：工作、名气、能力、秘密会怎样在这里出问题或出彩
- **人设**：这里最能放大角色哪个习惯、弱点、萌点或反差

**不要只想“能发生什么”，还要想“为什么偏偏在这里发生”。**

### 6. 做地点联动表
把两个地点两两配对，强行问一遍：
- A 地点的人会不会跑到 B 地点
- 从 A 到 B 的路上会不会出事
- 两个地点放在一起，能不能制造信息差、追逐、救场、误会或公开高光

常用联动：
- 住宅区 × 公园
- 工作地 × 老地标
- 学校 × 商圈
- 片场 × 大学城
- 帝都 × 州郡来客
- 府邸 × 节日集市

### 7. 给地点叠加“特殊事件”
在高频地点上覆盖一次事件：
- 比赛 / 活动 / 节日 / 校庆
- 事故 / 火灾 / 停电 / 封路
- 热搜 / 围观 / 偷拍 / 节目录制
- 限时活动 / 限量礼物 / 签售 / 集会
- 失物、抓贼、误闯、捡到东西、发现秘密

事件要满足两条：
1. 和地点本身有关
2. 能放大角色的人设或关系

### 8. 用环境差异竖人设
地图不只用来排路线，也用来做对比。

可以对比：
- 花红柳绿的繁华州郡 vs 单调封闭的府邸生活
- 热闹节日民风 vs 冷清压抑的权力空间
- 谁爱兰花、谁养獒犬、谁数屋里地砖、谁只看得到院中一排竹子

这些差异能顺手带出：
- 身份与处境
- 性格和审美
- 长期孤独或长期自由
- 两个角色世界观的反差

### 9. 设计“回归式重复”
挑 1—3 个有纪念意义的地点，安排在不同关系节点回访：
- 初见 / 第一次动心
- 暧昧 / 约会 / 确认关系
- 冷战 / 分离 / 和好
- 告白 / 求婚 / 收尾

每次回去都要至少变 1 样：
- 季节变了
- 身份变了
- 关系变了
- 看同一处风景的心情变了
- 同一动作被赋予了新意义

### 10. 最后挂回主线
把生成的地点梗再筛一遍，只保留能承担至少 1 项功能的场景：
- 推进感情
- 暴露人设
- 回收伏笔
- 增加生活气
- 制造冲突
- 给主角发高光

**主线不清的时候，地图联动只会越写越散。**

## 输出格式
1. 地图地点清单
2. 角色 A / B 活动轨迹
3. 单地点联想表
4. 地点联动梗 6—10 条
5. 特殊事件 3—5 条
6. 用环境差异竖人设的点
7. 回归式重复地点 1—3 组
8. 最终保留的可写场景

## 快速示例
### 输入
- 地点：住宅区、中心公园、早餐街、电视台、古建筑、大学城
- 角色 A：晨跑爱好者，公众人物，平时要躲媒体
- 角色 B：夜间工作多，常在清晨回家，嘴硬洁癖

### 输出
- 活动轨迹：
  - A：住宅区 → 中心公园 → 早餐街 → 返家
  - B：电视台 → 夜市路口 → 住宅区
- 联动梗：
  1. A 晨跑伪装出门，在早餐街被 B 连续偶遇
  2. 电视台车辆常去古建筑附近录节目，A / B 在同地多次出意外
  3. 大学城丢窃案一路跑进公园，A 当场制止，顺带上热搜
- 回归式重复：
  - 初见在湖边
  - 暧昧期一起夜走湖边
  - 冷战时一人独自回到湖边
  - 求婚时再次回到湖边

## 注意事项
- **禁止**把地图当摆设，只写“这里很美”
- **禁止**所有地点都只负责偶遇，没有后果
- **禁止**为了水场景硬塞无关地点
- **禁止**把州郡、帝都、府邸全都写上桌，却不区分前台和背景板
- 同一个地点反复出现时，**关系和意义必须升级**
