先把 CP 做成**一眼能懂、落笔就有火花**的一对，再补层次。**网文里的 CP，先要好记、好写、能起张力，不必一上来就追求两个人都极其复杂。**

## 适用情况
- 原创 CP 怎么写都不带感
- 同人转原创，不知道怎么把喜欢的角色感觉迁移过来
- 两个主角都写得很完整，却没有火花
- 角色只有标签，没有具体互动味道
- 想写热门人设组合，但一写就像空壳

## 输入
- 题材与目标风味（甜、爽、拉扯、救赎、强强、年下等）
- A / B 当前设想
- 想让读者最先吃到的感觉
- 可参考的同人角色、热门原型或现成梗（可选）

## 步骤
### 1. 每个人先只抓 1—2 个强标签
不要一上来给完整人物传记，先写读者第一眼最容易记住的点。

常用来源：
- 热门原型：乖、甜、狼狗、冰山、霸道、钓系、傲娇、温柔、疯批、清冷、嘴硬
- 关系原型：年上年下、强强、宿敌、救赎、先婚后爱、双向暗恋
- 读者熟悉的刻板印象：团宠小太阳、闷骚大佬、活泼话痨、温良少爷、阴险老狐狸

**先抓“容易记住的壳”，再补能站住的芯。**

### 2. 把标签翻译成可写行为
不要只写“乖”“冰山”“霸道”，要继续拆成：
- 这个标签在互动里会怎么说话
- 会怎么主动 / 被动
- 会怎么露出萌点
- 最容易写崩成什么

示例：
- **乖 / 甜**：认真、直球、被撩不自知、容易团宠、看着好欺负但不一定真软弱
- **冰山 / 闷骚**：对别人冷，对目标有例外；重点不是“没反应”，而是**克制下的偏爱和保护欲**
- **狼狗 / 暴躁**：主动、固执、追人时有少年气；重点不是乱吼，而是行动力强
- **霸道**：重点不是高高在上，而是会接管局面、宣示主权、在关键时刻替对方做决定或撑场

**写得带感的关键，不是标签本身，而是标签在场景里怎么动。**

### 3. 如果参考同人角色，只取“读者最吃的那一面”
从同人转原创时，不要整个人照搬，也不要把原作的全部复杂性硬搬过来。

更稳的做法：
1. 先找出读者最常记住、最常磕的那一面
2. 把这层印象压成原创角色的强标签
3. 再补 1 个能防止太脸谱化的深层点

例如：
- 不是照搬完整原著人格
- 而是提炼出“阴险老狐狸 × 活泼天然直球”
- 再各补一个真正影响选择的底层动机

**网文里先有辨识度，再谈立体度。**

### 4. 这对 CP 必须先有“为什么会起火花”
从下面至少选 1—2 条：
- 性格互补：一个稳，一个跳；一个克制，一个直球
- 权力差 / 气场差：一个控场，一个搅局
- 目标冲突：一个想靠近，一个不让靠近
- 需求互补：一个缺安全感，一个天生会给安全感
- 互动反差：对别人都一样，对对方不一样

用一句话写清：
`这对 CP 最带感的地方是____。`

### 5. 一定要有一方主动，或被迫主动
**两个人都不主动，故事根本启动不了。**

主动不一定是表白，也可以是：
- 明确追人
- 总是制造接触
- 因任务、婚约、治病、合作而被迫靠近
- 天性外放，天然会打破距离
- 为了求生、求助、报恩、抱大腿，不得不先迈步

如果 A、B 都是被动、克制、等对方开口，就要补一个“谁先动”的机制。

### 6. 用“秘密规则”制造长期张力
主角二人里，最好有一方是**相对坦诚的一张白纸**，另一方是**对关系有秘密的人**。

这个秘密可以：
- 读者知道，对方不知道
- 读者和对方都不知道，后面再揭
- 是身份、来意、旧事、暗恋、重生、任务、立场、病情、真实目的

常见好用写法：
- 他喜欢她 / 他重生了 / 他早就认识她，但她不知道
- 她接近他另有目的，但读者知道她的苦衷，他不知道
- 读者知道某人迟早掉马，另一个主角不知道

**秘密是矛盾核心。没有秘密的 CP，也能甜，但很难长期拉扯。**

### 7. 给两人分配“关系职能”
不要只写性格，要写这两个人在关系里各自负责什么：
- 谁负责主动推进
- 谁负责制造距离
- 谁负责提供安全感
- 谁负责制造危险感 / 新鲜感
- 谁更会装，谁更容易露馅
- 谁更清楚局势，谁更像一张白纸

这样互动才不是谁都能替换的模板场。

### 8. 先设计 3 场证明戏，再去写长线
至少给这对 CP 准备 3 类场面：
1. **初见 / 初次强记忆点**：让读者一下抓住这对组合的味道
2. **主动推进场**：谁先动、怎么动、另一个人怎么接
3. **秘密施压场**：因为秘密、误判、身份差、信息差，关系被拧紧

没有证明戏，标签永远只是标签。

### 9. 做“换人测试”
把 A 或 B 换成另一种常见性格，再问：
- 这场互动还会不会几乎原样成立
- 台词、动作、气氛是不是只改几个词就能继续用

如果答案是“会”，说明这对 CP 还不够贴人。

## 输出格式
1. A / B 的 1—2 个强标签
2. 每个标签对应的可写行为
3. 这对 CP 的核心带感点
4. 谁主动 / 为什么主动
5. 谁有秘密 / 秘密是什么 / 读者是否知道
6. 关系职能分配
7. 3 场证明戏
8. 最容易写崩的点

## 快速示例
### 输入
- 题材：电竞甜文
- 目标：写乖甜女主 × 冰山闷骚男主

### 输出
- 女主强标签：乖、甜
- 女主可写行为：认真、直球、容易让人想照顾、会在不自觉时撩人
- 男主强标签：冰山、闷骚、强控场
- 男主可写行为：对别人淡，对女主会额外留意、保护、替她兜底
- 核心带感点：她天然靠近，他明明克制却越来越管不住自己
- 主动方：女主天然会先说先做，男主在关键节点主动接管局面
- 秘密：男主空降战队其实是因为喜欢女主；读者知道，女主不知道
- 证明戏：初见时女主给他暖宝宝；比赛中男主替女主扛压；秘密快露时两人对彼此判断彻底改变

## 注意事项
- **不要**给两个人都写成高冷闷葫芦
- **不要**只给标签，不给行为和场面
- **不要**把“冰山”写成木头，把“霸道”写成没礼貌
- **不要**从同人搬整个人，只搬读者最吃的核心感觉
- **不要**两个人都没有秘密、没有主动、没有主导差
- CP 带感的本质不是设定多，而是：**好记 + 有火花 + 能长出剧情**
