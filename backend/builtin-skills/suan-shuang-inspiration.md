先别急着判断“这桥段合理不合理”。先从让人忍不住吐槽、转发、求文名的离谱内容里，提炼真正让人上头的记忆点，再把它改写成可控的爽点、笑点或反派操作。**目标不是照抄雷文，而是抓住“为什么它让人忘不掉”。**

## 适用情况
- 剧情太顺、太正、太像标准模板
- 想写狗血、神展开、黑色幽默，但怕翻车
- 角色总按最安全的套路做选择
- 有题材卖点，但桥段不够让人记住
- 反派不够疯，主角不够带感

## 输入
- 当前题材、主角定位、读者预期
- 想要的酸爽方向：狗血 / 猎奇 / 打脸 / 黑色幽默 / 暗黑反派 / 神展开
- 可用人物：主角 / 配角 / 反派
- 不能碰的尺度与底线
- 当前卡住的剧情节点（可选）

## 核心原则
1. **先找记忆点，再谈包装。**
2. 离谱桥段必须对应角色的欲望、恐惧、执念或恶念，不能只靠作者硬发疯。
3. **让人物按性格做选择，不按攻略做选择。**
4. 主角不适合做的事，可以交给反派、配角、世界规则或支线事件去做。
5. 高风险桥段最好安排一个“正常人视角”去吐槽、质疑、破坏或反击，避免作者和角色一起犯蠢。
6. **不是所有离谱都能成为酸爽。** 没有后果、没有反馈、没有回收的离谱，只会变成纯雷。

## 步骤

### 1. 从三个源头找材料
优先从下面三类素材中找 5—10 个候选点：

#### A. 雷文 / 吐槽帖 / 离谱名场面
记录：
- 哪个桥段最让人想挂论坛
- 它为什么能让人记住
- 读者是在“气”“笑”“震惊”还是“停不下来”

#### B. 游戏 / 互动剧情 / 多选分支
记录：
- 同一局面有哪些不同选法
- 不同选法背后暴露了什么性格
- 哪个选择最不按常理，但最有戏

#### C. 现实里不敢做、不能做、说不出口的欲望
记录：
- 想报复却不能报复
- 想发疯却必须体面
- 想越界却被道德 / 身份 / 规则拦住
- 想要极端结果却不敢真的承认

**这三类素材最值钱的地方，不是“离谱”，而是它们会暴露人真正想要什么。**

### 2. 把每个素材拆成 4 栏
对每个候选点，逐条写：
1. 表面离谱点  
2. 读者会产生的情绪：猎奇 / 爽 / 气 / 笑 / 后怕  
3. 背后的欲望、恐惧或禁忌  
4. 最值得保留的那一下

示例：
- 表面离谱点：主角一醒来就在刑场
- 情绪：猝不及防的紧张
- 背后机制：身份带来的立刻危机
- 必保留：开局就没有缓冲，必须马上想办法活

### 3. 做“同一局面，多人分叉”
挑一个关键场面，分别写主角、配角、反派会怎么选。

必须至少拉开 3 种路线：
- 最稳妥的选法
- 最符合本人物性格的选法
- 最有戏但仍然站得住的选法

如果所有人最后都走同一条路，说明人物还没长出区别。

可直接使用下列追问：
- 这个人最怕失去什么
- 这个人最想证明什么
- 这个人会不会为了爽、为了赢、为了恶念，选一个更极端的办法
- 如果这个人不按常规出牌，最可能干什么

### 4. 决定离谱点该交给谁来做
不是每个酸爽点都该落在主角身上。

#### 适合给主角
- 需要打破僵局
- 需要让主角显得带感、敢做、出乎意料
- 需要主角亲手拿回主动权

#### 适合给反派
- 主角做会毁基本盘
- 需要兑现恶感、疯狂、禁忌和压迫
- 需要把世界补完整：不是只有主角会狠，恶也真的存在

#### 适合给配角 / 支线
- 需要先试水高风险桥段
- 需要制造荒诞气氛
- 需要让主角作为见证者、吐槽者或回收者出手

### 5. 给高风险桥段加“安全阀”
下面至少选 1—2 个：
- 安排正常角色现场吐槽
- 安排明确代价和报应
- 让桥段落在配角 / 反派身上，不碰主角底线
- 用反讽、喜剧或黑色幽默包裹
- 让主角看穿荒唐逻辑，并顺势打脸
- 在桥段后立刻回收局势变化，而不是让离谱白发生

**如果桥段的效果只是“作者和角色一起发病”，就重做。**

### 6. 把素材改写成“酸爽桥段卡”
按下面格式生成 3—5 个可写桥段：
- 正常局面是什么
- 谁先做了离谱选择
- 这个选择暴露了什么欲望 / 恶念 / 执念
- 周围人怎么反应
- 谁负责吐槽 / 见证 / 加码
- 最后怎样回收到爽点、笑点、恶感或打脸
- 风险是什么，怎样降风险

### 7. 反查是不是“只剩雷，没有爽”
逐条检查：
- 删掉离谱外壳，这段还有人物逻辑吗
- 这个桥段会改写关系、局势或人物评价吗
- 主角是不是仍然站得住
- 有没有一个清楚的情绪方向：爽 / 笑 / 怒 / 惊
- 读者会记住的是“作者疯了”，还是“这桥段真有劲”

## 输出格式
1. 当前题材与读者预期
2. 5 个酸爽素材候选
3. 每个素材的 4 栏拆解
4. 关键场景的多人物分叉选择
5. 推荐保留的 3 个酸爽桥段卡
6. 每个桥段的安全阀
7. 风险最高的点与修正建议

## 快速示例
### 输入
- 题材：快穿打脸
- 方向：想写离谱男主逻辑被主角反讽

### 输出
- 素材：男主把自我感动当爱情，把受害者当理应牺牲的一方
- 背后机制：把荒唐价值观说得理直气壮，会让读者又气又想笑
- 改写：先让配角说出离谱台词，再让主角当场拆逻辑、翻旧账、反手打脸
- 安全阀：离谱言论不当真理写，必须被吐槽、被反制、被回收

## 注意事项
- **禁止**直接照搬雷文桥段
- **禁止**为了猎奇牺牲人物逻辑
- **禁止**主角和作者一起默认荒唐逻辑
- **禁止**只有离谱，没有后果和反馈
- 酸爽真正值钱的地方，不是“怪”，而是**怪得让人记住，且记住之后情绪有去处**。
