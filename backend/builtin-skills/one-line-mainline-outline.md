把一个梗或一句话主线，扩成“主线四格 + 故事骨架 + 主角轨迹 + 粗纲”。先钉死能贯穿到底的主线，再去补设定、支线和人物。

## 适用情况
- 只有一句梗，不会扩成长篇
- 文案有了，正文不知道怎么推
- 感情线、剧情线、单元剧不知道怎么串
- 想先做粗纲，再细化章节

## 输入
- 一句话梗或文案
- 题材与篇幅
- 主角 / CP
- 当前最想写的体验：甜、虐、爽、悬念、成长等

## 步骤

### 1. 先写主线四格：起因 / 目标 / 经过 / 结果
先别急着补世界观、人物表、支线设定，先把这四格写清：
- **起因**：故事为什么会开始
- **目标**：主角当下最明确想达成什么
- **经过**：为了做到它，至少要经历哪几次关键阻碍或转折
- **结果**：故事最后或阶段最后会落到什么局势

模板：
`因为____，主角决定/不得不____；过程中____；最后____。`

**只有一条能贯穿到底的主线，才能支撑支线。** 如果这四格还写不顺，先别继续堆设定。

#### 快速示例
- 起因：XX 死后发现自己是书里的恶毒配角，结局很惨，于是重生了
- 目标：远离原书男女主，过好自己的生活
- 经过：在远离过程中结识反派、发展事业，原书主角反复找茬
- 结果：主角改写命运，收获事业和感情，原书主角自食其果

### 2. 把四格压成一句话主线
用下面格式写一句：
`谁为了____，做了____，途中遇到____，最后____。`

要求：
- 至少写出主角、目标、阻力、结果中的 3 项
- **突出萌点或爆点**，不要把一句话写成设定说明书
- 如果是双主角，优先写最能代表全文的那条主线

### 3. 扩成五句故事骨架
按固定格式写 5 句：
1. 背景 / 起点
2. 灾难 1
3. 灾难 2
4. 灾难 3
5. 结尾

这 5 句以后也可以直接改成简介雏形。

### 4. 让三个灾难递进
三个灾难**必须是递进关系，不是并列清单**。

检查标准：
- 每个灾难都比前一个更难收拾
- 每个灾难都会改变关系、目标或局势
- 后一个灾难必须由前一个灾难推出来
- 如果删掉灾难 3，结局基本不变，重写

用这个句式自检：
`因为灾难1，人物/局势变成____，所以引出灾难2；因为灾难2，____，所以引出灾难3。`

### 5. 按题材理解“灾难”
#### 爱情线
把灾难理解为关系变化点。每次灾难后，两人的关系都要更复杂：误会、越界、吃醋、失去、公开站队、彻底决裂等。

#### 剧情线
直接写主线危机。灾难要能推动升级、调查、复仇、打脸或破局，而不是单纯倒霉。

#### 单元剧
写串联所有单元的**主线灾难**，不是把单元故事一条条摆出来。单元的影响要积累，最后在主线上爆发。

### 6. 给主要人物做轨迹表
至少给主角 / CP 各写一份：
- **人物动机**：他真正想要什么
- **人物目标**：为了得到它，他准备怎么做
- **人物冲突**：谁或什么在拦他，尽量写宽一点
- **人物感悟**：走完这条线后，他学会了什么
- **一句话人物线**：`他为了____，做了____，遇到____，最后明白____。`

如果是双主角，允许一人主扛剧情线，另一人主扛感情变化线。

### 7. 把人物线挂回主线
逐条检查：
- 故事骨架里的每个灾难，是否都推进了至少一位主角的人物线
- 人物动机是否足以解释他在灾难里的选择
- 人物感悟是否真是剧情逼出来的，而不是结尾硬写感想

### 8. 扩成粗纲
把五句骨架逐句扩成一段，每段至少写清：
- 这段谁想达成什么
- 被什么打断
- 结果如何改变下一段
- 这一段主要推进哪条线：剧情 / 感情 / 成长

### 9. 最后做一次删改
优先删掉：
- 不能推主线的设定说明
- 不能升级灾难的重复桥段
- 不能改变人物轨迹的空转事件

## 输出格式
1. 主线四格（起因 / 目标 / 经过 / 结果）
2. 一句话主线
3. 五句故事骨架
4. 主角 A 人物轨迹
5. 主角 B 人物轨迹（如有）
6. 灾难递进链
7. 粗纲段落摘要

## 示例
### 输入
- 梗：分手前夜，女主想和男主和好，却发现他一直在隐瞒一件大事

### 输出
- 主线四格：
  - 起因：两人冷战已久，女主准备在最后一次见面里把话说开
  - 目标：女主想弄清男主到底隐瞒了什么，并决定是否继续这段关系
  - 经过：男主失约、失联、出现却仍不解释，矛盾层层升级
  - 结果：真相揭开，两人重新定义关系
- 一句话主线：女主想挽回关系，却在一次次失望里逼出真相，最终决定是否原谅男主。
- 五句骨架：
  1. 两人冷战后约定最后见一面。
  2. 男主迟到失约，女主先压着火等他。
  3. 男主失联、电话不接，女主开始怀疑他另有隐情。
  4. 男主终于出现却仍不肯解释，女主当场爆发提出分手。
  5. 真相揭开，两人决定重新定义这段关系。
- 女主人物轨迹：
  - 动机：想保住这段感情，也想弄明白自己到底值不值得被坦诚对待
  - 目标：逼男主说真话
  - 冲突：男主隐瞒、旧矛盾、信任崩塌
  - 感悟：真正的和好不是忍下去，而是先看清问题

## 注意事项
- **禁止**先补设定补到停不下来，再回头硬套主线
- **禁止**把三个灾难写成三个差不多的重复受气点
- **禁止**只有事件，没有人物轨迹
- **禁止**把支线、设定、配角戏堆得比主线还清楚
- 主线越短，扩写时越要抓“递进”和“变化”
