写人物时，读者会把角色和反复出现的词、场景、道具、气质绑定在一起。**如果总把负面词和负面环境往角色身上堆，角色就会跟着一起发灰；如果必须写缺点，也要同步绑定力量感、脆弱美感、秩序感、例外感等正向印象。**

## 适用情况
- 角色设定不差，但一落笔就招烦
- 病弱、短命、残疾、不良、暴躁等设定写不出魅力
- 角色总在负面环境里出现，读者记不住优点
- 同样的设定，别人写得带感，自己写得掉价

## 输入
- 角色现有设定
- 角色当前最容易劝退的点
- 想让读者优先吃到的魅力
- 当前出场或关键场景

## 步骤
### 1. 先定“读者第一反应要记住什么”
从中选 1 个主方向：
- 力量感
- 安全感
- 脆弱美感
- 控制感
- 神秘感
- 少年气
- 反差萌
- 可靠、会照顾人

### 2. 盘点负面设定，分清“保留”和“换说法”
不是把缺点删除，而是换表达重心：
- 短命 → 时间感稀薄、活得锋利、每一次付出都更珍贵
- 残疾 / 病弱 → 脆弱感、美感、自持、让人想保护
- 校霸 / 不良 → 力量感、控制场面、替人扛事
- 暴脾气 → 反应快、护短、真性情（前提是别写成没素质）

### 3. 给角色绑定正面元素
从这些里选 2—4 个：
- 场景：花、雪、光、干净房间、秩序感环境、强势出场环境
- 道具：书、琴、刀、伞、眼镜、宠物、礼物、手套等
- 行为：护短、补位、沉得住气、照顾弱者、自己扛事
- 别人视角：被依赖、被信任、让场面稳定下来

**不要只让角色反复和脏乱差、抱怨、无能、刻薄绑定。**

### 4. 让正面感大于负面感
每写 1 个负面点，最好补 1—2 个正面回收：
- 有缺点，但有能力
- 有伤病，但有风骨
- 有危险感，但有底线
- 有冷硬，但有例外

### 5. 别反复强调同一个负面词
高危写法：
- 一直写“短命”“虚弱”“脾气差”“阴沉”“邋遢”
- 每次出场都先交代他哪里不行

改法：
- 把结论词换成可见场面
- 把负面点藏进后果里，不天天口头复读
- 让别人先感受到他的作用，再知道他的缺陷

### 6. 用“负面 + 补偿点”做组合
常用组合：
- 病弱 + 超常能力
- 不良 + 保护欲
- 高冷 + 稳定可靠
- 嘴毒 + 关键时刻真补位
- 残缺 + 极强自尊或审美感

### 7. 用场面证明，不用旁白夸
准备 2 场：
1. **绑定场**：让正面元素第一次和角色连上
2. **翻盘场**：当负面设定可能掉价时，用能力、选择或气质把它翻成魅力

### 8. 最后做“换词测试”
把描写里的高频词圈出来：
- 如果删掉名字，只看这些词，读者联想到的是“烦人、没用、灰扑扑”，还是“强、美、稳、危险但迷人”？
- 如果前者居多，就重写。

## 输出格式
1. 角色当前负面点
2. 目标魅力方向
3. 需要绑定的正面元素
4. 负面点的改写方向
5. 两个证明场面
6. 需要删掉的重复负面词
7. 最终读者印象

## 快速示例
### 输入
- 角色：病弱短命的贵公子攻
- 问题：总被写得很丧、很烦、魅力值低

### 输出
- 目标魅力：脆弱美感 + 自持 + 让人心疼的珍贵感
- 正面绑定：常在安静、干净、审美高的环境里出场；待人克制；关键时刻能看穿局势
- 改写方向：少复读“短命”，多写“他明知时间不多，仍把最稳的那一面留给别人”
- 翻盘场：发病后第一反应不是喊痛，而是先把最重要的信息交代完；别人这时才真正意识到他的分量

### 输入
- 角色：校园里的校霸男主

### 输出
- 目标魅力：力量感 + 控场能力 + 例外偏爱
- 正面绑定：解决麻烦、替人挡事、在混乱里最能镇场
- 改写方向：少写“混混”“痞”，多写“场面一乱下来，第一个能让所有人闭嘴的是他”

## 注意事项
- **禁止**把角色魅力全靠形容词硬夸
- **禁止**负面点只负责卖惨，不回收价值
- **禁止**把“正面表达”写成洗白，角色的代价和缺陷仍要存在
- 真正有魅力的角色，不是没有缺点，而是**读者先记住他的亮处，再接受他的暗处**
