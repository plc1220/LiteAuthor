把感情流写成“甜 → 误差 → 矛盾 → 激化 → 回收”的波形。**感情流不是一直发糖，也不是靠大虐硬拖；重点是两个人在学着去爱，关系一直在变化。** 真正能撑住篇幅的波折，来自人物性格、背景、价值观和未磨合处，而不是作者临时想来一刀。

## 适用情况
- 男女主互动很甜，但一直甜下去就空了
- 在一起后只会腻歪，不知道后面写什么
- 想写拉扯，却只会靠误会或强行虐
- 感情线有冲突，但吵完什么都没变
- 不确定自己的线更偏剧情流还是感情流

## 输入
- 双方当前关系阶段
- 双方性格、价值观、创伤点
- 双方成长背景和关键旧事
- 当前外部处境
- 这段希望把关系推到哪里

## 先判断是不是感情流
满足 2 条以上，可按本 Skill 处理：
- 冲突主要来自人物内心、性格、旧伤、价值观
- 剧情从人物背景和关系变化往外长
- 结尾的重点是人物成长、关系升温或关系重构

如果主要冲突来自外敌、升级、案件、任务，本 Skill 仍可用，但只作为感情副线节奏。

## 核心原则
1. **先想人物在想什么，再想这段要不要甜。**
2. **甜本身不是推进，关系变化才是推进。**
3. **波折要控制在不失控的限度里：小虐怡情，大虐伤身。**
4. 每次矛盾都要挖出更深一层的性格或背景问题。
5. 如果这段关系已经没有任何新变化可写，主感情线通常接近完结。

## 步骤
### 1. 找出双方还没磨合好的地方
按下面四类各列 1—2 项：
- 性格：控制欲强、嘴硬、逃避、迟钝、敏感、爱逞强
- 价值观：是否信任共同承担、如何看待责任、自由、牺牲
- 背景：家庭创伤、旧恋情、阶层差、身世、事业压力
- 关系现状：信息差、身份差、公开/不公开、信任度不对等

### 2. 把标签追问成因
不要只写“他不谈感情”“她讨厌被管”“两人都傲娇”，继续往下问：
- 为什么不谈感情
- 为什么讨厌独断
- 这种性格最怕被触发什么
- 遇到同一种刺激时会怎么反应

**成因越清，互动越自然。**

### 3. 用“情感正负值”排波形
把关系写成一连串 `+ / -` 变化，而不是平铺的甜。

基础波形：
- 亲近、偏爱、心动、被理解 `+`
- 小误差、小落空、没接住、试探失败 `-`
- 矛盾激化、旧伤被踩、立场冲突 `--`
- 解释、补偿、共同承担、重新选择 `++`

推荐节奏：
`甜 (+) → 小误差 (-) → 更深矛盾 (--) → 回收 (++)`

**回收后的正值，最好高于前面的平铺甜。**

### 4. 让“减法”来自具体日常，不只来自大事件
负值不一定靠生离死别，也可以来自很小的失落：
- 分享消息却没等到回复
- 对方回了别人，没回自己
- 关键时刻只收到一个冷淡的问号
- 明明在意，却装作无所谓

这类小减法最适合铺暧昧期和甜文中的拉扯。

### 5. 每次矛盾都写“表面问题”和“真实问题”
例如：
- 表面：为什么不回消息
- 真实：我在你心里到底重不重要
- 表面：为什么瞒着我去冒险
- 真实：你根本不相信我能和你一起扛

没有真实问题，吵架就只是吵架。

### 6. 在一起后，继续从背景和性格里挖戏
不要以为“在一起了”就只能写甜。
可继续挖：
- 控制欲 vs 讨厌被控制
- 原生家庭带来的回避、讨好、自卑
- 事业选择和生活排序
- 过去未处理完的人和事
- 身份公开、立场站队、共同承担外部压力

例如：角色是孤儿，亲生父母忽然找上门；矛盾自然就出来了。

### 7. 让互动必须符合人设
- 两个都很傲娇的人，不会轻易低头直球告白
- 一个平时强撑的人，关键时刻也更可能先嘴硬再行动
- 一个厌恶独断的人，不会因为恋爱就突然接受对方替自己做主

**互动方式要从人物性格长出来。**

### 8. 控制“虐”的剂量和回收速度
优先写：
- 短时间憋屈
- 明确升级
- 有效回收

避免：
- 长时间单边受气
- 一直减分不回收
- 只虐不改关系

读者能接受波折，前提是看得到变化和希望。

### 9. 检查这段关系还有没有新变化
如果以下都写完了，主感情线通常接近收束：
- 双方已完成主要磨合
- 关键秘密和旧伤都已回收
- 关系不会再因人物成长发生新变化
- 继续写只剩重复发糖或重复吵架

## 输出格式
1. 这条感情线更偏剧情流还是感情流
2. 双方未磨合点
3. 双方性格/背景触发点
4. 本段 `+ / - / -- / ++` 波形
5. 每个负值节点的表面问题 / 真实问题
6. 回收方式
7. 在一起后还能继续挖的背景矛盾
8. 这条线何时该收束

## 快速示例
### 输入
- 男主强势独断
- 女主最讨厌别人替自己做决定
- 两人刚在一起，作者不知道后面写什么

### 输出
- 未磨合点：控制欲 vs 自主权
- 波形：
  - 男主在危机里先护住女主 `+`
  - 男主擅自替女主推掉机会，女主炸 `-`
  - 吵到“你从来没把我当能并肩的人” `--`
  - 男主学会先征求意见，并在更大危机里与她共同决策 `++`
- 作用：关系不靠硬虐，而靠磨合升级

### 输入
- 暧昧期甜文，想制造一点拉扯

### 输出
- 小减法：
  - 一方试探性分享歌曲，对方没回
  - 共同好友却先收到了回复 `-`
  - 准备放弃时，对方回：“你怎么知道我在听这首歌。” `+`

## 注意事项
- **不要**把感情流写成“天天腻歪”
- **不要**只考虑甜不甜，不考虑人物在想什么
- **不要**每次矛盾都靠误会和不长嘴
- **不要**让波折失控到关系无法回收
- 感情流最好看的地方，不是他们一直很甜，而是**他们在一次次正负变化里，学会怎么爱对方**。
