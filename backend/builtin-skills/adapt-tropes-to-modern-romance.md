把别的题材里好用的人设爽点拆出来，再换成现代能成立的身份、关系和冲突。**不要直接搬古代身份壳、耽美相处壳或同人滤镜，要搬的是读者真正吃的权力感、稀缺感、上升感、技能感、忠犬感和对抗感。**

## 适用情况
- 想把古言、耽美、同人里的热门人设改到现言
- 知道某种人设很香，但一落到现代就违和
- 怕照搬设定踩伦理、现实或平台雷点
- 想借别的题材爽点，但不知道现代等价物

## 输入
- 原题材与原人设
- 读者最吃的点
- 目标题材与目标读者
- 想保留的关系模式、爽点或技能点
- 不能碰的现实或伦理限制

## 核心原则
1. **表象不是卖点，核心快感才是。**
2. 现代题材先过现实性、伦理和平台风险，再谈苏感。
3. 身份可以换壳，互动引擎也要一起重建。
4. 转化后仍要能稳定产出冲突、爽点和感情戏。

## 步骤
### 1. 先把原人设拆成两层
分别写：
- 外壳：身份、时代、制度、关系名目
- 内核：权力感、地位感、被偏爱、向上爬、强技能、强对抗、强忠诚、反差感

### 2. 抽出读者真正吃的爽点
常见可迁移的内核：
- 权力、金钱、地位
- 独立、强大、痴情、护短
- 在受限环境里找到生路并往上爬
- 技能稀缺、专业压场
- 对手式互动、互补式组队
**先回答：读者到底在爽什么。**

### 3. 标出不能直搬的外壳
高风险项通常包括：
- 皇帝、王爷、一夫多妻、妻妾等级
- 师生、近亲、婚外、多角踩伦理
- 过强的封建等级压迫
- 不合现代法律和平台要求的身份玩法
- 耽美里靠同寝同宿、全天贴身获得的天然亲密

### 4. 给它换现代等价物
常用替换：
- 王爷皇上 → 高资源、高地位、高掌控力的现代身份
- 宫斗宅斗 → 职场、豪门、家族企业、社交圈竞争
- 将军武将 → 纪律感、担当感、执行力强的现代职业身份
- 武林高手 → 专业强者、竞技强者、安保或调查型强者
- 古代才艺 → 非遗、艺术、专业技能、文化资本

### 5. 连互动机制一起改
别只换职业名。还要重建两人为什么总能相遇、对撞、并肩。
例如：
- 耽美里同宿舍的天然亲密，改成同项目、同团队、同案件、同创业
- 古言里的身份压制，改成资源差、圈层差、话语权差
- 同人里的原著滤镜，改成正文里可见的前史、名望或旧账

### 6. 做价值观校验
逐条问：
- 这层爽点放到现代，会不会变成雷点
- 读者喜欢的是强势，还是越线
- 读者喜欢的是生存翻盘，还是封建壳本身
- 如果去掉原题材滤镜，这个人还成立吗

### 7. 写三场验证戏
至少准备：
1. 现代版初见高光
2. 现代版冲突高光
3. 现代版关系推进高光
如果三场都写不出，说明只换了壳，没换成能跑正文的结构。

## 输出格式
1. 原人设外壳
2. 原人设内核
3. 真正爽点
4. 不能直搬的部分
5. 现代等价身份与冲突
6. 现代版互动引擎
7. 三场验证戏
8. 最容易翻车的雷点

## 快速示例
### 输入
- 想把古言里“大家贵女在压迫环境中步步上爬”的爽感改成现言

### 输出
- 外壳：封建家族、妻妾、内宅
- 内核：在被压迫环境中找到生存法则并往上爬，让别人不得不重视自己
- 现代改法：放进豪门或职场竞争，用资源、规则和社交手腕取代安顿小妾式剧情
- 风险：不能把“容忍小三小四”当成现代女主美德

## 注意事项
- **不要**直接搬身份名头
- **不要**把封建壳误当成爽点本身
- **不要**把耽美天然同频互动原样套到现言
- **不要**只想改设定，不重做冲突和关系结构
- 真正稳的移植，是：**保留读者最吃的那口，换成现代还能成立的玩法。**
