先给配角定功能，再补行为逻辑。**配角不该只是作者推剧情的手，也不等于一上来就要写成群像。最低标准不是多复杂，而是这个人为什么会这么做，读者能看明白。**

## 适用情况
- 配角一出场就像工具人
- 反派找茬像单细胞
- 为了推进剧情，配角被写得集体降智
- 一听不要工具人就想给所有配角开支线
- 配角一多就乱，不知道该写多少

## 核心原则
1. **不写工具人 ≠ 写群像。**
2. 配角可以戏少，但**行为必须有逻辑支撑**。
3. 重要配角先定功能，再定座次，再补逻辑。
4. 读者不一定非要喜欢配角，但最好不要一眼看出他只是来完成作者任务。
5. 修配角工具人感，优先补动机、立场、信息差、限制，不是先加大量戏份。

## 最低合格线
把配角创作粗分成四档：
1. 一眼看出是工具人
2. 不出戏，能正常完成任务
3. 鲜活，有记忆点
4. 能让人喜欢、能独立发光

大多数正文配角，至少要做到 **2**。
不是每个配角都必须写到 4。

## 步骤
### 1. 先定这个配角的功能
格式：
`这个配角存在，是为了让主角在____上发生____变化。`

常见功能：
- 提出问题
- 制造阻碍
- 提供帮助
- 见证高光
- 反衬主角
- 带出规则
- 推动关系线

答不出功能，先别给戏。

### 2. 再写行为逻辑链
至少补四格：
- **立场**：他站哪边
- **目标**：他此刻想得到什么
- **顾虑**：他怕失去什么
- **信息**：他现在知道什么、不知道什么

把这四格串起来：
`因为____立场 + 想要____ + 害怕____ + 只知道____，所以他会____。`

哪怕只是一场戏的配角，也最好能写出这条链。

### 3. 用去作者化检查工具人感
逐条问：
- 去掉作者想推进的那件事，他还会这么做吗？
- 他找主角麻烦，是出于自身立场，还是单纯为了让剧情动一下？
- 他夸主角、踩主角、背叛主角，是否符合前文行为模式？
- 他是不是只会在需要时突然变蠢、变坏或变好？

只要有两项答是，工具人感就很重。

### 4. 不要把修工具人误写成加戏
更稳的修法是补这四样：
- 一个明确诉求
- 一个现实限制
- 一个符合人设的行动方式
- 一个能看出他不是空壳的小细节

不一定要补：
- 长篇前史
- 独立副本
- 大量内心戏
- 完整群像支线

### 5. 给反派和阻力人物保住脑子
反派找茬至少要占 1—2 条：
- 他能从中获利
- 他真心认同自己的做法
- 他被误导或被逼迫
- 他对主角有旧怨、竞争、偏见或利益冲突

衡量反派强不强，不只看身份，要看：
- 他能从主角手里夺走什么
- 他能让主角付出什么代价
- 他能不能让主角原打法失效

**单纯因为需要有人使坏，最容易写成工具人。**

### 6. 重要配角再排座次
- 炮灰或功能型
- 小反派或小阻力
- 大反派或长线对手
- 关键盟友
- 对照组

座次不同，补逻辑的深浅也不同。
炮灰只要 1 条清楚动机就够，长线对手需要完整目标和打法。

### 7. 判断是不是已经够立体
满足下面任意 3 项，通常就不容易出戏：
- 行动符合自身利益或性格
- 不是只有一个标签
- 不是只在主角需要时才存在
- 与主角有明确纠缠点
- 做坏事或做好事都有原因
- 同类场景下反应基本一致

### 8. 控制笔墨，别让配角反客为主
问自己：
- 这段配角戏结束后，主角线有什么变化？
- 这场戏是让世界更真，还是在单独演另一部戏？
- 删掉这段，主线会不会更清楚？

配角可以鲜活，但正文重心还是要回到主角。

## 输出格式
1. 配角名单与座次
2. 每人的功能
3. 每人的行为逻辑链
4. 工具人风险点
5. 该补的最少信息
6. 该删或该压的多余戏份
7. 与主角的纠缠点
8. 修改后是否达到至少不出戏

## 快速示例
### 输入
- 男配总在关键时刻跳出来羞辱主角
- 但前文没立场、没利益冲突

### 输出
- 功能：阶段性阻力
- 行为逻辑链：他与主角竞争同一资源，担心被主角压过，又只知道主角的一部分信息，所以会先低估、再打压
- 修法：补竞争关系 + 误判来源 + 他惯用的打压方式，而不是额外给他一整段童年故事
- 风险：如果每次都只会无脑挑衅，且失败后毫无反思或新打法，工具人感仍然很强

## 注意事项
- **不要**把配角不工具人理解成每个配角都要写成主角
- **不要**为了修工具人感，先给所有人开支线
- **不要**让配角为了推进剧情突然失智
- **不要**把读者不喜欢某个角色，直接等同于这个角色是工具人
- 稳的配角不是戏越多越好，而是**出场时有理由，行动时有逻辑，用完后不出戏**
