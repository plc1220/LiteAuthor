把抽象的“压抑、诡异、喜庆、肃杀、暧昧、悲凉”翻成读者能看见、听见、感觉到的细节。**气氛不是靠直接宣布，而是靠“表层场合 + 异样 + 细节选择 + 人物反应”形成。**

## 适用情况
- 一写气氛就只会说“气氛很压抑 / 很诡异 / 很热闹”
- 场景有事件，没有味道
- 开篇需要迅速定场
- 同样是婚礼、宴会、夜路、战场，写出来都一个味
- 想用细节给冲突加压，或给反转提前铺气味

## 输入
- 场景类型
- 表层气氛
- 里层真正藏着的情绪 / 危险
- 视角人物
- 当前要推进的动作或冲突
- 想让读者记住的一个点

## 核心公式
**1 个表层场合 + 1 个潜在异样 + 3 个有效细节 + 1 个人物反应**

填空：
- 表面像：____
- 实际在往：____走
- 最不对劲的一笔是：____
- 视角人物第一时间会注意到：____

## 核心原则
1. **先定气氛走向，再选细节。** 先问“这场要让读者感觉什么”，再问“靠什么让他感觉到”。
2. 细节要有功能，至少完成一项：**定场、加压、泄露信息、埋反转、钉情绪**。
3. **别平均铺五感。** 一场戏抓 3—4 个最值钱的细节，比五感大全更有效。
4. 细节要贴视角人物。胆小的人先注意暗处和声音，强势的人先注意异常和威胁，重生者会先注意触发前世记忆的那一笔。
5. **气氛要和动作绑在一起。** 别写成独立风景段。

## 步骤
### 1. 先拆“表层”和“里层”
把场景写成一句话：
`表面是____，底下其实是____。`

常见组合：
- 喜庆里藏危险
- 热闹里藏冷场
- 温柔里藏控制
- 平静里藏崩溃
- 庄严里藏荒诞
- 日常里藏不对劲

### 2. 选视角人物最会注意的细节方向
从下面选 2—3 类：
- **声音**：锣鼓、雨声、脚步、窃语、突然安静
- **光线 / 颜色**：烛火、灯影、血色、白幔、夕照、霓虹
- **触感 / 温度 / 重量**：闷、冷、黏、烫、沉、勒、刺
- **人物反应**：谁低头不语、谁脸色变了、谁装镇定、谁不敢抬头
- **物件**：盖头、酒杯、门、牌位、刀、请柬、药碗、手机

### 3. 先写正常，再写异常
推荐顺序：
1. 两笔正常场面细节，让读者知道这里原本应该是什么气氛
2. 一笔异常细节，把气氛拧歪
3. 一笔人物反应，让气氛落到人身上

例如：
- 喜烛、唱礼、红绸，是“正常”
- 满堂骤静、白幔、空席、血迹，是“异常”
- 主角掀盖头、指尖发冷、有人不敢抬眼，是“反应”

### 4. 优先选会参与剧情的细节
好细节不是漂亮，是有用。优先写：
- 会影响人物判断的
- 会推动下一步动作的
- 会提前泄露危险的
- 会让反转更重的

如果一笔细节删掉后，场面没任何变化，这笔通常可以压缩。

### 5. 用反差制造气氛
常用反差：
- **越热闹，越显得突然的安静可怕**
- **越华丽，越显得一点血、一点灰、一处破损刺眼**
- **越温柔，越显得一句不合时宜的话危险**
- **越正常，越显得礼数错位、表情不对、站位不对诡异**

### 6. 让细节贴着动作走
不要写：
- 先铺半页环境，再让人物动

优先写：
- 人物动作 + 细节接触 + 气氛变化

示例：
- 她抬手掀开盖头，喜娘的惊呼还没落下，满堂忽然安静得只剩蜡泪滴落的声音。
- 他推门进去，屋里药味压得人发闷，案上的灯芯炸了一下，照出床边那只没来得及收起的绣鞋。

### 7. 控制密度
- 开篇定场：2—4 笔就够
- 高潮正中：只留会影响当前判断的细节
- 余波：补一笔残局或残响，钉住情绪

**不要**每句都在渲染，不然会糊成一团。

## 常用模板
### 模板 1：正常两笔 + 异常一笔
`先写场合本该有的样子，再用一个不合时宜的细节把气氛拧歪。`

### 模板 2：感官先行
`先从声音 / 触感 / 气味切入，再让视线落到关键物上。`

### 模板 3：人物反应钉气氛
`环境细节只写到能让人物做出反应为止，不单独写满。`

## 输出格式
1. 表层气氛
2. 里层情绪 / 危险
3. 视角人物最先注意什么
4. 可用细节 3—4 个
5. 最关键的异常细节
6. 人物反应
7. 组合后的段落或句群
8. 可删的装饰性细节

## 快速示例
### 输入
- 场景：婚礼现场
- 表层气氛：喜庆热闹
- 里层：新娘重生，准备拒婚
- 视角：新娘
- 要推进：她当场翻局

### 输出
- 可用细节：唱礼声、凤冠压得额角生疼、红盖头挡住视线、对面那只递来的手、满堂骤静
- 异常细节：她掀盖头后，喜乐声里忽然只剩一片死寂
- 人物反应：她把凤冠摘下来放在桌上，声音平静却不容置疑
- 组合段落：先用唱礼和盖头定“喜”，再用头痛、手势和死寂把气氛翻成“逼迫”和“决裂”

## 注意事项
- **不要**直接写“气氛很诡异 / 很压抑”，却不给细节支撑
- **不要**把无关漂亮摆设当成气氛本体
- **不要**平均铺声音、颜色、触感、气味、动作各一笔，写成清单
- **不要**细节很多，但没有一个和当前冲突有关
- 真正能留住人的气氛，是**读者先感到不对，再看见不对，最后等着人物出手。**
