先定这个角色在故事里负责什么，再倒推底色、经历和具体反应。**角色不是先有一堆标签，再逼剧情迁就；更稳的写法是先保住这场戏真正要的效果，再让人设把中间的合理性补齐。**

## 适用情况
- 先想到桥段、冲突或角色功能，不擅长先写完整人物小传
- 配角总是只有标签，没有可写反应
- 暴君、疯批、高冷、傲娇等设定一落笔就空
- 角色在不同关系里表现太单一
- 想让角色推动剧情，但又怕 OOC

## 输入
- 核心梗或当前大剧情
- 已想好的高光场面 / 冲突场面
- 这个角色在故事里的功能
- 这场戏希望他 / 她做到的结果
- 当前最怕写崩的边界

## 核心原则
1. **先定角色功能，再补血肉。**
2. **角色要服务故事，但不能硬掰到崩。**
3. **先问“故事要 TA 做什么”，再问“TA 在什么条件下会这么做”。**
4. **同一个标签，在不同关系里会有不同表现。**
5. 普通配角不用写满一整本小传，只保留会影响剧情的部分。

## 步骤
### 1. 先写这个角色在故事里负责什么
常见功能：
- 阻碍主线
- 推动 CP
- 对照主角
- 递资源 / 递信息
- 提供价值观冲突
- 放大主题
- 充当短期反派 / 长期对手

写成一句：
`这个角色存在，是为了____。`

### 2. 再写这场戏到底要 TA 做到什么
不要一上来就问“按设定 TA 应该怎么做”，先写：
`在这场戏里，我希望 TA____。`

比如：
- 留下男配，不是杀光所有人
- 坚决阻挠男女主，不马上退场
- 在关键处站到主角对面
- 明知在意主角，嘴上仍不承认

**先定要什么结果，再倒推它的合理性。**

### 3. 从“要做到的结果”倒推大人设
问自己：
- TA 更主动还是更被动
- 更狠、更稳、还是更会装
- 最重视面子、利益、感情，还是控制权
- 遇到刺激时先动手、先算计、先退、还是先嘴硬

把答案压成 3—5 条大人设。

### 4. 给人设补“为什么会这样”
每条大人设都补成因：
- 出身
- 家庭影响
- 关键旧事
- 长期缺失
- 学会的生存方式
- 最怕失去什么

句式：
`因为过去____，所以 TA 一遇到____，就会____。`

### 5. 写不同关系下的不同面
至少分别想一遍：
- 对长辈 / 权威
- 对同辈 / 对手
- 对喜欢的人 / CP
- 对弱者 / 小孩
- 对自己需要利用的人

**高傲不是 24 小时同一张脸。**
同一底色，在不同对象前会有不同表现方式。

### 6. 配角只保留“会出戏的设定”
对普通配角，优先写：
- 名字 / 记忆点
- 和主角的关系
- 一条功能明确的背景
- 一条会影响行为的执念 / 伤口 / 欲望

如果这些已经足够让你脑补出 TA 会怎么说、怎么做，就够了。
**不要给每个配角都写满无用档案。**

### 7. 如果情节和人设打架，按三种方式修
1. **改情节**：这场戏不值当，就删
2. **补触发**：给足刺激、代价、前史，让这次反应站得住
3. **调人设**：如果新反应更能统一前后，就微调底色

不要僵在“这个标签的人绝不会这样”。  
更有效的问法是：**TA 在什么条件下会这样，而又不崩？**

### 8. 用样本找感觉，但别照搬
如果抓不准某类角色：
- 找 1—3 个类似角色样本
- 观察他们在不同关系里的反应
- 记录语气、决策、情绪触发点
- 再反问：如果换成自己的角色，哪里会一样，哪里会不同

### 9. 做“换人测试”
把角色名字遮掉，问：
- 这场反应是不是换谁都能用
- 这句台词是不是任何高傲 / 暴君 / 傲娇都能说
- 如果能，说明角色还不够具体

### 10. 最后回收成可执行资料
至少整理出：
1. 角色功能
2. 这场戏要的结果
3. 大人设 3—5 条
4. 成因链
5. 不同关系下的表现
6. 关键边界
7. 最容易 OOC 的点

## 输出格式
1. 角色功能
2. 这场戏目标
3. 大人设
4. 成因链
5. 关系面向表
6. 最少必备设定
7. 情节与人设冲突的修法
8. OOC 风险点

## 快速示例
### 输入
- 角色：阻挠男女主的女主闺蜜
- 需求：她不能很快退场，还要让读者觉得她不是纯工具人

### 输出
- 角色功能：持续阻挠 CP，放大女主在友情和爱情之间的选择
- 这场戏目标：她表面温和，实则一次次把女主往男主对立面拉
- 大人设：占有欲强、待人得体、内里偏激、对女主高度依赖
- 成因：长期缺乏家庭陪伴，唯一稳定关系就是女主，所以会把男主视为夺走资源的人
- 关系面向：对外会装得很好，对女主会控制，对男主会带刺
- 结果：角色既有功能，也有来路

### 输入
- 角色：暴君男主
- 需求：要在不崩的前提下，写出他为了女主狠狠干回去

### 输出
- 先定故事结果：这场戏要突出他的偏执和爱，而不是讲道理
- 再倒推底色：强控制欲、护短、极端报复、对失去高度恐惧
- 再补触发：女主受伤 / 被夺走，直接打到他的底层禁区
- 结论：重点不是“暴君按教科书会怎么做”，而是“在什么刺激下，他会做出更夸张但仍然成立的反应”

## 注意事项
- **不要**先写一堆生日血型，再发现剧情根本用不上
- **不要**把角色写成只会执行剧情的木偶
- **不要**把标签当结论，不补成因和边界
- **不要**让所有关系面向都一模一样
- 角色越要服务剧情，越要把**功能、底色、触发条件**三样一起钉死
