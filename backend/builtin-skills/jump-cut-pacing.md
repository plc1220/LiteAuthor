先确定本场戏真正要交付什么，再把不增值的顺时序过程剪掉，直切重点。**跳笔不是漏写，也不是偷懒；跳笔是把篇幅让给最值得看的地方。**

## 适用情况
- 一写场景就顺着时间线往下记，像流水账
- 高潮明明很大，写出来却慢、散、没劲
- 过渡戏太多，人物一直在走路、准备、解释
- 想提升镜头感和文字凝练度
- 明知道这段该快，但不知道该省哪部分

## 输入
- 当前场景的一句话梗概
- 本场戏最重要的目标或情绪
- 角色必须知道的信息
- 现有顺时序流程
- 这一段写完后要落到的结果

## 步骤

### 1. 先写本场戏真正要交付的东西
只保留 1 个主交付，最多再留 1 个副功能：
- 主交付可能是：救人、对峙、翻盘、告白、决裂、情绪升华
- 副功能可能是：立人设、补背景、埋伏笔

如果这场戏说不清“到底最该看什么”，它就已经散了。

### 2. 把内容分成三栏
把现有流程拆成：
- **必须写**：关键决定、冲突、反转、情绪变化、重要后果
- **一句带过**：时间流逝、移动过程、重复准备、次要交涉
- **可以省略**：不影响理解、只是在走流程的部分

判断标准只有一个：
**删掉这一段后，读者会不会失去关键理解、关键情绪或关键后果。**

### 3. 决定从哪里切
常用切法：
- **直切危机点**：省掉前面的大量铺垫，开场就进冲突
- **直切结果前一秒**：把篇幅留给最关键的动作和选择
- **直切后果**：先让结果落地，再用短句补足过程

常见模板：
- `交代危险已发生 → 直切营救 / 对峙`
- `一句补足中间过程 → 直切情绪爆点`
- `先写结果改写局势 → 再倒补必要信息`

### 4. 用桥接句补上被剪掉的时间
跳笔后不能让读者迷路。用 1—2 句桥接句补清楚：
- 过了多久
- 人到了哪
- 事情已经推进到哪一步
- 为什么会直接进入这一幕

常用句式：
- `等他赶到时，____已经____。`
- `半个时辰后，____。`
- `中间那些试探和推诿都被她略过，开口便是重点。`
- `再见面时，局面已经和出发前完全不同。`

### 5. 把 70% 的篇幅砸在重点上
不要平均分配笔墨。准备、过程、结果三段里，真正重要的那段至少占大头。

例如：
- 结局重点是双救赎，就少写绑架准备和追逐过程，多写救下之后的情绪回收
- 重点是打脸见证，就少写后台运作，多写公开翻盘和群体反应

### 6. 让一场戏同时干三件事
用最少的情节发挥最大的作用。优先让一场戏同时完成 3 项以上：
- 推剧情
- 立人设
- 放大矛盾
- 补背景
- 埋伏笔
- 回收情绪

如果一场戏只负责“从 A 走到 B”，优先考虑压缩。

### 7. 借影视化的“切”，但别把小说写没了
可以借影视思路去想：
- 这一幕最该被看见的画面是什么
- 镜头应该落在哪个动作、表情、声音、空间细节上
- 哪个点一切进去，节奏最利落

但要保留文字的优势：
- 小说**可以直接概括**过程，不必把每个动作都演一遍
- 小说**可以直接点明心理**，不必全靠摔杯子、瞪眼睛、抠椅子代替
- 小说**可以用精练词语**，不必全篇都用冗长白描去替换精准概括

**影视化是借节奏，不是放弃文字本身。**

### 8. 反查是不是“剪过头了”
跳笔后再检查：
- 读者还能不能看懂因果
- 关键转折有没有被写到
- 被省略的过程里，有没有其实才是爽点 / 泪点 / 悬念点
- 角色态度变化是不是突然了

如果过程本身就是看点，就不要硬跳。

## 输出格式
1. 本场戏主交付 / 副功能
2. 必写 / 一句带过 / 可省略三栏
3. 切入点
4. 桥接句
5. 重点笔墨分配
6. 一场戏承担的多重功能
7. 跳笔后的连续性风险

## 快速示例
### 输入
- 结局：女主被反派绑走，男主营救，最后完成感情升华

### 输出
- 主交付：灾难后的双向救赎
- 必写：营救成功前后的选择、两人的情绪爆点、局势结果
- 一句带过：反派前期准备、绑架全过程、无效周旋
- 切入点：直接从营救行动最危险的时刻切入
- 桥接句：`等男主找到她时，废墟深处只剩最后一扇快要塌下来的门。`
- 笔墨分配：追逐与交手 30%，救下后的情绪和关系回收 70%

## 注意事项
- **禁止**把所有过程都剪掉，导致剧情像提纲
- **禁止**为了快而快，把真正有看点的过程一起省掉
- **禁止**全篇都用“去解释化”白描，放弃本来更有力的精练表达
- 跳笔最适合处理无效过渡，不适合处理本身就承载悬念与变化的核心过程
