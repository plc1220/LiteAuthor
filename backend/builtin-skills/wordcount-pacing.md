先按总字数决定节奏模型，再给每段分字数。**节奏不是写到哪算哪，而是让每一个阶段都知道自己该快、该慢、该在哪儿爆。** 中短篇和百万长篇的排法不一样：前者靠有限事件写足分量，后者靠循环推进和不断换包装。

## 适用情况
- 写了很多字，主线成果还没落地
- 20—30 万字的文写得很薄，或者很拖
- 百万字长篇越写越像重复打怪
- 不知道一个情节该占几章，结果全凭感觉
- 写几章后总觉得节奏忽快忽慢

## 输入
- 预计总字数
- 一句话主线
- 故事开始点和结束点
- 题材类型（感情、升级、经营、权谋等）
- 当前已经想到的关键桥段

## 核心原则
1. 先定总篇幅，再定阶段数量。
2. 每个阶段都要有**可见结果**，不是模糊状态词。
3. 大节奏由很多小起伏组成；单个桥段也要有自己的起伏。
4. 20—30 万字文重在“有限事件写出分量”；百万字文重在“循环推进但每轮有新意”。
5. 在还没有节奏直觉前，先用字数硬控。

## 步骤
### 1. 先按总字数选节奏模型
#### A. 20—30 万字中短篇
适合：
- 时间跨度相对集中
- 核心事件不算太多
- 重点靠关系变化、心理变化、关键节点推进

写法要点：
- 先定明确的开始点和结束点
- 中间只放 4—8 个大节点
- 因为大剧情数量有限，**每个节点要写细**
- 细的重点不是废话，而是对话、心理、选择、关系变化和局势后果

#### B. 80—150 万字以上长篇
适合：
- 升级流、经营流、长期成长线、宫斗宅斗、长线事业文

写法要点：
- 不靠一条直线撑全文，靠阶段循环
- 常用循环：`积累 → 受压/挑战 → 反击/突破 → 升级/换阶段`
- 循环可以反复，但**包装必须变**
- 每一轮至少换 1—2 项：舞台、对手、代价、资源、见证者、成果形式

### 2. 写出开始点和结束点
按这个句式写：
- `故事从____开始`
- `故事到____阶段性结束`

开始点要能立刻入局，结束点要是一个清晰结果。
例：
- 从“穿越后发现自己已经在泰坦尼克号上”开始
- 到“获救并在美国开始新生活”结束

没有开始点和结束点，字数很容易空转。

### 3. 把中间拆成阶段里程碑
每个里程碑都写成：
`X 万字前完成____`

好例：
- 3 万字前：主角成功结识关键人物
- 6 万字前：第一次重大误会和关系破裂
- 10 万字前：关系反转或第一次关键胜利

坏例：
- 继续成长
- 感情升温
- 主角更强了

### 4. 每个里程碑再拆成“小起伏”
每一段都至少走完：
`开端/目标 → 受阻/冲突 → 爆点/反击 → 转折/余波`

例：
- 隐姓埋名入学
- 被人轻视
- 关键人物无视或挑衅
- 主角展现能力
- 对方改观，局势变化

**大节奏不是单独存在的，是很多小起伏拼起来的。**

### 5. 中短篇：给每个大节点分章数
20—30 万字文特别需要做这一步。
给每个节点写：
- 预计多少章
- 这一段重点写什么
- 是该细写还是带过

判断方法：
- 大剧情少，就把重点放在关系和情绪的具体变化上
- 不是所有桥段都该均分篇幅
- 越靠近本书卖点的桥段，越该多给字数

例如 24 万字、每章 3000 字，约 80 章：
- 入局：2—4 章
- 认识关键人物：4—8 章
- 第一次冲突：3—6 章
- 情绪低谷：6—10 章
- 关系反转或重大危机：8—12 章

### 6. 长篇：给每一轮循环换包装
对每一轮循环都写 5 件事：
1. 这一轮主角要积累什么
2. 压制或挑战来自哪里
3. 主角靠什么突破
4. 这次升级具体升了什么
5. 和上一轮比，新意在哪儿

包装差异可从这些方向换：
- 换场域：学校 → 战场 → 朝堂 → 商场
- 换对手：同辈 → 上位者 → 规则本身
- 换代价：输名声 → 失资源 → 伤关系 → 破底线
- 换结果：涨实力 → 涨地位 → 涨影响力 → 涨掌控权

**如果每一轮都只是“富二代踩人，主角打脸”，读者会先麻。**

### 7. 用字数逼自己做取舍
每写 3—5 章，检查一次：
- 离下个里程碑还剩多少字
- 最近几章是不是在重复同一种冲突
- 有没有桥段删掉后，主线几乎不受影响
- 本阶段最值钱的高潮有没有留够篇幅

赶不上时优先：
- 删枝杈
- 合并功能相近的桥段
- 把无效过渡一句带过
- 把笔墨让给真正的卖点

### 8. 判断“快写”和“细写”
#### 该快写的
- 路上转场
- 重复解释
- 无新变化的准备过程
- 已经写过同类功能的桥段

#### 该细写的
- 第一次关键相遇
- 关系转折
- 主角第一次真正翻盘
- 阶段性最大危机
- 阶段成果落地和余波

### 9. 阶段结束时必须有回收
每到一个里程碑，至少明确 3 件事：
- 主角拿到了什么
- 失去了什么
- 下一阶段为什么必须继续

没有回收，字数只会像流水账。

## 输出格式
1. 总字数与节奏模型
2. 开始点 / 结束点
3. 里程碑表
4. 每个里程碑的小起伏
5. 每段预计章数或字数
6. 需要快写的部分
7. 需要细写的部分
8. 长篇循环的新意表
9. 当前节奏风险

## 快速示例
### 示例 1：24 万字中短篇
- 开始点：主角醒来时已经在沉船上
- 结束点：获救并开始新生活
- 里程碑：求生 → 接近关键人物 → 关系升温 → 灾难爆发 → 逃难 → 获救
- 处理：因为宏观事件不多，重点细写关系变化、心理焦虑和选择代价

### 示例 2：100 万字升级文
- 循环 1：小舞台积累 → 被同阶打压 → 初次突破 → 出圈
- 循环 2：进入更大舞台 → 被规则压制 → 换打法突破 → 拿到身份
- 循环 3：旧敌回收 → 更高层斗法 → 付代价突破 → 改写格局
- 处理：每轮都保留“受压—爆发—升级”，但换对手、换资源、换结果

## 注意事项
- **禁止**把所有桥段平均分配篇幅
- **禁止**中短篇只列事件，不写关系和情绪分量
- **禁止**长篇只会重复同一种压制和打脸
- **禁止**阶段成果迟迟不落地，只靠“以后会更精彩”硬拖
- 节奏的本质不是字数多寡，而是**每一段都在朝一个更高的结果推进**
