围绕一个重点反复出现，但每次换角度、加信息、改承载方式。**重复不是复制粘贴，而是让读者真正记住：这件事很重要。**

## 适用情况
- 关键剧情写过了，读者却没感觉
- 初见心动、角色落魄、重要伤势、打脸余波等情节点不够重
- 聊天场景很多，不知道聊什么，容易写水
- 想延长某个瞬间的冲击力
- 需要让读者牢牢记住一个物件、一句台词或一种处境

## 输入
- 本段最想让读者记住的重点
- 重点属于：情节 / 情绪 / 关系 / 处境 / 物件 / 细节动作
- 预计能给它几次出场机会
- 当前阶段的主线作用

## 步骤

### 1. 只选一个值得反复写的重点
先写一句：`这一段最该让读者记住的是____。`

不要一章里反复强调五六个重点，重点一多，就谁都不重。

### 2. 明确读者要感受到什么
同一个重点，可以服务不同感受：
- 心动
- 落魄
- 危险
- 羞辱
- 偏爱
- 失去
- 胜利后的余波

先定感受，再决定怎么重复。

### 3. 安排 2-3 次重复节点
最稳的数量通常是 2-3 次：
- 第 1 次：让重点出现
- 第 2 次：让重点变重
- 第 3 次：让重点被确认或带来后果

**三次通常够了。** 再多就容易像念经。

### 4. 每次重复必须新增一种价值
重复时至少新增一项：
- 放大某个细节
- 补一条新信息
- 换一个观察者
- 补一个后果
- 做过去与现在的对比
- 让外界反应给出确认

如果只是把原句换词再写一遍，读者只会觉得水。

### 5. 常用重复路径
- **角色回想**：事后回放某个细节，放大当时没说透的感受
- **他人追问 / 转述**：让另一个角色把重点重新提出来，并追加信息
- **生活细节对比**：用今昔反差加重某种处境
- **路人 / 对手反应**：让重点从主观感受变成客观确认
- **余波再显**：事情已经发生，但影响继续扩散

### 6. 细节瞬间可用“分段微重复”
一个耳光、一个眼神、一句告白、一次摔倒，如果想拉长冲击力，可以拆成 2-3 小段：
- 声音 / 动作本身
- 身体或心理反应
- 他人反应或环境停顿

这样能延长读者的阅读停顿，让这个瞬间更扎实。

### 7. 对话不知道写什么时，围绕同一重点展开
如果人物必须聊天，就让聊天继续服务同一个重点，而不是东拉西扯。
例如：
- 不是随便寒暄
- 而是继续放大“他真的落魄了”“她真的动心了”“那场打脸真的传开了”

### 8. 重复要跳着用，不要每段都上
重复法则适合重点剧情，不适合全文每一件小事。
每个剧情都“发生一次，再解释一次，再回忆一次”，读者很快会疲劳。

### 9. 反查是否变成水
逐条检查：
- 这次重复比上次多了什么
- 读者理解有没有更深一层
- 情绪有没有更重
- 局势有没有更清楚

有两项以上答不上来，就该删。

## 输出格式
1. 重点是什么
2. 读者该感受到什么
3. 第 1 次怎么出现
4. 第 2 次怎么加重
5. 第 3 次怎么确认 / 回收
6. 每次新增的信息或情绪
7. 使用风险

## 快速示例
### 示例 1：男主落魄
- 第 1 次：住处简陋，和从前形成反差
- 第 2 次：路人轻慢，外界确认他的处境
- 第 3 次：旧识嘲讽，逼出他真正的反应

### 示例 2：初见心动
- 第 1 次：当场被对方一个细节击中
- 第 2 次：回家后放大那个细节
- 第 3 次：朋友点破对方身份或分量，让心动变得更具体

## 注意事项
- **禁止**原句复制粘贴
- **禁止**一章里反复强调太多重点
- **禁止**所有剧情都套重复法则
- **禁止**把无聊内容反复写很多遍
- 重复的目标是加重，不是凑字数
