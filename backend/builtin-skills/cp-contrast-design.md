先把两个人最像的地方找出来，再故意拉开他们的发力点。**CP不是把两张差不多的人设卡摆在一起，而是让两个人在同一场戏里能彼此照亮、彼此顶起来。**

## 适用情况
- 攻受 / 男女主写得越来越像
- 两个人都强、都冷、都温柔，结果没区别
- 同框很多，但没有火花
- 一写互动就温吞、没张力

## 输入
- 双方当前标签
- 故事类型与主卖点
- 当前关系阶段
- 想强化的 CP 感：拉扯 / 互补 / 宿敌 / 救赎 / 并肩

## 步骤
### 1. 先列重叠点
把两个人最像的地方写出来：
- 都冷
- 都强
- 都聪明
- 都嘴硬

然后问：**他们像到什么程度会互相吞没？**

### 2. 从六个维度至少拉开三个
可选维度：
- 目标：一个要控制，一个要自由
- 情绪表达：一个收着，一个炸着
- 社交姿态：一个上位感强，一个平视甚至犯冲
- 决策方式：一个先算后动，一个先动后补
- 照顾方式：一个默默安排，一个直接护短
- 风险偏好：一个稳，一个敢赌

### 3. 同类标签也要写出不同落点
可以都高冷，但：
- 一个冷在克制和礼貌
- 一个冷在傲慢和不耐烦

可以都强势，但：
- 一个强在掌控全局
- 一个强在不肯低头

**相同标签，不等于相同味道。**

### 4. 给每人一个 只有他会这么做 的动作和说话路数
分别写清：
- 他第一反应怎么出手
- 他平时怎么关心人
- 他生气时怎么压人
- 他嘴硬时会说什么类型的话

如果换个人也能这么说、这么做，说明还不够分开。

### 5. 把差异落到四类场面里
至少分别检查：
- 初见：两个人先注意到对方什么
- 冲突：谁先逼近，谁先后退，谁先发难
- 照顾：一个怎么护，一个怎么接
- 决策：谁扛结果，谁拧方向，谁最后让步

### 6. 保留一个隐藏同频点
如果只有反差没有同频，容易像硬配。
给他们一个真正能靠近的点：
- 同样怕失去
- 同样重承诺
- 同样不信权威
- 同样想保护某种东西

只是外在做法不同。

### 7. 做换人测试
把两人的台词、动作、场面功能互换一遍。
如果基本还能成立，说明这对 CP 还写得太像。

## 输出格式
1. 双方重叠点
2. 需要拉开的 3 个维度
3. 各自的标签落点
4. 各自专属动作和说话路数
5. 初见 / 冲突 / 照顾 / 决策的差异写法
6. 隐藏同频点
7. 换人测试结果

## 快速示例
### 输入
- 两个人都想写成高冷强者

### 输出方向
- A 冷在克制，习惯少说、说就只说重点，照顾人也不张扬。
- B 冷在傲慢，看谁都带压迫感，动怒时会直接把场面控住。
- A 关心人时先处理问题，B 关心人时先把挡路的人清掉。
- 两人都不爱解释，但一个是懒得说，一个是觉得没必要说。

## 注意事项
- **禁止**只靠外貌或身份差来冒充反差
- **禁止**为了突出一方，把另一方写扁
- **禁止**两个人连说话节奏和做事习惯都一样
- **禁止**只有反差，没有真正能互相理解的点
- CP 好嗑的关键之一，是**这两个人不能互相替代**。
