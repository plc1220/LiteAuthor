按下面的表，把一个网文项目先搭成“能写”的状态。先做 100-200 字梗概，再补人设、事件线、成长线和互动点，不要一上来就无边无际地想剧情。

## 适用情况
- 新手第一次搭完整大纲
- 脑内有很多片段，但写着写着互相不照应
- 会想单个梗，不会把感情线、事业线和成长线排成一条
- 容易想到哪写到哪

## 使用原则
- **前 6 项为必做项**
- 后面的波折、闪光场合、资料清单、主题升华为附加项
- 能力还不稳时，先把必做项做好，不要一口气炖八宝粥

## 步骤
### 1. 写 100-200 字全文梗概
必须包含：
- 主角起点处境
- 核心矛盾
- 关系或主线如何推进
- 结局方向

### 2. 做主角 A / B 基本人设
每人至少补齐：
- 年龄 / 身份
- 性格表层 + 内里
- 外形或第一记忆点
- 弱点
- 爱好
- **1 个大亮点 + 3 个小亮点**
  - 例：专业能力、特殊天赋、气味 / 听力 / 口才 / 身体条件、社交特质等

### 3. 写“起承转合”事件线
用 4 段写完整体推进：
- **起**：两人如何被拉到一起，哪些设定开始展开
- **承**：如何持续绑定，关系和主线如何升温
- **转**：最大的换挡点是什么，身份、关系、立场或舞台如何变化
- **合**：结局前的核心回收是什么

每段都要写**具体事件**，不要只写“感情升温”“冲突升级”。

### 4. 写成长线
分别为主要角色补两条：
- 性格怎么变
- 能力 / 事业怎么变

最好标清时间段或阶段节点。  
如有事业线，再列一个“成就 / 荣誉 / 资源”时间表。

### 5. 写“人设 × 剧情”的碰撞点
按起承转合四段分别写：
- 角色的哪个特质会在这段被放大
- 这个特质会制造什么具体戏
- 这个碰撞点如何推进关系或主线

**每一段至少写 2 个碰撞点。**

### 6. 写感情线阶段互动
至少补齐这些格子：
- 萌生好感
- 欢乐互动
- 单箭头
- 双向暗恋 / 暧昧
- 表白
- 确认关系后互动 A / B

每格至少写一个**可直接拿来写的具体梗**。

### 7. 附加题：波折 / 反派 / 干扰
选做。可写：
- 重大波折
- 小误会
- 小闹剧
- 反派的核心作用
- 外部干扰者
- 会真正改写关系或局势的冲突

原则：**别用力过猛。** 能打一棒子给个枣，就别一路爆虐到失控。

### 8. 附加题：闪光场合
分别为主要角色列 2-3 个“狂刷好感”的高光：
- 职业高光
- 人设高光
- 危机处理高光
- 美感 / 魅力高光
- 公开场合高光

写清楚“在哪里、做什么、谁看见、为什么帅 / 苏 / 带感”。

### 9. 附加题：资料和素材清单
如果哪类戏最虚，就先囤素材：
- 行业资料
- 地点资料
- 服饰 / 场景描写
- 颁奖 / 综艺 / 朝堂 / 战斗等模板段
- 剧中剧 / 戏中戏梗概

### 10. 附加题：主题和表达
只在能稳住时再做：
- 想在哪些情节里写人性
- 想批判什么现实
- 三观和立场如何通过剧情表达

新手不要一上来就靠说教抬主题。

## 注意事项
- **禁止**只写设定词，不写具体事件
- **禁止**开局为了一个“意外原因”啰嗦半章；能一句交代，就一句交代
- **禁止**什么都想写。能力没到时，先把 2-3 个最值钱的点踩实
- 这份模板是为了防止“想一出是一出”，不是让正文变成表格
