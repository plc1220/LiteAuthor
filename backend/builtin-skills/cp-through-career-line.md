把感情戏挂在事业事件上写。**同一件事最好同时推进事业线和关系线，不要各写各的。**

## 适用情况
- 事业线和感情线像两篇文
- 两个人有互动，但 CP 感不强
- 事业剧情很多，不知道怎么顺手刷感情值
- 想写嘴硬心软、暗中帮忙、表里不一的糖

## 输入
- 当前事业事件
- 双方关系阶段
- 双方表面态度
- 谁更容易嘴硬 / 谁更容易暗中付出
- 这一段想让关系升温还是加深拉扯

## 步骤
### 1. 先选一个事业事件做载体
不要凭空加糖，先找事件：
- 对方落魄了
- 对方升职 / 翻车 / 被刁难
- 对方拿到新机会
- 对方在主线里遇到困难
- 对方必须做出一个事业选择

### 2. 给另一方安排“当面反应 + 背后动作”
最常用的组合：
- 当面吐槽 / 讽刺 / 不耐烦
- 背后关心 / 打听 / 想办法 / 找人帮忙 / 暗中兜底

表面是嘴硬，背后是真在意，这种最容易刷 CP 值。

### 3. 把“吐槽”理解成所有表达意见的动作
这里的吐槽不只是否定，还包括：
- 关心
- 提建议
- 提醒风险
- 替对方骂人
- 事后追问
- 帮对方补后路

重点不是解决所有问题，而是让人物对这件事有反应、有站位、有偏心。

### 4. 每个事业节点都尽量回收一次关系变化
写完一场后，至少检查两件事：
- 事业线有没有往前走半步
- 关系线有没有往前走半步

如果只推进事业，不推进关系，这场戏太“干”；
如果只推进关系，不碰事业，这场戏又容易飘。

### 5. 用“当面一套，背后一套”制造糖点
常用句式：
- 当面：`不就是个破工作，有什么好高兴的。`
- 背后：`她在你那边过得怎么样？别让人欺负她。`

这种反差既能立人，也能让同一事业事件长出感情戏。

### 6. 不要一次帮到底，留一点后劲
如果另一方一出手就把问题全解决，事业线容易塌。
更稳的写法是：
- 只帮半步
- 只提供机会，不替对方完成
- 只兜底，不抢主角功劳

### 7. 每次吐槽都要换角度
不要反复只写一种关心方式。
可以轮换：
- 明面讽刺
- 私下打听
- 替对方求人
- 出主意
- 吃醋式吐槽
- 替对方出气

## 输出格式
1. 当前事业事件
2. 当面反应
3. 背后动作
4. 这次关系增量
5. 这次事业增量
6. 下一个可以继续联动的事件
7. 风格提醒：甜 / 拉扯 / 嘴硬心软 / 暗恋感

## 示例
### 输入
- 女主失业，男主嘴硬心软

### 输出
- 当面反应：男主嘲她“大小姐也有今天”
- 背后动作：回头去找朋友问有没有适合她的工作
- 事业增量：女主得到新工作入口
- 关系增量：读者看见男主嘴硬心软，CP 值上升
- 下一步：女主上班后，男主继续追问她有没有被欺负

## 注意事项
- **别把感情线从事业线里剥出去单写**
- **别让帮忙直接代替主角成长**
- **别只会嘴硬，不给实际关心动作**
- 事业线里最顺手的糖，往往不是表白，而是“我嘴上嫌你，行动上却站你这边”
