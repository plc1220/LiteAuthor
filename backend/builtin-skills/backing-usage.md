让后台托底，不让后台代打。**爽文里有后台不等于主角可以躺赢。后台最稳的用法，是给主角身份、资源、威慑和善后，但关键对撞、关键选择和最终成长必须落在主角自己身上。**

## 适用情况
- 主角有家世、师门、大佬、家族、组织撑腰
- 写着写着主角像挂件，什么都靠后台解决
- 敌人太强，需要合理借力
- 想写狐假虎威、先打脸后补刀这类爽点

## 输入
- 主角的后台是什么
- 当前对手和主角的差距
- 这场冲突最想给读者的爽感
- 后台能提供的资源
- 这条线后面是否还要让主角亲自清算

## 核心原则
1. **后台给托底，不抢主角的主功。**
2. **打反派这件事，能让主角亲自动手就别让后台代打。**
3. **后台最适合提供身份、物品、情报、规则保护、威慑和善后。**
4. **真正的大账，后面要让成长起来的主角自己去打。**
5. **后台每出手一次，都要顺手抬高主角下一次独立作战的能力。**

## 先分清：后台能给什么，不能抢什么
### 能给
- 身份背书
- 资源 / 物品 / 人脉
- 情报和门路
- 规则保护
- 出场威慑
- 事后更重的处罚

### 不能抢
- 主角的主动选择
- 主角的关键高光
- 主角的核心仇恨回收
- 主角的成长证明
- 主角和读者一起等了很久的那口气

## 步骤
### 1. 先写清这场冲突，主角到底要亲自完成什么
填空：
`这一场就算有后台，主角也必须亲自完成____。`

常见答案：
- 当场反击
- 拿证据
- 做决定
- 扛风险
- 公开打脸
- 给最后一击
- 拿回主动权

### 2. 再决定后台只负责哪一层
从下面选 1—2 层，不要全包：
- 让主角有资格上桌
- 给主角一个关键物品或情报
- 阻止对方无脑碾压
- 在主角赢完后追加制度处罚
- 为不可能赢的局面托底一次

### 3. 三种稳妥用法
#### A. 托底型
后台给身份、物品、规矩保护，主角自己去打。

适合：
- 主角不该毫无门槛就接触更高层对手
- 主角需要一个合理入场券

#### B. 狐假虎威型
主角要对上远强于自己的敌人，后台出现或被提及，让对方误判主角不好惹，从而不敢硬来。

适合：
- 前中期差距过大
- 需要先保住场面和主角体面

**但后面这笔账要让主角亲自补回。**

#### C. 二连环型
主角先自己打脸、翻盘、拿回理，后台再出手给反派更重处罚。

适合：
- 想让主角先爽一轮
- 又想把报应再抬一层
- 让读者同时吃到“主角能打”和“后台够硬”

### 4. 给后台出手设置边界
每次后台出手前，先答 3 个问题：
- 如果后台不来，主角会不会直接死局
- 如果后台全包，主角会不会显得废
- 后台这次出手后，主角下一步能不能更独立

答不稳，就把后台的作用收窄。

### 5. 安排“借力后的成长回收”
后台帮过之后，不要只停在安全过关。至少回收 1 项：
- 主角学会一套规则
- 主角拿到关键资源
- 主角建立新名声
- 主角看清更大的敌人
- 主角下次不用同样方式也能打

这样后台不是代打工具，而是成长跳板。

### 6. 长线大账一定留给主角本人
如果对手是长期反派，稳写法是：
- 前期：后台托底一次，避免主角被秒
- 中期：主角自己反击几次，逐渐能独立对线
- 后期：这笔账由成长后的主角亲自清算

**前面借力可以，最后收账别让后台抢。**

## 常用模板
### 模板一：后台给身份，主角自己出手
`主角拿着后台给的资格入场 → 自己在场上赢下对撞 → 后台只负责挡掉规则外碾压`

### 模板二：主角先打脸，后台再补刀
`主角公开翻盘 → 反派吃第一轮亏 → 后台出面追加更重处罚 → 读者吃到二连爽点`

### 模板三：前期狐假虎威，后期亲自清算
`前期强敌太高，后台震住场面 → 主角保住命和面子 → 中后期主角成长 → 回头亲自打掉这笔账`

## 输出格式
1. 主角必须亲自完成的部分
2. 后台本场只负责哪一层
3. 选择的用法：托底型 / 狐假虎威型 / 二连环型
4. 当前场景的具体分工
5. 借力后的成长回收
6. 这条账后面如何由主角亲自结算
7. 主角被写成挂件的风险点

## 注意事项
- **不要**一遇强敌就让大佬替主角把仗打完
- **不要**让主角只负责嘴硬，后台负责真正解决问题
- **不要**长期靠后台撑起主角的“强”
- **不要**把终局级仇恨交给后台代收
- 后台真正好用的地方，不是“替主角赢”，而是**让主角赢得更稳、更大、也更有层次**。
