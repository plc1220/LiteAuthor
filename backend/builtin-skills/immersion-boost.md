让每一段关键描写都落回角色身上。**不要只写外部发生了什么，还要写这件事经过角色心里时变成了什么。**

## 适用情况
- 读者总觉得能看见作者在背后讲解
- 场景写得清楚，但不带感
- 人物会说会做，却像空壳
- 段落太客观，像监控录像或剧情梗概
- 明明事情不小，读起来却像白开水

## 输入
- 需要修改的段落或场景概述
- 当前主视角角色
- 角色此刻最强烈的欲望 / 担心 / 偏见
- 这一段希望读者感到什么
- 若已有原文，先标出最平、最像说明书的 3—5 句

## 步骤
### 1. 先找“作者在场感”从哪来
通常有 5 种：
1. 只写外部事实，不写角色怎样接收到这些事实
2. 所有信息一个轻重，像摄像头平扫
3. 动作做完就结束，没有心理回弹
4. 情绪只靠旁白宣布，如“她很难过”“他很愤怒”
5. 段落里看不出角色现在最在意什么

先逐句标记：哪些是外部信息，哪些是真正属于角色的感受或判断。  
**如果整段几乎全是外部信息，代入感一定弱。**

### 2. 给每个关键小节补一个“角色落点”
把重要描写改成这条链：
`外部变化 → 角色判断 → 情绪/身体反应 → 下一步动作`

不是每句话都加心理，而是**每个关键动作、关键描述、关键转折后，都要让读者知道这件事压在角色身上是什么感觉。**

### 3. 用角色目的筛选细节
同一个场景，不同角色会看到完全不同的东西。  
按主视角角色当下的目的选细节：

- 想自保的人，会先注意危险、眼色、退路
- 想复仇的人，会先注意破绽、把柄、谁在站队
- 想示弱的人，会先注意谁能利用、谁会心软
- 想表白的人，会先注意对方的反应、距离、语气

**不要把场景里所有细节平均写出来，只写会刺中这个角色的细节。**

### 4. 心理描写要短、准、当下
优先写下面 4 类，而不是长篇分析：
- 第一反应
- 当下判断
- 突然冒出来的欲望或担心
- 自我压住情绪的一瞬间

可直接套用：
- `她先是____，随即意识到____。`
- `他本能想____，却硬生生忍住了。`
- `这句话落进她耳里，不像劝，更像____。`
- `他明知道____，还是忍不住____。`

**心理描写的任务是让场景更有体温，不是给角色写小论文。**

### 5. 动作和心理要交替，不要各写各的
高频好用顺序：
- 动作 → 心理一笔 → 动作
- 描述 → 判断一笔 → 对话/动作
- 对话 → 误读/反应一笔 → 反击/沉默

如果连续大段心理不落地，场景会飘。  
如果连续大段动作没有感受，人物会空。

### 6. 把情绪藏进具体反应里
少写结论词，多写可见反应：

- 不直接写“紧张”，改写手指、呼吸、停顿、视线
- 不直接写“委屈”，改写喉咙发紧、嘴硬、话到嘴边咽回去
- 不直接写“愤怒”，改写措辞变硬、动作变慢、笑意消失
- 不直接写“心动”，改写多看一眼、回避视线、注意到平时不会注意的细节

**让读者自己感觉到情绪，比作者替读者下结论更有代入感。**

### 7. 写完一大段就切成读者视角回看
至少查 3 项：
1. **通顺易懂**：读者第一遍能不能顺着读下去
2. **情感真实**：这个反应像不像这个人、像不像这个处境
3. **印象深刻**：这一段有没有留下一个能记住的点  
   可能是强冲突、强欲望、独特细节、情绪反转、人物选择

如果三项里有两项不达标，不要硬往下写，先回修。

### 8. 练习“不要一气呵成地冲到底”
训练顺序：
- 一段一回头：每写完一段，检查有没有角色落点
- 两段一回头：检查前后情绪是不是连着的
- 三段一回头：检查这一小段到底在让读者代入谁、感受什么

**修改不是打断灵感，而是把“会写”练成“写得进人心里”。**

## 输出格式
1. 作者在场感最重的句子
2. 应补的心理或情绪落点
3. 建议保留的外部细节
4. 建议删掉的空说明
5. 改写后的段落
6. 改写说明：代入感增强在哪里

## 快速示例
### 输入
- 段落：  
  “门开了，屋里站着很多人。沈昭走进去坐下。有人让她认错，她没有说话。”

### 输出
1. 作者在场感最重的句子：  
   “屋里站着很多人。” 只有客观信息，没有角色过滤。
2. 应补落点：  
   沈昭进门时的第一判断；她为什么沉默；沉默里是怕、恨，还是硬撑。
3. 改写：
   “门一开，满屋目光齐齐压过来。沈昭脚下顿了一瞬，指尖却先一步掐进掌心——这阵仗不像问话，倒像等着看她低头。她还是走了进去，慢慢坐下，脊背挺得发疼。‘认错。’上首的人开口。她喉头发紧，最终什么也没说，只把下唇咬得更白。”
4. 改写说明：  
   补上了角色判断、身体反应和硬撑的意志，读者更容易站进她的处境里。

## 注意事项
- **禁止**每句话都塞心理，写成流水账式自言自语
- **禁止**让角色知道自己不可能知道的信息
- **禁止**用“她很悲伤”“他很愤怒”代替具体反应
- **禁止**心理描写和当前动作脱节
- 代入感不是多写内心，而是**让读者始终知道：这件事落在角色身上，正在怎样发热**
