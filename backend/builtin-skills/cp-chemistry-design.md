写男女主或 CP 时，先别急着表白。**CP 感最稳的来源是“例外感”——两个人在彼此面前会变得不一样，而这种不一样不会给普通配角。**读者一旦看出这种区别，就会默认这段关系有恋爱可能。

## 适用情况
- 两人互动不少，但像朋友、队友、同事
- 表白、承诺很多，读者还是不嗑
- 想写暧昧期、动心期、重逢期的恋爱感
- 角色对谁都一样温柔或一样嘴欠，看不出偏爱

## 核心原则
1. **恋爱感不是“说喜欢”，而是“把对方当例外”。**
2. **对别人无所谓的事，对对方会在意。**
3. **拘束、紧张、维护形象，往往比直白告白更像心动。**
4. **深情藏着写，行动优于语言。**
5. 配角也能制造糖，但**主 CP 必须有只有他们才会出现的反应**。

## 输入
- A / B 平时对外的默认态度
- A / B 在对方面前最容易失衡的点
- 当前关系阶段（刚认识 / 暧昧 / 已恋爱 / 破镜重逢）
- 这场戏要让读者意识到什么：心动 / 偏爱 / 余情未了 / 吃醋 / 不舍

## 步骤
### 1. 先列“默认状态”
分别写清 A / B 面对普通人时：
- 说话风格
- 距离感
- 会不会自恋、嘴欠、强势、散漫
- 会不会在意形象
- 会不会主动照顾别人

### 2. 再列“例外状态”
只挑 2—4 个最明显的不同：
- 平时邋遢，在对方面前会整理衣服、在意气味
- 平时嘴贫，在对方面前突然收着说
- 平时谁都能怼，在对方面前会留余地
- 平时不解释，在对方面前会多解释一句
- 平时不爱麻烦别人，在对方面前却会下意识求助
- 平时不管别人死活，在对方面前会先护住、先回头、先注意细节

### 3. 用“同类场景对比”放大例外感
优先写同一种处境下，对配角和对 CP 的不同反应。

例如：
- 都是刚运动完：对别人满不在乎，对喜欢的人却担心汗味
- 都是被夸：对别人顺势接，对喜欢的人反而别扭
- 都是求帮忙：对别人嫌麻烦，对喜欢的人先答应再嘴硬

### 4. 把深情藏进小动作
优先使用：
- 停顿
- 整理衣领、擦手、拉开距离又忍不住靠近
- 明明能说狠话，却换了 softer 的说法
- 下意识替对方挡一下、留门、偏伞、记住小习惯
- 想照顾，却先找借口

**不要**一上来就让人物把目的挂嘴上。

差写法：反复强调“我要和你结婚”“我最爱你”  
稳写法：先有气氛、铺垫、反应，再让读者自己看出这人栽了。

### 5. 写“在意形象”的反应
恋爱中的人常见信号：
- 担心自己不够好看、不够体面、身上有味道
- 说出口前先犹豫
- 明明平时厚脸皮，到了对方面前反而拘束
- 说完一句话会偷偷观察对方反应
- 被对方看见狼狈样时会更在意

### 6. 配一个第三方观察点
让配角或旁观者只看见“他们俩不对劲”，但不要替读者总结“他们相爱了”。

旁观者可观察：
- 语气差别
- 站位差别
- 忍让差别
- 对别人和对对方的反应速度差别

### 7. 每场戏只推进半步
从下列变化里选 1 个：
- 普通 → 在意
- 在意 → 偏心
- 偏心 → 吃醋
- 吃醋 → 不敢承认
- 冷着脸 → 余情未了
- 嘴硬 → 肢体先露馅

## 输出格式
1. A / B 的默认状态
2. A / B 的例外状态
3. 适合做对比的场景 3 个
4. 每个场景里的关键动作 / 语气差别
5. 第三方能看出来的异常
6. 本场关系推进半步是什么
7. 需要删掉的直白台词

## 快速示例
### 输入
- 男主平时自恋又邋遢
- 女主是他真正喜欢的人
- 目标：写出暧昧期的 CP 感

### 输出
- 默认状态：男主面对普通女配，浑身汗味也无所谓，照样凑近开玩笑
- 例外状态：男主面对女主，会先闻自己衣服、下意识拉开一点距离、说话没平时那么放肆
- 对比场景：刚打完球 / 挤地铁 / 下雨共伞
- 关键动作：女主说冷时，他第一反应不是把带汗的外套塞过去，而是先犹豫，怕她嫌弃，再想别的办法
- 第三方异常：朋友发现他第一次在见人前照镜子
- 关系推进：读者意识到男主已经把女主当成例外

## 注意事项
- **不要**让角色对所有人都一样特别
- **不要**只写表白，不写例外反应
- **不要**为了甜硬塞亲密动作，先让人物“在意起来”
- **不要**让深情只停在口头承诺里
- 好嗑的 CP 感，本质上是：**这两个人在彼此面前，都不再是对别人那副样子。**
