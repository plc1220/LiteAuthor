为多人箭头场面设计差异化的人设反应、争端焦点和断章位置。**修罗场不刺激，通常不是人数不够，而是所有人都在用同一种语气问同一个低价值问题。真正带感的修罗场，是不同关系、不同欲望、不同态度同时撞向主角。**

## 适用情况
- 多人喜欢同一个角色，但场面写出来不紧张
- 修罗场只剩“你和谁更好”这种重复问答
- 每个情敌都像同一个人，只会吃醋
- 不知道该把冲突放在刚见面、刚对峙，还是主角回应前
- 场面热闹，但关系没推进、断章也没力

## 输入
- 参与修罗场的人物清单
- 箭头关系：谁喜欢谁、谁敌视谁、谁和谁是炮灰 / 主力
- 当前会面的场景
- 想打出的味道：酸爽、压迫、暧昧、敌意、真香、爆笑等
- 这场之后希望关系或局势变成什么样

## 步骤
### 1. 先画箭头图，不要直接写台词
至少写清：
- 谁是主角视角真正会在意的人
- 谁只是功能性炮灰
- 谁和谁之间除了喜欢主角，还有没有别的竞争、优越感或旧账

**没有箭头图，修罗场很容易写成一群人围着主角吵。**

### 2. 给每个人分一种“情感模式”
三个人都喜欢主角，不代表三个人都只能写成吃醋。

优先把模式拉开：
- **我行你不行**：优越感、看不起对手
- **嫉妒**：因为对方比自己强 / 更近 / 更被信任
- **隐忧 / 不屑**：觉得主角的做法危险、荒唐、不靠谱
- **伪善 / 温柔**：表面体贴，实则想抢位置
- **交付 / 崇拜**：我完全信你，愿意把选择权交给你
- **占有 / 控制**：不一定表白，但想主导主角的行动
- **绵里藏针**：话说得轻，刀却递得稳

**每个人最好只有一个主模式，再带半个副模式，不要全都一样。**

### 3. 选一个真正会炸的争端焦点
别急着写“和谁一起吃饭”“你更喜欢谁”。这类问题如果没有更深 stakes，通常不够刺激。

更稳的焦点是：
- 主角即将做一件危险或高价值的事
- 主角该信谁、不信谁
- 谁更有资格陪主角去 / 知道秘密 / 站在身边
- 主角的选择会改写安全、前途、资源、关系
- 主角刚完成一件了不起的事，几方都要借机表态

**修罗场的核心不是“选边站”的字面问题，而是“谁有资格影响主角的行动和心”。**

### 4. 让所有人围绕同一件事表态，而不是轮流问废话
最稳的写法是：
- 主角先做出一个动作 / 决定
- 其他人针对这同一件事给出截然不同的反应

示例模板：
- A：`这也敢做？` —— 隐忧 / 不屑
- B：`太危险了，不如和我一起。` —— 伪善 / 温柔
- C：`我信你，你决定。` —— 交付 / 崇拜

这样读者看的不是“谁又吃醋了”，而是**谁怎么看主角、谁最懂主角、谁最想控制主角**。

### 5. 让冲突既对主角，也对彼此
修罗场不能只有“所有人问主角”。还要让几方之间互相顶牛。

可用冲突来源：
- 资格之争：你凭什么替她决定
- 能力之争：你配不上 / 你护不住
- 立场之争：你根本不是真信她，只是想利用她
- 优越感之争：你行我不行？凭什么
- 嫉妒之争：凭什么她先信你 / 先看你 / 先选你

**主角是焦点，但修罗场的火不能只往主角一个人身上烧。**

### 6. 让主角的回应成为场面落点
多人表态之后，主角必须有一个会改关系的动作，不然这场戏就只是热闹。

可用落点：
- 主角明确偏向其中一人
- 主角拒绝所有人，自己做决定
- 主角当场拆穿某人的伪善
- 主角奖励真正信任自己的人
- 主角成功后，对不同反应给出不同回收

**修罗场最值钱的不是别人怎么吵，而是主角最后怎么处理这些态度。**

### 7. 断章优先断在“所有矛头同时指向主角”的一拍
常见候选：
- 刚见面：通常太早，情绪还没抬起来
- 互相放完两轮暗刺：可用，但还差一个集中点
- **所有人都把问题、敌意、示好或期待压到主角身上，而主角还没回应：最稳**

如果是多方对峙场面，优先断在：
- 所有人同时等待主角选择
- 某句带刀的话说完，主角还没回
- 主角做出关键动作前一秒

**断在将爆未爆，不断在刚进场。**

### 8. 压缩低价值声音，别让炮灰抢戏
如果场上有 4—6 个人，不要每个人都给同等分量。

优先保留：
- 真正会影响主角判断的人
- 情感模式最不一样的人
- 最能抬高冲突 stakes 的那一两个

可压缩：
- 功能重复的人
- 只会跟风吃醋的人
- 不会改关系走向的炮灰

### 9. 如果觉得不够刺激，先换问题，不先加人数
高危信号：
- 三个人同时问主角一个日常问题
- 三个人的态度都只是“你别理他”
- 场面吵完以后，谁都没更近、没更远、没更恨

这时优先改：
- 问题的 stakes
- 每个人的态度模式
- 主角的最终回应

不要先想着再塞一个人进来。

## 输出格式
1. 箭头图
2. 每个人的情感模式
3. 争端焦点
4. 各方反应
5. 角色之间的互顶点
6. 主角最终回应
7. 最佳断章点
8. 同质化风险

## 快速示例
### 输入
- 三个角色都单箭头主角
- 主角准备独自去做一件有风险但必须做的事

### 输出
- A 模式：隐忧 / 不屑，觉得主角太冒险
- B 模式：伪善 / 温柔，提出“和我一起，我能保护你”
- C 模式：交付 / 崇拜，直接说“你决定，我配合你”
- 互顶点：A 觉得 B 装好人，B 反咬 A 只会泼冷水，C 质疑两人根本不是真信任主角
- 主角回应：主角谁都不选，只带 C 提供的关键东西，自己去做
- 断章：三人同时看向主角，等她回答那一刻

## 注意事项
- **禁止**所有人都用同一种方式吃醋
- **禁止**把修罗场写成轮流问同一个问题
- **禁止**只让别人闹，主角却没有任何落点动作
- **禁止**刚见面就断章，情绪还没到峰值
- **禁止**吵完之后关系和局势毫无变化
- 修罗场最带感的时候，不是人多，而是**每个人都从不同角度逼出主角和关系的真相**
