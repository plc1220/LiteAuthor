按起因、经过、结果拆开一段情节，再决定哪里详写、哪里简写、哪里扩写。**一场戏不是每个部分都该写一样重。真正值钱的笔墨，优先给读者最想看的那一口：主角高光、反派吃瘪、甜点回响、沙雕反杀、围观见证。**

## 适用情况
- 一章内容塞太满，自己也分不清该多写哪里
- 重点明明有，但写出来像流水账
- 打脸、甜戏、对手戏总是一闪而过
- 该扩的地方没扩，该简的地方反而写了很多
- 同一种套路在同一段里反复用，读者容易疲

## 输入
- 当前要写的事件
- 这一段最想给读者的感受
- 主角在这段里的高光点
- 反派或配角的作用
- 这段要不要承接后续余波

## 核心概念
- **详写**：正面写清，让读者看懂因果、压迫、选择或过程
- **简写**：只保必要动作和结果，避免拖慢节奏
- **扩写**：把已经发生的关键点继续写出回音、反应、后果和余味

## 步骤
### 1. 先把事件拆成三段
写成：
1. 起因
2. 经过
3. 结果

不拆开，就很容易整段一样重，最后全都不重。

### 2. 先定本场真正重点
只选一个优先级最高的重心：
- 为什么会这样
- 主角怎么破局
- 破局之后有多爽、多甜、多痛快

**一场戏可以有多个功能，但笔墨重点最好只有一个主项，最多一个主项 + 一个辅项。**

### 3. 给三段分别标：详写 / 简写 / 扩写
常见排法：

#### A. 起因详写，经过简写，结果扩写
适合：
- 打脸
- 受辱后反击
- 先压后扬
- 需要先把憋屈和不公立住

写法：
- 起因写清楚对方怎么压人、为什么可恨
- 经过不要拖泥带水，让主角一击即中
- 结果重点写对方反应、路人见证、余波扩散

#### B. 起因简写，经过详写，结果中写
适合：
- 智斗
- 谈判
- 追逐
- 修罗场
- 推拉感强的对手戏

写法：
- 起因只交代必要信息
- 重点写应对、拆招、试探、临场变化
- 结果写到确认输赢或关系变化即可

#### C. 起因简写，经过简写，结果扩写
适合：
- 爽点回收
- 掉马
- 公开打脸
- 论坛体、围观体、群众反应

写法：
- 前两段只保最必要的进入
- 把笔墨集中到“回响”上
- 重点写看客、舆论、对手失态、主角形象抬升

### 4. 把“读者舒服的内容”单独圈出来
优先扩这些：
- 主角解决麻烦
- 主角形象变强、变苏、变带感
- 反派被反噬、被自己人打脸
- CP 的偏爱、护短、嘴硬心软
- 沙雕反常规
- 路人点评、论坛水贴、围观见证

**让读者舒服的内容不是“重复同一个梗很多遍”，而是把最该爽的那一下写满。**

### 5. 同一种套路，一段里不要打两次
如果 A 剧情里已经用了 A 套路达成效果，就不要在同一段里再原样来一次。
可重复的是“口味”，不是“招式”。
- 可以在 B 剧情换汤不换药
- 但 A 剧情内部不宜连续复读

### 6. 配角只在能放大主角时扩写
扩写配角时先问：
- 他是在见证主角高光吗
- 他是在放大反派难堪吗
- 他是在自然带出下一步冲突吗

如果只是为了热闹多写几句，删。

### 7. 检查有没有把“应该略写”的地方写成注水
逐段问：
- 删成一句话，读者会不会照样懂
- 这段有没有新的冲突、信息、情绪或回响
- 这段是不是只是在重复已经成立的东西

能一句说完的经过，就别写成三段。

## 输出格式
1. 事件的起因 / 经过 / 结果
2. 每段标记：详写 / 简写 / 扩写
3. 本场真正重点
4. 读者舒服点
5. 该扩写的反应与余波
6. 该压缩的部分
7. 重复套路风险

## 快速示例
### 输入
- 主角被年长者当众苛责为难

### 输出
- 起因：详写。写清主角为什么被针对，对方如何倚老卖老、卖弄压制
- 经过：简写。主角直接一句反击或一个动作破局
- 结果：扩写。写对方失态、路人侧目、旁人点评、后续风声
- 重点：不是“主角到底说了几句话”，而是“这口气如何出得掷地有声”

## 注意事项
- **禁止**三个部分一样重，写成平铺直叙
- **禁止**把主角高光一笔带过，却把过渡动作写很多
- **禁止**在同一段里反复使用同一招式制造同一种爽点
- **禁止**扩写与主角无关的配角热闹
- 详略安排的目标不是省字数，而是**把读者最想看的地方写值钱**
