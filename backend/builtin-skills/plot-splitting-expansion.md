先把一句梗拆成动作，再把动作拆成矛盾。**拆分不是注水；每多一层，都要新增冲突、人物信息、后续钩子或情绪变化。**

## 适用情况
- 只有一个梗或一个画面，不知道怎么写成一场戏
- 场景太薄，只有结果没有过程
- 一扩写就变流水账或注水
- 想写“见面、吃饭、吵架、起床、告白”这类日常桥段，却生不出戏
- 有片段灵感，但放不进大纲

## 输入
- 一句话场景或梗
- 参与人物
- 当前关系 / 立场
- 这场戏想达成的结果
- 可用场景或道具（可选）

## 步骤

### 1. 先写基础句
用一句话写清：`谁在什么场景里，想做什么。`

没有“想做什么”，就很难拆。

### 2. 把大动作拆成 2-4 个小动作
例如：
- “起床”可以拆成：醒来 → 找衣服 → 穿衣服 → 冲出门
- “一起吃饭”可以拆成：发出邀请 → 对方是否答应 → 吃饭过程中出状况 → 饭后余波

先拆动作，后加戏。

### 3. 每个小动作都加一个阻力
常用阻力：
- 环境阻力：迟到、下雨、堵车、地点不合适
- 人物阻力：对方拒绝、插入第三人、立场冲突
- 内心阻力：尴尬、心虚、嘴硬、不想示弱
- 信息阻力：误会、隐瞒、答非所问、没说清

**没有阻力，动作只是在走流程。**

### 4. 补“为什么是现在”
逐条问：
- 为什么偏偏现在做这件事
- 为什么是这个人来做
- 不做会怎样

这一步用来补动机，避免桥段像作者硬安排。

### 5. 给场景加失败点或意外岔路
不要只想“成功怎么写”，还要想：
- 邀请没成功怎么办
- 计划被打断怎么办
- 本来想示好，结果演成出丑怎么办
- 本来想整人，结果自己被反整怎么办

失败点一出来，场景就会长出转折。

### 6. 提前放“目标预告”
让读者知道这场戏值得往下看什么：
- 会不会出丑
- 会不会打脸
- 会不会露馅
- 会不会关系变好 / 变坏
- 会不会引出下一场更大的事

这是场景的期待感来源。

### 7. 补情绪变化
至少写清三点：
- 开始前是什么情绪
- 进行中怎么变
- 结束后留下什么余波

同一件事，情绪不变，戏就容易平。

### 8. 区分主线和细节
- **主线**：一改动，后面大片剧情都要跟着变
- **细节**：改了不会伤到整条故事骨架

写大纲时先保纯主线，细节单独分类。不要让细节把主线淹掉。

### 9. 反查有没有注水
每个新增的小动作至少要完成一项：
- 推进主线
- 制造矛盾
- 揭示人物
- 埋伏笔 / 回收伏笔
- 改变关系或局势

答不上来，就删或并。

## 输出格式
1. 基础句
2. 动作链
3. 每步阻力
4. 动机
5. 失败点 / 意外岔路
6. 目标预告
7. 情绪变化
8. 这场戏属于主线还是细节
9. 可直接开写的场景提纲

## 快速示例
### 输入
- 场景：女主请男主吃饭

### 输出
- 基础句：女主为了道歉，想请男主吃饭
- 动作链：开口邀请 → 男主拒绝 → 女主换理由再约 → 男主答应 → 吃饭时出状况 → 饭后关系变化
- 阻力：男主不想赴约；女主心虚；餐厅选错；第三人突然出现
- 失败点：本来想赔礼，结果因为一句话又吵起来
- 目标预告：读者会期待这顿饭到底是修复关系，还是把矛盾闹得更大
- 情绪变化：心虚 → 紧张 → 破防 / 缓和

## 注意事项
- **禁止**为了扩写而平空加无用动作
- **禁止**只有结果，没有过程里的阻力和变化
- **禁止**把细节写得比主线还重
- **禁止**一拆分就忘了这场戏到底要达成什么
- 拆分的目标不是写长，而是把场景写活
