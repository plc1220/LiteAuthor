排查卡文时，先别继续硬写，先把卡点归类。**卡文通常只有两大类：一类是根上没矛盾，所以没剧情；一类是知道要去哪里，但中间没有桥。**

## 适用情况
- 开文几章就没东西写
- 有大纲，但正文一句都落不下来
- 明明知道下一步要升温/翻脸/打脸，却不会写过程
- 中段日常戏、感情戏、升级线总卡壳

## 输入
- 当前卡住的章节或阶段
- 核心设定 / 一句话主梗
- 最近一个已完成的剧情点
- 下一个明确想写的转折点
- 主要人物关系与当前矛盾
- 现在最具体的卡点：没事件 / 没桥 / 没细节 / 不知改哪

## 先判型
### A. 完全没东西写
常见表现：
- 写两章就把梗用完了
- 只能让人物重复聊天、吃饭、赶路
- 一问“接下来写什么”，脑子是空的

### B. 有纲但写不出来
常见表现：
- 知道“这里要心动/和好/翻车/公开打脸”
- 但从上个剧情点走不到这个点
- 只会写结果，不会写过程

## 步骤
### 1. 先写一句当前卡点
填空：

`我卡在：已经写到____，下一步想写____，但不会写____。`

## A类：完全没东西写

### 2A. 先查核心设定有没有矛盾
连续回答 4 个问题：
1. 主角现在最想要什么
2. 谁或什么在拦
3. 不做会失去什么
4. 为什么必须现在动起来

**空掉 2 项以上，通常不是文笔问题，是设定没搭好。**

### 3A. 再查是不是“梗用太快”
问：
- 核心卖点是不是前几章就用完了
- 有没有把所有信息一次性讲完
- 主角是不是很快就失去新的阻力和目标

如果是，说明节奏过快，后面自然会空。

### 4A. 用“设定矛盾”补根
从下面选 1—3 个补进去：
- **改主角**：补一个更明确的欲望、不得不、执念、软肋
- **改对手**：让关键人物和主角有更直接的吸引、利用、竞争、误判、敌意
- **改背景**：加规则、惩罚、期限、外部压力、身份障碍
- **改关系**：让核心人物不得不频繁产生交集
- **改后果**：让“不行动”的代价变具体

### 5A. 把大梗拆成 3—5 个小梗
每个小梗都写清：
- 当前小目标
- 当前阻力
- 小梗结束后，关系或局势改了什么

**没有变化的小梗，不算梗。**

### 6A. 快速修设定模板
按下面格式补：
- 主角想要：
- 阻力来源：
- 不做的代价：
- 不得不绑定的对象：
- 背景施压方式：
- 第一阶段 3 个小梗：

## B类：有纲但写不出来

### 2B. 先钉死这段终点
用一句话写：

`到这段结尾，必须被炸出来的信息/变化是____。`

例如：
- A 开始习惯 B 的存在
- 女主终于察觉男主偏爱
- 小团体第一次公开站队主角
- 对手第一次真正忌惮主角

### 3B. 反推：这个转折在解决什么矛盾
转折不是空降结果，而是**某个矛盾被解决**。
继续写：
- 之前卡住的矛盾是什么
- 解决后会出现什么明确变化

如果答不上来，说明这个“转折”还只是结论，不是剧情。

### 4B. 找一个“最小梗”当桥
选一个非常小、非常具体的场景，去承载转折。  
优先选：
- 吃饭 / 早饭 / 点外卖
- 顺路 / 送东西 / 借物
- 同住 / 同桌 / 同任务
- 帮忙 / 打掩护 / 临时求助
- 日常习惯 / 小道具 / 重复出现的偏好

**最小梗越小，越容易一梗多用。**

### 5B. 用“前置信号 → 小变化 → 明确信息”铺转折
给这个最小梗写 3 层：

#### 前置信号
前面已经出现过的习惯、偏好、物件、嘴硬反应、关系状态

#### 小变化
有人开始注意、默认、破例、容忍、靠近、别扭、吃醋

#### 明确信息
一个具体动作，把转折正式炸出来。  
例如：递纸、打掩护、主动维护、失去分寸、改站队

### 6B. 让一场小戏至少一梗多用
每场最小梗尽量同时完成 3 项以上：
- 刻画人物特质
- 体现核心冲突
- 做有效互动
- 埋下后文信息
- 推进情感或主线
- 引出下一梗

### 7B. 不会写时，先写“信号”再写“结果”
按这个顺序列 5 格：
1. 这一场的前提
2. 双方各自想要什么
3. 会反复出现的细节信号
4. 结尾要炸出的明确变化
5. 下一场从什么余波接

**先有信号，再有转折；没有信号的转折，只会像作者按头。**

### 8B. 如果仍然卡，优先补具体情景，不要补空心理
继续追问：
- 这个人会摸什么、看什么、顺手做什么
- 这个小道具会不会再次出现
- 谁会先破例
- 谁会先看见不对劲
- 这一场删掉后，会少哪一步变化

## 快速诊断表

| 症状 | 常见根因 | 优先处理 |
|---|---|---|
| 几章就没剧情 | 核心设定没矛盾 / 节奏太快 | 补设定矛盾，拆小梗 |
| 知道转折不会写 | 转折之间没桥 | 找最小梗，补信号 |
| 场面总是空的 | 只有结果没有过程 | 写前提、细节、明确动作 |
| 感情线不动 | 互动没结论 | 每场都写变化与余波 |

## 输出格式
1. 卡文类型：A / B
2. 根因判断
3. 缺失的矛盾或缺失的桥
4. 修设定方案或最小梗方案
5. 下一步 3 个可写场景
6. 每个场景的前提 / 信号 / 结果
7. 最容易继续卡住的点

## 快速示例
### 输入
- 已写：两人同住屋檐下
- 想写：高冷方慢慢喜欢上活泼方
- 卡点：不知道中间写什么

### 输出
- 类型：B 类，有纲写不出来
- 终点：高冷方开始习惯对方的存在，并第一次为对方破例
- 最小梗：早餐
- 前置信号：活泼方吃东西有固定习惯，高冷方一直嫌弃
- 小变化：高冷方开始不再吐槽，还会顺手递纸
- 明确信息：对方迟到时，高冷方居然主动帮忙打掩护
- 结果：关系从“嫌弃”推进到“习惯 + 破例”，还能顺手接下一场道歉或补偿戏

## 注意事项
- **禁止**一卡就全面推翻整本书，先查是根的问题还是桥的问题
- **禁止**把“心动了”“关系更好了”当成剧情本身
- **禁止**只列转折点，不补前置信号和具体场景
- **禁止**每场都只聊天，不让局势发生变化
- 卡文多数时候不是“不会写”，而是**没把矛盾搭明白，或没把转折之间的桥搭出来。**
