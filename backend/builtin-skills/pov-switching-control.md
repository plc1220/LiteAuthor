先问这次切视角到底补了什么。别人的心理不是不能写，但**非主视角人物的心理越多，代入感越容易散；能用感官、体感、动作、停顿和说话方式带出的，就别整段读心。** 全文通常以主视角为主，其他视角只负责补一刀更值钱的信息。

## 适用情况
- 总忍不住写男主、反派或配角心理
- 一章里频繁换脑袋，代入感弱
- 想写悬疑、解谜、多线并进，不知道何时切视角
- 配角戏越写越多，主角反而被挤掉
- 经常靠心理补丁解释人物为什么这么说、这么做

## 核心原则
1. 主视角以外的视角，通常只为**补充和丰富**。
2. 只为接续场景而切视角，通常得不偿失。
3. 能用主视角的观察、对白、动作、误判解决，就别切。
4. 前中期尤其要克制重要角色的完整心路，防止泄底、掉神秘感。
5. 可先用经验尺自检：**主视角 : 其他人物 : 全知 ≈ 7 : 2 : 1**。这不是铁律，但失衡时很好用。
6. **非主视角人物优先写感官、体感、动作和当下判断，少写完整心理链。**

## 什么时候该切
- 主视角之外的人物发起了重要情节点
- 需要补一块主视角拿不到、但读者知道后会更精彩的关键信息
- 需要拼出完整真相，或加深悬疑、解谜感
- 插叙必须补因果，不补会看不懂
- 某个配角或反派视角能明显改写读者理解

## 什么时候别切
- 只是为了让场景不断档
- 只是想顺手解释别人为什么这样说
- 只是想提前摊开 CP / 反派的好感、苦衷或秘密
- 只是主视角表达不准，所以拿别人心理打补丁
- **只是想把非主视角人物“此刻到底在想什么”全写出来**

## 切前四问
1. 这段新视角独有的信息是什么？
2. 这信息主视角真的拿不到吗？
3. 切过去后，读者理解会被明显改写吗？
4. 不切的话，能不能用动作、对白、场面反应解决？

四问里答不上两问以上，先别切。

## 实操步骤

### 1. 给每场戏标主脑袋
填写：
- 本场主视角：
- 本场想让读者知道什么：
- 这信息为什么只能从这个人这里知道：

### 2. 控制切换粒度
- 一场戏尽量一个主视角
- 必须换时，优先在**场景边界、章节边界、小结果落地后**切
- 切完第一句就让读者知道：现在是谁在看、为什么值得看

### 3. 非主视角先写“外显四件套”
若必须短切到配角或对手视角，优先只给：
1. 他看见 / 听见了什么
2. 身体哪里发紧、发麻、发疼、发热
3. 他做了什么动作，或没来得及做什么
4. 他的第一反应和最直接判断

**少写“他其实一直在想……”这种完整心路。**  
尤其感情戏里，非主视角人物的反应，优先写停顿、失语、回避、手上动作、体感变化。

反例：
- 他心里一惊，忽然意识到自己早就喜欢她，只是不敢承认。

改写：
- 她话音落下，他指尖一顿，半天没把杯子放回去。喉结动了动，却只别开眼说了句：“别胡说。”

### 4. 插叙和补心理，先问“有没有因果价值”
插叙常见合理用途：
- 补前因
- 补缺失信息
- 解谜回收
- 让重要人物的行动逻辑成立

无因果价值、只为抒情或解释的回忆，尽量删。

### 5. 做比例检查
统计最近 3—5 章：
- 主视角是否明显被配角挤占
- 配角视角是否长期比主角更有戏
- 是否总靠别人心理来替主角补信息
- 非主视角人物是不是经常被写得像全程可读心

如果答案是“是”，优先把镜头收回主线人物。

### 6. 若总想插播心理，先改写主视角表达
按顺序补：
1. 动作
2. 语气
3. 停顿
4. 视线
5. 场面结果

还不够，再考虑心理。很多“忍不住切视角”，本质是外显表达不够准。

### 7. 局部统一，长期也统一
- 一个场景里先保局部统一，不同段别乱换头
- 一段剧情里，尽量让读者知道“这几章主要跟着谁”
- 如果全文是多视角，也要有明显主轴；**80% 的正文最好仍围着主线人物展开**

## 输出格式
1. 当前视角分配
2. 每次切视角的理由
3. 可删的无效切换
4. 非主视角可改成外显的心理点
5. 需要保留的关键切换点
6. 调整后的比例建议
7. 神秘感 / 代入感风险

## 快速示例
### 原写
女主和反派对话两句，立刻切到反派心理解释“其实他是为了保护她”。

### 调整
保留女主视角，只写反派突然改口、压下视线、故意说反话。把“为什么保护”后置到更关键的揭晓点。

### 原写
为了把一场晚宴从头写完，女主视角写到一半，切男主视角，再切反派视角，最后切路人视角。

### 调整
整场晚宴只保留女主主视角；男主视角改成章尾单独一段，只补“他看见了女主没看见的关键动作”；反派和路人反应压进女主的观察里。

## 注意事项
- **不要**把第三人称写成全员随时可读心
- **不要**让配角视角长期抢走主角戏份
- **不要**靠心理补丁代替精准表达
- **不要**把“结构复杂”误当成“乱切视角”
- 视角切换的目标不是显得高级，而是**让信息更值钱、节奏更集中、代入感更稳**。
