以“被宠到了”为目标设计甜戏。优先用小任性索取和提前照顾，不要只会送钱、送资源、救场。

## 适用情况
- 想写甜宠，但只会写买买买
- 女主 / 受方总像无底线索取，读者容易烦
- 男主 / 攻方总像工具人保姆，甜味不对
- 想写“她被好好放在心上”的感觉

## 输入
- CP 设定与当前关系阶段
- 这一场甜戏想让谁更有“被宠感”
- 可用场景：吃饭、夜宵、生病、通勤、学习、任务后休整等
- 想要的甜味：软甜 / 傲娇甜 / 暗甜 / 反差甜

## 步骤
### 1. 先定这场戏的被宠核心
用一句话写清：
`这一场要让读者感到：TA被宠，是因为____。`

常见核心：
- 小小任性也会被包容
- 没开口的需要也会被照顾
- 不舒服、不方便、不体面时也被安稳接住
- 对方连自己没注意到的细节都先想到了

### 2. 先选甜点来源：索取 / 照顾 / 混合
#### A. 小任性索取
让一方提出**小、轻、无伤大雅**的要求。

适合：
- 想吃点不该吃的
- 想多要一点陪伴
- 想讨一个亲亲、一个抱抱、一个例外
- 想在对方面前稍微任性一点

#### B. 提前照顾
让另一方在对方没开口前，先把细节安排好。

适合：
- 饮食、温度、作息、伤病、通勤、情绪波动
- 收尾动作：递水、换口味、垫外套、留热饮、备药、善后

#### C. 混合
先有一个小索取，再在兑现时暗藏更高一级的照顾。**这类最容易出甜味层次。**

### 3. 写“索取”时守住四条线
- **必须小。** 让人觉得可爱，不让人觉得得寸进尺。
- **必须自然。** 只能落在信任关系里，像撒娇，不像命令。
- **必须补一笔懂事。** 抗议可以有，但本质不能真作天作地。
- **最好有回应。** 得到包容后，给一笔乖乖配合、害羞回看、偷偷开心，让读者知道这不是单向索取。

可直接检查：
- 这个要求换成陌生人提，还成立吗？如果不成立，说明亲密感对了。
- 这个要求会不会让对方付出过大代价？如果会，缩小。
- 删掉“她其实很懂事”这一层，角色会不会像作精？如果会，补。

### 4. 写“照顾”时不要只写结论，要写加码细节
不要只写“他很细心”“他很体贴”，改写成：
- 为什么会担心
- 具体调整了什么
- 做完后有没有继续善后

常见加码法：
- 把对方要的东西换成更适合她的版本
- 处理口感、温度、软硬、分量
- 做完主动作第二步收尾
- 一边嫌麻烦，一边还是照顾到底
- 外表平静，内里一直在算她会不会难受

### 5. 让“被宠感”落在细节链上，不要只落在结果上
优先写成链条：
`小欲望 / 小不适 → 对方注意到 → 悄悄加码 → 收尾照顾 → 受方感到自己被放在心上`

比起“他给她买了很贵的东西”，更有效的是：
- 她想吃夜宵，他嘴上不赞成，手上已经把更合适的版本做好
- 她喝了不爱喝的东西，他还记得给她补热水、盯着她喝完
- 她没说冷，他先摸到她手凉，直接把热饮塞过来

### 6. 按男主类型换写法
- **暖男**：周到、细致、提前想到
- **傲娇**：嘴硬、动作快、做完就跑
- **高冷**：话少，但处理得很稳
- **狼狗 / 小狼狗**：手忙脚乱，但很拼命
- **粗线条男主**：不够细腻也没关系，重点写“笨拙地想照顾好她”

**不要把“照顾型甜”局限在暖男。**

### 7. 视角决定甜味
- 用受方视角写：重点是“我被宠到了”
- 用照顾方视角写：重点是“他表面没说，心里已经替她想完了”
- 围观视角只作点缀，不要喧宾夺主

### 8. 控制度，避免翻车
高危信号：
- 全文只会一种宠法
- 女主什么都不做，只负责提要求
- 男主无微不至到像保姆 / 奴仆 / 舔狗
- 每次都靠花钱、送礼、替做事表达爱

改法：
- 一场戏只点一到两笔
- 索取和照顾穿插写，不要次次堆满
- 让被宠一方也给出信任、回应、偏爱或理解
- 多用留白，别把每一步都喊出来

### 9. 甜戏结束后至少回收一项
- 关系更近
- 受方更敢任性一点
- 照顾方更确定自己在意
- 旁人开始看出不对劲
- 之后某个冲突因为这场被宠感而更有重量

## 输出格式
1. 本场被宠核心
2. 甜点来源：索取 / 照顾 / 混合
3. 小要求或小不适
4. 照顾方的加码细节
5. 受方的反应
6. 这场戏的分寸提醒
7. 余波

## 快速示例
### 输入
- 设定：豪门大小姐 × 温柔男主
- 需求：写一场夜宵甜戏，不能落入炫富

### 输出
1. 被宠核心：她在他面前可以小小任性，他也会把她照顾好
2. 甜点来源：混合
3. 小要求：她嘴馋，非想吃一碗不太健康的夜宵
4. 加码细节：他嘴上说不行，手上却把做法改得更温和，分量减半，口感调软；吃完后又顺手补一杯热饮，怕她胃不舒服
5. 受方反应：先抗议，再乖乖喝完；表面还挑两句，心里已经被哄顺了
6. 分寸提醒：她可以馋、可以磨，但不能写成无底线折腾人
7. 余波：她更确定自己在他这里是被特别照顾的

## 注意事项
- **禁止**把索取写成狮子大开口
- **禁止**把照顾写成男主单向当保姆
- **禁止**只写“宠”，不写受方感受到什么
- **禁止**把“作”误当成“甜”
- **很多时候点一笔比堆满更甜**
