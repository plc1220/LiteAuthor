把主角写成“读者看得见行为和矛盾，却暂时看不透底牌”的状态。**神秘感来自信息克制，不来自故弄玄虚。**

## 适用情况
- 感情流、苏文、强设定主角文里，想让主角更耐看
- 开篇需要追文钩子，但不想靠硬塞大事件
- 主角一出场就把想法全说透，魅力掉得太快
- 想让读者产生“还想继续了解这个人”的欲望

## 输入
- 主角定位、题材、核心卖点
- 当前开篇或场景梗概
- 主角有哪些信息必须先藏住
- 哪些魅力点必须尽快交付
- 预计在哪些节点回收真相

## 核心原则
1. **先让读者看见矛盾，再让读者慢慢看懂原因。**
2. **先写可见表现，再写主角内在。**
3. **神秘感必须逻辑自洽。** 读者可以暂时不知道答案，不能感觉作者在乱藏。
4. **藏“为什么”，不藏“有什么可看”。**
5. **心理描写越多，神秘感掉得越快。**

## 步骤
### 1. 先写读者此刻最想知道的 3 个问题
用问句列出：
- 主角到底是什么人？
- 他为什么这样做？
- 他和谁、和过去、和眼前矛盾之间到底有什么关系？

后文所有信息投放，都围绕这 3 个问题服务。

### 2. 把主角信息分成三层
#### 必须立刻给
不立住就没人追的东西：
- 气场
- 当前处境
- 最抓人的外显特征
- 和主线直接相关的异常点

#### 先给半口
能引发猜测，但不必解释透的东西：
- 反常的选择
- 暧昧的态度
- 和身份不匹配的生活细节
- 别人看不懂、但明显有故事的行为

#### 暂时藏住
真正决定神秘感寿命的内容：
- 真实动机
- 完整过去
- 真正立场
- 感情底牌
- 关键秘密

### 3. 先用“外显矛盾”立钩子
优先组合两类看似不搭的信号：
- 很穷的处境 + 很贵的细节
- 很冷淡的态度 + 不合常理的维护
- 很普通的身份 + 明显高位者的反应
- 很被动的表面 + 实际掌控局势的动作

**不要一上来解释矛盾成因，先让读者记住这个矛盾。**

### 4. 写场面时，优先用外部镜头
优先顺序：
1. 别人看到主角什么
2. 主角做了什么动作或选择
3. 场面因此出现了什么反应
4. 最后才给主角一笔必要的内在反应

常用载体：
- 眼神、停顿、站位、习惯动作
- 对别人不一样的反应
- 不合身份的熟练 / 生疏
- 别人的误判、猜测、被吸引

### 5. 心理描写只留“一笔带方向的”，不要整段剖白
优先写：
- 一瞬间的犹豫
- 被刺中的反应
- 强行压下去的情绪
- 选择之后留下的余味

少写：
- 把真实动机一口气讲完
- 长篇解释“其实他是因为……”
- 直白宣布“他很爱她”“他很痛苦”

### 6. 每场戏至少保留一个“为什么”
写完一个场景后检查：
- 读者有没有得到新线索
- 读者是不是还想继续问一个更深的问题

如果一场戏把主角解释得干干净净，神秘感就断了。

### 7. 用“小回收 + 更深问题”续命
每 3—5 章，至少回收一层答案，同时抛出更深一层问题。

示例：
- 先让读者知道主角不是普通人
- 再让读者想知道主角为什么沦落至此
- 再让读者想知道他究竟站哪边

**只藏不回收，会变成拖；只回收不升级，会很快失效。**

### 8. 关键节点再开主角内里
适合真正放开主角视角或心理的时机：
- 感情转折点
- 价值选择点
- 身份真相快揭开时
- 必须让读者彻底共情的场面

在这些节点之前，尽量多让主角“被看见”，少让主角“自我解释”。

### 9. 可参考的视角比例
开头和前半段可尝试：
- **外部观察 / 导演镜头：约 30%**
- **贴近主角的跟随视角：约 69%**
- **直白心理解释：约 1%**

这是参考，不是铁律。重点是：**外部可见信息多一些，解释性心理少一些。**

### 10. 排雷
出现下面情况就重做：
- 把一句能说清的话，硬拆成大段神态外貌凑“高级感”
- 只有神秘，没有逻辑
- 主角什么都不想、什么都不说、什么都不做，只剩空壳
- 藏得太狠，导致读者根本不知道这人有什么可追
- 过早把主角完整剖开，后面魅力迅速掉平

## 输出格式
1. 读者最想知道的 3 个问题
2. 必须立刻给 / 先给半口 / 暂时藏住
3. 这一段的外显矛盾
4. 推荐视角安排
5. 该删掉的泄底句
6. 该补上的外部线索
7. 小回收节点
8. 一个可直接改写的开场示例

## 快速示例
### 直给版
- 小少爷受不了总裁，离家出走去奶茶店打工。

### 神秘版
- 奶茶店来了个新员工，长得很干净，手腕上那块表抵得上店里半年流水。他不会给公交卡充值，却认识所有奢侈品牌的隐藏款。每天晚上门外都有一辆豪车停着，车里的人从不下来，只在他下班前离开。

### 改写说明
- 不先交代完整前史
- 先给“处境反差”和“异常关系”
- 让读者先产生“他到底经历过什么”的追问

## 注意事项
- **禁止**把“神秘感”写成“故意不说明白”
- **禁止**频繁长篇内心戏把魅力说没
- **禁止**只靠旁白夸主角苏
- 真正能钩人的主角神秘感，是**信息不全，但行为、反应和逻辑一直成立**。
