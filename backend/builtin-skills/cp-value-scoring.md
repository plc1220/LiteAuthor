用分数检查 CP 感，不凭感觉。把每个章节、场景、回忆、误会、想念和路人视角都换算成关系增减，看这对人在读者心里到底有没有持续存在感。**分数不是数学游戏，而是逼自己看清：他们有没有互动、牵绊、偏爱和后果。**

## 适用情况
- 官配设定不差，但读者不嗑
- 两人同框很多，CP 感仍然弱
- 中段经常各写各的，关系像断线
- 需要检查哪些桥段该加分、哪些桥段在扣分

## 输入
- 当前大纲或章节摘要
- 双方当前关系阶段
- 题材和基调
- 预计要检查的范围（单章 / 一卷 / 全文）

## 步骤
### 1. 先给关系定一个起始分
可自定区间，例如 `-20 ~ 100`，也可简单用 `-10 ~ 10`。重点不是绝对值，而是增减逻辑是否连贯。

### 2. 给事件分三类记分
#### A. 直接互动
同框争执、约会、联手、表白、背叛、护短、误会。强事件可大幅加减分。

#### B. 间接存在感
即使没见面，也能记分：
- 吃到好东西，第一个想到对方 `+0.5`
- 做选择时受对方影响 `+0.5 ~ +2`
- 内心否定对方三观 `-0.5 ~ -2`
- 想起共同回忆、下意识找对方 `+1`

#### C. 外部刷存在感
用别人、舆论、环境提醒读者“他们是一组”：
- 闺蜜 / 同事追问两人进展
- 路人误会他们有情况
- 网友嗑 CP
- 别的角色一眼看出谁在意谁

这类分数通常不大，但能补连续性。

### 3. 大小分要和事件重量匹配
参考：
- 小心动、小想起、小照顾：`±0.5 ~ ±2`
- 明显护短、吃醋、站队、误会解除：`±3 ~ ±8`
- 大背叛、大真相、大和解、大牺牲：`±10` 以上

**严重事件要拉开差距，小事件只负责持续刷存在感。**

### 4. 允许“不同步加分”
同一场戏不一定双方都同时上涨。可以写成：
- 男主这场 `+3`，女主 `+0.5`
- 表面关系没变，内里其中一方已经失衡

### 5. 分开场景也要补分
如果一段时间不能同框，强制补至少一种：
- 想到对方
- 对话里提到对方
- 因对方改变决定
- 路人视角把两人并提

**长时间零分，读者就会忘了这是 CP 文。**

### 6. 看总分，更要看曲线
重点检查：
- 有没有长期不涨不跌的平直段
- 有没有刚涨一点就被无意义扣回去
- 高潮前有没有积累足够分数
- 和好或在一起之前，前面的扣分是否真的回收了

### 7. 用分数反查缺戏点
如果这几章总分不动，优先补：
- 牵绊
- 互动后果
- 想起对方的瞬间
- 路人视角
- 公开或私下的偏心

## 输出格式
1. 起始分
2. 事件记分表
3. 累计分曲线
4. 断档章节
5. 推荐补分点
6. 风险扣分点

## 快速示例
### 输入
- 女主今天吃到好吃的，想着下次带男主来
- 当晚两人没见面

### 输出
- 记分：`+0.5`
- 原因：虽然没同框，但女主已经把男主放进日常分享名单
- 建议：下一章可用闺蜜一句“你又想到他了？”再补一个外部存在感点

### 输入
- 女主发现男主可能是杀父仇人
- 后来查明是误会，男主还为她背了锅

### 输出
- 扣分：`-10`
- 回收：误会解除 `+8`，愧疚与心疼 `+7`
- 说明：重大负面必须有更强后续回收，不然 CP 会直接塌

## 注意事项
- **禁止**把分数当死公式硬套
- **禁止**只有同框才算分
- **禁止**只加不减，或只减不回收
- 分数的目的不是精确计算，而是检查这对人有没有在读者心里持续“活着”
