分析用户提供的故事梗概、正文摘要或他文剧情，提炼真正的主干剧情，并把主干、二级核心、细节支线分开整理。

## 适用情况
- 大纲越写越散，写着写着全是细枝末节
- 只会拆前三章，不会看全文主干
- 写到一半卡文、写崩，发现自己并不知道这文真正要往哪走
- 需要把长文压成投稿版、千字版或一句话大纲
- 主线、暗线、感情线混在一起，越写越乱

## 输入
- 故事一句话梗概
- 已有大纲、剧情列表或正文摘要
- 题材、篇幅、卖点
- 当前分不清的情节点（可选）
- 如果是研究他文，提供全文摘要或分卷摘要

## 核心原则
1. **先抓主干，再补细节。**
2. **改动后会牵动全文的，才算核心主干；只影响局部承接的，归细节支线。**
3. **一级核心只写总结句，二级核心才写关键剧情。一级、二级禁止混写。**
4. **主线、暗线、感情线、事业线等先分开写完整，再融合。**
5. **千字版大纲只保留核心，不准用细枝末节凑字数。**
6. 细节可以精彩，但没有主干时，细节只会把文写散。

## 步骤

### 1. 先判断这次是在“研究他文”还是“整理自文”
- **研究他文**：如果目标是学剧情，不要只拆前三章。先按全文总结主干，再回头看章节节奏和细节。
- **整理自文**：先停止扩写细节，把现有情节压缩成最少的信息量版本。

### 2. 先做一版 1000 字主干摘要
把整篇故事强行压进约 1000 字内，只写：
- 主角最核心的目标
- 主要阻力
- 关键转折
- 终局回收

**凡是写进这 1000 字里还嫌占地方的，多半不是主干。**

### 3. 用“改动波及范围”判断主干和细节
逐条检查每个情节：
- 改掉它，全文主线、终局、人物关系都要跟着重排 → **核心主干**
- 改掉它，只影响前后几章的承接和气氛 → **细节支线**
- 改掉它，只是少了一个热闹场面或一段说明 → **可删细节**

判断句式：
`如果删掉/改掉这段，全文最核心的故事还成立吗？`

### 4. 把核心再分成一级核心和二级核心
- **一级核心**：对一整段剧情的高层概括，一句话能统领一串同方向事件
- **二级核心**：一级核心下面的关键推进点，是具体剧情节点

写法顺序固定为：
1. 先列一级核心
2. 再在每个一级核心下面列二级核心
3. 最后才把细节支线另存

**不要把一级核心和二级核心写在同一层，否则节奏会乱。**

### 5. 把同方向剧情统一归总成一句话
把多个相似作用的剧情，归并成一个更高层的主干表达。

示例：
- 男主送礼
- 男主替女主做事
- 男主为女主受苦
- 男主为女主放弃重要机会

先分别记为二级核心，再统一归总为一级核心：
`男主持续为女主付出、奉献或牺牲。`

### 6. 把不同线先分开写完整
遇到主线、暗线、感情线、事业线等并行内容时，先分别整理：
- 每条线的起点
- 每条线的因果顺序
- 每条线的阶段目标
- 每条线的阶段回收

**不要一边写 A 线，一边顺手把 B 线逻辑揉进去。**  
先把每条线单独写顺，再决定它们在哪些节点交叉。

### 7. 再做“融合版”顺序
把分开的线重新合并时，只保留会互相改写局势的交叉点：
- 哪个节点必须由主线带出暗线
- 哪个节点必须由感情线放大主线
- 哪个节点能自然形成章节钩子或卷末转折

如果两条线并排存在，却互不影响，多半是散线，不是融合。

### 8. 细节单独放进另一个文档
把这些内容单独归档，不要混进主干大纲：
- 氛围描写
- 小道具、小误会、小插曲
- 具体台词和场面设计
- 阶段性角色的琐碎背景
- 不改变全文走向的支线补充

**主干文档负责方向，细节文档负责血肉。**

### 9. 做一次“有没有拿细枝末节充数”的检查
重点排查：
- 大纲里是否出现大量“发生了很多事”，但删掉也不影响全文
- 是否把某个局部冲突写得比真正主线还详细
- 是否用背景说明、日常流程、枝节人物来撑篇幅
- 是否因为不会写主干，就拿细节填空

### 10. 输出两版结果
至少同时给出：
- **完整版主干大纲**：用于自己写作
- **压缩版大纲**：用于投稿、文案、向编辑说明卖点

## 输出格式
1. 一句话总主干
2. 1000 字主干摘要
3. 一级核心列表
4. 每个一级核心下的二级核心
5. 主线 / 暗线 / 感情线 / 事业线分线整理
6. 应移出主干、单独存放的细节内容
7. 当前最容易导致卡文或写崩的散点
8. 100 字投稿版大纲

## 快速示例
### 输入
- 现有剧情很多：相遇、暧昧、误会、家族压力、事业竞争、旧案伏笔、多人配角日常
- 作者的问题：越写越散，不知道什么是主干

### 输出
1. 一句话总主干：女主为了翻身并查清旧案，被迫与男主结盟；两人在合作与试探中感情升级，最终一起掀翻幕后势力。
2. 一级核心：
   - 女主翻身夺回主动权
   - 女主与男主从互用到结盟
   - 旧案真相逐步逼近并引爆终局
3. 二级核心示例：
   - 翻身线：保住资格、第一次反杀、公开夺回资源
   - 感情线：被迫合作、互相试探、第一次明确偏向
   - 旧案线：发现线索、误判对象、锁定真凶
4. 应移出主干的细节：
   - 某次宴会上的衣着描写
   - 某个小配角的完整家庭背景
   - 与终局无关的日常闲聊

## 注意事项
- **不要**为了显得丰富，把一级核心和二级核心混成一团
- **不要**一边理主线，一边把所有细节一股脑塞进去
- **不要**只会分析前三章，不看全文怎么收束
- **不要**拿细枝末节假装自己有大纲
- 主干写不清时，先修主干，不要先修文笔
