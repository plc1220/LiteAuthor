围绕“他们为什么现在还不能顺利在一起”搭冲突，不要只靠一个薄弱误会硬拖。**感情线最怕两件事：核心矛盾太早解决，和每一步都靠硬冲突硬拽。甜宠也不是没有矛盾；矛盾不一定尖刻痛苦，很多时候只是两个人设放在一起后，自然流出来的差异、电压和失衡。**

## 适用情况
- 前期互动有了，中段却发空
- 核心矛盾解决太早，后面没钩子
- 每个阶段都硬塞刺激点，关系反而显得生硬
- 甜宠文很甜，但两个人总像没戏
- 想写攻守逆转、地位变化，却总像硬拗
- 表白、在一起或高强度亲密来得太早，后续索然无味

## 输入
- CP 双方人设、目标、抗拒点
- 关系起点、预期终点
- 当前题材与核心卖点
- 预计篇幅
- 已有矛盾 / 卡住的阶段

## 核心原则
1. **先写清“他们为什么还不能顺利在一起”。**
2. **矛盾不等于互相伤害。** 甜宠也需要矛盾，但可以是节奏错位、边界碰撞、表达差异和关系失衡。
3. **矛盾不是固定一攻一守。** 一动一静可以有张力，两个都强、谁也不让也可以有张力。
4. 一篇文里通常要有**一个最大的核心冲突**，最好吊到后面再真正解决。
5. 其他冲突负责递进、波动和承托，**不要几个大冲突挤在一起。**
6. 冲突没有固定次数要求。过多冲突只会让读者觉得像坐在颠簸车上。
7. 攻守逆转、地位转变可以反复用，但**每次都必须比上一次多一点进展。**

## 步骤
### 1. 先写一句主问题
按模板写：
`他们明明互相吸引，却暂时不能顺利在一起，因为____。`

如果这句只有一个很容易解决的理由，继续补，不要只靠单一薄弱矛盾拖全书。

### 2. 把矛盾分成 4 类，至少占 2 类
- **外部现实**：家庭、身份、立场、事业、任务、距离
- **过去包袱**：旧伤、旧情、黑历史、承诺、创伤
- **心理问题**：自卑、控制欲、不信任、责任感过重、害怕失控
- **关系问题**：信息差、节奏错位、价值观冲突、边界冲突

### 3. 先找“人设差异电压”
对双方分别写清：
- 气质差：温润 / 暴躁，沉静 / 张扬，冷淡 / 热烈
- 表达差：直球 / 克制，嘴硬 / 会哄，会撩 / 不接球
- 处事差：先冲 / 先忍，先控场 / 先观察
- 节奏差：想快一点确认关系 / 想慢一点试探
- 自尊与边界：爱掌控 / 不肯低头，容易越界 / 极重分寸

再判断这对关系更像哪种态势：
- **一攻一守**：一方主动推进，一方本能防守
- **王不见王**：双方都强，都想控场
- **表面平和，暗里拉扯**：嘴上不吵，行动和节奏始终不在一拍

张力不是作者宣布“他们有火花”，而是这些差异一碰就会起反应。

例如：
- **温润沉静 × 暴躁张扬**：张力来自一个稳、一个炸
- **骄傲锋利 × 暴躁张扬**：张力来自谁也不让谁

### 4. 先分清“温和转变”和“剧烈转变”
#### A. 温和转变
适合写：
- 对对方印象慢慢改观
- 从不顺眼到愿意多看一眼
- 从防备到默认对方可靠

这类转变**不一定要靠正面冲突**，可以靠长期交流、细节观察、一次恰到好处的理解慢慢完成。

#### B. 剧烈转变
适合写：
- 忽然意识到自己喜欢对方
- 决定承认自己爱上了原本不该爱的人
- 因某件大事彻底改写原有立场

这类转变通常要配一个更强的事件、抉择或代价。

**能慢写就别乱炸；要炸，就让它真有分量。**

### 5. 设计“大小冲突梯子”
按“先轻后重、先表后里、先外后内”排 3—5 级：
1. 第一级：让两人难靠近
2. 第二级：靠近后也不敢确认
3. 第三级：确认前后必须付代价
4. 第四级：真正触到核心抗拒或现实阻力

每解决一级，都必须顶出下一级。**最大的冲突尽量留到后面，小冲突负责一路抬温。**

### 6. 控制冲突密度
逐条检查每个冲突：
- 它会不会真的改关系，而不只是拖时间
- 它和前一个冲突相比，是递进、波动，还是纯重复
- 连着放会不会让角色显得一惊一乍、情绪太满

如果几个大冲突挤在一起，优先拆开，中间穿插：
- 日常交流
- 轻度误判
- 小范围吃醋或护短
- 非感情线的小事件

### 7. 不一定只写“感情冲突”
有些关系推进，不一定非要靠两个人互相闹。

可用的辅助冲突：
- 第三方原本讨厌某个主角，后来改观
- 外部任务把两人绑在一起
- 一方暴露不得已，给另一方和读者一次重新认识的机会

**目的不是让配角抢戏，而是给主角关系提供变化条件。**

### 8. 设计攻守逆转时，必须写出“前后差”
常用顺序：
`一方主动追 → 另一方逐渐软化但嘴硬 → 主动方因某事退半步 → 另一方出现危机感 → 另一方开始行动`

关键不是“谁追谁”，而是：
- 前面有没有软化和铺垫
- 逆转后有没有真实行动
- 下一次类似冲突时，人物能不能少走一点旧流程

例如：
- 第一次危机：还要经过“思想挣扎”才出手
- 第二次危机：直接跳过挣扎，立刻护人或宣告主权

这样读者才会感觉：**这段关系有进展，不是在原地重放。**

### 9. 检查关键亲密节点
逐一写明：
- 什么时候第一次明显心动
- 什么时候第一次默认对方特殊
- 什么时候表白 / 在一起
- 什么时候发生更高强度亲密
- 这个节点之后靠什么继续钩住读者

如果在一起后没有新阶段、新代价或新问题，宁可后置。

### 10. 互动推进不动时，先回人设，不要硬推
出现这些信号就先查人设：
- 互动全靠作者安排，不像人物自己会做的事
- 想推进关系时，总觉得别扭、违和
- 两人都冷、都闷、都不接球

优先补：
- 一动一静
- 一撩一防
- 一强一强但会对撞
- 一方先主动，一方后失衡

### 11. 反查“冲突过量”
出现下面情况就重排：
- 每个节点都非吵不可
- 一点情绪变化都要靠大事触发
- 前后冲突只是换皮重复
- 角色像永远在坐过山车，没有喘息和余韵
- 一次冲突结束后，人物毫无变化，下次又从头来

## 输出格式
1. 关系一句话主问题
2. 多重矛盾清单
3. 人设差异电压表
4. 温和转变点 / 剧烈转变点
5. 3—5 级冲突梯子
6. 最大冲突与后置理由
7. 攻守逆转节点与铺垫
8. 关键亲密节点表
9. 每级解决后顶出的新钩子
10. 人设与互动风险

## 快速示例
- 核心问题：两人互相有好感，但一方习惯掌控关系，另一方又极度抗拒被安排，关系无法轻易推进
- 人设差异：一方稳、会哄、擅长掌控节奏；一方炸、嘴硬、反感被看透
- 温和转变：从互看不顺眼，到开始默认“出事时找他 / 她”
- 剧烈转变：其中一方在一次危机后第一次承认“我不是想赢你，我是在怕失去你”
- 冲突梯子：
  1. 被迫接触，互相试探
  2. 相处后改观，但都嘴硬
  3. 一方退半步，另一方开始危机感
  4. 核心抗拒真正爆开，关系被迫正面选择

## 注意事项
- **不要**把矛盾理解成只有吵架和互虐
- **不要**只靠一个薄弱误会拖全书
- **不要**把每个阶段都写成硬冲突爆点
- **不要**把几个大冲突连着砸
- **不要**让攻守逆转没有前因，只靠作者宣布
- 感情线稳不稳，关键看：**每解决一个问题，关系有没有真的挪半步；每一次大变化，前面有没有足够铺垫。**
