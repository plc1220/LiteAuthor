拆解同题材参考文的单章骨架，不抄句子，只学结构。**同样三千字，有的章像活人，有的章像流水账，差别常常不在“写了什么”，而在“信息怎么排、谁先感受、谁来反应、有没有余波”。**

## 适用情况
- 一章里明明很多内容，读起来却不鲜活
- 不知道同样字数为什么别人更饱满
- 自己的章节总是大段叙事，缺少起伏
- 想用参考文校准单章节奏和构成
- 想知道一章到底该由哪些部分组成

## 输入
- 自己要优化的章节或片段
- 参考文 1—3 篇，**必须同题材、同风格、同频道、同字数区间**
- 本章目标：推进剧情 / 推进感情 / 落爽点 / 埋钩子
- 章节大概字数
- 当前最不满意的问题

## 步骤
### 1. 先找对参考物
优先选同题材、同风格、完成度高、成绩明显更好的样本。

如果目标是学商业写法，优先看成熟爆款或金榜文；不要拿和自己同样发空的样本互相参考。

### 2. 先拆“这一章分几部分”
不要一上来抠句子，先看这章大结构。

至少写出：
- 上半章在干什么
- 下半章在干什么
- 章尾落在什么点上

常见稳定写法：
- 上半章发生一件事
- 下半章写这件事的冲击、余波和众人反应

### 3. 再给每一段打功能标签
把参考章按句群或段落标成这些功能：
- 叙事
- 角色感受
- 对话
- 心理活动
- 肢体互动
- 他人反应
- 信息补充
- 环境/气氛

### 4. 统计顺序，不要只统计数量
重点看它是怎么排的。

例如一段可能是：
`[叙事][角色感受][对话][对话][肢体互动][心理活动]`

而自己的写法可能是：
`[叙事][叙事][叙事][对话][对话][叙事][对话]`

顺序一拉出来，就知道缺的不是“内容”，而是“构成”。

### 5. 找出这一章为什么“像活人”
重点观察 4 件事：
1. 事件发生后，角色有没有立刻产生感受
2. 感受后，有没有对话或动作把它外化
3. 这件事有没有带出他人反应或关系变化
4. 章尾是不是停在最值钱的变化点上

很多鲜活感都来自：
`事件 → 感受 → 对话/动作 → 余波`

### 6. 用同样方法拆自己的章节
对自己的章节做同样的功能标记和顺序统计，再和参考物并排对照。

### 7. 只学骨架，不抄内容
优先借的是：
- 一章分几部分
- 每段承担什么功能
- 事件和反应怎么交替
- 哪些信息放前，哪些信息放后

**不要照抄桥段、金句、设定和具体台词。**

### 8. 把差距翻成改稿动作
常见问题和改法：
- 叙事太多 → 插入角色当下感受和反应
- 对话在飘 → 补动作、停顿、心理落点
- 只有事情没有余波 → 补别人怎么看、局势怎么变
- 章尾太平 → 把最值钱的变化往后挪到章尾附近

## 输出格式
1. 参考章大结构
2. 参考章功能顺序
3. 自己章节的大结构
4. 自己章节的功能顺序
5. 关键差距 3—5 条
6. 可直接执行的改稿动作
7. 新章尾建议

## 快速示例
### 参考章
- 上半章：主角处理一件事
- 下半章：众人受冲击后的反应
- 段内顺序：叙事 → 感受 → 对话 → 肢体 → 余波

### 自己章节
- 全章都在交代事情经过
- 几乎没有角色感受、动作和他人反应

### 改法
- 每个关键事件后补一个“谁被触发了什么感受”
- 对话之间补动作和停顿
- 章尾停在反应最强或局势刚变的地方

## 注意事项
- **禁止**拿风格完全不同的文硬套骨架
- **禁止**只拆内容梗概，不拆段落功能
- **禁止**把参考文的句子和桥段直接搬走
- 真正有用的不是“别人写了什么”，而是**别人怎么安排同样的内容，让它更有呼吸感**。
