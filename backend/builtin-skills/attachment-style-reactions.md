写分离、冷淡、重逢、示好、安抚这类感情戏时，先别急着写“生气”“委屈”“撒娇”，先给角色定依恋类型，再反推他的本能反应。**同一件事，不同依恋类型会做出完全不同的动作、台词和求爱 / 防御方式。**

## 适用情况
- 同一段感情戏换谁来写都差不多
- 角色一被冷落就只会哭闹或沉默
- 想把傲娇、缺爱、嘴硬、懂事写得更有根基
- 需要设计离别、重逢、冷战、求和、安抚桥段

## 输入
- 当前关系阶段
- 触发事件：出差、失联、冷淡、误会、回归、示好等
- 角色既往经历：被稳定照顾 / 被忽视 / 被拒绝 / 关系混乱等
- 想让读者吃到的感觉：心疼、可爱、拉扯、安全感、虐感等

## 先定依恋类型
### 1. 安全型
- 核心：相信关系稳定，也相信自己值得被爱
- 常见表现：会想念，但不失控；能继续过自己的生活；愿意直接表达需求
- 适合写：省心、治愈、稳定偏爱、成熟温柔

### 2. 焦虑型
- 核心：怕被抛下，想靠近又怕不被珍惜
- 常见表现：反复确认、嘴硬黏人、表面攻击实际求安抚、情绪起伏大
- 适合写：傲娇、缺爱、吃醋、拉扯、可怜感

### 3. 回避型
- 核心：怕依赖、怕失控，先把自己藏起来
- 常见表现：被伤到后先退、装不在乎、拒绝接近、把需要感藏住
- 适合写：冷淡、克制、疏离、慢热、破防瞬间

### 4. 混乱型 / 恐惧回避型
- 核心：既想靠近又怕受伤，反应容易混乱
- 常见表现：忽冷忽热、刚求抱又立刻推开、情绪和行为不一致
- 适合写：创伤型角色、强烈不安全感、重度拉扯

## 步骤
### 1. 先写“这件事会触发角色最怕什么”
不要只写事件本身，要写它刺中了什么：
- 被冷落 → “我是不是不重要”
- 长时间分开 → “我会不会被忘掉”
- 对方回来 → “我还能不能相信这次不会再走”
- 对方示好 → “我要不要把软处露出来”

### 2. 按四层写反应
每个角色至少写清 4 层：
1. **第一反应**：立刻做什么  
2. **防御动作**：嘴硬、躲开、挑刺、假装忙、过度体贴  
3. **真实需求**：想被抱、想被解释、想确认位置、想先一个人缓冲  
4. **被安抚后的松动**：会不会靠近、说软话、收礼物、愿不愿解释

### 3. 让“攻击”和“求爱”同时存在
尤其焦虑型和混乱型，不要只写一种表面情绪。
例如：
- 嘴上赶人，身体却不躲
- 不接礼物，但偷偷留下
- 说不想见，对方真要走时又慌

### 4. 安排“同类型才会做”的细节
- 安全型：继续生活、准备礼物、愿意正面沟通
- 焦虑型：回消息忽快忽慢、故意呛人、测试对方会不会追
- 回避型：避开眼神、减少接触、把在意写成公事
- 混乱型：前一秒靠近，后一秒翻脸；一边求救，一边自毁

### 5. 让安抚方式对型
安抚不等于统一抱一抱。
- 安全型：坦诚沟通、稳定兑现
- 焦虑型：给明确偏爱、及时回应、身体安抚
- 回避型：给空间但别消失，用行动稳住
- 混乱型：先降低刺激，再给清晰边界和稳定陪伴

### 6. 回收成关系变化
一场依恋反应戏至少要留下 1 项后果：
- 更信任
- 更依赖
- 更怕失去
- 更清楚对方软肋
- 暂时更近，或暂时更疏远

## 输出格式
1. 触发事件
2. 角色依恋类型
3. 被刺中的核心恐惧
4. 第一反应
5. 防御动作
6. 真实需求
7. 安抚方式
8. 关系后果

## 快速示例
### 输入
- 事件：A 因工作出差两个月后回家
- 关系：热恋期

### 输出
- 安全型：B 会照常生活，准备礼物或学做饭；见面后先高兴，再自然分享这段时间的生活
- 焦虑型：B 先冷脸、不接礼物、故意呛 A，但 A 一抱过来就会慢慢软掉；核心需求是“确认自己没被丢下”
- 回避型：B 会说“回来就回来，和我没关系”，主动减少身体接触；但 A 真准备离开时，B 的失落会从细节里漏出来
- 混乱型：B 先扑上去确认人真的回来了，下一秒又因为委屈翻脸；需要先被接住，再慢慢解释

## 注意事项
- **禁止**把依恋类型写成标签说明书，必须落到动作、台词、节奏上
- **禁止**所有角色都用同一种吃醋、闹别扭、求和方式
- **禁止**只写表面攻击，不写真正需要
- 依恋类型不是判案结论，而是帮助反推出角色在亲密关系里的本能模式
