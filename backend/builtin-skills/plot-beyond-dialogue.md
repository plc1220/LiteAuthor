先决定这段剧情除了对白，还能靠什么推进；如果必须写成长对话，也要让对白自己带出情绪、信息和节奏。**对话不是原罪，空对白才是问题。**

## 适用情况
- 整章几乎都靠人物说话
- 一删掉「愤怒地说」「生气地说」，台词就没情绪
- 长对话写着写着发水、跑偏、没层次
- 多人场景里只有两个人轮流说话
- 对话很多，但看完不知道推进了什么

## 核心原则
1. **能从台词里听出来的情绪，不再用情绪副词重复说明。**
2. 每段对话至少承担 1 项功能：推进情节、暴露心理、交代背景、投放线索、改变关系。
3. **长对话可以存在，前提是每几轮都有新信息、新压力或新态度。**
4. 对话不一定句句配动作，但必须有换气点，不然就会发闷。
5. 对话必须服从故事大纲和人物命运，不能写成漫无边际的闲聊。

## 先删掉的低效写法
- 他愤怒地吼道
- 他生气地说
- 她伤心地回答

先检查：如果把「愤怒地」「生气地」「伤心地」删掉，台词还成立吗？
- 成立：直接删
- 不成立：说明情绪没有写进台词和语境里，要重写台词，不是补副词

## 长对话三种稳写法
### 1. 纯对白推进
适合冲突强、信息密度高、台词本身有张力的场景。
要求：
- 每轮都在改变局势、立场或关系
- 台词里带潜台词，不是把已知信息再说一遍
- 至少每 3—5 轮出现一次新的压力点

### 2. 对话 + 反思 / 回忆 / 心理
说一小段，立刻插入人物的联想、判断、记忆或自我辩解。
适合：
- 让一句话长出更深的意味
- 写嘴上说一套、心里想一套
- 在不打断场面的前提下增加深度
注意：
- **反思必须服务当下对话，不要借机开长篇论文。**
- 反思越多，趣味性风险越高；通俗文里要控制篇幅

### 3. 对话 + 景物切换
说几句，镜头切到环境、天气、灯光、路人、器物，再切回来。
作用：
- 给读者换气
- 暗示人物情绪
- 稍微放慢节奏，让对话更有余味
要求：
- 景物必须和当前气氛有关
- 切景不是为了凑字，是为了加暗示或停顿

## 常用展开通道
当对白太单薄时，从下面补 2—3 个：
1. 主角内心判断
2. 对方神态与反应
3. 路人或旁观者反应
4. 隐情或背景的短句投放
5. 环境与器物暗示
6. 主角下一步打算

## 实操步骤
### 1. 先写这段对话的任务
用一句话写清：
`这段对话写完后，什么必须发生变化？`
只从下面选 1—2 项：
- 人物关系变了
- 主角知道了新信息
- 读者看懂了人物心理
- 冲突升级了
- 下一步行动被顶出来了

### 2. 只保留核心台词
先把最关键的台词单拎出来。
再问：
- 哪句能删
- 哪句在重复
- 哪句只是在解释上一句

### 3. 选一种换气法
- 台词足够强：纯对白推进
- 需要深度：插反思/心理
- 需要停顿和暗示：切景物
- 多人场景：再加旁观者反应或局势观察

### 4. 检查每一轮有没有新东西
逐段问：
- 这段新增了什么
- 这段把谁逼到了什么位置
- 删掉它，剧情会不会照样成立

### 5. 多人场景定主焦点
- 主焦点负责主线冲突
- 次焦点负责补刀、误导、证实或制造气氛
- 其余人用概述或反应带过
**不要一群人都完整说一遍。**

### 6. 最后查跑偏
如果这段对话只是作者舍不得停、舍不得虐、舍不得让角色吃亏，就容易发水。
把作者和角色稍微拉开：
- 角色可以动情，作者要知道这段戏在干什么
- 不要因为偏爱人物，就让对话一直原地打转不进入冲突

## 快速自查
- 台词本身能不能读出情绪
- 每段对白是否有明确功能
- 长对话中间有没有换气点
- 多人场景是不是仍然只有两个人在说
- 删掉一半对白后，信息会不会更集中
- 这段是否仍在朝大纲推进

## 输出格式
1. 这段对话的任务
2. 需要保留的核心台词
3. 选用的换气法
4. 需要补的内心 / 景物 / 旁观者反应
5. 建议删除的重复台词
6. 改写后的段落结构

## 示例
### 输入
- 两个人吵架，整章全是对白
- 台词后面全是「她生气地说」

### 输出
- 任务：把关系从僵持推进到决裂
- 处理：删掉情绪副词，让台词本身带火药味
- 换气法：对白 + 景物切换
- 补法：争执到一半切一笔窗外雨声和杯沿轻响，再切回两人对话
- 检查：每轮争执都要让旧账再翻出一点，而不是反复说同一句狠话

## 注意事项
- **禁止**用情绪副词代替台词质量
- **禁止**把对话写成闲聊
- **禁止**景物切换纯为凑字
- **禁止**反思一开就停不下来
- **禁止**因为偏爱人物，让对白绕圈不落事
