把角色做成“读者隔着很多章也能一眼认出来的人”，并在中后期持续稳住味道。**人设不是资料表，也不是开头惊艳一次就算立住；真正稳的人设，要在剧情线和感情线里被反复证明。** 如果后期读者觉得“没崩，但没前面带感”，通常不是线太多，而是最近没有发生足够贴标签的情节。

## 适用情况
- 角色设定不少，但读者记不住
- 开头人设很鲜明，中后期却越来越淡
- 感情线一多，主角像只剩“谈恋爱”
- 剧情线一忙，角色反应开始谁来都行
- 只会重复“温柔地笑了笑”“冷冷地说道”这类空形容
- 写着写着手感丢了，不知道怎么把角色写回去

## 核心原则
1. **人设不是一句标签，而是一套稳定的反应方式。**
2. **剧情线和感情线都必须服务人设。** 不管写事业、恋爱、冲突，读者都该看到“这个人就是会这样处理”。
3. 重复人设，不是反复说“她很温柔”“他很高冷”，而是**换场景反复证明同一特质**。
4. 一个角色至少要有：核心标签、处理事情的方法、说话手感、在亲密关系里的例外。
5. 后期人设发飘时，先回看最初出场，重新抓感觉，再补贴标签的情节。

## 步骤
### 1. 定 1 个核心标签 + 3 个辅助标签
要求：
- 核心标签负责第一印象
- 辅助标签负责层次和反差
- 标签不要全是近义词

差：
- 高冷、冷漠、淡漠、话少

好：
- 核心标签：掌控型
- 辅助标签：克制、护短、审美挑剔

### 2. 把标签翻成可见反应
每个标签都写成 3 类东西：
1. 平时怎么说话
2. 遇事怎么处理
3. 压力来了怎么反应

例如“温柔”不要只写“温柔地笑了笑”，而要写成：
- 平时：说话留余地，不抢人难堪
- 处理事：先安抚情绪，再解决问题
- 压力下：自己难受也会先顾别人，但底线被踩会突然硬起来

### 3. 给角色准备 3—5 个可回踩抓手
从这些里选：
- 习惯
- 喜好 / 厌恶
- 小癖好
- 说话口头禅
- 固定技能
- 特定弱点
- 饰物 / 用物 / 审美

这些抓手的作用不是丰富资料卡，而是给正文提供反复落地的支点。

### 4. 安排三轮以上回踩
至少安排：
1. **第一次硬证明**：出场就让读者看见这个人设点
2. **第二次换角度证明**：隔几章换个场景再证明一次
3. **第三次在压力里证明**：让这个点参与解决问题或制造冲突
4. **关系里的例外版本**：同一个人设在亲密关系里出现偏爱、松动或反差

不是一句台词说三遍，而是同一个点换着法子出现三遍以上。

### 5. 让人设长在关键戏里
每写一场主线戏，检查四件事：
- 这场里，角色做了什么只有 TA 会做的选择
- 这场里，角色怎么处理问题，是否符合人设
- 这场里，角色的说话手感有没有留下记忆点
- 这场里，别人会因 TA 的做法产生什么反应

如果一场高潮换成任何角色都能成立，这场戏还没长在人设上。

### 6. 中后期人设发飘时，用“回看第一章找感觉”
按这个顺序补救：

1. **回看第一章或前三章**  
   找出当时读者最容易喜欢上的那口味，不要只看设定表，要看实际场景。

2. **抄出最抓人的 3 样东西**  
   例如：
   - 说话手感
   - 处理冲突的方式
   - 最鲜明的价值取向 / 边界

3. **检查最近 5—10 章**  
   问自己：
   - 最近有没有发生特别贴这个标签的情节
   - 感情线和剧情线里，TA 有没有做出“只有 TA 会做”的反应
   - 最近是不是只剩功能推进，没有人味和个性

4. **补“贴标签情节”**  
   优先补进这三类场面：
   - 主线决策
   - 冲突处理
   - 亲密关系互动

   不是硬插一场“专门立人设”的独立戏，而是让主线推进本身顺手把人设踩实。

5. **删掉空复读**  
   如果最近一直在写“温柔地笑”“冷冷地看”，却没有对应的选择、动作和后果，就删掉，改写成具体反应。

### 7. 做“可替代性测试”
问自己：
- 这场戏换一个性格完全不同的人来做，还成不成立
- 去掉这个人设点，这段剧情还会原样发生吗
- 别人为什么会因为 TA 而改变态度

如果都成立，人设还没真正参与剧情。

### 8. 做“遮名测试”
把角色名字遮掉，再看一段戏：
- 只看台词和动作，能不能认出是谁
- 把名字互换，这段戏还成不成立
- 提到某个习惯或处理方式，能不能立刻想到这个角色

不能的话，就继续补可见反应和回踩抓手。

## 输出格式
1. 核心标签 + 3 个辅助标签
2. 每个标签的可见反应
3. 3—5 个可回踩抓手
4. 三轮以上回踩节奏
5. 最近最能立住人设的 3 场戏
6. 中后期发飘诊断
7. 需要补的“贴标签情节”
8. 遮名测试结果

## 快速示例
### 输入
- 女主开头因为“嘴甜但记仇、会装乖反杀”很讨喜
- 中后期感情线一多，只剩“温柔会撒糖”，读者觉得没崩但味淡了

### 输出
- 回看第一章，提炼出：
  1. 她会用软话包刀子
  2. 吃亏后一定记账
  3. 面对强者也会装乖找机会反击
- 检查最近章节：她虽然一直在谈恋爱，但几乎没有做出这三类反应
- 补法：
  - 在感情戏里，让她不是单纯害羞，而是用装乖套话、顺手反制
  - 在剧情戏里，让她面对压力时仍保留“记仇并找机会回敬”的处理方式
  - 删掉泛泛的“温柔”描述，改成具体语言和动作
- 结果：读者会重新觉得“这还是最开始那个很有味道的女主”

## 注意事项
- **不要**做人设只做表格，不做回踩计划
- **不要**靠作者旁白反复宣布角色是什么样的人
- **不要**把感情线和剧情线写成与人设无关的功能场
- **不要**只会用副词重复同一个标签
- 真正稳的人设，是**读者换个场景、隔着很多章，仍然能一眼认出“这就是 TA”**。
