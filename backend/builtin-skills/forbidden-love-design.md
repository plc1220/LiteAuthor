围绕“不被允许的爱情”生成情节骨架，重点写清禁忌来自哪里、社会怎样施压、恋人如何应对，以及这段关系最终要付出什么代价。优先把阻力写成具体的社会现实，而不是空喊“世俗不容”。

## 适用情况
- 想写婚外情、师生恋、门第禁忌、同性受压、忘年恋、伦理禁忌等关系
- 需要把“不能爱”写成真正的外部压力和内部撕裂
- 想写秘密恋情、公开反抗、被迫分离或代价型结局
- 不想把禁忌之恋写成纯猎奇设定

## 输入
- 禁忌类型：法律 / 伦理 / 宗教 / 家族 / 婚姻 / 阶层 / 身份
- 故事时代与社会环境
- 双方原有关系与社会身份
- 这段关系最吸引彼此的地方
- 双方选择：秘密继续 / 公开反抗 / 一明一暗
- 周围人的态度：支持者、旁观者、阻止者
- 结局倾向：代价型 HE / BE / 幻灭式开放结局

## 步骤
### 1. 先写清禁忌从哪里来
明确这段关系到底违反了什么：
- 社会习俗
- 法律或制度
- 家族伦理
- 婚姻契约
- 宗教禁令
- 阶层与身份秩序

**禁忌越具体，后续压力越好写。**

### 2. 交代关系起点与社会环境
在开端先让读者明白：
- 两人是什么关系
- 周围环境如何看待这种关系
- 一旦暴露，最现实的风险是什么

不要只顾恋爱本身，必须先交代“他们所处的世界”。

### 3. 决定恋人的应对策略
分别写清两人对禁忌关系的态度：
- 谁更愿意隐瞒
- 谁更想公开反抗
- 谁更怕失去体面、家庭、地位或安全

这一步决定后续冲突是“对外抗争”还是“对内撕裂”。

### 4. 设计世俗回应
让周围人的反应具体落地，可从中选 3—5 种：
- 流言
- 家庭施压
- 制度处罚
- 监视与隔离
- 道德羞辱
- 现实报复
- 经济或名誉损失

**不要只写一句“所有人都反对”。要写他们怎么反对。**

### 5. 先给短暂的平和期
在压力真正压下来前，先给一段平和或欢畅：
- 秘密幽会
- 短暂同盟
- 彼此确认关系
- 对未来抱有幻想

这样后面的破裂才有代价感。

### 6. 让压力同时打在外部和内部
禁忌之恋的高压，不只来自旁人，也来自恋人自己：
- 恐惧
- 羞耻
- 自我怀疑
- 幻想破灭
- 对未来失去信心

可以决定这段感情是“越压越烈”，还是“在现实与心理重压下逐渐消退”。

### 7. 设计关系破裂或硬抗升级
中段至少安排一个转折：
- 关系在压力下更深，决定公然对抗
- 关系在现实面前开始松动
- 一方撑不住，另一方还在坚持
- 表面继续，内里已经裂开

这样结局才不是突然到来。

### 8. 用代价型高潮收束
高潮优先考虑：
- 死亡
- 永久分离
- 社会性死亡
- 家庭与身份彻底破裂
- 幻想破灭后只剩幸存者余痛
- 即便在一起，也要失去重要代价

**禁忌之恋通常不能无伤通关。即使给 HE，也必须付出足够重的代价。**

### 9. 检查是否写成猎奇消费
逐条检查：
- 关系是否真的成立，而不只是设定刺激
- 世俗压力是否具体可见
- 代价是否落在人物生命、名誉、未来或精神上
- 读者是否能同时感到爱与毁灭风险

## 输出格式
1. 禁忌来源
2. 社会环境与暴露风险
3. 双方应对策略
4. 周围人反应清单
5. 平和期设计
6. 外部压力与内部撕裂
7. 关系转折点
8. 高潮代价
9. 结局余波
10. 翻车风险

## 快速示例
### 输入
- 已婚贵族夫人 × 平民画师
- 社会环境：严苛家族名誉体系

### 输出
1. 禁忌来源：婚姻契约 + 阶层秩序
2. 暴露风险：夫人会失去名誉与继承权，画师会被流放
3. 应对策略：夫人想隐瞒，画师更想公开抗争
4. 世俗回应：家族监视、社交圈羞辱、经济封锁
5. 平和期：两人在画室里拥有短暂乌托邦
6. 转折：家族察觉后，夫人开始动摇，画师拒绝退让
7. 高潮：画师被逼离城，夫人保住身份却彻底失去爱情
8. 余波：幸存者长期活在幻灭中

## 注意事项
- **禁止**把禁忌之恋写成纯噱头
- **禁止**只有旁人反对，没有恋人自己的心理代价
- **禁止**中段一直甜，结尾突然强行悲剧
- **禁止**让社会压力只停留在口头评价，不落到现实后果
- 好看的禁忌之恋，不只是“他们爱得危险”，而是**他们每靠近一步，都更清楚自己在拿什么下注**。
