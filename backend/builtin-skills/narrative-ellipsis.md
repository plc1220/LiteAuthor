别把小说写成说明书。**先写清必须让读者知道的东西，再故意留出一截让读者自己补完。留白不是少写，而是不把能被读者自然推出来的部分全说尽。**

## 适用情况
- 总怕读者看不懂，于是把前因后果和情绪全解释一遍
- 场景之间非要加很多连接句，读起来发黏
- 感情戏、暧昧戏写得太实，没有回味
- 对话、动作都写了，还要再补一段总结
- 正文像说明书，读者没有脑补空间

## 输入
- 当前场景或段落
- 这段必须让读者明白的信息
- 当前写得过满的版本
- 想留下的情绪：暧昧 / 心酸 / 爽 / 悬念 / 余味
- 读者最该自己补完的那一层

## 核心原则
1. **核心信息必须清楚，边缘信息可以不说尽。**
2. **留白要建立在“读者能推出来”之上，不建立在“作者自己没写清”之上。**
3. 读者最容易脑补的，通常是关系、情绪余波、动作背后的意思。
4. 同一句话能靠上下文看懂，就不要再解释第二遍。
5. 留白最值钱的地方，不在省字，而在回味。

## 步骤
### 1. 先列“必须写明”的 3 件事
通常只保留：
- 谁在场、谁在说 / 做
- 这一刻真正发生了什么变化
- 这一段结束后，关系 / 局势变成什么样

写不进这三件事里的信息，先默认可删或后移。

### 2. 把“说明书味”最重的句子标出来
高危句通常是：
- 重复解释前一句已经能看懂的情绪
- 用大段心理总结动作的意思
- 把场景之间每一道连接都交代得过满
- 生怕读者不懂，于是把潜台词直接翻译出来
- 把余波、后劲、尴尬全部亲口说破

### 3. 决定这段留白留在哪一层
常用 5 种：
1. **情绪留白**：不总结“他很难过 / 很心动”，让动作和反应代替
2. **关系留白**：不明说“他们更近了”，停在一个只对彼此成立的动作 / 称呼
3. **动机留白**：先给行为，原因后置或只给半截
4. **转场留白**：省掉能被上下文自然补完的连接句
5. **结果留白**：不把一场戏的余味解释完，让读者自己接后劲

### 4. 用“删一句还懂吗”测试
逐句问：
- 删掉这句，读者还能不能知道谁在说话、谁在做什么？
- 删掉这句，关系和局势会不会还是成立？
- 这句是不是只把读者已经能推出来的东西再说一遍？

如果连续两问都能过，就删或压缩。

### 5. 让留白落在可见物上
最稳的收法不是抽象总结，而是落在：
- 一个动作
- 一个停顿
- 一个物件
- 一个没说完的句子
- 一个只在此刻有意思的称呼
- 一个别人看不出、读者看得出的差别待遇

### 6. 感情戏优先留“怎么爱”，少讲“多爱”
不要反复解释“他很在意她”。优先写：
- 记住对方忌口
- 对别人不会做、对他才会做的照顾
- 嘴上敷衍，动作先一步
- 话没说满，但态度已经偏过去

### 7. 留白之后，再补一次“清楚度”
重点检查：
- 读者能不能分清谁在说话
- 关键因果有没有断
- 场景是不是从含蓄写成了含糊
- 该明说的设定 / 规则 / 关键信息有没有被错删

**留白的底线是：可以让读者自己补完，不能逼读者猜作者想写什么。**

## 输出格式
1. 这段必须写明的信息
2. 当前说明书味最重的句子
3. 适合留白的层：情绪 / 关系 / 动机 / 转场 / 结果
4. 建议删除或压缩的句子
5. 留白后的改写版本
6. 仍需补清楚的点
7. 余味最强的收尾点

## 快速示例
### 原写
他其实很在意她，只是不知道怎么表达。看到她发烧，他心里特别懊恼，也很心疼，所以虽然嘴上不说，还是把药买回来了，想让她明白自己对她是不同的。

### 改写
药放桌上。  
“别传染给我。”  
他说完就走，门却没关严。

### 原写
她说完那句话后有些后悔，担心自己是不是把气氛弄得太僵，所以想找补几句，但是又不知道说什么，只能低着头不说话。

### 改写
话音落下，她先低头去拧瓶盖。  
半天没拧开。

### 原写
两人聊完以后，他们之间的关系其实已经变得更暧昧了一些，只是彼此都没有挑明。

### 改写
他把最后一个包子里的馅挑出去，放进她碗里。  
这次她没说谢谢。

## 注意事项
- **不要**把留白写成信息缺失
- **不要**怕读者脑补，就把每层意思都翻译出来
- **不要**场景已经很清楚了，还再补一段作者总结
- **不要**为了显高级故意写得云里雾里
- 真正有用的留白，是**读者看得懂，但还会自己往下想一截**。
