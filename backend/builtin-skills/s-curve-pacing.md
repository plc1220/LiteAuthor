先把全文压成一句话卖点，再用“分阶段推进 + 大小高潮 + 裁剪支线 + 张弛交替”把剧情压成 S 型曲线。**想让剧情紧凑，先学会舍得砍。所有不服务主主题的情节、配角和视角，都是候选删减项。**

## 适用情况
- 剧情散、拖、注水，高潮推不起来
- 同时想写很多主题，结果主线不聚焦
- 爽点、萌点很多，但串不成持续推进的剧情
- 配角和支线越写越多，盖过主线
- 多视角一切就乱，章节写完没推进

## 输入
- 一句话故事卖点
- 预计字数或分卷数
- 主角 / CP / 核心冲突
- 已有的高光桥段或必写爽点
- 当前最拖的部分（可选）

## 步骤

### 1. 先用一句话概括全文核心情节
格式：
`这篇文主要在写____。`
要求：
- 必须能概括整篇文最值钱的那条线
- 如果有两个主题，先分主次，只保留一个主主题
- 不能一句话说清，说明故事还没想明白

### 2. 根据这句话圈出“必写重点”
问自己：
- 这句话里最值钱的动作、结果或情绪是什么
- 为了把这个点写爽，前面必须铺哪些情节
- 哪些设定只是调味，不能抢篇幅

### 3. 用“性格 + 背景”把主线扩成 3—5 个阶段
每个阶段都写：
- 阶段目标
- 主要阻力
- 一个小高潮
- 阶段结果

主线只是方向，不够撑篇幅。要靠主角性格、处境和背景把每阶段长出来。

### 4. 每阶段先定高潮，再反推铺垫
先写：
`这一阶段最炸的点是什么？`
再反推：
- 为了到这个点，前面必须先发生什么
- 哪些暧昧、冲突、误会、对抗要提前埋
- 哪些信息要故意留到高潮再揭

**不要先平铺一堆过程，再临时想高潮。**

### 5. 视角优先从简
- 能单主视角，就别乱跳
- 同一场景尽量跟着一个主角跑
- 不要为了图方便解释误会而切视角
- 如果已经写成多线多视角，就把 2—3 条线逐步引向同一场景，让它们“多线合一”后再分开

视角越乱，越容易把人物写崩、把节奏写散。

### 6. 配角和支线必须有功能
给每条支线 / 每个重要配角补一句：
`它存在，是为了____。`
常见有效功能：
- 推主线
- 衬主角
- 制造主线阻力
- 放大关系冲突
- 提供阶段性回收

答不上来，就删、并线或后置。**除非准备换主角，否则不要让配角长期抢戏。**

### 7. 在大纲里直接标出“提气点”和“放气点”
提气点可用：
- 欲扬先抑
- 正衬 / 反衬
- 气氛烘托
- 前后呼应
- 角色高光 / 萌点 / 爽点
- 章尾钩子

放气点可用：
- 余波
- 关系回响
- 角色喘息
- 信息回收后的短暂平缓

**S 型曲线不是一直往上冲，而是提一口气、松一口气、再往上提。**

### 8. 分清小高潮和大高潮
- **小高潮**：冲突更简单、人物更少、篇幅更短，只解决一部分问题，并留下后续空间
- **大高潮**：集中回收一个阶段最核心的矛盾、关系或目标，情绪和局势都要明显抬高

不要把所有力气都用在小高潮上，不然后面会后继乏力。

### 9. 正文阶段做字数和章节控制
每卷 / 每阶段先定：
- 这一段大约多少字
- 大高潮放在哪
- 中间安排几个小高潮
- 每章推进哪一步

每写完一章都自检：
- 这一章有没有推进主线或主要关系
- 是不是铺垫过多
- 有没有为了凑字数而注水
- 如果超出字数，怎样尽快拽回主线

### 10. 剧情太紧时，不是加水，而是补“有功能的缓冲”
如果读者觉得“太赶”“喘不过气”，优先补：
- 人物塑造
- 情绪余波
- 关系变化
- 下个高潮的反向铺垫

但缓冲章也必须有功能，**不是停下来写无关日常。**

## 输出格式
1. 一句话主主题
2. 主次说明
3. 3—5 个阶段表
4. 每阶段的高潮与铺垫
5. 视角方案
6. 配角 / 支线保留与删减表
7. 提气点 / 放气点标注
8. 大小高潮规划
9. 字数与章节推进计划
10. 容易注水或跑偏的风险点

## 快速示例
### 输入
- 主题：女主甩了渣男

### 输出
- 一句话主主题：这篇文主要在写女主认清渣男并彻底甩掉他
- 必写重点：渣男如何渣、女主为何忍、什么刺激她醒悟、甩掉后的回收
- 小高潮：第一次看清渣男真面目
- 大高潮：当众甩掉渣男并完成局势翻盘

## 注意事项
- **禁止**一边嫌剧情拖，一边什么都舍不得砍
- **禁止**把两个平级主题平均用力，导致主线失焦
- **禁止**让配角、攻二、路人支线盖过主角主线
- **禁止**每章都只在解释设定，不推进事情
- 有新灵感可以加，但**不能改掉当前阶段必须回收的方向**
