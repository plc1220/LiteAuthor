先决定这一章或这一段，到底要让读者憋什么、等什么、最后宣泄什么。**高潮不是“主角赢了”，而是矛盾被推到最尖后，情绪终于炸开。**“压”负责攒共情和期待，“扬”负责宣泄和兑现。

## 适用情况
- 单章或连续几章需要明显的先抑后扬
- 明明安排了打脸、反转、和好、翻盘，读者却觉得不够解气
- 高潮只有事件结果，没有情感释放
- 想把事业线、感情线、打脸线揉成同一波高潮
- 感情戏想写出甜—险—甜—险的追读感
- 想让高潮后顺手落主题和立场

## 核心原则
1. **“扬”首先是情感宣泄，其次才是事件结果。**
2. **“压”不是单纯虐主，而是给后面的宣泄攒共情筹码。**
3. **反派不一定是具体的人，也可以是环境、规则、时代、舆论，甚至主角自己的缺点。**
4. **越现实、越具体、越全面的问题，越容易激发共情。**
5. **高潮结束后，最好让人物立场或作者价值取向顺手落地。**

## 步骤

### 1. 先写一句：这次到底在宣泄什么
从下面选 1—2 项：
- 利益受损
- 尊严被踩
- 感情破裂
- 理想受挫
- 亲友受伤
- 长期压迫
- 自我怀疑
- 大环境的无力感

句式：
`主角前面一直在忍____，这一段要让TA终于____。`

### 2. 先定冲突对象
可选：
- 具体反派
- 利益共同体
- 舆论 / 行业规则
- 家庭 / 阶层 / 时代环境
- 主角自己的执念、缺点、心魔

**别把“反派”理解得太死。只要能把矛盾推到临界点，它就能成为高潮的对撞面。**

### 3. 给“压”准备共情筹码
按需选 3—6 项：
- 主角与目标之间的巨大差距
- 主角曾受过的白眼、轻视、羞辱
- 发奋的契机
- 长期努力的过程
- 多次反复的波折和磨难
- 眼看要成时又被卡一道
- 主角为了别人或原则不肯退的代价

**没有这些铺垫，后面再大的赢也容易空。**

### 4. 先选站队视角和信息差
- **主角 / 受害者视角**：适合拉心疼、拉共情
- **反派视角**：适合提前拉仇恨、埋阴招
- **围观 / 全知视角**：适合让读者比场内角色多知道半步，转出期待感

判断句：
- 想让读者担心主角，就让读者看到危险已经来了
- 想让读者等反杀，就让读者知道主角有底牌，但反派不知道

### 5. 给每一下“压”标功能
常见功能：
- 让读者心疼主角
- 让读者恨反派
- 拉高悬念
- 提前埋期待
- 爆感情矛盾
- 暗示更大危机

写时别只问“主角又惨了没”，要问：
`这一压，给后面的扬多攒了哪一口气？`

### 6. 把“扬”写成三层回收
一波像样的“扬”，至少回收 2 层：
- **情绪层**：委屈、愤怒、压抑被宣泄
- **事件层**：困局被解决，反派被反噬，误会被掀开
- **关系层**：改站队、改认知、改感情、改权力关系

### 7. 分清“悬念”和“期待”
- **悬念**：读者只知道危险，不知道怎么解
- **期待**：读者知道危险，也知道主角有方案、底牌或援手，只等它什么时候爆出来

悬念让读者绷紧，期待让读者想看主角怎么收拾场面。两者交替用，波形会更顺。

### 8. 常用波形
- 短章：`压 → 扬`
- 中段高潮：`压 → 扬 → 再压 → 再扬`
- 大高潮：`多次小压 → 一次大扬 → 余波再回收`
- **感情章**：`甜/舒适 → 小翻车 → 再靠近 → 新麻烦/新误会`
  - 先给一口能吃到的甜或短暂顺遂，证明这段关系/目标值得追
  - **处境在落时，关系最好给一点起；关系在退时，外部局势最好给一点进**
  - 章尾优先落在“还得继续见面、继续解决”的问题上

每次再压，都要多一个新变量：
- 更高层的阻碍
- 更重的代价
- 更公开的场合
- 更难退让的情感选择

### 9. 把“扬”落到场面，不要只用旁白总结
优先写：
- 围观者的反应
- 反派的失态
- 主角终于做出的选择
- 重要关系的变化
- 实际利益回到主角手里

少写：
- “她终于出了一口气”
- “所有人都被震惊了”
- “他赢了”

### 10. 高潮后补一句“这件事说明了什么”
可落到：
- 主角终于守住了什么
- 主角看清了什么
- 这个世界 / 环境的问题在哪里
- 作者想让读者认同什么

这一步很适合接卒章显志、人物立场、下一阶段主题。

## 输出格式
1. 这次高潮要宣泄什么
2. 冲突对象
3. 共情筹码
4. 站队视角与信息差
5. 每一下“压”的功能
6. 每一下“扬”的回收
7. 推荐波形
8. 高潮后的价值落点
9. 风险点

## 快速示例
### 输入
- 种田文：主角想买下自己一直工作的酒楼

### 输出
1. 宣泄：从长期贫穷、被轻视，到终于翻身
2. 冲突对象：同行打压、贫穷处境、旧身份束缚
3. 共情筹码：
   - 店小二和掌柜收入的巨大差距
   - 做店小二时受过的白眼
   - 发奋契机：被客人踩住手背
   - 多年攒钱和反复波折
4. 扬的场面：
   - 最终成交
   - 曾经看不起他的人改口
   - 主角第一次以掌柜身份站进柜台里
5. 价值落点：努力未必改天换地，但至少能改写自己这一生

## 注意事项
- **禁止**只压不扬，或压十章扬半章
- **禁止**扬完只有嘴上震惊，没有实际回收
- **禁止**反派翻车全靠天降意外，不靠主角行动
- **禁止**把高潮写成单纯热闹，不写主角真正憋了什么
- 爽感最足的时候，不是主角赢了，而是**读者和主角一起把那口气吐出来了**
