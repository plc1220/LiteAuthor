先确定这段最想让读者看到什么，再只写服务这个中心的细节。**描写不是把脑海里的所有东西都端上来，而是把最该被看到的那一笔钉住。**

## 适用情况
- 写景一写就几百字，画面很满但不抓人
- 不知道哪些细节该写，哪些该删
- 人物外貌总写成五官报菜名
- 明明想烘托气氛，却被杂物、路人、摆设分走注意力
- 想加快文字节奏，让描写更快进主题

## 输入
- 描写对象：人物 / 景物 / 事件现场
- 这段最想传递的东西：情绪、气氛、人物印象或情节信息
- 脑海中的元素清单
- 这段后面要接什么剧情

## 核心原则
1. **先问“要表达什么”，再决定写什么。**
2. **只留第一眼最抓人的 1—3 个元素。**
3. 次要细节要么一句带过，要么直接删。
4. **给读者留想象空间**，不要把每一根头发都写死。
5. 写人时优先用“整体印象 + 一处特点”，不要五官报菜名。

## 步骤
### 1. 用一句话写中心
写成：
`这段存在，是为了让读者第一眼感到/看到____。`

例：
- 压抑感已经到位，接下来要让读者看到“坏事要发生”
- 不是为了写完整湖景，而是为了写“云帆破开云层的壮阔”
- 不是为了说明她五官多标准，而是为了立住“清冷里带一点锋利”

### 2. 列出原始画面元素
把脑海里出现的景、人、物、动作全部列出来，不急着写正文。

### 3. 给每个元素打标签
只用四类：
- **核心镜头**：不写就丢中心
- **辅助烘托**：可写 1 笔，但不能抢镜
- **一句带过**：知道有就够了
- **删除**：与当前中心无关

判断标准：
- 这是不是读者**第一眼**该看到的东西？
- 它是否直接服务当前情绪、人物印象或情节？
- 它会不会抢走真正的重点？
- 它后面会不会被剧情回收？

### 4. 只保留 1—3 个核心元素
推荐组合：
- 远景定场 1 个
- 主体镜头 1 个
- 强化细节 1 个

写景示例：
- 原始元素：帆船、晚霞、倒影、浮桥、小孩、猫
- 目标：写恢弘壮观
- 保留：晚霞的天空、破开云层的云帆
- 删除/弱化：倒影、浮桥、小孩、猫

### 5. 按“定场 → 主体 → 进主题”落笔
常用顺序：
1. 用 1 句定场
2. 用 1—2 句打主镜头
3. 立刻接人物反应、事件推进或情绪落点

**不要**把镜头里所有东西按从上到下、从左到右报一遍。

### 6. 写人物外貌时只深化一处
先给整体印象，再挑一处最能记住的点。
- 大印象：明艳 / 清冷 / 温软 / 狼狈 / 凌厉
- 细节点：眼神、笑意、伤疤、酒窝、唇色、步态、手上的痣等

示例：
- 差：头发、眼睛、鼻子、嘴、脸型、长腿全写一遍
- 好：先给“她一进门，整间屋子都亮了一寸”的整体印象，再抓“眼尾那颗痣”或“笑时压不住的虎牙”

### 7. 留白检查
复查时删这三类：
- 只是说明“现实里确实存在”的背景物
- 与当前中心无关的装饰细节
- 会拖慢进主题速度的重复修饰

问自己：
- 删掉这句，中心会不会更清楚？
- 读者是不是已经能脑补，没必要替他全部补完？
- 这段能不能更快进入人物或事件？

## 输出格式
1. 这段中心
2. 原始元素清单
3. 保留 / 一句带过 / 删除清单
4. 核心镜头组合
5. 改写版
6. 为什么这样取舍

## 快速示例
### 输入
- 目标：写一艘云帆冲破晚霞云层的壮阔
- 元素：帆船、晚霞、湖面倒影、浮桥、小孩、猫

### 输出
- 中心：让读者第一眼看到“云帆破云”的恢弘
- 保留：晚霞、云帆
- 删除：倒影、浮桥、小孩、猫
- 改写：晚霞压满天幕，一艘云帆自云层里破出，像把整片天空硬生生劈开。

## 注意事项
- **禁止**为了“画面完整”把镜头里每样东西都写一遍
- **禁止**把无关细节当成文笔好
- **禁止**人物外貌写成五官报菜名
- **禁止**只顾华丽，不顾这段到底要推进什么
- 好描写不是“写得多”，而是**读者一下就看到重点**。
