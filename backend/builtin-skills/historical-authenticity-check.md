先确定这到底是“正史背景严肃写法”，还是“借史壳的言情写法”。**正史背景最怕的不是少写，而是把不懂的东西写死，还写错时代气质。**

## 适用情况
- 写真实朝代、真实历史人物、真实制度背景
- 想要历史感，但担心被挑出硬伤
- 不知道哪些地方必须查，哪些地方可以不写
- 容易把现代观念、现代用语塞进古代背景

## 输入
- 朝代与具体时间点
- 主要人物身份
- 关键场景（朝堂/军旅/婚嫁/日常/宫廷/市井等）
- 剧情里必须出现的制度、器物、称谓
- 作品的严肃程度

## 核心原则
1. **正史背景先求不犯大错，再求细节漂亮。**
2. 最重要的是**时代环境、制度边界、人物精神风貌**。
3. **古代人要有古代人的样子**，别让现代语感和现代价值判断乱入。
4. 不熟的细节，宁可少写、泛写，也不要编硬知识。
5. 器物服饰不是重点时，可以不细写；写了就尽量别错。

## 步骤
### 1. 锁定时间断面
不要只写“唐朝”“汉朝”“明朝”。
必须补清：
- 具体皇帝/前后期
- 当时政治气氛
- 社会风气
- 制度状态是否处于变化中

**同一朝代前后差异可能很大，不能用一个模糊印象覆盖。**

### 2. 先查四层“硬骨架”
优先核查：
1. 官制、兵制、法律、田制等与剧情强相关的制度
2. 礼法、身份边界、男女社交边界
3. 语言称谓、官职叫法、常见禁忌
4. 关键历史事件与真实时间线

这四层比首饰纹样、帘帐花色更重要。

### 3. 再查“时代气质”
写每个主要角色前，先回答：
- 这个时代的人最自然的荣辱观是什么
- 面对婚姻、忠孝、君臣、阶级时，默认反应是什么
- 什么话在当时说得出口，什么话说出来就很违和

**历史感不只在名词，更在思路和反应。**

### 4. 把细节分成三类
#### 必查
- 会直接影响剧情结果的制度、时间、身份、称谓
- 关键器物、礼制、军政流程
- 一旦写错就会出戏的常识性历史点

#### 可泛写
- 不影响剧情推进的一般衣食住行
- 普通摆设、非关键器物
- 只负责陪衬气氛的细枝末节

#### 干脆不写
- 自己完全不熟，还会增加错误率的知识型细节
- 为了显得考据而硬塞的说明

### 5. 处理语言和对话
逐句检查：
- 有没有明显现代口语、网络词、现代思维套话
- 有没有后世才出现的制度叫法或词义
- 有没有把现代恋爱逻辑、生存逻辑硬套给古人

不要求文言，但要避免**现代人穿戏服说现代话**。

### 6. 只让有意义的器物入镜
如果某件器物对剧情没有作用，只需一笔带过或不写。
如果器物是关键线索、信物、身份象征，再单独查它：
- 当时有没有
- 适不适合这个身份使用
- 名称是否准确

### 7. 不熟就换安全写法
当资料不够确定时，优先：
- 改写成不易出错的泛称
- 把镜头落回人物行为和关系
- 避免强行展示专业细节

**历史小说不是器物说明书。**

### 8. 最后做“硬伤扫描”
逐项检查：
- 时间线对不对
- 身份边界对不对
- 制度和官职对不对
- 语言和时代精神对不对
- 关键器物有没有时代错位
- 重大历史事件有没有张冠李戴

## 输出格式
1. 朝代与时间断面
2. 必查制度清单
3. 时代气质与人物精神风貌
4. 必查 / 可泛写 / 不写清单
5. 高风险语言与称谓
6. 关键器物核查表
7. 当前文稿的硬伤风险

## 快速示例
### 输入
- 正史背景言情，发生在某朝中后期

### 输出
- 先锁具体时间点，而不是笼统写“某朝”
- 必查：官制、婚嫁礼法、交通与出行边界
- 可泛写：普通室内摆设、日常衣料纹样
- 高风险：现代语气、过于现代的男女相处方式、后世制度叫法提前出现
- 安全策略：不熟的器物不硬写，把重点放回人物处境与制度压力

## 注意事项
- **禁止**一边写严肃历史，一边让人物满口现代腔
- **禁止**为了显得考据，硬塞一堆与剧情无关的名词
- **禁止**没查清就写死关键制度和器物
- **禁止**把同一朝代的不同时期混成一个模糊背景
- 正史背景写得稳，靠的不是炫资料，而是**环境真、制度稳、人物像那个时代的人**。
