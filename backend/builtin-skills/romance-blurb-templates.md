按 CP、人设、大纲为感情线文案选模板，再补上回应弧线和结尾钩子。**感情线文案不是把恋爱过程按时间顺序讲完，而是让读者先嗑到关系、矛盾和最想看的那一步。**

## 适用情况
- 感情文文案写成剧情摘要
- 看不出 CP 感，不知道这对为什么好嗑
- 只有单向表白，没有回应
- 无法在文案里同时交代人设、CP 和主线
- 剧情主感情辅时，CP 突然出现在后半截

## 输入
- 题材与频道
- CP 组合 / 是否无 CP
- 双方人设
- 当前关系阶段
- 主要感情矛盾
- 是否感情主 / 剧情主
- 正文风格
- 结尾想留的钩子

## 步骤
### 1. 先定重心
只在下面二选一：
- 感情主，剧情辅
- 剧情主，感情辅

规则：
- 感情主：文案主段落写关系、拉扯、矛盾，剧情只用来托举关系
- 剧情主：文案主段落写主线和爽点，但**前面就要埋一笔 CP**，不能到结尾才突然提感情
- **禁止**把“剧情版”和“感情版”硬分成两段

### 2. 先列三样材料：CP、人设、大纲
不管选哪种模板，先各写一句：
- CP 看点：这对为什么好嗑
- 人设看点：双方最抓人的标签或反差
- 大纲看点：当前最大的矛盾或变化

如果这三句里有一句很弱，文案就会空。

### 3. 从四种模板里选一种
#### 模板 1：CP 眼中的你
适用：
- 想突出一方怎么看另一方
- 想快速立出 CP 火花
- 想用“误判 → 改观 → 关系变化”制造钩子

结构：
1. 一方先给另一方下判断  
2. 用 2—3 个具体细节证明这种判断  
3. 让判断失效：告白、反救、掉马、重逢、身份暴露、关系倒转  
4. 用一句新问题或新关系收尾  

骨架：
`A 一直觉得 B ____。`
`因为 B 总是 ____ / ____ / ____。`
`直到 ____。`
`A 才发现，原来 ____。`
`可这时，____。`

#### 模板 2：花式表白
适用：
- 需要直接卖 CP
- 文案里已经有表白、追求、告白场面
- 想写“死缠烂打 / 反复靠近 / 生死关头表态”这类情绪点

核心规则：
- **必须有回应**
- 单方面表白不够，读者要看见另一方的感情开始变化
- 回应弧线尽量按：`拒绝 → 动摇 → 动心 → 更明确的选择`
- **禁止**从“拒绝”直接跳到“生死不渝”，中间没有过渡

结构：
1. 一方主动靠近 / 表白 / 撩拨  
2. 另一方拒绝、嫌弃、嘴硬  
3. 用一场有分量的事件改写关系  
4. 补一笔回应或松口  
5. 用更大的阻碍或更重的选择收尾  

#### 模板 3：我想对你说的话
适用：
- 第一人称文案
- 想强打代入感
- 想把人物心理直接摆出来

核心任务：
- 同时交代人设、CP、大纲、心理
- 不只写“我做了什么”，还要写“我为什么在乎、我怕什么、我想得到什么”

结构：
1. 对“你”说一句最核心的情感句  
2. 写为了这段关系付出了什么 / 失去了什么  
3. 写当前最大的痛、误解或无望  
4. 用一句转折或执念收尾  

骨架：
`____，你一定不知道，我为了你 ____。`
`我以为 ____。`
`可现在，____。`
`如果还能 ____，我 ____。`

#### 模板 4：对感情的总结
适用：
- 感情线已经有明确矛盾
- 想写克制、概括型、偏正剧或偏文艺文案
- 想用“关系判断 + 矛盾 + 心动”快速成文

结构：
1. 先写这段关系本来不可能 / 不该开始  
2. 写把两人牵到一起的事件  
3. 写为什么不能接受 / 不敢靠近 / 必须拒绝  
4. 写内心已经被撬动  
5. 用尚未解决的阻碍收尾  

### 4. 套常用句数模板
#### A. 七句模板：最稳的言情 / 耽美文案骨架
1. A 对 B 产生兴趣 / 在意  
2. B 的人设亮点  
3. B 对 A 的态度  
4. 感情上的矛盾  
5. A 为解除矛盾做了什么  
6. 做完之后产生了什么新变化 / 新问题  
7. A 或整段关系的总结 + 钩子  

#### B. 六句模板：双人设 + 双总结
1. 主角 A 人设  
2. 主角 B 人设  
3. CP 关系  
4. 感情矛盾  
5. A 对这段感情的总结  
6. B 对这段感情的总结 / 反转句  

#### C. 四句模板：剧情主、感情辅
1. 主角人设  
2. 主角行动目标的前因后果  
3. CP 一句  
4. 事业线 / 主线结果一句  

适用提醒：
- 女频剧情文就算剧情主，通常也离不开 `CP + 人设 + 大纲`，只是比重不同
- 如果是无 CP 文，把第 3 句改成“主角和核心对手 / 阵营 / 同伴的关系火花”

### 5. 结尾钩子：要“好奇”，不要只留问号
常用钩子：
- 原本想接受，结果对方消失了
- 本以为就此别过，多年后却重逢
- 刚松口，新的立场矛盾出现
- 明明该结束，却发现最关键的真相还没揭开

判断标准：
- 看完后读者想的是“我想知道后面怎么变”
- 不是“这不就是普通疑问句吗”

### 6. 让心理可见，别只剩事件
高频问题是：文案只写人物做了什么，不写人物怎么想。  
改法：
- 每版文案至少有一句明确情绪：怕、恨、动心、执念、迟疑、后悔、舍不得
- 至少有一句能看出“这段关系对主角意味着什么”
- 偏第一人称时尤其要把心理落地，不然只像喊口号

### 7. 避免割裂和流水账
- 不按时间顺序把恋爱全过程讲完
- 不把剧情和感情拆成两篇文
- 不只写告白，不写回应
- 不让感情跨度一下跳太大
- 不用太多背景说明抢掉 CP 感

### 8. 最后检查
逐条确认：
- 看完能不能立刻知道 CP 怎么搭
- 人设有没有落到具体关系里
- 感情矛盾是否明确
- 回应弧线是否自然
- 结尾有没有钩子
- 如果是剧情主，CP 是否提前埋过
- 如果是第一人称，心理是否真的可代入

## 输出格式
1. 文案重心：感情主 / 剧情主
2. 选用模板
3. CP / 人设 / 大纲三句材料
4. 句子骨架
5. 文案 2 版
   - 标准版
   - 更有钩子版
6. 可删内容
7. 最容易翻车的点

## 快速示例
### 输入
- 设定：清冷师姐 × 话痨师弟
- 需求：剧情主，感情辅

### 输出
- 重心：剧情主，感情辅
- 模板：CP 眼中的你
- CP 看点：一个冷、一个吵，却总在最危险的时候站到一起
- 骨架：先写师弟眼里师姐难接近，再写并肩破局里的默契和特殊照顾，随后点出身份 / 立场矛盾，最后用一次未说完的回应收尾

## 注意事项
- **禁止**只写单向表白
- **禁止**从拒绝直接跳到生死不渝
- **禁止**把心理全省掉，只剩事件清单
- **禁止**把剧情版和感情版分成两篇
- **必须**让读者看完知道：这对 CP 怎么搭、卡在哪、下一步最想看什么
