把景物写进人物视角和当前动作里，用镜头顺序完成“景—人—情绪”的切换。**景物最好不是作者自己在看，而是角色在看；如果不是角色在看，也要有清晰的镜外观察位置。**

## 适用情况
- 大段写景容易脱离人物，像风景作文
- 明明想烘托气氛，写出来却很做作
- 想把人物和景物自然连在一起
- 情绪描写反复、直白，想改成借景表达
- 场景有环境，但读者记不住画面

## 输入
- 当前场景正在发生什么
- 当前动作或冲突点
- 视角人物 / 是否使用抽离镜头
- 当前情绪与基调
- 可用的环境元素
- 这段景物想完成什么：定场 / 衬情绪 / 加压 / 托高潮 / 留记忆点

## 核心原则
1. **先定谁在看，再决定怎么写景。**
2. **同一处景，在不同情绪下必须不一样。**
3. 景物至少完成一项功能：定场、衬情绪、制造阻碍、放大气氛、留记忆点。
4. 镜头移动要有次序，别东一下西一下。
5. 情绪写到发腻时，可以**移情于景**，但仍要贴当前场面。

## 步骤
### 1. 先选镜头位置
二选一：
- **主观镜头**：透过人物眼睛看景。适合情绪强、代入感强的场面。
- **抽离镜头**：像镜外人在看。适合开场定调、拉开关系、制造冷感或宿命感。

先确定机位，再写景。

### 2. 先写“这段景物要帮什么忙”
填空：
`这段景物存在，是为了让读者感到 / 看见 ____。`

例如：
- 平静里藏伤心
- 热闹里藏疏离
- 夜色很美，但人物心情发冷
- 高潮将至，气氛渐渐绷紧

### 3. 选会被情绪染色的 1—3 个景物
优先选这些：
- 光线
- 天气
- 声音
- 温度
- 水、风、月、花、灯、门窗、廊檐等

**不要把环境清单全端上来。** 只选最能被当前情绪“看出不同”的几笔。

### 4. 用镜头路径写，不要乱切
常用路径：
- **远到近**：先定场，再拉到人物
- **近到远**：先抓人物感知，再带出环境
- **人到景**：先写人物动作，再顺着视线落到景物
- **景到人**：先写环境压过来，再落到人物反应

一段里尽量只用一条主路径。

### 5. 把景物绑到动作和感知上
稳妥写法：
- 人物动作 + 景物接触
- 人物视线 + 景物变化
- 人物情绪 + 景物印象
- 景物异常 + 人物反应

例如：
- 她站在廊下没动，雨丝一阵阵斜进来，打湿了半边衣袖。
- 他抬头看天，同样一片晴空，此刻只觉得亮得刺眼。

### 6. 直接写情太满时，改成借景侧写
当人物情绪已经很明显，再继续直写容易重复。可改为：
- 把心情折到天气、光影、远近层次上
- 用一两笔外在环境替代长段心理剖白
- 让景物成为人物情绪的回声，而不是独立节目

### 7. 高潮段让景物托气氛，不要抢戏
高潮处的景物重点不是“美”，而是：
- 紧张感
- 压迫感
- 失控感
- 冷寂感

**景物要托戏，不要盖戏。**

## 常用模板
### 模板 1：人物眼睛看景
`人物先感到什么，再顺着视线落到景物，景物因此带上情绪。`

### 模板 2：景物压到人身上
`先写风、雨、灯、夜色等环境压过来，再写人物的站位、动作和反应。`

### 模板 3：移情于景
`本来该直接写情，改成写人物眼中的景物变化。`

## 快速示例
### 输入
- 场景：一对热恋中人和一个刚失恋的人，看到同一片蓝天碧水
- 目标：写出“同景异感”

### 输出
- 热恋视角：天色明净、风也轻，水面亮得像在发光
- 失恋视角：天太亮，水光刺眼，周围一切都显得空
- 重点不在景物本身变了，而在**看景的人变了**

## 注意事项
- **禁止**脱离人物，单独写一大段景物
- **禁止**正在冲突时停下来做风景介绍
- **禁止**一边写悲痛，一边塞无关华丽摆设
- **禁止**同一段里远近、主客观镜头乱切
- 目标不是景物更多，而是**景物一出现，就和人、情绪、剧情拧在一起**。
