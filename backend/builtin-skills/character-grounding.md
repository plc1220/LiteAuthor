把可爱、疯批、高冷等标签变成有剧情作用的角色。**角色不是先定一个漂亮形象，再强行往上贴剧情；真正立得住的人设，至少要有成因、有用处、有关键时刻的表现。** 当角色可爱但无用、疯但不带感、强行贴剧情写得痛苦时，用这个 Skill。

## 适用情况
- 角色设定看着香，正文却很纸片
- 只会写“很可爱”“很疯”“很美”，不知道怎么落地
- 角色为了符合标签被写得像白痴或工具人
- 剧情和人设总在互相拉扯
- 想让角色互动更符合人设

## 输入
- 角色定位与戏份
- 3—5 个核心标签
- 角色从出生到出场前的重要经历
- 当前剧情里需要他解决或推动的问题
- 需要重点卖的关系或场面

## 核心原则
1. **先让角色有用，再让角色可爱/疯/苏。**
2. **背景成因决定反应方式。**
3. **标签必须落到关键动作、选择和后果上。**
4. 极端标签要有支点：天赋、实力、坚韧、勇敢、品味、边界。
5. 互动要符合细化后的性格成因，不要只套大标签。

## 步骤
### 1. 先问：这个角色在故事里有什么用
至少写清两件事：
- 剧情用处：推进什么、阻碍什么、托住什么、改变什么
- 关系用处：让谁动心、失衡、成长、崩防或被治愈

如果答不出来，这个人设大概率还只是形象贴纸。

### 2. 从“出生到出场前”补背景链
至少补：
- 家庭/成长环境
- 关键旧事
- 最在乎什么
- 最怕失去什么
- 什么刺激最容易触发他

句式：
`因为过去____，所以现在一遇到____就会____。`

**不补成因，标签就会发空。**

### 3. 给角色一个“有用支点”
优先从下面选 1—2 项：
- 天赋点
- 专长/技能
- 解决问题的办法
- 关键时刻扛事的能力
- 稳定的判断力

如果剧情暂时给不了高光能力，就给：
- 坚韧
- 勇敢
- 能在关键时刻承担别人承担不了的东西

### 4. 把标签改写成关键场面
不要只写：
- 他很可爱
- 她很疯
- 他很温柔

改写成：
- 在什么场面下，他会做出什么只有他会做的事
- 这个动作会怎样改变局面

例如：
- 软白甜不是只会哭，而是**哭着也敢捅对手十八刀**
- 平时软乎乎的人，关键时刻反而最能扛事

这样角色才既可爱又有用。

### 5. 极端标签要加“屏障”
像疯批美人、病娇、狂人这类标签，至少补 2 项：
- **实力屏障**：他为什么可以肆无忌惮
- **时髦值/审美点**：穿着、动作、场面、说话方式、品味
- **选择性**：他不是无差别乱疯，而是在特定刺激下发作

没有这些，角色就会只剩“纯疯”，不带感。

### 6. 用“正面 + 侧面 + 后果”三线落地
#### 正面
角色自己怎么说、怎么做、怎么选

#### 侧面
别人为什么会觉得他可爱、危险、可靠、难忘

#### 后果
他这次出手后，剧情和关系发生了什么变化

### 7. 互动必须从人设成因里长出来
- 两个傲娇，不会突然来一段低头认错式告白
- 一个“不谈感情”的人，不代表永远不会亲密；要先细化他为什么不谈、怕什么、卡在哪
- 一个“不在意美色”的人，也要写清他到底更看重灵魂、权力、责任，还是有旧伤

**标签越抽象，越要往具体原因拆。**

### 8. 反查是不是在“为了形象强行写剧情”
出现下面情况就要重做：
- 先定一个形象，再硬塞无效情节证明他很可爱/很疯
- 角色除了标签，没有实际作用
- 角色的反应谁来都行
- 删掉形容词后，人物立不住

稳的写法是：**先有背景和用途，再让形象从场面里长出来。**

## 输出格式
1. 角色定位与剧情/关系用处
2. 背景链：过去 → 触发点 → 当前反应
3. 有用支点
4. 核心标签对应的关键场面
5. 正面 / 侧面 / 后果落地法
6. 互动禁区与自然互动方式
7. 这个角色最容易纸片化的风险

## 快速示例
### 输入
- 角色：软白甜小哭包
- 目标：写得可爱，但不想写成白痴

### 输出
- 有用支点：医术很强；关键时刻敢拼命
- 关键场面：平时容易红眼眶，但真出事时能一边哭一边处理伤口，还敢下狠手救人
- 后果：读者会觉得他不是废物萌，而是软外壳下有硬骨头

### 输入
- 角色：疯批美人反派
- 目标：写得美又疯，不想只剩发疯

### 输出
- 背景链：长期高压环境养成控制欲
- 有用支点：战力/权力极强
- 屏障：有实力兜底，能承担发疯成本
- 时髦值：固定场面、说话方式、审美习惯
- 选择性：只在被挑衅到底线时失控
- 效果：角色是危险又亮眼，不是单纯闹腾

## 注意事项
- **不要**只给人物贴可爱、疯批、清冷这类词
- **不要**为了让角色讨喜，把他写得没有作用
- **不要**让极端标签只靠作者旁白硬夸
- **不要**把互动写成人设失真
- 真正落地的人设，不是“看起来像”，而是**一出手就知道这个人能干什么、为什么会这样、会把局面推到哪**。
