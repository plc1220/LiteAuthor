写开篇时，先别急着背“黄金三章”。**开头最重要的不是把所有技巧都塞进去，而是让读者尽快吃到这本书真正的卖点。卖点不同，节奏就不同；硬用同一套快节奏公式，反而容易把开头写碎、写飘、写假。**

## 适用情况
- 开头拼命追求节奏，结果很乱
- 前三章钩子、伏笔、反转很多，卖点却没立住
- 打脸开场不够爽，人设开场不够带感
- 总想模仿榜单文的“黄金三章”，但自己一用就失灵
- 跟风题材时，只跟了壳子，没跟到核心快感

## 核心原则
1. **先定卖点，再定时速。**
2. 开篇只需要把最值钱的一口先交出来，不需要一次交完全部能力。
3. **无脑甜不是作者不用动脑，而是读者不用费力；细节要由作者负责看清。**
4. **打脸开场要快，但不能薄；人设开场可以慢，但不能散。**
5. 跟风题材时，跟的是读者想要的体验，不是题材名词本身。

## 步骤
### 1. 先写开篇承诺
用一句话写：
`前 1—3 章必须让读者吃到____。`

可选：
- 打脸爽感
- 主角带感
- 甜感 / 拉扯感
- 悬念追问
- 世界观新鲜感

只选 1 个主项，最多带 1 个副项。

### 2. 分清自己到底在卖什么
不要写题材壳，写读者体验。

差写法：
- 豪门老男人
- 校园甜文
- 逆袭爽文

好写法：
- 在熟悉场景里被高位者偏爱、被宠、被托底
- 看两个人在日常接触里慢慢变甜
- 看主角被压后快速反打、让观众破防

**题材只是壳，真正决定开篇节奏的是这口体验。**

### 3. 按卖点选择开篇时速
### A. 打脸 / 爽点开场
处理原则：**快、准、反应足。**

具体做法：
- 多短句、多分段
- 少解释，先把挑衅和局面立住
- 用多角度反应放大爽感：围观、炮灰、懂行人、主角自己的态度
- 不要只写“主角一巴掌过去”，要写完整链条：
  `挑衅 / 误判 → 主角出手 → 全场反应 → 对手破防 → 余波`

### B. 卖主角人设 / 卖苏感开场
处理原则：**慢半拍，把细节扣清。**

具体做法：
- 放慢动作、神态、语气、观察点
- 把主角与众不同的地方落在具体行为里
- 让读者从细节里感到“这个人带感”，而不是作者先宣布
- 不要一上来切太多场面，主角还没立稳就跑走

### C. 甜感 / 暧昧开场
处理原则：**关系张力比事件数量更重要。**

具体做法：
- 先写接触感、站位、语言火花、微小偏爱
- 少用生硬大事件硬推感情
- 把读者的注意力锁在“这两个人有戏”上

### D. 悬念 / 设定开场
处理原则：**只挂一个主问题。**

具体做法：
- 先抛一个明确追问
- 先藏半步原因，不要把背景全讲完
- 让主角尽快因为这个问题行动起来

## 4. 识别“黄金三章误用”
出现下面情况，通常说明你不是节奏快，是节奏乱：
- 开头同时塞起承转合、悬念、伏笔、反转、设定介绍
- 一章里切很多场，读者记不住主角在干什么
- 觉得“我已经放了钩子”，但钩子和卖点无关
- 每章都想断在悬念处，结果下一章又不回收

**对大多数作者来说，开头更稳的做法不是再快一点，而是先把卖点写实。**

### 5. 打脸开头别只写结果，要写体感
如果开篇卖“无脑爽文”，重点不是让主角立刻赢，而是让读者感到爽。

至少补齐：
- 对手怎么误判
- 主角怎么轻松反打
- 观众怎么看见
- 对手怎么破防
- 这次之后局势变了什么

### 6. 无脑甜别理解成“随便写写”
如果卖点是轻松甜、日常甜、互动甜：
- 不要偷懒写成平铺聊天
- 主动把细节放大给读者看
- 用动作、停顿、语气、照顾方式、视线变化来交糖

**读者不用费脑，不代表作者可以不做观察。**

### 7. 跟风先跟“核心快感”，不是跟题材壳
跟风时先问：
`这类题材真正爆的，到底是哪一口？`

例如：
- 不是“豪门老男人”四个字本身值钱
- 而是“在熟悉生活场景里，被高位者偏爱、被宠、被托底”的满足感值钱

如果只抄壳子，不拆快感，通常跟到的只是表面热词。

### 8. 最后做一次开篇检查
逐条问：
- 前 3 章最主要在卖什么
- 这个卖点是否已经在具体场面里被读者吃到
- 开篇是不是为了追求快，把真正该慢写的地方写薄了
- 是不是挂了很多悬念，但没有一条比卖点更值钱

## 输出格式
1. 开篇承诺
2. 主卖点 / 副卖点
3. 题材壳与核心快感
4. 本开篇的时速选择
5. 前 1—3 章场景安排
6. 应快写的部分
7. 应慢写的部分
8. 当前“黄金三章误用”风险

## 快速示例
### 输入
- 爽文开篇，主角前两章已经出手三次，但读者还是不觉得爽

### 输出
- 主卖点：打脸爽感
- 问题：只有结果，没有围观和破防反应
- 修改：压缩解释，把“挑衅 → 出手 → 反应 → 破防 → 余波”链条写全；多用短句和分段，把主角的从容感写出来

### 输入
- 甜文开篇，想学黄金三章，结果前三章切了五个场面

### 输出
- 主卖点：主角带感 + CP 有戏
- 修改：删掉多余切场，留 1—2 个核心场景；放慢人物动作、语气和接触细节，让读者先记住主角，再期待两人后续互动

## 注意事项
- **不要**把“开头快”误当成“开头好”
- **不要**所有题材都套同一种黄金三章模板
- **不要**打脸只写结果，不写过程和反应
- **不要**甜文只写聊天，不写细节和关系张力
- **不要**跟风只跟题材名词，不拆核心快感
- 开篇真正值钱的不是热闹，而是**让读者在最短时间内吃到这本书最该卖的那一口。**
