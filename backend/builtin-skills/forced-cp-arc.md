写捆绑 CP 时，先把两人绑在一起，再让感情错位增长。围绕舆论、工作、婚约、任务、家族安排等外力，安排两人从公事公办、互相误读、单方动心、另一方避嫌，走到真正在意。**核心不是“突然相爱”，而是“先被绑，再有人先失衡”。**

## 适用情况
- 被迫营业、炒 CP、综艺绑定、合约恋爱
- 先婚后爱、政治联姻、任务搭档
- 想写“被迫捆绑成一对”再转真感情
- 追妻火葬场想要前期过渡更顺

## 输入
- 绑定原因
- 双方强弱和公众地位
- 初始印象 / 偏见
- 谁先动心、谁后知后觉
- 想走甜、拉扯还是火葬场

## 步骤
### 1. 先写清“为什么分不开”
绑定条件必须具体：
- 工作合作
- 舆论炒作
- 家族 / 婚约
- 任务同组
- 利益交换
- 身份限制

**没有强绑定，就很难持续写这类 CP。**

### 2. 定初期态度，但别嫌弃过头
前期可以写：
- 公事公办
- 冷淡
- 不耐烦
- 怀疑
- 保持距离

但**不要**把一方写成极端羞辱、恶意贬低、把对方当脏东西。前面嫌弃越狠，后面越难洗白。

### 3. 先写一方动心，再写另一方
最稳的顺序是：
`被迫同框 → 负面印象修正 → A 先动心 → B 因避嫌/误判/自保而后退 → A 更失衡 → B 后知后觉`

**不要两个人同时一起开窍。**

### 4. 用“重新认识”当第一个转折
前期最有效的加分点不是表白，而是推翻偏见：
- 以为对方炒作，结果对方其实很有分寸
- 以为对方虚荣，结果对方其实正直能扛事
- 以为对方软弱，结果对方有自己的硬骨头

### 5. 让先动心的人先示好，后动心的人先避嫌
这类文很吃错位：
- 先动心的人开始给台阶、照顾、偏心
- 后动心的人因为怕再被误会、怕越界、怕影响工作，反而躲着走
- 路人先看出端倪，当事人还不承认

### 6. 路人和环境可以提前“刷真相”
可用：
- 经纪人 / 朋友提醒
- 观众 / 网友乱磕
- 同事起哄
- 家人误会
- 第三人先发现谁在失衡

这类外部压力既能制造同框，也能把隐藏情绪逼出来。

### 7. 如果要写追妻火葬场，补“认错 + 补偿”
火葬场不能只有告白，要补：
- 认清自己前期误判了什么
- 明确具体伤害在哪里
- 用行动补偿，不只用台词后悔
- 让被伤的人有拒绝权和观察期

## 输出格式
1. 绑定原因
2. 初期关系状态
3. 初期偏见
4. 先动心的人与动心点
5. 后动心的人与避嫌理由
6. 路人助攻点
7. 关系转折表
8. 若走火葬场，补偿动作

## 快速示例
### 输入
- 影帝男主 × 新人女主，被公司强绑营业

### 输出
- 绑定原因：新戏宣传和公司策略
- 初期状态：男主公事公办，女主怕被误会主动道歉
- 偏见：男主以为女主在借自己上位
- 转折：合作中发现女主处事干净、工作认真，男主先动心；女主却因舆论开始躲他
- 路人助攻：工作人员先看出男主会下意识照顾女主
- 后续：男主越靠近，女主越避嫌，直到男主失衡摊牌

## 注意事项
- **禁止**绑定原因太弱，只靠作者硬凑同框
- **禁止**前期嫌弃过猛，后期突然深情
- **禁止**双方同步动心、同步表态
- **禁止**火葬场只会“我错了”，没有可见补偿
