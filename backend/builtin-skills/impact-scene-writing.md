把耳光、巨响、突袭、猝然亲吻这类“发生得很短、存在感很强”的瞬间，拆成几层去写。**拉长的是读者接收这个瞬间的过程，不是事件本身的时长。** 用词要简单、句子要短、层次要分明，靠多角度接力，而不是靠一大段花哨长句。

## 适用情况
- 耳光、爆炸、撞击、摔落、开门见血这类短促强刺激画面
- 突然亲吻、突然靠近、突然说破等情绪爆点
- 明明脑海里画面很猛，写出来却轻飘飘一句过去
- 读者容易看漏瞬间动作

## 输入
- 这一瞬到底发生了什么
- 当前主视角角色
- 想让读者重点感受到什么：惊吓 / 爽感 / 甜炸 / 羞耻 / 震惊
- 这一瞬结束后要接的动作或后果

## 核心原则
1. **用词尽量简单，别把短瞬间写成抒情长段。**
2. **每一层只负责一个点：声音 / 触感 / 认知 / 情绪 / 动作。**
3. 让读者和角色按顺序“接收”这个瞬间。
4. 重要瞬间可以分层；不重要瞬间一笔带过就够了。
5. 感叹号是加重工具，不是满屏装饰。

## 步骤
### 1. 先用一句最短的句子打出去
优先用：
- 拟声词
- 一个直接动作
- 一个最醒目的外部现象

例如：
- “啪！”
- 门外轰然一响。
- 他忽然低头亲了上来。

### 2. 第二层写“外部感官”
只写一层最直接的感官：
- 声音
- 光影
- 震动
- 温度
- 疼痛 / 压迫 / 失衡

### 3. 第三层写“角色终于明白发生了什么”
重要的是“认知抵达”的那一下：
- 刚才是她打了我？
- 门外是什么炸了？
- 他刚才是不是亲了我？

### 4. 第四层再放情绪
等读者先接住事实，再放：
- 震惊
- 羞耻
- 愤怒
- 爽感
- 不敢置信
- 心跳失控

### 5. 第五层补对应动作
情绪落地后，人物要有反应：
- 扑过去
- 后退
- 捂脸
- 闭眼
- 抬手
- 说话
- 失语

### 6. 用“不同角度”而不是“同一句反复说”
同一个事件可以拆成：
- 拟声词
- 外部声音
- 身体触感
- 事实确认
- 心理反应
- 后续动作

**禁止**只是换个说法反复宣布同一句话。

### 7. 感叹号使用规则
- 句子描述很普通时，可用感叹号提气
- 句子里已经有强烈情绪词时，可不用，避免满屏惊叹号
- 真要双倍爆发，只挑最该爆的那一句

### 8. 可选写法：让读者和角色一起“后知后觉”
如果想更强代入，可先只给画面，不立刻解释。  
让读者先疑惑，角色也先疑惑；等角色反应过来，读者也同时反应过来。

## 常用模板
### 模板 A：物理冲击
1. 拟声词 / 动作
2. 外部感官
3. 身体反应
4. 认知抵达
5. 情绪爆发
6. 对应动作

### 模板 B：情绪爆点
1. 客观发生了什么
2. 主角感到哪里不对
3. 主角明白发生了什么
4. 情绪失控
5. 后续动作或台词

## 输出格式
1. 事件核心
2. 分层清单
3. 改写后的段落
4. 这一瞬结束后如何接下去

## 快速示例
### 示例 1：耳光
- “啪！”
- 空气里炸开一声脆响。
- 她捂住脸，掌心下火辣辣地疼。
- 女主刚才打了她一巴掌。
- 她竟然敢打她？
- “我跟你拼了！”她猛地扑了上去。

### 示例 2：突然亲吻
- “吧唧。”
- 男主笑着，慢慢退开。
- 女主怔在原地，指尖下意识碰了碰嘴唇。
- 唇上还残着一点温热。
- ……刚才，是被亲了？
- 她耳根轰地烧起来，整个人都僵住了。

## 注意事项
- **禁止**用大段华丽长句破坏“短暂”的特性
- **禁止**每层都堆满修饰词
- **禁止**还没让读者看懂发生什么，就先长篇抒情
- **禁止**把所有短瞬间都慢写，只有关键镜头才值得分层
