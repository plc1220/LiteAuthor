先从人设出发，再决定他们在哪里、因为什么、以什么方式发生互动。**不是先摆一个场景把两个人塞进去，而是让场景和互动都像这两个人自己长出来的。**

## 适用情况
- 不会写男女主或双主角互动
- 角色一见面就像在完成任务，没有自然张力
- 同样的搭配换谁都能说同样的话
- 总爱把人物塞进一个现成场景里硬互动
- 初见、试探、暧昧场写得像模板

## 输入
- A / B 的核心人设
- 当前最重要的性格切片
- 两人的当前处境
- 当前关系阶段
- 这场互动想完成的目标

## 步骤
### 1. 先提炼这场戏真正相关的人设切片
不要把整个人设卡全搬进来，只提这场最有用的那一部分：
- 他此刻最显眼的性格是什么
- 她此刻最容易被触发的反应是什么
- 谁更主动，谁更防备，谁更会装

**同一个人物在不同阶段，最该拿来推动互动的人设切片也不同。**

### 2. 用“人设 + 处境”反推相遇场景
不要先想“让他们在哪谈恋爱”，先问：
- 这种人，在当前剧情里最可能出现在哪里
- 这种性格，在这种处境下最可能做什么
- 他们为什么会在这个节点碰上

场景要像人物自己走进去的，不要像作者把他们拎进去的。

### 3. 先定互动质地，再写台词
同样是初见，不同人设会长出完全不同的互动气质：
- 风流型人物 → 更可能调笑、观察、逗弄
- 老谋深算型人物 → 更可能克制、试探、隔着距离施压
- 天真娇俏型人物 → 更容易慌乱、顶嘴、露出鲜活反应
- 冷淡强势型人物 → 更可能先控场、后露破绽

**先定“这场戏是什么味道”，再定他们具体说什么。**

### 4. 按人设写四个“第一反应”
分别写清：
- 第一眼先注意到对方什么
- 第一反应先做什么动作
- 第一时间会说什么类型的话
- 第一层会藏住什么心思

互动带感，往往靠的就是这些“第一反应”够像本人。

### 5. 让场景动作也贴人设
除了台词，还要看动作和距离：
- 谁会逼近
- 谁会后退
- 谁敢居高临下地看人
- 谁会借规矩、身份、地形保持距离
- 谁会嘴硬，谁会露馅

**动作、站位、视线，本身就是互动的一部分。**

### 6. 做“换人测试”
把其中一方换成另一个性格完全不同的人，再问：
- 这场戏还会不会原样成立
- 台词、动作、气氛是不是几乎不用改

如果答案是“会”，说明这场互动还太模板，不够贴人。

### 7. 避免“相亲式空场景”
高风险写法：
- 单纯把两个人关进一个房间聊天
- 没有处境压力，只靠作者强行制造暧昧
- 场景和剧情无关，只为让他们对话

除非故事本身就是相亲、约会、谈判，否则优先用：
- 任务
- 误会
- 冲突
- 目击
- 救场
- 共同处境
来生成互动。

### 8. 一场互动结束后必须留下变化
至少留下 1 项：
- 对对方有了新判断
- 关系偏移半步
- 埋下下一次见面的理由
- 暴露了某个软肋或魅力点
- 带出新的剧情入口

如果只是说了几句带感的话，关系和局势都没变，这场戏就不够值钱。

## 输出格式
1. A / B 的场景人设切片
2. 最自然的相遇场景
3. 这场互动的气质
4. 双方四个第一反应
5. 关键动作与站位
6. 互动后的关系变化
7. 换人测试结果
8. 模板化风险点

## 快速示例
### 输入
- 男主：风流倜傥
- 女主：天真娇俏
- 场景目标：初见有火花

### 输出
- 相遇场景：不是安排一场干聊，而是在女主偷偷跑出来、狼狈失态时被男主看见
- 互动气质：轻佻调笑 + 少女羞恼
- 男主第一反应：先看戏，再出声
- 女主第一反应：先慌，再嘴硬
- 关系变化：两人都对对方留下了强记忆点

## 注意事项
- **禁止**先定甜场面，再硬改人设去配合
- **禁止**所有人都用同一套暧昧台词
- **禁止**互动只靠嘴说，不写动作和距离
- **禁止**为了让他们见面，安排一个和人物、剧情都无关的空场景
- 互动最值钱的地方，不是“终于说上话”，而是**只有这两个人才会这样说、这样动、这样看彼此**
