把甜宠写成“偏爱和在意”，不是写成“有钱”“很强”或“别人多震惊”。**所有送礼、护短、救场、打脸、吃醋、照顾，重点都要落在：这个人为什么偏偏在意 TA，这种在意怎么变成具体动作、具体情绪和关系变化。**

## 适用情况
- 甜宠场面很多，但读者只觉得苏，不觉得甜
- 男主 / 女主的宠爱方式高度重复，只会送、买、救、护
- 吃醋桥段模板化，换谁都能用
- 写照顾像家长，写撒娇像作精
- 甜戏里围观群众很忙，CP 自己没情绪

## 输入
- CP 设定与当前关系阶段
- 一方当前能提供的优势：钱、权、实力、身份、资源等
- 这一场想写的甜点
- 想让读者吃到的情绪：甜 / 酸甜 / 暗甜 / 护短 / 笨拙宠 / 反差宠
- 当前角色风味：傲娇、暖男、高冷、小狼狗、年上、腹黑等

## 步骤
### 1. 先写这场甜戏的“在意点”
用一句话填空：
`这个人此刻真正放在心上的，是对方的____。`

常见在意点：
- 对方开不开心
- 对方会不会受委屈
- 对方喜不喜欢自己准备的东西
- 对方有没有受伤
- 对方会不会误会自己
- 对方在别人眼里会不会被轻视
- 对方会不会不舒服、受冻、挨饿、难堪

没有在意点，只剩外在动作，甜味就会很淡。

### 2. 把“钱和强”翻译成“在意的动作”
不要只写 TA 做了什么，更要写 TA 做这件事时的情绪和视线落点。

改写方法：
- 送礼，不只写礼物多贵，要写 TA 怎么挑、怎么犹豫、怎么偷看反应
- 出手护短，不只写 TA 多强，要写 TA 第一眼先确认对方有没有事
- 公开撑腰，不只写别人多震惊，要写 TA 为什么偏偏在这一刻站出来

例：
- 霸总送礼，不只写豪华，要写 TA 表面不经意递过去，实际从拆礼盒那一刻起一直在观察反应
- 剑修解围，不只写反派如何吃瘪，要写 TA 处理完麻烦后，视线第一时间落回对方身上

### 3. 优先写双方反应，再写围观反应
甜戏里的主次顺序通常是：
1. 在意的人做了什么
2. 被在意的人怎么感受到
3. 围观者怎么震惊、羡慕、误会

围观反应是调味料，不是主菜。

### 4. 先轮换基础甜味，不要全文只有一种糖
至少轮换 3 种：

#### 明甜
公开护短、站队、偏爱、当众撑腰。

#### 暗甜
记住口味、默默照顾、第一反应想到对方、话没说满但动作先到了。

#### 笨拙甜
明明很会处理别的事，碰到对方却紧张、怕说错、怕送错、怕被拒绝。

#### 嘴硬甜
嘴上冷淡，行动却全是偏爱。

#### 酸甜
带一点点吃醋、失落、试探、紧张，再用后续补偿回收。

**上等甜文不是纯糖水，而是酸酸甜甜，口感有变化。**

### 5. 再补三种高频可复用甜法
#### 索取型甜
让一方提出一个**可爱、自然、无伤大雅**的小要求，另一方选择包容、答应或半推半就地答应。

重点：
- 索取要小，不要一上来就高代价、高负担
- 索取要贴关系阶段，不要越线
- 索取者最好本质懂事，只是在亲近感里偶尔撒娇
- 甜点不在“要到了”，而在“对方愿意给”

适合写：
- 想吃某样东西
- 想让对方陪自己做件小事
- 想要一个拥抱 / 一个吻 / 一个承诺式的小动作

#### 提前照顾型甜
对方什么都没说，另一方已经先想到并处理好了。

重点：
- 写对方**还没开口**，TA 已经注意到隐形需求
- 照顾要落在细小处：怕胃不舒服、怕手冷、怕太累、怕尴尬、怕难堪
- 这种甜可以和任何角色风味叠加：傲娇 + 照顾、冷脸 + 照顾、笨拙 + 照顾都成立

真正的甜点不是“我照顾了你”，而是“我连你没说出来的部分也放在心上了”。

#### 隐晦吃醋
少直接大喊“你离他远点”，多写试探、绕弯、抢位置、给替代方案、安静地把两人隔开。

重点：
- 吃醋方式必须贴人设
- 温柔型可以绕着问、顺手接管
- 傲娇型可以嘴硬、找借口、暗暗较劲
- 高冷型可以不说破，但用行动重新划边界
- “万人迷”写不好时，不妨从“人见人醋”去写，而不是“人见人爱”

配角甚至同性配角的吃醋，有时也能起到点睛作用。

### 6. 给强者加“在意时的失衡”
越强、越有钱、越万能的人，越要写 TA 在在意对方时出现的失衡：
- 明明平时果断，这次反复挑礼物
- 明明习惯用钱解决问题，这次发现钱摆不平，只能小心试探
- 明明平时冷静，这次却一直盯着对方反应

失衡会让甜味落地。

### 7. 让甜戏落在具体细节上
优先使用可见小动作：
- 替对方擦掉脸上的血迹或灰尘
- 递药、递水、留位置
- 听见别人提起对方就先抬眼
- 先护住，再收手，再若无其事装冷淡
- 记得对方一句随口说过的话
- 冬天提前准备热饮，夏天顺手把冰的换成温的

小动作越具体，甜味越真。

### 8. 控制甜和爽的比例
如果一场戏同时要甜和爽：
- 先给关键情绪动作
- 再给爽点回收
- 最后用一句或一个细节把甜味钉住

不要全篇幅都拿去写别人震惊、反派吃瘪，结果 CP 自己像背景板。

### 9. 同一种宠法不要连续复读
逐条检查：
- 是不是每次都靠买东西表达爱
- 是不是每次都靠英雄救场表达爱
- 是不是每次都靠一句“有我在”表达爱
- 是不是每次都只会写一种吃醋反应

如果是，就必须换口味、换场景、换在意点。

### 10. 甜戏结束后要有余波
一场甜戏结束后，至少回收 1 项：
- 关系更近了一步
- 一方更确定自己在意对方
- 一方开始迟疑自己是不是动心了
- 旁人看出两人关系不一样
- 后续冲突因此变得更有分量

### 11. 别把甜法写死
同一场戏可以同时是：
- 傲娇 + 提前照顾
- 嘴硬 + 隐晦吃醋
- 索取型甜 + 笨拙甜

不要把“甜”和“宠”硬切得太死。真正好吃的甜戏，往往是**角色风味 + 在意动作 + 情绪回响**一起生效。

## 输出格式
1. 本场甜点类型
2. 在意点
3. 角色风味
4. 具体动作
5. 双方情绪反应
6. 围观反应（可选）
7. 余波
8. 下次建议换的甜味

## 快速示例
### 输入
- 设定：暖男男主半夜给大小姐女主煮面
- 目标：写出索取型甜 + 提前照顾型甜

### 输出
- 在意点：男主既在意她馋到了，也在意她胃会不会难受
- 甜点类型：索取型甜 + 提前照顾型甜
- 具体动作：女主磨着想吃泡面，男主嘴上不赞同还是答应；表面煮的是“泡面”，实际把面换成更软的面；吃完又递热牛奶
- 情绪回响：女主因为“他会担心我难受”而乖乖喝下；男主表面温柔，内心一直盯着她有没有皱眉
- 余波：以后男主会先记她的胃口和忌口，女主也更敢对他提一点小要求

### 输入
- 设定：学霸男主看见女主一直向男 B 请教问题
- 目标：写隐晦吃醋

### 输出
- 在意点：他不想女主把求助和依赖给别人
- 具体动作：不直接发火，而是先问她在问什么，再顺手把答案补得更完整，最后淡淡补一句“下次这种问题来问我”
- 余波：女主开始察觉他的边界感，男主的吃醋也更贴人设

## 注意事项
- **禁止**把甜宠写成单纯炫富或炫强
- **禁止**只写围观者震惊，不写角色彼此的情绪
- **禁止**把索取写成无底线伸手，把照顾写成把对方当弱智
- **禁止**全文只有一种甜味
- **禁止**甜戏结束后关系毫无变化
- 真正的甜必须让读者看到：**偏爱是怎么发生的，对方又是怎么感觉到的**
