先定这身衣服要让读者看到什么，再决定写哪几笔。现代文写穿搭，**重点不是把品牌和材质报菜名，而是让衣服同时服务身份、品味、年代和场景。**

## 适用情况
- 想写学生、白领、明星、名媛、富豪等现代人物出场穿着
- 一写名牌就像暴发户堆 logo
- 想写时尚高品味角色，却怕搭配土、过时、不贴人
- 需要用衣着侧写人物的富有、职业和审美

## 输入
- 角色身份、年龄、职业、经济层次
- 当前场景、季节、年份
- 想通过穿搭传递的核心印象
- 观察者是谁，认不认识这些品牌或风格
- 是否需要突出富有、时尚、专业、反差

## 步骤
### 1. 先定这身衣服的任务
只选 1-2 个主目标：
- 显年轻
- 显专业
- 显富
- 显时尚
- 显松弛
- 显危险
- 显克制
衣服不是独立展柜，要服务人物。

### 2. 先判断“适不适合”，再判断“贵不贵”
检查四件事：
- 身份是否匹配：学生不像贵妇，白领不像红毯
- 性格是否匹配：张扬的人和克制的人不会穿成一个味
- 场合是否匹配：通勤、约会、宴会、发布会不同
- 年代是否匹配：别把 05 年的流行写进 07 年的时髦角色

### 3. 写普通穿搭，用“款式 + 颜色 + 版型 + 一处亮点”
不要把每件单品都报一遍。优先写：
- 整体轮廓
- 一两件关键单品
- 最能说明品味的一处搭配

### 4. 写富贵感，少堆品牌，多写“一个识货的人怎么看”
最稳的写法是：
- 只挑 **一个** 最值钱或最有门槛的单品
- 借另一个角色的视角去看
- 带上情绪：惊讶、艳羡、鄙夷、攀比、见怪不怪
**品牌本身不会自动产生贵气，别人的反应才会。**

### 5. 让“谁在看”决定写法
- 同阶层看同阶层：不会因有钱本身惊讶，只会比较品味高低
- 普通人看有钱人：可能认不出牌子，只会觉得贵、讲究、距离感强
- 白领或中产看有钱人：会识货，会艳羡，但不一定大惊小怪
- 身份反差揭露时：同一套奢侈品才会真正产生冲击
**先定观察者，才能定反应。**

### 6. 用一句能落地的价值信号，别写显微镜说明书
少写：
- 一眼根本看不出的材质细节
- 整串品牌名平铺直叙
- 把每个零件都拆开解释
多写：
- 价格对比
- 稀缺性
- 定制门槛
- 别人识货后的失态、沉默、改口
读者更容易记住“这一副眼镜够买两辆车”，而不是一串材料名。

### 7. 一件关键单品立住后，其他奢侈品点到为止
如果眼镜、包、腕表、鞋子、香水全堆上来，容易变土。**先用一个高门槛单品定级，再默认其他不会太差。**

### 8. 写时尚角色，优先找现实参考
当需要写：
- 时尚女大学生
- 女明星
- 名媛
- 设计师
- 潮流男主
先找参考图、秀场图、时装杂志、街拍或同年份造型，再按图改写成文字。

记录这些信息：
- 年份
- 场合
- 轮廓
- 主色
- 亮点单品
- 妆发感觉
闭眼硬编最容易土和过时。

### 9. 输出时给两版
- **设定版**：这身衣服怎么搭、为什么适合角色
- **正文版**：一段可直接入文的描写，必须带观察者视角或人物反应

## 输出格式
1. 角色与场景定位
2. 穿搭目标
3. 核心单品与整体风格
4. 观察者视角与情绪
5. 可直接入文的描写
6. 需要避开的土味写法

## 快速示例
### 输入
- 角色：隐藏身份的豪门继承人
- 场景：第一次公开露面
- 目标：写出贵而不俗

### 输出
- 核心单品：一副极难买到的定制眼镜
- 观察者：懂行但买不起的白领同事
- 正文版：他抬手摘下眼镜时，同事先愣了一下，像是终于认出那副镜架的来头，后半句寒暄都卡在喉咙里。那不是商场里摆出来撑门面的货色，订一副要等很久，价钱够普通人把车库停满。可他只是随手搁在桌边，像放下一支再普通不过的钢笔。

## 注意事项
- **禁止**平板罗列品牌名
- **禁止**写一眼根本看不出的材质细节
- **禁止**只顾贵，不顾好不好看、适不适合人
- **禁止**把所有奢侈品一次性堆满
- 现代服饰描写最重要的是：**合人、合时、合场，再侧写出阶层和品味。**
