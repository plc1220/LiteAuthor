从角色一个具体想要的东西、关系或结果出发，往下挖到真正缺的是什么。**表层需求只是入口，深层动机才决定剧情。**

## 适用情况
- 角色会做事，但行为总显得空
- 不知道角色为什么偏偏在意这件事
- 写甜文甜不到点，写虐文虐不到点
- 角色的欲望、恐惧和旧事还没串起来
- 想把一个小需求写出人物层次和后续剧情

## 输入
- 角色定位
- 当前具体想要的东西 / 关系 / 结果
- 触发这个需求的场景
- 已知的人设、过去经历、当前关系
- 想推进的情节方向：甜 / 虐 / 爽 / 成长

## 步骤

### 1. 先把“想要什么”写具体
不要写“她想变好”“他想幸福”，改写成：
- 想要一瓶香水
- 想拿下这次晋升
- 想让某个人来接自己
- 想留住一只猫
- 想得到一句认可

**越具体，越容易往下挖。**

### 2. 从表层需求往下挖 4 层
围绕这个具体需求，依次回答：

#### ① 功能层
这个东西能帮 TA 做什么？
- 更好看
- 更体面
- 更安全
- 更省事
- 更容易接近某个人

#### ② 情感层
TA 想通过它得到什么感觉？
- 被喜欢
- 被重视
- 被肯定
- 被羡慕
- 被安慰
- 被理解

#### ③ 身份层
TA 想借这件事证明自己是什么人，或摆脱什么位置？
- 我不是土气的人
- 我配得上那个圈子
- 我值得被爱
- 我不是废物
- 我不是会被抛下的人

#### ④ 旧事层
过去发生过什么，让 TA 偏偏对这件事执着？
- 曾被嘲笑寒酸
- 曾被拒绝、抛弃、否定
- 曾在某段关系里吃过亏
- 曾被某件象征物安慰过
- 曾失去过类似的东西或人

### 3. 一次不要只信第一个答案
至少列出 3 个不同动机版本，再比较。

同样是“想要一瓶香水”，可能是：
- 想在约会时更有魅力
- 想融入讲究格调的圈子
- 想拿它撑场面，掩盖自卑
- 因为某本书、某部电影赋予了它特殊意义
- 只是单纯喜欢这种味道

**不同动机，会长出完全不同的性格和剧情。**

### 4. 给每个动机补三件事
对每个候选动机，都补上：
1. 这暴露了角色什么性格
2. 这背后可能藏着什么故事
3. 这会把剧情推向哪里

示例：
- 为了显得有魅力 → 可能有自卑或求认同需求 → 感情线更敏感、更在乎回应
- 为了炫耀 → 可能虚荣、怕输、怕被看低 → 容易长出攀比、冲突、打脸
- 为了纪念某段故事 → 可能念旧、重情、理想化 → 容易长出回忆线、救赎线、执念线

### 5. 选“主动机 + 次动机”
角色可以复杂，但不要一口气全塞。

推荐保留：
- 1 个主动力：最核心的真需求
- 1 个次动力：让角色更像真人的补充欲望

**主动力负责推进剧情，次动力负责增加层次。**

### 6. 把动机转成甜点、虐点、爽点
如果已经知道角色真正想要什么，情绪就更容易打中。

#### 写甜
优先写：
- 得到想要的
- 被看见真正的需求
- 没开口就被回应
- 以为得不到，结果被稳稳接住

#### 写虐
优先写：
- 被夺走最想要的
- 眼看要得到却失之交臂
- 真需求被误解、被践踏
- 在“想要的”和“必须守住的”之间二选一

#### 写爽
优先写：
- 夺回失去的
- 证明自己配得上
- 把曾经的羞辱翻盘
- 终于不再靠讨好去换取想要的东西

**不知道角色怕失去什么，就写不准虐；不知道角色缺什么、爱什么，就写不准甜。**

### 7. 把动机写成行为链
按这个顺序展开：
`想要什么 → 为什么想要 → 会怎么去拿 → 造成什么后果 → 暴露更深的缺口`

这样角色的行为就不是孤立动作，而是会自然长剧情。

### 8. 用“给他 / 夺走 / 误解 / 延迟满足”四种方式测情节
围绕同一个需求，至少测试四种变化：
- **直接给他**：他会不会不敢信，或反而更依赖？
- **直接夺走**：他最痛的是失去东西，还是失去背后的意义？
- **被人误解**：别人把他的需求看成虚荣、作、冷漠，会怎样？
- **延迟满足**：明明差一点就得到，角色会更执着还是更退缩？

### 9. 最后检查动机是否真能解释角色
逐条检查：
- 这个动机能不能和过往挂上钩
- 它能不能解释当前这场戏里的选择
- 它和角色现有价值观是否冲突
- 如果冲突，有没有足够的触发和代价
- 换个角色，这个动机还会不会原样成立

## 输出格式
1. 表层需求
2. 功能层 / 情感层 / 身份层 / 旧事层
3. 3 个候选动机
4. 主动机 + 次动机
5. 这暴露出的性格和旧事
6. 可发展的甜点 / 虐点 / 爽点
7. 行为链
8. 最容易写偏的风险

## 快速示例
### 输入
- 角色：都市文女配
- 需求：想买一瓶很贵的香水

### 输出
- 表层需求：买香水
- 功能层：让自己在约会时更有魅力
- 情感层：想被那个男人认真看见
- 身份层：想证明自己不是廉价替代品
- 旧事层：曾被前任嘲笑“你永远上不了台面”
- 主动机：渴望被认同
- 次动机：想靠消费补体面
- 甜点：有人注意到她真正的不安，不是只夸她喷得香
- 虐点：约会前香水碎了，她想起旧日羞辱
- 爽点：后来她不再靠香水撑场面，而是靠能力和选择赢回尊严

## 注意事项
- **禁止**只停在“想要这个东西”，不追问为什么
- **禁止**把所有动机都塞给一个角色
- **禁止**甜虐只写结果，不写角色在意的点
- **禁止**把需求写成空泛大词，落不到具体场景
- 每个动机背后最好都能长出一个故事；**人物就是靠这些故事丰满起来的。**
