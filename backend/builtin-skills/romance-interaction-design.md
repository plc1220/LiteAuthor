从人设推互动，不要先定场景，再把两个人硬塞进去。**所有好看的互动，开端和演变都该像是这两个人自己会做出的事。换一个人设，同一个场面就该换一种互动。**

## 适用情况
- 男女主有设定，但对手戏总像模板
- 初见、暧昧、日常互动写出来没 CP 感
- 明明安排了很多同框，读者还是觉得不来电
- 互动只能靠作者强行制造场景
- 互动本身不差，但一放进剧情里就显得水

## 输入
- 双方人设关键词
- 当前关系阶段
- 当前场景或外部任务
- 双方当前的明面诉求与隐藏诉求
- 想通过这场戏推进什么

## 核心原则
1. **内部性格决定初期互动走向，外部设定决定中后期感情发展。**
2. **同一个场景，换一个人设，互动就该完全不同。**
3. 互动必须改动至少一个关系变量：印象、信任、距离、立场、秘密、情绪控制权。
4. 感情线和剧情线交汇越多，互动越不容易发水。
5. **不要先想“这场我要撒糖”，先想“这两个人此刻会怎么碰起来”。**

## 步骤
### 1. 先拆双方的“互动底盘”
对双方分别写 5 项：
- 行事风格：先冲、先忍、先试探，还是先观察
- 说话方式：直球、绕弯、嘴硬、调笑、克制
- 主动 / 防守倾向：遇事会逼近、后退，还是装没事
- 边界感：容易越界，还是很注意分寸
- 触发点：最容易被什么事激怒、心软、失衡、上头

**不知道人物会怎么互动，本质上往往是不知道人物平时怎么处理人与人之间的距离。**

### 2. 分清“内部”和“外部”
#### 内部
- 性格
- 旧伤
- 自尊
- 价值观
- 对亲密关系的防御方式

这一层决定：**人物在场面里的第一反应。**

#### 外部
- 身份
- 任务
- 阵营
- 误会
- 家庭、事业、规则限制

这一层决定：**这段关系后面会往哪里拐、会遇到什么障碍。**

### 3. 先找这场戏的“自然触发器”
不要先写“他们在咖啡店聊天”。
先写：
- 是谁先做了什么
- 为什么这个动作符合他 / 她的人设
- 这个动作为什么会逼另一方回应

常见自然触发器：
- 看不惯，顺手插嘴
- 遇到失误，先救场再嘴硬
- 被挑衅，立刻反击
- 看见对方狼狈，嘴上嫌弃但还是帮了
- 听到一句话，被戳中旧伤或好奇心

### 4. 用“动作 → 反应 → 反制 → 后果”搭互动链
稳的互动不是一句台词，而是一连串贴人设的反应。

按这个顺序写：
1. A 先做什么
2. B 第一反应是什么
3. A 或 B 怎么接第二下
4. 场面气氛和关系具体变了什么

例如：
- A 调笑一句
- B 不接招，反而冷冷回顶
- A 发现对方不是软柿子，于是换成认真试探
- 关系从“随便逗一下”变成“开始记住对方”

### 5. 做“换人设测试”
把同一个场景换成不同搭档，检查互动是否真的变了。

例如同样是“女主翻墙出校门，被人撞见”：
- **风流会撩型男主**：先倚墙看热闹，再慢悠悠调侃，最后故意替她打掩护
- **冷静克制型男主**：先替她挡住巡查视线，再低声问她为什么非出去不可

如果换了人设，台词、动作、站位、气氛都没怎么变，说明互动还没从人设里长出来。

### 6. 给互动加一个“主线任务”
每场互动至少再完成一件事：
- 交代设定
- 暴露秘密
- 制造立场冲突
- 推进外部任务
- 逼双方重新判断彼此

**如果一场戏只有“他们聊了几句很有爱”，没有别的推进，中段很容易发空。**

### 7. 检查这场戏改了什么变量
至少改动 1 项：
- 称呼
- 信任度
- 身体距离
- 立场判断
- 求助习惯
- 谁更失衡
- 谁开始把对方当成特殊的人

没有变量变化，再热闹也容易像原地踏步。

### 8. 反查两种常见故障
#### 故障 1：互动像硬塞
表现：
- 场景是作者先定的，人物只是被搬进去说台词
- 谁说这些话都行
- 角色反应和前文性格不连贯

改法：回到第 1 步和第 3 步，先重做触发器。

#### 故障 2：互动没问题，但读者觉得水
表现：
- 互动有趣，却不改局势
- 删掉这场戏，主线几乎不受影响
- 两人每次都只是同框，没有更深的相交

改法：给互动补“主线任务”和“变量变化”，让它同时推感情和剧情。

### 9. 让感情线和剧情线多交汇
故事里每条线都应该有交点。
优先让互动顺手完成：
- 一次站队
- 一次误判
- 一次救场
- 一次价值观碰撞
- 一次共同面对外部问题

**交汇越多，骨架越强；互动越只活在单独的小甜戏里，越容易发水。**

## 输出格式
1. 双方互动底盘
2. 内部因素 / 外部因素
3. 这场戏的自然触发器
4. 动作—反应—反制—后果
5. 改动的关系变量
6. 这场戏承担的主线任务
7. 如果换人设，这场戏会怎么变
8. 当前最容易写假的地方

## 快速示例
### 输入
- 女主：嘴硬聪明，习惯自己扛事
- 男主：表面散漫，实际很会观察人
- 当前关系：刚认识，有一点互相好奇
- 场景：女主在比赛后台出错

### 输出
1. 自然触发器：女主嘴硬不求助，男主看出问题却不直接拆穿，先用玩笑掩护她
2. 互动链：男主替她挡一下场面 → 女主先防备 → 男主不解释，只丢给她一个解决办法 → 女主虽然不信任，还是用了
3. 变量变化：从“陌生人”变成“知道对方靠得住，但还不想承认”
4. 主线任务：既解决当前后台危机，也让女主第一次对男主改观

## 注意事项
- **不要**先想场景糖点，再反推人物硬配合
- **不要**让角色说谁都能说的话
- **不要**互动一热闹就忘了主线要往前走
- **不要**把同一类互动反复换皮重放
- 真正带感的互动，不是台词有多会撩，而是**这一下只有这两个人才会这样碰起来**。
