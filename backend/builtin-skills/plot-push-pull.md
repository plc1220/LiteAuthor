用“先给预期—再短暂打破—最后更有力拉回”的方式制造波折和抓人感。**平淡的本质不是没有事，而是箭头方向一直不变。** 推拉不是乱反套路，而是在不伤核心卖点的前提下，让剧情短暂偏开，再拉回主线，放大读者情绪。

## 适用情况
- 情节一条直线往前走，顺但没劲
- 想写波折，一写就生硬、刻意、为了反而反
- 感情戏太顺，甜得没层次
- 爽点套路齐全，读者情绪却起不来
- 章节钩子只会靠“下一章更精彩”

## 核心原则
1. **先有稳定预期，才有推。**
2. 推是“短暂改向”，不是换主线，不是背刺文案承诺。
3. 拉必须比推更值钱：更甜、更爽、更准、更接近主线回报。
4. 推拉要控制在读者可接受范围内：**不伤人设逻辑，不伤核心关系，不为波折而波折。**
5. 每次推拉至少改动一项：关系、局势、认知、期待。

## 步骤
### 1. 写出当前读者的默认预期
用一句话写：
`如果不做推拉，这段大概率会写成____。`

例如：
- 两人和好后继续甜下去
- 主角按计划顺利打脸
- 这场谈判会正常谈成

没有默认预期，推拉只会显得乱。

### 2. 选一个“推点”
常用推法有 4 类：

1. **临门一脚出岔**
   - 快要兑现时，突然冒出一个新阻力
   - 适合爽点、告白、和解、翻盘

2. **短暂反向动作**
   - 角色做出一个和表面预期相反的选择
   - 但这个选择必须符合更深的人设或局势

3. **先给甜 / 顺，再立刻加小刺**
   - 先让读者放松，再轻轻打破和谐
   - 适合感情戏、日常戏、轻剧情

4. **局势暂时背离**
   - 大方向没变，但局部结果先往反方向走
   - 适合打脸、竞争、调查、事业线

### 3. 给“推”补底层理由
每个推点都要回答：
- 角色为什么会这么做
- 外部什么东西逼出了这一下
- 这一下和前文铺垫能不能对上

答不上来，说明这不是推拉，是硬拐弯。

### 4. 提前设计“拉回点”
推之前就先想好怎么拉回来。常用拉法：

- **情节拉回**：主角换招、拿出底牌、事情更接近真正目标
- **情绪拉回**：给一口更值钱的甜、护、认同、安抚
- **认知拉回**：让读者发现前面的反向动作另有原因，人物反而更立体
- **关系拉回**：看似后退，实际让两人更清楚彼此的底线或在意

没有拉回点，推得再热闹也只会耗读者耐心。

### 5. 控制推拉幅度
用这组问题检查：
- 这一下会不会伤到主角 / CP 的核心魅力
- 读者能不能接受它是“暂时的”
- 这口气能不能在 1—3 场戏或一个小阶段内回收
- 拉回来以后，回报值够不够高

稳的写法通常是：**小推快拉，大推早铺垫，重推给重回报。**

### 6. 按四拍排节奏
把一轮推拉写成：
1. **稳**：先给一个可预期的方向
2. **推**：让箭头短暂改向
3. **悬**：让读者担心 / 好奇 / 期待怎么回
4. **拉**：回到主线并给更大的情绪收益

写完后再补一句：
- 这轮推拉之后，什么东西比原来更强了

### 7. 章尾留“下一轮更高预期”
好的推拉不是一轮做完就彻底平掉，而是：
- 这一轮拉回了
- 但读者已经知道后面还能更大、更狠、更甜

这样剧情才会一直有抓力。

## 快速自查
- 删掉“推”以后，这段是不是更顺，但也更无聊
- 删掉“拉”以后，这段是不是只剩作妖
- 这一下是不是只为了反套路，而不是为了更好吃
- 这轮推拉有没有真的改动关系、局势或期待

## 输出格式
1. 当前默认预期
2. 选用的推点
3. 推点的底层理由
4. 拉回点
5. 推拉幅度评估
6. 稳 / 推 / 悬 / 拉 四拍
7. 这轮之后被放大的东西
8. 章尾下一轮期待

## 快速示例
### 示例 1：感情戏
- 默认预期：两人刚缓和，接下来会继续发糖
- 推点：其中一人因为旧习惯说错一句话，气氛立刻降下去
- 底层理由：这不是硬误会，而是人物长期防御方式的自然外露
- 拉回点：关键时刻 TA 转头站到对方这边，替对方扛下一波压力
- 效果：比单纯发糖更能证明在意

### 示例 2：爽点戏
- 默认预期：主角会拿出证据公开打脸
- 推点：第一层证据先被压掉，反派短暂占上风
- 拉回点：主角改用第二套规则和更硬的证据反杀
- 效果：爽点更高，不是平铺直叙地把底牌递出去

## 注意事项
- **不要**为了反套路而反套路
- **不要**每次都靠大误会或强行虐一下
- **不要**推得太久不拉
- **不要**让推拉损坏核心人设和文案承诺
- 真正好用的推拉，是**让读者觉得“差点偏了”，但最后吃到的那口比直着写更香。**
