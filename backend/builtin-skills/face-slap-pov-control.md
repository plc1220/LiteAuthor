把打脸、洗冤和当众反击场景写成“压力连续推进 → 主角出手 → 回响放大”，不要一上来把所有人的想法和计划解释完。

## 适用情况
- 证明清白、反陷害、公开反杀、当众翻盘
- 打脸段落很长却不爽
- 读者比主角先知道太多，主角显得磨叽或不聪明
- 场上角色很多，反派、路人、帮手都在说话，主角反而发虚
- 最爽的地方总是一笔带过，前面无关解释却写很长

## 输入
- 当前场景一句话
- 主角要打回去的点
- 反派 / 配角名单
- 主角掌握的线索
- 读者现在必须知道 / 暂时不能知道的信息
- 当前版本最拖沓或最平的段落（可选）

## 核心原则
1. **本场唯一主跟随对象默认是主角。**
2. **冤枉、嘲笑、逼问、愤怒最紧的那一段必须不断开。** 不要拿大段解释、群像心理、路人闲聊去截断情绪。
3. **反派的完整动机能不直写就不直写。** 先让反派做事，再让主角抓破绽。
4. **解释尽量后置。** 很多解释不该写在前面，而该放到主角反杀后，由主角亲口说出来。
5. **最爽的部分要拉长。** 主角出手、反派失衡、围观改口、高位者偏向，这些比前面的憋屈更值得给字数。
6. **围观者和帮手是放大器，不是主菜。** 他们只负责见证、补刀、改口，不负责替主角抢戏。
7. 一场打脸至少要改掉一项：**评价、站队、资源、关系或地位**。

## 步骤

### 1. 先定本场最值钱的一下
填一句：
`这场戏最值钱的不是____，而是____。`

例如：
- 不是反派有多坏，而是主角当众夺回评价权
- 不是计划多复杂，而是反派以为稳赢却翻车
- 不是大家都在骂主角，而是主角怎么把局面反掀回来

### 2. 给在场角色分功能，不分长篇心路
按功能表整理：
- **主角**：发现问题 / 判断 / 出手 / 补刀 / 定义真相
- **反派**：施压 / 误导 / 抢话语权 / 自乱阵脚
- **帮手**：递证据 / 见证 / 补一句关键话
- **围观者**：质疑 / 动摇 / 改口 / 传播回响
- **高位角色或男主**：在结果出来后，放大主角含金量

如果某个角色只有“我也想说几句”，删或并。

### 3. 只写主角有资格知道的东西
列三栏：
- **主角已知**
- **主角可推断**
- **作者知道但现在不要写**

第三栏常见内容：
- 反派完整计划
- 反派详细动机
- 围观者的整段内心活动
- 帮手已经猜到真相的全套分析

**第三栏先藏住。** 打脸戏里，读者可以与主角同层，或略领先主角一点，但不要远远领先。

### 4. 把整场拆成两个阶段来写

#### A. 压制期：情绪连续推进，不要断
这一段只写：
- 反派怎么发难
- 主角怎么承压
- 主角注意到了什么异常
- 场上风向怎么对主角不利

这一段**不要**塞进：
- 反派为什么这么做的长解释
- 大段路人插嘴聊天
- 主角停下来用几百字想计谋

核心要求：**憋屈、愤怒、针锋相对这段情绪要一口气推上去。**

#### B. 反杀期：先做，再让场面追着主角改口
这一段优先顺序：
1. 主角先行动
2. 场上立刻出现反应
3. 主角再解释或补刀
4. 围观者改口
5. 高位角色偏向或盖章

**不要**写成“主角先在脑内把一切想完，再慢慢出手”。

### 5. 删掉三类最伤爽感的内容
1. **反派和小配角的大段心理描写**：会把主角的聪明感吃掉
2. **与翻盘无关的路人对话**：会截断对抗情绪
3. **过长的前置解释**：会让真正的反击显得轻飘

检查句子时重点删：
- `其实女反派早就计划好了……`
- `围观群众心里都在想……`
- `主角迅速想到了 A、B、C、D 四个步骤……`

### 6. 用“动作 → 反应 → 解释”替代“先想明白再做”
更稳的写法是：
`主角先拿证据 / 做动作 → 反派和围观者先变脸 → 主角再顺势解释`

这样写的好处：
- 现场不会突然停住
- 主角显得更果断
- 解释本身也能变成爽点的一部分

例如：
- 差写法：主角先想清楚反派的全部计划，再开始证明自己
- 稳写法：主角先拿出另一个关键道具自证，等众人愣住后，再点破反派真正的算计

### 7. 把“解释真相”变成第二个爽点
主角翻盘后，不要急着收尾。把这些内容交给主角输出：
- 反派的破绽
- 反派的动机
- 反派原本打算怎么坑人
- 为什么反派现在翻车了

**让主角说，而不是作者讲。**

这样能同时拿到两层爽感：
1. 主角已经赢了
2. 主角还在继续压着对方打

### 8. 围观者和高位角色，等到情势缓下来再上场
冲突最紧的时候，路人和围观者的话通常是噪音。
更稳的使用时机是：
- 主角已经完成关键反击
- 反派开始失衡
- 场上风向准备翻转

这时再让围观者：
- 改口
- 震惊
- 互相确认刚才发生了什么
- 吹主角
- 衬托高位角色或男主的偏向

**围观者出现得越晚，越像加成；越早，越像打断。**

### 9. 连载章法：让每章都有明确回收
如果这一场戏足够大，可以这样切：
- 上一章压到主角开始反击的瞬间
- 下一章重点拉长反杀、补刀、改口和余波

但必须保证：
- 每章都有明确推进
- 不要一整章只负责憋屈，不给回收
- **一章最好只交一个最清楚的爽点**，不要同时硬塞三四个半吊子高光

### 10. 最后做“主角还聪明吗”检查
逐条回答：
- 不看旁白解释，主角能不能靠场内信息完成反击
- 读者是不是在跟着主角一起发现关键点
- 反派翻车是不是因为主角抓住破绽，而不是作者突然公布答案
- 删掉配角内心后，这场戏会不会反而更紧
- 最值钱的反杀部分，字数是不是明显多于无关解释

任一答“不能”，重做视角和详略分配。

## 输出格式
1. 本场最值钱的一下
2. 角色功能表
3. 主角已知 / 可推断 / 暂不揭露
4. 压制期写什么、反杀期写什么
5. 需要删掉的上帝视角句子
6. 主角的动作 → 反应 → 解释链
7. 围观回响和高位角色回响
8. 本场改掉了什么局势
9. 重写建议

## 快速示例
### 问题版
女主在剧组被诬陷偷东西。作者同时写女反派为什么陷害、助理有多心虚、男主赶来前已经猜到真相、群众各自内心怎么站队。结果读者比女主先知道一切，只剩下等她慢慢证明自己。

### 改法
- 跟随对象：女主
- 已知：自己没偷；道具位置不对；助理指控太快
- 暂不揭露：女反派完整计划、男主真实判断
- 压制期：女反派发难、助理补刀、众人怀疑，女主只观察异常，不插入长解释
- 反杀期：女主先拿出另一个道具自证 → 众人愣住 → 女主再点破助理和女反派的算计 → 围观者改口 → 男主只在结果出来后公开偏向女主，继续放大爽感
- 重点：先让主角做，再让全场追着主角改口

## 注意事项
- **禁止**同段乱切多人心理
- **禁止**为了显得复杂，把所有动机一次写穿
- **禁止**让打脸戏主要篇幅花在围观路人身上
- **禁止**把主角最爽的反杀写得比前面的解释还短
- 打脸戏最带感的时候，不是作者知道真相，而是**主角把真相当场掀出来，而且掀出来以后场面真的变了**
