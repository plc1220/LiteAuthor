用三个反问把一个平设定掰出成因、反转和惊喜点：**为什么不……呢？如果反过来会怎么样？这里发生什么会让主角或读者惊讶？** 先把思路放开，再把“不可能”扭成“成立且有戏的可能”。这个方法可以同时用在人设、剧情、桥段和开篇设计上。

## 适用情况
- 设定太模板：总裁、影后、师尊、校草，一看就像老梗
- 只有一句人设或梗概，扩不成故事
- 脑洞总停在第一层，写出来太平
- 想做反套路，但不知道从哪翻
- 一上来就把大纲写死，结果自己都不想写了

## 输入
- 一句话平版设定 / 人设 / 桥段
- 题材和想要的风味
- 已有约束：世界观、主线、CP 方向、不能改的设定
- 当前最想强化的点：冲突 / 反差 / 爽点 / 感情火花

## 步骤

### 1. 先写出“最平的版本”
不要美化，直接写老梗原型。
例如：
- 沉默寡言的人
- 害羞的明星
- 总裁男主，影后女主，破镜重圆
- 高冷师尊收了个活泼徒弟

只有先看见“平”，后面才知道该从哪掀。

### 2. 第一问：为什么不……呢？
这一步专门找**成因、限制、前史、隐藏条件**。
不要只找最常规的解释，要追问：
- 他为什么会变成这样
- 这件事为什么偏偏发生在他身上
- 有没有更有戏、更能长出后续的原因
- 这个设定背后是不是藏着另一个问题

适合扩的方向：
- 童年经历
- 身份秘密
- 职业代价
- 性格防御方式
- 某个外人不知道的现实困难

### 3. 第二问：如果反过来会怎么样？
这一步专门找**反差、反套路、位置倒置**。
优先反这几类：
- 强弱反过来
- 表里反过来
- 主动被动反过来
- 身份高低反过来
- 目的和结果反过来
- 第一印象和真实状态反过来

常见写法：
- 看起来高位的人，其实已经快撑不住
- 看起来很会说的人，后来反而最沉默
- 看起来在掌控局面的人，其实才是被逼着演的人

### 4. 第三问：这里发生什么会让主角或读者惊讶？
这一步专门加**事件钩子**。
不是单纯搞怪，而是加一个会改写关系或局势的点：
- 一个突如其来的真相
- 一个不合常规但合人物的选择
- 一次把人看穿的相遇
- 一场打破原有定位的翻车
- 一个会让关系立刻有火花的场面

要求：
- 惊讶点必须能继续长剧情
- 惊讶点最好一出现就改写双方关系
- 惊讶点后面要能补出原因，不要纯猎奇

### 5. 每一问先放量，再筛
对每一问至少列 3 个方向，再挑最有戏的。
筛选标准只看 4 条：
- 能不能长出新的人设层次
- 能不能制造持续冲突
- 能不能带来具体场面
- 能不能和题材爽点 / 甜点接上

**只新奇、不长剧情的点，直接淘汰。**

### 6. 选中一个方向后，立刻往下接成故事
按这条线补：
`过去发生了什么 → 现在处在什么状态 → 当下有什么麻烦 → 他会吸引或惹怒谁 → 下一层问题是什么`

不要只停在“这个设定很妙”，要马上问：
- 这种人现在最容易遇到什么问题
- 这种人最不该遇见谁
- 谁最能激发他的另一面
- 这个设定第一场戏该怎么亮相

### 7. 从“人设点”长出“剧情点”
如果起点是一个人物标签，继续追问：
- 这种人怎么坐上现在的位置
- 这种人最容易在哪件事上翻车
- 这种人面对爱情、利益、羞辱时最特别的反应是什么
- 这种人会制造什么麻烦，而不是只会被剧情砸中

**好脑洞不是设定摆在那里，而是设定一动就有戏。**

### 8. 惊喜要成立，不要只求怪
每个被选中的脑洞都过三遍：
- 前面能不能补出成因
- 后面能不能接出后果
- 换个场景，这个人还是不是他自己

能圆上的是脑洞，圆不上的是噱头。

### 9. 大纲没定死前，先留弹性
前期先把“最好玩的那根线”找出来，不要急着把所有细节写死。
尤其别一上来就逼自己：
- 先用 30 字概括全书
- 先把每个角色全生命周期写完
- 先把所有支线顺序排死

先从一个有戏的点扩，等核心矛盾和人物火花出来，再回头补完整大纲。

## 输出格式
1. 原始平版设定
2. 第一问候选
3. 第二问候选
4. 第三问候选
5. 选中方向及理由
6. 过去 / 现在 / 当前麻烦
7. 关系火花
8. 可直接写的 3 个场面
9. 后续还能怎么升级

## 快速示例
### 输入
- 破镜重圆：男主总裁，女主影后

### 输出
- 原始平版设定：霸总前任 + 影后前任
- 第一问：一个总裁身上发生什么更有戏？  
  - 他其实已经破产，只是表面光鲜  
  - 他不是冷静，是长期高压下学会装稳
- 第二问：如果反过来会怎么样？  
  - 众人面前他沉稳强势，一个人时却抓狂算账  
  - 看起来像高位掌控方，实际最怕被人发现自己已经撑不住
- 第三问：什么会让读者惊讶？  
  - 女主是第一个撞见他狼狈一面的人  
  - 她本以为他过得很好，结果发现他一直在硬撑
- 关系火花：前任重逢不再只是旧情，而是“看穿体面”和“被迫暴露真实”
- 可写场面：酒会强撑、独处破防、被女主抓到最狼狈的一次

## 注意事项
- **禁止**只为了反套路而反套路
- **禁止**脑洞只停留在“设定好怪”，不去接剧情
- **禁止**每个情节都硬反常规，结果人物和世界一起失真
- **禁止**一边想脑洞，一边过早把所有细节定死
- 好用的脑洞不是最怪的，而是**最能长出故事的**
