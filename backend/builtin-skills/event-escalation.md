把“提一下就解决”的桥段，改成会失控、会逼人做选择、还能留下余波的戏。**先拿掉最顺手的解决办法，再补一种真正会让局势变糟的阻力。**

## 适用情况
- 情节平推，读者感觉“这也算搞事？”
- 作者已经在制造事件，但刺激度不够
- 装逼 / 打脸桥段很长，却不爽
- 角色说几句话，矛盾就莫名其妙结束

## 输入
- 当前平版桥段
- 这场戏必须保留的结果
- 在场人物与当前关系
- 这场戏想给的情绪：紧张 / 爽 / 尴尬 / 失控

## 步骤
### 1. 先写出“最平版”
用一句话写清：
`谁遇到了什么事，最后怎么解决。`

先把无聊版本摆出来，才能知道该动哪一刀。

### 2. 找出最无聊的顺利解法
常见表现：
- 东西一递就结束
- 误会一句话说开
- 反派听一句劝就退
- 主角随口一说，别人就信
- 风险被立刻消掉，没有代价

### 3. 先拿掉顺利解法
优先从下面选 1 项：
- 东西不见了
- 关键人没到场
- 信息晚了一步
- 第三方撞见了
- 事情被误解了
- 主角暂时没资格、没证据、没筹码

**不要一口气全加。只加最能让这场戏长出行动的一项。**

### 4. 补一个“现在不处理就更糟”的后果
后果常见来源：
- 时间：再晚一步就来不及
- 面子：会当众丢脸
- 利益：会失去机会、资源、信任
- 危险：有人会受伤、失控、被抓住把柄
- 关系：误会会坐实，之后更难挽回

### 5. 让人物必须做选择，不要只做说明
好搞事不是“作者增加说明”，而是“角色被迫选边、冒险、硬顶、隐瞒或反击”。

连续问 3 句：
- 现在最急的人是谁？
- 他必须立刻做什么？
- 他这么做会不会让局面更糟半步？

### 6. 如果是装逼 / 打脸，再补 3 个条件
1. **主角凭什么能赢**：武力、地位、证据、规则、见证者，至少占一项  
2. **对手凭什么不会一句话认输**：对手也要有利益、面子或误判  
3. **谁看见了这次反击**：没有见证，爽感会打折

### 7. 结果不要只回到原点
结尾至少回收 1 项：
- 关系变了
- 局势变了
- 新误会埋下了
- 主角被更多人看见
- 下个麻烦被引出来

## 输出格式
1. 平版桥段
2. 最无聊的顺利解法
3. 升级后的核心阻力
4. 立即后果
5. 角色被迫做出的选择
6. 升级版情节链
7. 余波

## 快速示例
### 输入
女主捡到男主的救命药，见面后归还。

### 输出
- 平版：女主把药还给男主，结束
- 顺利解法：东西一递就没事
- 升级阻力：女主准备归还时，药先丢了
- 立即后果：男主现场发病，周围人误会女主害他
- 角色选择：女主必须一边救场，一边顶住误会去找药
- 余波：男主记住女主；女主也被卷进更大的麻烦

## 注意事项
- **不要**为了搞事硬加无关事故
- **不要**把“字数变长”误当成“刺激度变高”
- **不要**让反派为了送脸突然降智
- 如果这场戏本来就是安静过渡，不必强行升级
