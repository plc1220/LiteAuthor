先测核心梗，再钉主矛盾，再用“六段骨架 + 串珠事件”把故事撑开。**剧情流不是想到哪写到哪，也不是先堆很多设定再等剧情自己长出来；更稳的做法，是先让一个足够值钱的矛盾成立，再让合适的人设和小事件不断把它推到更大。**

## 适用情况
- 只有一个设定或一个名场面，撑不起整本书
- 知道开头和结尾，中间全空
- 感情流写久了，想学怎么先用矛盾带剧情
- 有很多桥段，但主线不清
- 中段情节发散，副本和支线越开越多

## 输入
- 一句话核心梗/基础点
- 题材与卖点
- 主角、对手或关键关系
- 预计篇幅
- 想写到的结局状态

## 步骤
### 1. 先做“一句话梗 + 标题测试”
用一句话总结：
`这篇文是关于____在____矛盾里，最后____的故事。`
再把它改写成一个足够抓人的标题/卖点句，问自己：
- 如果这是一个帖子标题或短视频标题，会不会点开
- 核心快感是苏、爽、甜、悬念，还是别的
- 不解释半天，能不能让人马上看出有戏

如果一句话里只有题材名词，没有矛盾和结果，先别往下做。

### 2. 先定“剧情真正靠什么矛盾往前跑”
剧情流更常见的起手不是“先有两个人”，而是“先有一组会自然升级的矛盾”。
优先写清：
- 主角想要什么
- 谁或什么在拦
- 这个阻力为什么会持续存在
- 这组矛盾凭什么能不断升级

常见矛盾来源：
- 竞争同一个位置/机会
- 两个阵营天然对撞
- 目标一致但方法冲突
- 世界或规则本身在压人
- 关系想靠近，但客观矛盾不允许顺利靠近

### 3. 主线用 3 个部分钉死
把主线写成：
- **人物/关系**：谁和谁
- **核心矛盾/动作**：在争什么、较什么劲、解决什么事
- **结果/变化**：最后变成什么

例如：
- 两个互看不顺眼的学霸，在暗中较劲的过程中爱上对方
- 女主进入娱乐圈，在资源争夺和舆论围剿里一路翻盘
- 少年拜入宗门，在升级与阵营冲突里争第一

### 4. 只补“会影响剧情选择”的人物信息
人设不用一上来写成百科，但至少补这几项：
- 外形记忆点
- 人物小传：家庭、性格、擅长、讨厌什么
- 生命中的重大事件
- 关键关系网
- 结局或最终想走到的状态

**重点不是资料全，而是这些信息会不会影响TA怎么选、怎么输、怎么赢。**

### 5. 先分清感情流和剧情流的发动方式
- **感情流常见起手**：先有人，再看人物如何推动关系
- **剧情流常见起手**：先有矛盾，再选最能把矛盾推到最大的背景和人设

如果想把故事写得更有剧情感，优先问：
- 什么客观矛盾天然存在
- 什么样的人设放进去，会把这个矛盾推得最狠
- 主角和对手各自为什么不会轻易退

### 6. 按六段搭主线骨架
依次写：
1. **起因**：主角被拉进核心问题
2. **过程**：主角开始推进，局势扩大
3. **低谷**：主角遭遇关键打击
4. **转折**：新信息/新选择/新同盟让主角重新站稳
5. **高潮**：主角正面解决核心矛盾
6. **结局**：局势回收，关系和主题落地

每一段都写清 4 件事：
- 阶段目标
- 关键事件
- 因果关系
- 阶段结果

### 7. 中段不会填，就用“串珠法”
把全文最重要的那根线先写出来，例如：
- 两个人互看不顺眼
- 女主一路翻盘
- 主角查案逼近真相
- 反派和主角不断升级对撞

然后给这根线串上 5—12 颗“珠子事件”。每颗珠子都必须服务同一根线。每颗珠子至少写：
- 发生了什么小事件
- 谁先出招
- 另一方会怎么反应
- 矛盾比上一颗珠子更深在哪里

示例：
- 珠子 1：转学生一来就夺走第一名
- 珠子 2：篮球赛再次压过对方
- 珠子 3：生活细节里互相刺激、互相较劲
- 珠子 4：关键考试出意外，矛盾升级
- 珠子 5：高潮处把前面积累的情绪和关系一起拉断

**珠子不是零散桥段，而是同一条矛盾线上的连续施压。**

### 8. 每颗珠子都要过“因果 + 人设”检查
逐条问：
- 这颗珠子为什么会发生
- 是谁的性格和处境把它顶出来的
- 删掉它，主线会不会少一层升级
- 换个人，这颗珠子还能不能原样成立

如果能，说明它还不够贴人、贴主线。

### 9. 高潮要像“拉断线”，不是突然拔高
前面串珠时不断累积：
- 不爽
- 误会
- 胜负欲
- 仇恨
- 心动
- 利益冲突

到高潮时，一次事件把这些累积一起引爆。稳的高潮通常同时回收：
- 前面积累的情绪
- 前面的伏笔/信息
- 角色关系变化
- 主角的关键选择

### 10. 支线只留三类
支线存在的理由，只优先保留：
- 推主线
- 丰满关键人物或关系
- 埋伏笔/回收前文

答不上来，就删、并线或后置。

### 11. 只有“结果很爽”，没有“怎么走到那”，就反向拆招
把对手或局势的压制手段逐条列出来，再写主角如何一层层拆：
- 对方用什么卡主角
- 真正打击到主角什么
- 主角靠什么信息、资源、能力或人脉反拆
- 拆完后局势怎么变

### 12. 不要贪多设定，先确定篇幅
新手最容易的问题不是没设定，而是设定过多却兜不住。提前写清：
- 这个阶段预计多少篇幅
- 这段戏在总框架里有多重要
- 哪些设定是必须的
- 哪些只是好看但暂时用不上

**篇幅和详略不明确，中段最容易散。**

## 输出格式
1. 一句话梗/标题测试
2. 核心矛盾
3. 主线三段式
4. 关键人物最少信息表
5. 六段主线表
6. 串珠事件链
7. 每颗珠子的因果与人设说明
8. 高潮“拉断线”设计
9. 必留支线/可删内容
10. 设定与篇幅控制

## 快速示例
### 输入
- 校园感情文，想写两个学霸从互看不顺眼到相爱

### 输出
- 一句话梗：两个互看不顺眼的学霸，在不断较劲和被迫靠近中爱上对方
- 核心矛盾：竞争第一名 + 性格都傲 + 客观场景不断逼近
- 串珠事件：
  1. 转学来抢第一
  2. 球场再较劲
  3. 日常里不断互相刺激
  4. 关键考试出意外
  5. 一方在高潮处做出超出预期的让步/牺牲
- 高潮回收：竞争线、情绪线、心动线一起引爆

## 注意事项
- **禁止**只有设定名词，没有核心矛盾
- **禁止**支线越开越多，主线失踪
- **禁止**珠子事件互不相干，只是热闹
- **禁止**高潮靠突然天降，而不是前面积累
- 剧情流最稳的写法，不是“事件很多”，而是**同一条矛盾线被不断推高，最后在高潮处一起炸开。**
