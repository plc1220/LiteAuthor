先把故事拆开，再谈技巧。**人物和剧情是故事本体；冲突、目标、动力是要素；反转、倒叙、卡章、视角是工具；钩子、黄金三章、节奏、代入感是读者感受。不要倒过来写。**

## 适用情况
- 记了很多技巧名词，一落笔还是乱
- 为了钩子、反转、卡章硬掰剧情
- 开篇像在完成“黄金三章作业”
- 觉得哪哪都该加技巧，却越来越不像故事
- 明明学了很多方法，写出来还是不顺

## 五层拆分
### 1. 故事本体
- 人物：谁想要什么
- 剧情：眼下发生什么，局势怎么变

### 2. 要素
- 目标
- 冲突
- 阻力
- 动力

### 3. 工具
- 反转
- 倒叙 / 插叙
- 视角安排
- 呼应
- 卡章
- 伏笔

### 4. 感受
- 钩子
- 黄金三章
- 悬念
- 节奏
- 代入感
- 爽感 / 心疼感

## 核心原则
1. **先把人物和情节写对，再考虑用什么工具写得更刺激。**
2. 工具只能放大效果，不能代替故事本身。
3. **感受是结果，不是硬套的公式。**
4. 同一个情节可以有多种写法，先选最服务故事的一种，不选最像技巧展示的一种。

## 输入
- 一句话梗概
- 当前卡住的场景 / 大纲 / 开篇
- 作者最想要的读者感受
- 已经想加的技巧：反转、双视角、倒叙、卡章等

## 步骤
### 1. 先写故事本体
用一句话填空：
`____因为____，必须____，否则____。`

如果这句写不顺，先别急着加技巧。

### 2. 分开列“人物核心”和“情节核心”
#### 人物核心
- 谁
- 他 / 她最想要什么
- 最怕失去什么
- 为什么偏偏会这么选

#### 情节核心
- 当前发生了什么
- 谁在阻拦
- 局势接下来会怎么变

### 3. 再列要素，不要直接列技巧
先补：
- 目标是什么
- 冲突点在哪里
- 阻力来自谁 / 什么
- 动力为什么成立

很多所谓“不会写钩子”，其实是目标和冲突都还没立住。

### 4. 先定想要的感受
用一句话写：
`这一段最想让读者感到____。`

例如：好奇、心疼、爽、压迫、失控、浪漫、反差。

### 5. 再从工具箱里选 1—2 个最合适的
对每个候选工具都回答：
- 它服务的是哪个人物或情节
- 不用它，这场戏还能成立吗
- 用了它，是更准了，还是只是更花
- 它会不会扭曲人物逻辑

### 6. 检查有没有本末倒置
#### 好写法
- 因为前 3 章人物立住、冲突起来、信息合理，所以读起来像“黄金三章”
- 因为这个情节本身适合用反转表现，所以反转更刺激

#### 差写法
- 为了达成“黄金三章”，硬塞三连钩子
- 先定“这里必须惊天反转”，再反推一个生硬情节

### 7. 给同一段至少做两版
- **直写版**：不额外上技巧，只把故事写清
- **加强版**：只加 1—2 个最合适的工具

比较：
- 哪版人物更顺
- 哪版情节更准
- 哪版读者感受更强

### 8. 删掉和故事无关的“技巧欲”
如果某个技巧只带来“像在写技巧”，不带来更强的人物 / 剧情效果，就删。

## 示例
### 示例 1：黄金三章
差：为了黄金三章，第一章硬塞打脸、反转、系统说明、反派亮相、悬念五连发。

好：先让前三章完成这四件事——人物立住、当前冲突起来、必要信息交代清楚、下一步值得看。做到了，读者自然会说“这开头很抓人”。

### 示例 2：反转
差：先想“我要写一个惊天反转”，再去硬找理由。

好：先确认情节本身成立，再判断用反转写，是不是比顺写更能放大刺激和落差。

## 输出格式
1. 故事本体一句话
2. 人物核心
3. 情节核心
4. 目标 / 冲突 / 阻力 / 动力
5. 想要的读者感受
6. 候选工具与服务对象
7. 直写版 vs 加强版比较
8. 应删掉的公式化部分

## 注意事项
- **不要**把技巧清单当成大纲
- **不要**为了套公式，反过来扭曲人物和情节
- **不要**把“感受名词”误当成“写法本身”
- 真正稳的写法，不是技巧最多，而是**故事先站住，技巧再放大**
