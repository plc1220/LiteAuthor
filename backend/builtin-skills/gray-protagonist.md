围绕“主角说了读者想说却不敢说的话”设计角色立场。重点不是把主角洗成好人，而是让他的难听话、黑色幽默和不体面选择，都能在剧情里被另一立场、真相反转和人性证据支撑起来。**读者可以不完全认同，但必须觉得：这人说得很狠，却不是胡说。**

## 适用情况
- 想写毒舌主角、反英雄主角、灰度主角
- 主角有黑点、有争议观点，容易招骂
- 想让角色“坏得有道理”，而不是单纯三观歪
- 想写价值观碰撞、人性复杂度、结尾反转

## 输入
- 主角的争议观点或不体面做法
- 主角最想刺穿的社会假象 / 人性假象
- 主流立场或对照角色
- 可用于反转的真相、被遮蔽的受害者、隐藏利益链
- 希望读者最终态度：认同 / 理解 / 又怕又服 / 不认同但觉得他说得对一半

## 步骤
### 1. 先定主角代言哪句“难听的真话”
用一句话写清：
`主角最想说破的是：____。`

要求：
- 必须戳中读者熟悉但平时不会直说的心理
- 最好和权力、利益、同情、伪善、双标、欲望有关
- 不能只是“世界很黑暗”这种空话

示例：
- “很多人嘴上讲正义，实际只是在享受站在高处同情别人。”
- “所谓双赢，很多时候只是让更弱的人继续忍。”

### 2. 让主角先有黑点，再让黑点和洞察绑定
可用黑点：自私、刻薄、贪财、毒舌、腹黑、手黑、现实。

要求：
- 黑点必须服务他的洞察力或生存方式
- **禁止**把“没素质、乱发疯、无差别伤人”当灰度
- 主角可以不好相处，但不能低级

### 3. 前半程先让主流立场看起来更占理
先给读者一个容易站队的表象：
- 某人看起来绝对弱势
- 某件事看起来有完美双赢解
- 主角的说法显得难听、冷血、过分

这样后面的反证才有力度。**不要一上来就宣布主角全对。**

### 4. 挖出“另一立场”或“被遮蔽的真相”
让剧情自己替主角说话，而不是主角长篇演讲。

优先补：
- 另一方的利益和委屈
- 表面和谐背后的利用、操控、绑架
- 受害者也可能在伤害别人
- 所谓善意背后其实有居高临下的傲慢

**主角的狠话必须有剧情证据，不然只是作者借角色说教。**

### 5. 用结尾反转证明：他不是全对，但绝不是胡说
推荐结构：
`表面正义 / 和谐 → 主角说出难听话 → 读者反感 → 真相翻面 → 读者重新判断`

反转要满足两点：
- 前面埋过线索
- 反转后能解释主角为什么早就那样看人

### 6. 配一个“看似正确”的对照角色
常用对照：
- 追求双赢的理想主义者
- 讲程序正义的正派角色
- 天真、热血、相信人性本善的人

作用：
- 给主角难度
- 给读者提供主流视角
- 让价值观碰撞不是自说自话

注意：**对照角色不能写成傻白甜工具人。** 他必须真有吸引力、真有道理，主角的胜出才有分量。

### 7. 允许多套三观并存，不要替作者盖棺定论
写法重点：
- 可以让主角更有压场感，但别把其他立场全写成笑话
- 可以证明主角看得更深，但别把他写成永恒真理
- 作者要包容角色，不要急着给“正确答案”

这样读者会觉得故事在讨论人，而不是上价值课。

### 8. 给主角留一条“人味”或狭窄底线
至少留 1 项：
- 对某类弱者仍有同情
- 只伤该伤的人，不滥伤无辜
- 嘴毒，但真到关键处会兜底
- 可以利用规则，但不以凌辱弱者为乐

没有人味，角色只会变成刻薄机器。

### 9. 最后检查是否站得住
逐条问：
- 删掉反转和证据，这套观点还剩什么
- 主角是不是只会输出金句，不会改变局势
- 对照角色是否真的有说服力
- 主角的黑点是不是已经低级到赶客
- 读者最终是不是“未必认同，但能理解他为什么这样说”

## 输出格式
1. 主角最难听却最真的一句话
2. 主角黑点
3. 主流立场 / 对照角色
4. 被遮蔽的真相
5. 反转节点
6. 主角的人味 / 底线
7. 读者最终站位
8. 容易翻车的点

## 快速示例
### 输入
- 想写一个毒舌律师男主，观点很不讨喜

### 输出
- 难听真话：很多人口中的正义，只是高位者的自我感动
- 黑点：自恋、毒舌、贪财、手段强硬
- 对照角色：理想主义实习律师，凡事都想双赢
- 表象：案件里弱势方看起来完全值得同情
- 被遮蔽的真相：弱势方也在利用舆论和情感绑架，把代价全压给别人
- 反转：结尾揭开完整事实，主角前面的冷话一下成立
- 人味：男主虽然嘴坏，但真正被制度压死的人，他会想办法保到底

## 注意事项
- **禁止**只让主角输出阴暗观点，剧情却从不验证
- **禁止**靠把所有配角写蠢，证明主角高明
- **禁止**把灰度写成单纯恶臭或没底线
- **禁止**让反转全靠临时加设定
- 灰度主角最带感的地方，不是他坏，而是**他把别人不肯承认的真相说破了**。
