先决定这个缺点是为了让人物更活，还是为了把剧情或关系炸起来。**好缺点不是简单“抹黑”，而是会在关键时候发作，逼角色失误、暴露软肋、改变关系。**

## 适用情况
- 主角太完美，像工具人
- 缺点写在设定表里，却进不了正文
- 不知道该给角色什么缺点，怕一写就劝退
- 想让缺点推动 CP 或剧情，而不只是贴标签
- 想写身体或处境缺陷，却怕只剩卖惨

## 核心原则
1. **缺点必须影响选择、关系或情节。**
2. 性格缺陷最稳，身体或处境缺陷也能用，但**必须带来剧情作用**。
3. **缺点像埋在故事里的炸点**：读者要能感觉到它迟早会出事。
4. 先给人物足够的魅力、能力或可怜点，再让缺点发作，更容易让读者吃进去。
5. **观感型缺点可以保留，原则型缺点不能当萌点。**涉及道德和法律底线的缺点，不能靠“可爱”糊过去。

## 输入
- 角色定位
- 这条缺点想服务什么：鲜活 / 剧情 / CP / 成长
- 当前题材和读者口味
- 角色最重要的关系与冲突

## 步骤
### 1. 先选缺点功能
#### A. 鲜活型缺点
作用：让人物有味道，有人味。

常见例子：
- 爱面子
- 嘴硬
- 小心眼
- 怂
- 轻微洁癖
- 爱吃醋

#### B. 成长型缺点
作用：会真伤人、真坏事，必须进成长线。

常见例子：
- 自负
- 逃避
- 控制欲强
- 偏见重
- 过度自私
- 不愿信人

#### C. 连接型缺点
作用：把两个人或一条剧情线绑定起来。

常见例子：
- 回避型人格
- 过度骄傲导致反复错过
- 某种身体后遗症、残疾、创伤反应
- 情绪障碍、社交障碍、极端防御心

**连接型缺点不是为了惨，而是为了制造“不得不面对”的关系变化。**

### 2. 判断伤害级别
先问三个问题：
- 这个缺点会不会持续伤到别人
- 会不会妨碍角色完成主要目标
- 会不会让读者失去站队理由

如果多项答案是“会”，就按成长型处理，不能只当作小可爱。

### 3. 补成因，但不要拿成因洗白
写清：
- 这个缺点从哪来
- 它曾经如何保护过角色
- 现在为什么开始反噬

例如：
- 自负曾经是能力强带来的底气，现在却让角色低估别人
- 回避曾经能自保，现在却让角色错过真正重要的人

### 4. 写出“缺点发作链”
至少安排 3 次：
1. **日常发作**：让读者认得这个缺点
2. **压力下发作**：让它坏一次事
3. **关系里发作**：让它改变 CP、友情或站队

如果三次发作都没改变任何事，这个缺点就不值钱。

### 5. 身体或处境缺陷怎么用
可以用，但必须同时满足：
- 不只是为了卖惨
- 它会改变角色行动方式
- 它会制造依赖、误会、照顾、补偿或限制
- 它和当前关系或主线有实际连接

例如：
- 失明、旧伤、腿伤，不是单纯让读者心疼，而是让角色必须重新分配信任、依赖和误解
- 角色因为后遗症离不开某人照顾，关系才被强行绑定

### 6. 先给读者一个“吃得下去的入口”
如果缺点比较刺激观感，先给人物补 1—2 个可接受点：
- 能力强
- 有原则
- 对特定人很真
- 脆弱点清楚
- 某方面特别讨喜

**先让读者对角色有好感，再让缺点发作，容忍度会高很多。**

### 7. 判断要不要改
#### 可以不改
- 只是生活性毛病
- 不伤底线
- 反而能稳定形成人物味道

#### 必须改
- 会持续伤害别人
- 明显踩道德或法律底线
- 再不改会毁掉关系或主线结果

### 8. 按“炸点”去安排后果
缺点不是摆着好看，要问：
- 它什么时候炸
- 炸完谁受伤
- 关系怎么变
- 角色会不会因此被迫面对自己

稳妥链条：
`缺点发作 → 付出代价 → 被迫看见问题 → 做出修正 → 用新行为证明`

## 输出格式
1. 缺点名称
2. 缺点功能：鲜活型 / 成长型 / 连接型
3. 成因
4. 三次发作场景
5. 对剧情或关系的影响
6. 读者吃下这个缺点的入口
7. 是否必须改
8. 后果与修正链
9. 容易写崩的点

## 快速示例
### 输入
- 角色：高智商、很自负的主角
- 需求：让缺点推动 CP

### 输出
- 缺点：自负
- 功能：成长型 + 连接型
- 成因：长期靠智商碾压别人，习惯把所有人算进自己的掌控里
- 日常发作：说话太满，轻视别人判断
- 压力下发作：误判局势，第一次翻车
- 关系里发作：把 CP 也当成自己能操控的一环，结果被对方正面打脸
- 作用：CP 成为唯一的意外，也成为主角必须调整自我认知的起点

## 注意事项
- **不要**把严重恶习包装成“可爱任性”
- **不要**只有缺点标签，没有发作场景
- **不要**身体缺陷只负责卖惨，不负责推动关系或剧情
- **不要**让人物先只剩讨厌点，后面再想补救
- 好缺点的标准不是黑，而是：**既真实，又会带出戏。**
