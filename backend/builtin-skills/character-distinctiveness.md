先做一张角色区分表。**角色辨识度不是给每个人贴一串形容词，而是让读者提到某个轮廓、某个道具、某种反应，就能立刻想到这个人。**

## 适用情况
- 角色一多就撞脸，读者分不清谁是谁
- 主配角都有设定，但读完记不住
- 很多人都在用同一类外貌、同一类属性、同一类说话方式
- 想给角色加标志道具或梗，但怕越写越乱

## 输入
- 角色名单与戏份轻重
- 每个角色当前的一句话设定
- 题材、时代、世界观
- 现有外貌、属性、道具、习惯、关系功能
- 当前最容易混淆的几组角色

## 步骤

### 1. 先定每个角色在故事里的位置
先给每个主要角色写一句：
`这个人是靠什么被记住的？`

可从这几类里选：
- 主线功能：主角、搭档、对照组、反派、情感推动者
- 气质功能：稳场、搅局、压迫、治愈、搞笑、点火
- 关系功能：谁负责拉扯主角，谁负责托底，谁负责制造威胁

**功能先分开，后面才不容易写成同一种人。**

### 2. 每个主要角色只定 1 个“主记忆点”
主记忆点只保留一个最强的，其他都是辅助。

常见主记忆点来源：
- **轮廓型**：身高体型、发型、站姿、穿衣轮廓、整体气场
- **道具型**：某个稳定出现、能代表角色的物件
- **反应型**：固定的小动作、说话节奏、处理问题的方法
- **关系型**：只对某个人会露出的特殊反应

**一人一个主点，比一人五个散点更容易记。**

### 3. 审核外貌是否撞脸
把主要角色并排检查：
- 身高体型是否太像
- 发型发色是否太像
- 第一眼气质是否太像
- 穿衣和站姿是否太像

高频撞脸项要特别警惕：
- 都是娇小型
- 都是黑长直
- 都是硬汉脸
- 都是清瘦高冷挂

如果外貌确实要重合，**必须**在其他方面强行拉开：
- 一个沉稳，一个急躁
- 一个讲礼数，一个没耐心
- 一个护短，一个利己
- 一个说得少但出手快，一个说得慢但算得深

### 4. 审核属性和处理方式是否重合
不要只查外貌，还要查“站姿”。

重点比较：
- 遇事先冲、先忍、先算、先装没事
- 在乎体面、结果、感情、利益，优先级分别是什么
- 说话是直给、绕弯、毒舌、冷淡、碎嘴，还是装轻松
- 面对强者、弱者、亲近者、讨厌的人，有没有差异

**主要角色尤其不要“同一件事下反应像一个人写的”。**

### 5. 给标志道具时，守住四条线
标志道具可以强记忆，但别乱发。

#### 道具必须满足
- **独特**：同一阶段最好不要和别的主要角色撞
- **可重复**：不是只出现一次
- **可叙事**：能承接剧情、关系或伏笔，不是纯装饰
- **有来路**：最好和职业、过去、家庭、执念有关

#### 道具高危用法
- 只是好看，没有剧情用途
- 一堆角色都有“专属道具”，读者只会疲劳
- 道具和角色关系太浅，换个人拿也成立

经验上，**同一部书里强记忆型道具角色不要太多，1—3 个最稳。**

### 6. 作者必须知道角色的过去和家庭背景
就算正文暂时不写，作者也要知道：
- 这个人过去发生过什么
- 家庭背景如何影响了他的性格
- 这个背景会不会在后面长成梗、冲突或反转

同样是傲娇、温柔、强势，**不同家庭和旧事会让他们长出完全不同的表现。**

### 7. 做一张“角色区分矩阵”
给每个主要角色至少填这 6 项：
1. 主记忆点
2. 外貌轮廓
3. 属性 / 处理方式
4. 标志道具或动作
5. 背景关键词
6. 与主角的关系功能

如果两个人在其中 3 项以上都接近，就要继续拆开。

### 8. 每个角色都安排一个“证明场面”
不要只在设定表里区分，正文里也要让读者看见。

为每个主要角色各安排 1 场：
- 最能体现主记忆点的出场
- 最能体现处理方式的冲突
- 最能体现关系功能的互动

**角色的区分度，必须靠场面兑现。**

### 9. 最后做“遮名测试”
把角色名字遮掉，再看这段戏：
- 只看动作和台词，能不能认出是谁
- 提到某个道具或习惯，能不能想到具体角色
- 两个角色互换名字，这场戏会不会几乎不受影响

如果会，说明还不够分明。

## 输出格式
1. 角色清单
2. 每个角色的一句话功能
3. 主记忆点
4. 外貌 / 属性 / 道具 / 背景 / 关系矩阵
5. 容易撞的角色组
6. 调整建议
7. 每个角色的证明场面

## 快速示例
### 输入
- 角色 A：女主，记者
- 角色 B：闺蜜，律师
- 角色 C：反派，富二代

### 输出
- A 主记忆点：旧录音笔；遇事先追真相
- B 主记忆点：极稳的西装和冷静语速；遇事先找规则漏洞
- C 主记忆点：银色打火机；表面体面，实则喜欢把人逼到失态
- 撞点：A 和 B 都是都市精英女性，容易都写成“干练漂亮”
- 调整：A 强调疲惫、急、总在赶路；B 强调整洁、稳、说话留三分

## 注意事项
- **禁止**所有人都靠“高冷、温柔、强势”区分
- **禁止**给每个角色都发一个花哨道具
- **禁止**外貌区分开了，反应却还像同一个人
- **禁止**只做设定表，不给正文证明场面
- 真正有辨识度的角色，是**外貌能分、行为能分、关系能分、提到一个点就能想起这个人**。
