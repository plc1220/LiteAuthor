先按层级和功能配反派，不要所有对手都长一个模子。**好看的反派，不是“都很坏”，而是每一层反派都在不同地方逼主角、照主角、抬主线。**

## 适用情况
- 反派全像送头炮灰
- 大反派存在感弱，中反派又太单薄
- 不知道什么时候该用小反派，什么时候该上大反派
- 反派和主角没关系，冲突像平空发生
- 想写多层次反派，但总写成同一种人

## 输入
- 主角人设、背景、核心目标
- 故事主线与阶段节奏
- 预计篇幅
- 想给读者的爽感、压迫感或拉扯感
- 关键关系与主题冲突

## 步骤
### 1. 先从主角身上反推反派
优先问：
- 主角最容易刺痛别人什么
- 主角最容易被打击什么
- 主角的身份、家世、能力、性格，会自然引来哪类敌人

**反派最好从主角的世界里长出来，而不是凭空掉下来。**

### 2. 先分层，再定人
常用四层可选，不必全用，但至少要分清功能。

#### A. 炮灰反派
特点：
- 动机简单，恶得直接
- 戏份短，主要负责起冲突、送爽点、立主角

适合：
- 开局立威
- 小阶段打脸
- 让主角能力和手段迅速被看见

#### B. 伪友 / 嫉妒型中反派
特点：
- 早期可能是朋友、同伴、熟人
- 因羡慕、嫉妒、攀比、想取代主角而变坏
- 表面维持关系，暗里使坏

适合：
- 制造背刺和信任崩塌
- 打主角低谷
- 放大“知人知面不知心”的痛感

#### C. 观念对立型大反派
特点：
- 有完整来路和弧光
- 和主角站在对立面
- 不是单纯坏，而是和主角信的东西不一样
- 可能彼此欣赏，甚至惺惺相惜

适合：
- 扛全文主线
- 扛终局冲突
- 扛主题对撞

#### D. 美强惨魅力反派
特点：
- 外形、能力、经历都很有记忆点
- 很坏，但因为魅力和惨烈过去容易让读者记住

适合：
- 制造复杂情绪
- 放大阵营拉扯
- 提供“又恨又爱看”的阅读感

**这类反派要有魅力，不等于要洗白。**

### 3. 给每层反派写 6 件事
- 反派类型
- 想要什么
- 为什么盯上主角
- 主要手段
- 对主角造成的伤害
- 最终怎么下场或被结算

### 4. 让不同层反派打不同位置
优先分工：
- 小反派打脸面和短期利益
- 中反派打信任和关系
- 大反派打立场、观念和终局
- 魅力反派打情绪复杂度和记忆点

如果所有反派都只会“挑衅 + 陷害”，很快会重复。

### 5. 反派犯多大错，就付多大代价
尤其是爽文里，检查：
- 反派要毁主角什么：名声、资源、前途、关系、性命
- 主角反击后，反派失去了什么

**如果反派下手很重，结果只伤了个小喽啰，爽感会发虚。**

### 6. 给重要反派留记忆点
哪怕不是大反派，也至少给一个能被记住的点：
- 明显性格缺陷
- 独特习惯
- 特定说话方式
- 明确软肋或执念

避免反派全是同一张“恶毒脸”。

### 7. 检查反派是不是在抬主角和主线
逐个问：
- 这个反派下场后，主线是否前进了
- 他有没有逼出主角的新选择
- 他是不是只来凑热闹

如果打完一个反派，故事几乎没变，他多半层级和功能没设计好。

## 输出格式
1. 主角最容易引出的敌意来源
2. 反派层级表
3. 每层反派的功能
4. 每个重要反派的动机 / 手段 / 伤害 / 下场
5. 反派与主角的对应关系
6. 代价结算检查
7. 容易重复的风险点

## 快速示例
- 炮灰反派：跋扈同事，负责开局给主角制造第一波难堪，随后被快速打脸
- 中反派：表面姐妹、暗里攀比的闺蜜，负责背刺与低谷
- 大反派：行业大佬，和主角信奉完全不同的规则，扛终局对撞
- 魅力反派：能力极强、经历惨烈的对手，负责制造复杂情绪和阵营拉扯

## 注意事项
- **禁止**所有反派都只会无脑挑衅
- **禁止**大反派只有地位，没有来路和立场
- **禁止**小反派戏份抢太多，挤掉主线
- **禁止**魅力反派一有悲惨过去就强洗白
- **反派分层的目的，不是凑齐类型，而是让每一层都打在不同地方**
