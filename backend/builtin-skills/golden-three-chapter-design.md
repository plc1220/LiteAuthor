把前三章写成“读者愿意继续翻页的连续反应”，不要写成设定说明书，也不要只靠空悬念硬吊着。**大设定不一定是开头重点，开头真正该顶上来的，是主角眼前不得不处理的那件事。**

## 适用情况
- 开头有复活、穿越、失忆、退婚等大设定，但读起来不抓人
- 第一章解释很多，真正有戏的部分反而靠后
- 前三章只会硬卡悬念，不知道怎么让人觉得有趣
- 有多个梗，不知道该先写哪一个
- 开头想重写，却始终抓不住问题核心

## 核心原则
1. **前三章先写主角眼前的事，不先讲设定原理。**
2. 前三章的任务是：立主角、挂冲突、露卖点、给追读理由。
3. **好前三章不一定全靠悬念。** 有趣、新颖、带感，本身就能拉住读者。
4. 如果有多个梗，先列出来，再按“读者最想先看哪个”排序。
5. 大梗可以当背景，但真正的开场重心，优先放在能立刻演成场景的那一件事上。

## 步骤
### 1. 先列一张“梗名单”
把开头能用的东西全部列出来：
- 大设定
- 关系变化
- 当前困境
- 爽点 / 苏点 / 甜点
- 悬念
- 反转

然后逐项打分：
- 能不能立刻写成场景
- 能不能直接作用于主角
- 能不能在前三章给到回报
- 读者会不会当场想看下去

**排序最高的，才是开头重心。**

### 2. 把“大设定”降为背景，把“眼前困局”抬到台前
问自己：
- 这件大设定，读者现在最在意的是“它怎么发生的”，还是“主角现在因此怎么办”？
- 如果把设定说明删掉一半，场景会不会更抓人？

多数时候，开头更该先写：
- 主角怎么应对
- 主角现在最怕什么
- 主角此刻做了什么决定

而不是先把设定讲全。

### 3. 用“被迫应对”开场
优先从主角的当下动作切入：
- 假装失忆
- 临时撒谎
- 被迫站队
- 赶紧圆谎
- 当场接招
- 先保命 / 保关系 / 保局面

这种开法比单纯交代背景更容易起戏。

### 4. 第一章先给一个具体问题
不要只写“发生了大事”，要写：
- 主角现在必须糊弄谁
- 主角现在必须躲开什么
- 主角现在必须拿到什么
- 主角现在必须先过哪一关

**越具体，越容易抓人。**

### 5. 前三章不必死卡“悬念流”
如果当前梗不适合靠悬念拖，可以改用：
- 新鲜感
- 反差感
- 主角的机灵应对
- 特别的关系火花
- 眼前的小危机

只要读者觉得“这一段有意思”，就会往下看。

### 6. 前三章节拍参考
#### 第 1 章
- 主角进场
- 当前麻烦抛出
- 主角立刻做出应对
- 留下一个更大的后续问题

#### 第 2 章
- 放大阻力
- 补最关键的前情和关系
- 让主角继续主动应对
- 再露出一个题材卖点

#### 第 3 章
- 解决一个小回合
- 给一次小回报 / 小爽点 / 小翻车
- 抛出下一轮更大的矛盾

### 7. 如果写不清，就回到“梗排序”重排
当开头卡住时，不要一直改句子，先重排重心。
重点检查：
- 是不是把看似最大的设定，当成了最该先写的内容
- 是不是把真正有吸引力的梗压后了
- 是不是主角当下的反应写得太少

## 输出格式
1. 开头可用梗清单
2. 按吸引力排序后的前三项
3. 真正的开头重心
4. 第一句场景切入口
5. 第一章具体问题
6. 第 1 / 2 / 3 章节拍
7. 第三章章尾钩子
8. 当前开头最该后置的说明内容

## 快速示例
### 输入
- 开头：女主死而复生，被迫装失忆

### 输出
- 梗清单：死而复生、装失忆、身边人试探、女主真实目的
- 开头重心：不是“她怎么复活的”，而是“她现在只能装失忆应对盘问”
- 场景切口：
  - “——你还记得我是谁吗？”
  - 她定了定神，抬眼反问：“你是谁？”
- 后续：顺着这次装失忆，把更大的矛盾带出来

## 注意事项
- **不要**前三章都在解释设定
- **不要**把最大的场面一上来就用完
- **不要**为了悬念故意写得含糊
- **不要**让主角只站着听别人说话
- 前三章最值钱的，不是信息量大，而是**主角一出场就有事要扛，读者看完知道后面还会更有戏。**
