把抽象情绪、过滤词和说明味重的句子，改成能看见的动作、神态、身体反应、空间位置和动词句，让开篇、对白、冲突场景更有画面。**抽象词可以点一下，但只能当配料，不能当主料。**

## 适用情况
- 情绪只会写“难过、开心、不可置信、痛心”
- 句子里全是“觉得、看起来、显然、有些、似乎”
- 对话场景只有皱眉、脸红、看着对方
- 开篇不知道怎么让读者迅速看清时间、地点、人物
- 动作很多，但画面还是像说明书

## 核心原则
1. **少总结，多展示。**
2. **能写成动词句，就别绕成说明句。**
3. **别只盯五官，要写全身动作、站位、道具和环境。**
4. **重要人物、重要时刻多给笔墨；杂鱼和次要场面可以一句带过。**
5. **画面感不是辞藻堆砌，而是读者能直接看见发生了什么。**

## 步骤

### 1. 先圈出高危句
优先找这几类：

#### A. 抽象情绪词
- 难过、开心、愤怒、羞愧、不可置信、不可思议、惊恐万分、悲喜交加

#### B. 过滤词
- 觉得、感到、看到、发现、看起来、显然、有些、似乎、好像、仿佛、认为、一看……的样子

#### C. 说明味重的句型
- 把 / 被 / 将 / 让
- “正在……的某人”
- “用手 / 用脚 + 动词”
- 量词 + 动词：一手端着、一脚踢开
- “然后、随即”连缀动作
- 关联词和模糊词过多：因为、所以、虽然、但是、大概、可能、也许、几乎、貌似、算是

**这些不是绝对禁词，但只要不影响理解，就优先删、并、改。**

### 2. 先把句子改回动词句
优先去掉说明味和绕弯感。

- 他把门关上 → 他关上门
- 恶狗被他一脚踢开 → 他一脚踢开恶狗
- 他将孩子带回家 → 他带孩子回家
- 爸爸让他狠狠揍了一顿 → 爸爸狠狠揍了他一顿

把定语堆叠改回动作链：

- 听到声音，正在开窗的少年回过头来  
  → 少年正开窗，听到声音，回过头来

能省的身体部位就省：

- 用手推开门 → 推开门
- 用脚踢开门 → 踢开门

量词和身体部位只在**确实需要镜头感**时保留：

- 单手举杯、单脚踢门、指尖敲桌

### 3. 把“怎么样”改成“怎么表现出来”
把情绪词落到身体和动作上。可用这条链：

`情绪 → 身体反应 / 动作 → 想说没说的话或内心一闪`

例如：
- 他不可置信地看着她  
  → 他两眼一下睁圆，嘴唇动了动，半天没发出声音。
- 他有点难过  
  → 他肩膀慢慢塌下去，手指在衣角上揉了两下，还是没把那句解释说出口。

### 4. 别只写脸，补全身体和空间
如果一段里全是：
- 皱眉
- 抿唇
- 红了脸
- 眼睫轻颤
- 看着对方

就补这些：
- 站位：逼近、后退、挡住、侧身、绕开
- 手部：攥拳、蜷指、放下杯子、抓住门框
- 身体：僵住、俯身、转身、往后倒、扑过去
- 道具：烟、门、椅子、酒杯、伞、手机
- 环境：门外有没有人、房间里安不安静、光线和天气如何

**只盯五官，画面容易打不开。**

### 5. 对话场景用“加层法”
裸台词先别急着直接扔出去，按需要补层：

1. 台词本身  
2. 声线或情绪  
3. 一个有效动作  
4. 站位 / 距离变化  
5. 环境或旁物  
6. 必要时补一笔内心一闪

示例：

- 他说：“我们分手吧。”
- 他闷声说：“我们分手吧。”
- 他压低了声音，闷声说：“我们分手吧。”
- 他左右看了一圈，确认没人，才俯身凑到她耳边，声音压得发闷：“我们分手吧。”
- 他喉结滚了滚，还是压低声音，凑到她耳边：“我们分手吧。”

**动作不是装饰。** 每个动作至少回答一个问题：
- 谁在压人，谁在退让？
- 这句话是试探、心虚、安抚还是威胁？
- 这一下有没有改变气氛和关系？

### 6. 开篇或新场景先定场，再推进人物
需要定场时，用 1—2 句先让读者知道：
- 现在是什么时候
- 人在哪
- 眼前大概是什么画面

推荐顺序：
- 远景 → 近景
- 景物 → 人物
- 时间 → 空间 → 当前动作

例如：
- 金乌西垂，晚霞压住山脊。城门下人声正闹，她就是在这时候进的城。

**网文写景重在定场和烘托，不是长篇作文。** 三言两语够了，再多就拖。

### 7. 删掉不必要的弱化词和连接词
复查时重点看：
- 大概、可能、似乎、好像、几乎、也许、貌似、算是
- 因为、所以、虽然、但是
- 然后、随即
- “了”

处理原则：
- 不影响句意就删
- 能并成更利落的动作链就并
- 打斗、追逐、正在进行的场面，少用“然后 / 随即 / 了”去打断连续感

### 8. 最后做一次“能不能拍出来”检查
逐句问：
- 这句是作者在总结，还是读者真能看见？
- 如果删掉“难过、愤怒、不可置信”，场面还能不能立住？
- 这段里有没有至少一个记得住的动作、站位或道具？
- 这段画面是不是都堆在脸上了？

## 输出格式
1. 原段落问题
2. 高危词和高危句型
3. 需要保留的关键信息
4. 展示化改写
5. 改成动词句的地方
6. 新增的身体 / 空间 / 道具层
7. 删掉的弱化词与连接词
8. 画面感增强点

## 快速示例
### 输入
- 原句：他不可置信地看着她，觉得自己大概真的要失去她了。

### 输出
- 问题：抽象情绪词、过滤词、模糊词过多
- 改写：他眼睛猛地睁圆，嗓子像被什么堵住，手指抬了一下，又僵在半空。她已经转过身去，连背影都没给他留。
- 增强点：把“不可置信”和“要失去她”的感觉落到了身体反应和场面结果上

## 注意事项
- **禁止**把所有抽象词都当成绝对禁词；它们可以点一下，但别整段都靠它们撑
- **禁止**一段里塞满动作，却没有态度和关系变化
- **禁止**只会写脸部小动作，不写身体和空间
- **禁止**为了“有画面”乱堆形容词
- 真正的画面感，不是词难，而是**读者能直接看见发生了什么**
