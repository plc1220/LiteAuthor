先把一句话拆开，再决定回哪一格。**接话不要只顺着原句回答“是 / 否”“好 / 不好”，而要针对主体、动作、对象、时间地点、程度和预设条件分别回应。** 这样同一句话就能长出暧昧、反撩、沙雕、转折、补背景和埋误会。

## 适用情况
- 写对话时总卡在“下一句怎么接”
- 暧昧戏只会写“可以 / 不可以”
- 写委托、命令、任务对白时只会“接 / 不接”
- 想制造反撩、错频、转折、吐槽
- 人设已经有了，但台词总不带感

## 输入
- 原句
- 说话双方当前关系
- 想要的氛围：暧昧 / 反撩 / 沙雕 / 克制 / 挑衅 / 试探 / 质疑 / 反转
- 双方人设与说话习惯
- 这句对话希望推进什么：升温 / 拉扯 / 逗笑 / 埋误会 / 带背景 / 推剧情

## 步骤

### 1. 先原样抄下原句
不要急着接。

例如：
`我可以亲你吗？`
`去一趟某某城拿回一样东西。`

### 2. 把原句拆成字段
优先拆这些格子：
- 主体：谁？
- 许可：可不可以 / 要不要 / 能不能
- 动作：做什么？
- 对象：对谁 / 拿什么
- 时间：现在还是以后
- 地点：这里还是别处
- 程度：一下还是很久 / 多大代价
- 方式：怎么做 / 用什么做
- **预设条件**：为什么必须这样做？它本来是谁的？真的只能这么办吗？
- 真假 / 语气：申请、通知、试探、命令、撒娇还是诈人

### 3. 每个字段各写 1—2 个回答
不要立刻挑最好的一句，先铺开。

以“我可以亲你吗”为例：
- 回主体：`我可以啊。`
- 回许可：`看你表现。`
- 回动作：`你想亲哪里？`
- 回对象：`你确定是我吗？`
- 回地点：`要在这里亲？`
- 回时间：`现在就要？`
- 回程度：`一下还是两下？`

### 4. 也拆“命令 / 委托 / 任务句”
很多任务对白也不是只能回答“好 / 不行”，因为它内部也藏了预设。

例如：
`去一趟某某城拿回一样东西。`
- 回动作：`去一趟？为什么一定得我去？`
- 回地点：`某某城？我不认路。`
- 回对象：`什么东西？长什么样？`
- 回归属：`拿回？那东西本来是你的？`
- 回方式：`让他们送过来不行吗？`
- 回限制：`现在去，还是明天去？一个人去？`

**一旦开始拆，对话就不只是在接话，还能顺手做 4 件事：**
- 补背景
- 质疑前提
- 埋谎言 / 误会
- 把剧情分出新岔路

### 5. 按氛围选路线
#### 暧昧直球
顺着动作或程度回，让气氛升温。
- `你想亲哪里？`
- `一下不够。`

#### 反撩夺权
把主导权抢回来。
- `你不行，我可以。`
- `要亲也是我先。`

#### 装傻错频
故意回错字段，制造沙雕感。
- `你不行，它可以。`
- `我妈不给亲。`

#### 设限吊胃口
不直接答应，但给条件。
- `回家再说。`
- `先把门关上。`

#### 质疑 / 反转
去回预设条件，让对白长出剧情。
- `拿回？谁说那是他的？`
- `去某某城可以，你确定你给的地图没错？`

### 6. 给台词补动作和停顿
一句好接话，最好配一个动作、停顿或表情。

- `你想亲哪里？` 她指尖轻轻划过他的唇角。
- `拿回？` 他掂了掂手里的东西，笑了一下，`你先告诉我，这玩意真是你的？`

**动作决定这句是撩、是逗、还是在拆台。**

### 7. 让接话带出人设
同样一句“你想亲哪里”，不同人要说得不一样：
- 高冷型：字少、停顿多、动作克制
- 嘴硬型：先怼一下，再偷偷放水
- 沙雕型：故意跨频，跑偏但不脱线
- 机敏型：先抓前提漏洞，再反问回去

### 8. 最后检查这句到底推进了什么
至少命中 1 项：
- 关系更近了
- 谁先露馅了
- 谁拿到主导权
- 气氛变了
- 背景被带出来了
- 误会或悬念被埋下了

如果什么都没变，这句就只是机灵话。

## 输出格式
1. 原句
2. 拆句字段表
3. 各字段候选回答
4. 选中的回复路线
5. 搭配动作后的最终对话
6. 这句推进了什么
7. 可顺手带出的背景 / 冲突 / 误会

## 快速示例
### 示例 1：暧昧
- 原句：我可以亲你吗？
- 拆句：我 / 可以 / 亲 / 你 / 吗
- 候选：
  - 回主体：我可以啊
  - 回动作：你想亲哪里
  - 回程度：一下还是两下
- 最终：
  - 攻：我可以亲你吗？
  - 受：你想亲哪里？
  - 受抬手碰了碰他的下唇，像是在好心配合，又像在故意点火。
- 推进：受反客为主，暧昧瞬间升温

### 示例 2：任务
- 原句：去一趟某某城拿回一样东西。
- 拆句：去 / 某某城 / 拿回 / 东西
- 候选：
  - 回动作：为什么一定得我去
  - 回地点：某某城？那里不是封城了吗
  - 回归属：拿回？那东西本来是你的？
- 最终：
  - NPC：去一趟某某城，替我拿回一样东西。
  - 主角：拿回？
  - 主角把任务单折了两下，抬眼看他，`先说清楚，那东西真是你的？`
- 推进：顺手带出任务疑点和后续反转

## 注意事项
- **禁止**为了俏皮而完全脱离人设
- **禁止**每次都用同一种机灵句式
- **禁止**只写台词，不补动作和场面温度
- **禁止**把错频当万能招，导致对白全员像段子手
- 好接话的重点不是“聪明”，而是**贴人设、贴关系、贴当下剧情**
