先别急着立刻打大纲。先把自己想写这个梗的真正原因挖出来，再用片段试写找出能戳中的表达方式。**表层写法只是包装，真正能撑起大纲的是底层情感需求。**

## 适用情况
- 看中了一个梗，但一做大纲就歪
- 明知道这个题材很带感，却说不清爽在哪
- 不懂某类角色或关系，越写越想把 TA 拐成自己熟悉的类型
- 一卡住就往舒适区逃，疯狂加剧情、加设定、加外部事件
- 总觉得“梗不对纲”

## 输入
- 一句话主梗
- 当前最不会写的角色类型、关系模式或题材方向
- 已经想到的 3—10 个零碎场面、台词或设想
- 当前最怕写崩的点（可选）

## 步骤

### 1. 先承认自己到底卡在哪
写清楚：
- 不会写的是哪种人物 / 关系 / 题材
- 下意识会用什么熟悉写法逃避它
- 如果继续逃避，这个梗会被自己改成什么

### 2. 连续追问“为什么想写这个梗”
至少追问 3 轮，从表层一路往下挖：
1. 这个梗表面上最带感的是什么
2. 它让作者想补偿、发泄或体验什么
3. 如果删掉外壳，真正还想看的是什么
4. 最不愿放弃的名场面、关系或情绪是什么

把答案从“想写霸总”“想写强取豪夺”“想写拿乔”这类题材词，逼到“想看被高配对象偏爱”“想看主角站在有理位置又能拿到好处”“想看求而不得的失衡感”这类体验词。

### 3. 暂时不写大纲，先写片段
直接写 5—10 个零碎片段，每个片段只抓一个瞬间：
- 一个最想看的场面
- 一句最带感的台词
- 一次关系失衡
- 一次公开见证
- 一次主角获得心理补偿的瞬间

**先不要管这些片段能不能连成完整剧情。先看什么最能戳中自己。**

### 4. 给每个片段做爽点标注
逐条标记：
- 主角得到了什么：偏爱 / 主动权 / 道德高位 / 解释权 / 安全感 / 被看见 / 被需要
- 主角避免了什么：羞耻 / 主动求爱的不适 / 自尊受损 / 低配补偿 / 被看轻
- 谁在失衡：主角 / 对手 / 围观者 / 家人 / 规则
- 这个片段为什么想继续写

### 5. 从片段里找共同模式
对比所有片段，提炼：
- 哪些表面桥段不同，但满足的是同一种情感
- 哪种体验反复出现
- 哪种场面一旦拿掉，整组片段就不带感了

最后写成一句：
`这个梗表面在写____，本质在满足____。`

### 6. 分开“本质”和“表达方式”
至少列 3 种不同表达：
- 最初想到的主流写法
- 更适合当前手感的写法
- 能保住同样爽点的替代表达

**不要把某一种外壳误当成唯一正确写法。** 只要底层需求没变，包装可以换。

### 7. 反推这个爽点成立的前提
逐条问清：
- 需要有人围观，还是只要双方拉扯
- 需要权力差、身份差、信息差，还是误会差
- 需要主角被迫、不得不、顺水推舟，还是可以主动出击
- 需要什么借口、正当性、遮羞布，才能让读者安心代入
- 没有哪个条件，这个爽点就会塌

**很多梗不是写法难，而是前提条件没搭好。**

### 8. 选一个最能反复产出场面的核心矛盾
把前提翻成可持续的主矛盾，例如：
- 撒谎
- 伪装
- 被迫接近
- 利用与反利用
- 不能公开
- 先动心与不肯承认
- 想拿好处却不敢付代价

要求：
- 能连续制造场面
- 能逼出人物选择
- 能同时服务情感和剧情
- 能回收到结局

### 9. 再回头打大纲
只围绕以下 5 件事排：
1. 底层情感需求
2. 爽点成立前提
3. 核心矛盾
4. 最想写的 3—5 个片段
5. 哪条设定 / 剧情是在帮忙，哪条是在带偏

### 10. 反查是不是又在逃
出现下面情况就停下来重做：
- 一卡就开始狂加世界观和外部剧情
- 明明是人物 / 感情型梗，却把人物写成背景板
- 把不理解的角色硬拐成自己熟悉的类型
- 大纲越来越顺，但最初那个梗越来越淡

## 输出格式
1. 表层梗
2. 当前卡点
3. 灵魂拷问结果
4. 片段列表与爽点标注
5. 共同情感模式
6. 本质 vs 表达方式
7. 成立前提
8. 核心矛盾
9. 应删掉的跑偏方向
10. 可继续扩成大纲的骨架

## 快速示例
### 输入
- 主梗：强势大人物疯狂迷恋主角，但主角并不轻易回应
- 卡点：不会写“摆谱拿乔”型主角

### 输出
- 表层梗：高位者追爱、主角拿乔
- 片段共同点：主角先做了某些事让对方喜欢上自己，但始终站在“并没有故意勾引你”的位置上
- 本质：不是单纯拿乔，而是“既享受被高配对象偏爱，又保住主动权和道德高位”
- 前提：要有外部见证、足够大的权力差，以及让主角不得不接近对方的理由
- 核心矛盾：主角带着目的接近对方，同时持续撒谎，既想拿到结果，又不想付出情感

## 注意事项
- **不要**一上来就问“主流写法是什么”，先问“真正想吃哪口”
- **不要**因为不会写某种表层表现，就误判自己完全不适合这个题材
- **不要**先搭一整套大设定，再回头硬塞爽点
- **不要**把“作者自己的舒适区”误当成“这个梗最适合的写法”
