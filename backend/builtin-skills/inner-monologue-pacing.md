把心理描写放在该停的地方，不要一句对白配一段解读。**高节奏场景里，读者更想看动作、结果和反应；心理写太碎，会把爽点和冲突都切断。**

## 适用情况
- 对话一多就忍不住每句后面补心理
- 爽点、打脸、对峙写着写着变慢
- 场景里全是“她想”“她觉得”“她心里一动”
- 读者说节奏拖、心理太密、人物磨叽

## 输入
- 场景类型：打脸 / 对峙 / 救场 / 比赛 / 表白 / 崩溃 / 回收等
- 视角人物
- 这一场真正的重点
- 当前想保留的心理句

## 先分场景速度
- **快场**：打脸、争执、比赛、行动、救场、追逐、调查推进
- **中场**：谈判、试探、暧昧拉扯、危机前夜
- **慢场**：崩溃、告白、重大真相、情绪回收、失去后的余波

快场最忌一句一心理，慢场才允许多停。

## 核心原则
1. **快场不要一句一心理。**
2. 同一类情绪，优先集中写 1—2 段，不要分散成十几次小感叹。
3. 能用动作、语气、停顿、旁人反应写出来的，不要再重复解释一次。
4. 打脸和对峙场面里，**被打脸的人、旁观者、局势变化**往往比主角反复心理更有看头。
5. 心理句优先留在：`决策前`、`转折点`、`结果落地后`。

## 步骤
### 1. 先写这一场最值钱的东西
填空：
`这一场读者最想看的是____。`

如果答案是“主角出手”“对方吃瘪”“真相落地”，就不要让心理描写一直拦在动作前面。

### 2. 列出动作链，再安排心理插点
先只写外部过程：
`发生了什么 -> 谁说了什么 -> 谁做了什么 -> 结果如何`

再从中挑 2—3 个位置放心理：
- 出手前一秒
- 看到关键结果时
- 事情结束后的余波

其余地方优先留给动作和对白。

### 3. 把碎心理并成整段
高危写法：
- 他说一句，她想一句
- 他再说一句，她又想一句

更稳的写法：
- 前面先顺着对话和动作往下走
- 到关键点，再用一小段集中写这一轮的情绪变化

### 4. 决定谁的反应更值钱
不是所有场景都要多写主角心理。
优先考虑：
- 被打脸者的破防
- 旁观者的震惊或改观
- 视角人物在关键转折点的判断
- 情势突然逆转时全场的停顿

### 5. 用外显代替重复解释
少写：
- 她很紧张
- 她很心动
- 她很生气

多写：
- 她把手里的杯盖捏变形了
- 她回话慢了半拍
- 她想接话，却先低头去看消息
- 她本来后退，听完那句又站住了

### 6. 快场常用配比
可先按这个粗比例自检：
- 动作 / 对话 / 场面结果：70%—85%
- 心理描写：15%—30%

不是死公式，但如果快场里心理已经比动作还多，通常会拖。

### 7. 写完后删三类句子
优先删：
- 重复同一种情绪的句子
- 动作已经说明了，还再解释一遍的句子
- 不影响决策和结果的碎感想

## 输出格式
1. 场景速度
2. 这一场最值钱的东西
3. 动作链
4. 需要保留的 2—3 个心理插点
5. 更值得写反应的人
6. 应删的重复心理
7. 调整后的快慢比例

## 快速示例
### 输入
- 电竞文
- 男主问女主要欺负她的人的 ID，准备帮她报仇
- 现写法是男主每说一句，女主都跟一段心理

### 输出
- 场景速度：快场
- 最值钱的东西：男主出手、局势推进
- 保留心理插点：
  1. 男主开口要 ID 时，女主短暂一怔
  2. 事情真的被男主接过去后，女主在段尾集中回收一段情绪
- 中间处理：删掉每句对白后的碎心理，改写成动作和对话一气呵成
- 可补反应：被报复对象或旁观队友的反应，会比女主每句都想更有意思

## 注意事项
- **不要**把每句对白都写成阅读理解
- **不要**在该快的地方反复停车
- **不要**只会写主角心理，不写场面结果
- **不要**把“心理多”误当成“细腻”
- 心理描写最稳的写法，不是写得满，而是**写在真正该停的地方。**
