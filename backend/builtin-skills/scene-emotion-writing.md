把情绪写成「刺激 → 反应 → 结果」的现场，而不是一句结论。**值得写的不是“她很生气”，而是这份生气让她当场做了什么、让局势变了什么。**

## 适用情况
- 情绪描写只剩“很生气”“很难过”“很喜欢”
- 场景有情绪，但没有临场感
- 读者知道角色怎么想，却没有共鸣
- 想立人设、推关系或推剧情，却只写成心理旁白

## 输入
- 当前场景
- 角色要吃到的核心情绪
- 这份情绪要服务什么：立人设、推剧情、推关系、制造钩子
- 角色性格与处境

## 步骤
### 1. 先判断这份情绪值不值得展开
先问：
- 这份情绪会不会改变角色的选择
- 会不会暴露人设
- 会不会让关系或局势发生变化

如果都不会，能一句带过就一句带过。

### 2. 先找刺激源，不要只写心情
把情绪落到一个可见的当场事件上：
- 谁说了什么
- 谁做了什么
- 环境怎么逼迫了角色
- 角色看见了什么无法忽视的细节

### 3. 把情绪翻成反应链
按这个顺序写：
`刺激 → 第一反应 → 动作 / 对白 / 表情 → 周围反馈 → 结果`

这样读者看到的不是标签，而是情绪在现场怎么长出来。

### 4. 用角色自己的方式发作
同样是生气，不同角色写法不同：
- 有人当场回怼
- 有人冷处理
- 有人阴阳怪气
- 有人立刻报复
- 有人越笑越危险

**不要** 把所有情绪都写成同一种哭、吼、愣住。

### 5. 先写当场，再写总结
少写：
- 她很生气
- 他很伤心
- 她忽然意识到自己讨厌他

多写：
- 她做了什么
- 他说了什么
- 周围人怎么被影响
- 这件事之后留下了什么后果

### 6. 拉开有看点的情绪，压缩太顺的情绪
优先展开：
- 会引发误会、冲突、真香、吃醋、报复、护短的情绪
- 能暴露角色风格和底线的情绪

谨慎展开：
- 太顺、太平、没有后劲的情绪
- 还没积累够就直接宣布深情

如果后面要写反转或关系变化，前期就不要把所有心思一次讲光。

### 7. 让每个情绪点有作用
写完后检查，这场戏至少完成 1 项：
- 立住人设
- 推动剧情
- 推动关系
- 提供钩子
- 制造新的局势

没有作用的情绪描写，优先删。

## 输出格式
1. 这份情绪值不值得展开
2. 刺激源
3. 反应链
4. 推荐的动作、对白、细节
5. 这份情绪要带来的结果
6. 可删的直白总结句
7. 改写后的场景

## 快速示例
### 差写法
一个早上，主角被隔壁吵醒了，她很生气。

### 改写
凌晨三点，隔壁的派对还没散。主角顶着一头乱发站在门口，拎着网购了半个月的唢呐，深吸一口气就吹。整层楼的声控灯全亮了，邻居的音乐当场断掉，门缝里有人骂了一句脏话。主角吹完转身就走，回屋时还顺手给自己点了个赞——结果第二天，她就被隔壁记住了。

这里真正值钱的不是“她生气”，而是**这份生气让她干了什么，并且引来了后果**。

## 注意事项
- **不要** 只写情绪结论，不写刺激和反应
- **不要** 凭空写情绪，场景里却没有支点
- **不要** 每种情绪都写满
- **不要** 让情绪和剧情、关系完全脱节
- 临场感的关键是：**让读者像在现场看角色反应，而不是听作者总结**
