把外部冲突里的爽点，写成感情线的推进器。**感情流是吃爽点的，但爽点最后要落回甜和关系变化。** 如果一场打脸只让主角赢了，却没让两人的关系更近，这口爽就容易跑偏成纯事业线。

## 适用情况
- 感情流 / 甜文想加爽点，又怕喧宾夺主
- 想写“CP帮忙打脸”“联手打脸”，但不知道怎么写得既爽又甜
- 事业线、校园线、娱乐圈线有冲突，却不会拿来推感情
- 两人还没在一起，想借外部事件推进关系
- 甜文写得太平，想加一口解气感

## 输入
- 当前关系阶段
- 这波外部冲突
- 谁受委屈、谁出手
- 这波之后关系想推进到哪里
- 主线主卖点（甜 / 拉扯 / 双向奔赴 / 先婚后爱等）

## 核心原则
1. **主线是感情，爽点只是推进器。**
2. 主角自己打脸是爽；**CP 帮忙打脸就是又爽又甜。**
3. 主角先打，CP 再补刀，通常比单人解决更带劲。
4. 如果两人还没在一起，**“没想到对方会帮”**最适合制造关系转折。
5. 这波结束后，读者不仅要看到事情解决，还要看到**关系变了**。

## 步骤

### 1. 先钉死这波的感情落点
填空：
`这波爽点结束后，两人的关系要从____变成____。`

例如：
- 普通同事 → 明显偏心
- 互相试探 → 开始信任
- 暧昧拉扯 → 明确站队
- 单向暗恋 → 一方第一次意识到自己被在意

### 2. 再写眼前冲突
把冲突写清三件事：
- 主角受了什么委屈
- 这口气为什么必须出
- 这件事和两人的关系有什么可连接的地方

优先选能带来关系波动的冲突：
- 被黑、被误会、被欺负
- 被人轻视、打压、抢位置
- 主角遇到困难，需要有人托底
- 外部人物挑事，逼 CP 站队

### 3. 选 CP 的参与方式
常用 4 种：
1. **直接撑腰**：CP 当场帮主角打脸
2. **联手打脸**：主角出手，CP 配合补位
3. **主角先打，CP 再补刀**：主角先赢一回合，CP 再给更重处罚
4. **事后托底**：当场先过关，CP 在后面把惩罚补齐、把后果处理干净

### 4. 给这次出手加“意外性”
如果两人还没在一起，最好让主角或旁人**没预料到 CP 会帮**。
这样打脸不只是解气，还会变成关系推进点。

可用写法：
- 主角本来准备自己扛
- 旁人默认 CP 不会站主角
- 主角以为这件事和 CP 无关
- 结果 CP 明确站出来，公开站队或补刀

### 5. 把爽点回收成感情推进
至少回收 1—2 项：
- 主角第一次意识到 CP 很在意自己
- CP 借出手暴露偏爱或保护欲
- 两人之间的信任更深
- 外人看出两人关系不一般
- 一方原本不敢靠近，现在敢了
- 一方原本嘴硬，现在开始动摇

**如果这场打脸结束后，两人关系完全没变化，这场多半只是事业爽点。**

### 6. 让最后一笔落在“甜”上
稳妥顺序：
`受压 → 出手/反击 → 反派吃瘪 → 惩罚落地 → 两人关系发生变化`

最后一笔优先写：
- 主角的意识变化
- CP 的偏爱被看见
- 一句让关系升温的台词
- 打脸之后的托底、安抚、照顾、默契

不要把全文篇幅都花在反派吃瘪和围观震惊上，结果 CP 自己像背景板。

### 7. 事业线爽点怎么服务感情线
如果主线是感情，但爽点发生在事业 / 校园 / 比赛 / 舆论场：
- 先让主角凭本事处理一部分
- 再让 CP 在关键处补位、托底、追加惩罚或站队
- 最后把这次事业线高光回收成感情变化

**事业线的爽点，可以是感情线的推进器；前提是最后的着力点落在关系上。**

## 常用模板
### 模板一：CP 帮忙打脸
`主角被针对 → 自己先反击 → 对方以为稳赢 → CP 出场站队/补刀 → 反派受罚 → 主角意识到 CP 在意自己`

### 模板二：联手打脸
`外部人物挑事 → 主角负责正面回击 → CP 负责托底/放大结果 → 事情解决 → 两人的默契或亲密上升`

### 模板三：事业爽点转感情推进
`主角在事业上遇到压制 → 主角自己扛住第一波 → CP 在关键处帮忙翻盘 → 翻盘结果变成关系变化`

## 输出格式
1. 当前关系阶段
2. 这波外部冲突
3. CP 参与方式
4. 爽点是什么
5. 甜点是什么
6. 关系推进到哪里
7. 最后一笔应该落在哪个情绪点
8. 跑偏风险

## 快速示例
### 输入
- 设定：主角被黑上热搜，CP 暂时还没表白
- 目标：这场既要爽又要甜

### 输出
- 爽点：主角自己放证据反击
- CP 参与：主角先打脸，CP 再公开站队补刀
- 关系推进：主角从“觉得对方只是合作伙伴”变成“开始意识到对方很在意自己”
- 最后一笔：不是停在黑子被骂，而是停在主角看见 CP 出手后的情绪变化

## 注意事项
- **不要**让爽点只剩事业结果，和感情线毫无关系
- **不要**每次都写成 CP 全权代打，主角像挂件
- **不要**只写围观者震惊，不写两人的关系变化
- **不要**为了甜而把冲突写得太轻，爽感会发虚
- 感情流的爽点，最好做到：**事情解气了，关系也更近了。**
