写故事时，先别追求“和现实一模一样”。**真实感的核心不是复刻现实，而是让设定、人物和后果在这个故事里能够自圆其说。**现实题材要讲现实边界，幻想题材更要讲世界规则；但不管什么题材，都只需要把**故事真正要用到的部分**写得站得住。

## 适用情况
- 设定很离奇，怕读者说假
- 现实题材一写就悬浮
- 为了“真实”塞进太多解释，正文变拖
- 世界观能想出来，但落不到生活体验
- 读者会说“这不真实”，却说不清是哪里不对

## 核心原则
1. **真实感 = 规则自洽 + 行为有因 + 结果有痕迹。**
2. **真实感 ≠ 所有细节都完全符合现实概率。**
3. **重点部分要严，非重点部分可以略。**
4. **逻辑要为题材和主题服务。**甜爽文、搞笑文、硬剧情文，对真实感的要求不同。
5. 读者最容易相信的，不是大段设定说明，而是**角色真的在被规则影响**。

## 输入
- 题材与类型（现实 / 奇幻 / 修仙 / 娱乐圈 / 校园等）
- 作品主卖点
- 当前最怕读者不信的设定或情节
- 这个设定会影响哪些人物、选择和后果

## 步骤
### 1. 先定“这篇文到底要真实什么”
先写一句：
`这篇文最需要站住的是____。`

例如：
- 校园文：学校环境、成绩压力、同学关系
- 娱乐圈文：圈层规则、资源逻辑、舆论反应
- 修仙文：修炼规则、境界代价、世界秩序
- 甜爽文：情绪兑现和类型快感比硬核考据更重要

**先定重点，再定考据深度。**

### 2. 把设定分成三层
#### A. 核心层：必须讲清
删掉就会影响主线成立的规则。

例如：
- 修炼体系怎么升级
- 魔法为什么能这样用
- 身份制度、资源门槛、比赛规则
- 关系成立的现实条件

#### B. 支撑层：讲到够用
只需要解释到读者能接受，不必写百科全书。

例如：
- 行业内部更多细节
- 次要配角的完整职业流程
- 世界运行的边角规则

#### C. 背景层：可以模糊
不会直接改变主线结果的部分，只要不明显打脸前文即可。

### 3. 检查“规则 → 行为 → 后果”链条
每个关键设定都过一遍：
- 这条规则是什么
- 角色会因此怎么做
- 这么做会付出什么代价或得到什么结果
- 其他人会如何适应这条规则

如果只有规则，没有行为和后果，设定就会像贴在墙上的说明书。

### 4. 给陌生世界找“现实对应物”
让幻想设定落到读者熟悉的体验上。

常用对应：
- 修仙宗门 ↔ 学校 / 门派体系
- 境界考核 ↔ 考试 / 晋升
- 娱乐圈资源竞争 ↔ 职场竞争
- 魔法课程 ↔ 日常学习训练
- 宗门偏见、师门压力 ↔ 现实中的阶层与规则

**形式可以陌生，本质最好能让读者认出来。**

### 5. 用“结果痕迹”证明真实感
不要只说设定存在，要写它留下了什么痕迹：
- 黑眼圈、旧伤、资源差距、名声变化
- 生活习惯、社会偏见、职业口气
- 规矩带来的顾忌、限制和代价
- 角色为某件事提前准备或反复避坑

读者往往是因为看见这些痕迹，才相信设定真的在运转。

### 6. 判断什么时候该让逻辑为主题让步
先问：
- 这个问题是不是故事真正的重点
- 讲清它会不会让主线更强
- 不讲清会不会直接破坏读者信任

如果答案是：
- **会破坏主线成立** → 必须讲清
- **只会把故事带偏** → 可以压缩或略过

例如：
- 奇幻世界里，修炼规则必须自洽
- 但如果某个现代社会细节不是主线重点，就不必把整套现实推演写满
- 娱乐圈甜爽文里，某些桥段可以适当服务爽感；但如果作品主打行业真实，就不能乱来

### 7. 及时处理破坏感强的 BUG
出现读者会立刻出戏的点时，优先用以下方式修补：
- 提前埋规则
- 增加限制条件
- 补一个代价
- 让角色意识到异常，而不是装没看见

**BUG 不一定要零，但至少要能圆回来。**

## 输出格式
1. 这篇文最需要站住的真实感重点
2. 核心层 / 支撑层 / 背景层
3. 关键规则的“规则 → 行为 → 后果”链条
4. 可借用的现实对应物
5. 需要写出来的结果痕迹
6. 可以略写的部分
7. 最容易出戏的 BUG 点
8. 修正建议

## 快速示例
### 输入
- 题材：修仙文
- 设定：人物能活上千年
- 问题：怕读者觉得太离谱

### 输出
- 重点：不是解释“现实里怎么可能活千年”，而是解释这个世界为什么允许、代价是什么、社会因此怎么运转
- 核心层：修炼体系、寿命与境界的关系、飞升条件
- 结果痕迹：宗门资源争夺更长线，人物对时间的态度不同，凡人与修士的生活节奏不同
- 风险点：前文一直是纯修仙逻辑，后文突然掏出现代热武器解决一切，会立刻破坏真实感

## 注意事项
- **不要**把“真实感”误写成“资料堆砌”
- **不要**让核心规则随剧情方便随时改
- **不要**所有问题都用“这是小说”糊过去
- **不要**把非重点现实细节写得比主线还重
- 好的真实感，不是像现实复印件，而是：**这个故事世界有自己的规矩，而且这些规矩真的在起作用。**
