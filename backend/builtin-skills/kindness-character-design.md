先把善良写成选择和代价，不要写成旁白夸奖。**讨喜的善良，是会主动伸手、分得清对象、愿意付一点代价，但不会无底线替所有人买单。**

## 适用情况
- 想给女主或男主立“善良、温暖、值得被爱”的底色
- 角色设定不差，但读者觉得没人格魅力
- 一写善良就像圣母、恋爱脑或工具人
- 需要把善良同时服务感情线、主线或世界观

## 输入
- 角色当前处境与核心卖点
- 想立住的善良类型：共情、愿意助人、护短、愿意牺牲、会安慰人
- 本段剧情里可用的对象：弱者、陌生人、老人、小孩、边缘角色、受伤同伴、对手中的无辜者
- 角色可承受的代价：时间、金钱、机会、面子、风险、与自身利益冲突

## 核心原则
1. **先写行为，再让别人下结论。**
2. **优先帮对自己没明显好处的人。**
3. **善良必须带一点代价，分量才够。**
4. **善良要有边界。** 能共情，不等于不判断。
5. **一场善良戏最好兼做两件事**：立人设 + 推关系 / 主线 / 世界观。

## 步骤
### 1. 先钉死这场要证明哪一种善良
用一句话写：
`这场戏真正想证明的是：TA 在____情况下，仍会主动____。`

例如：
- 自己也拮据时，仍愿意帮更弱的人
- 明知会失去利益，仍愿意帮重要的人脱困
- 对陌生人也能给体面和安慰

### 2. 优先选“低回报对象”
优先级：
- 老人、小孩、病弱者、亡者、边缘人
- 与主角没有直接利益交换的人
- 只能向主角求助、没有别处可去的人

**对男主或女主好，不足以证明善良；对无利可图的人好，才更有说服力。**

### 3. 给善良加一个小代价
从下面至少选 1 项：
- 错过自己的事
- 花自己的钱或资源
- 冒一点险
- 暴露软肋
- 放弃当前更有利的选择
- 让自己在别人眼里显得“吃亏”

### 4. 把善良写在“利益冲突点”上
不要写成平静时顺手帮忙，优先写成：
- 帮别人，自己会少一点好处
- 说真话，会失去当前位置
- 救人，自己要承担后果
- 安慰别人，自己其实也很难

**冲突越明确，善良越值钱。**

### 5. 让善良落到具体动作
少写“她很善良”“他很善良”，多写：
- 先听完对方的话
- 主动安抚、解释、递东西、送人回去
- 替对方说一句体面话
- 主动承担麻烦
- 明知自己会被牵连，还是站出来

### 6. 给善良保留边界
至少写清 1 项：
- 帮的是无辜者，不是反复作恶的人
- 愿意帮，但不会让同伴替自己埋单
- 可以心软，但会先判断真假
- 救急不救烂摊子

### 7. 让这场善良戏顺手推进别的东西
至少回收 1—2 项：
- 某个配角对主角改观
- 感情线升温
- 世界观被自然带出
- 后文有回响：祝福、助攻、报恩、站队
- 主角的价值观被明确

### 8. 善良分层兑现
按强度从低到高安排：
1. **日常善意**：愿意帮陌生人、照顾弱者
2. **有代价的善意**：牺牲时间、面子、利益
3. **核心善意**：明知自己会失去什么，仍想帮最重要的人

## 输出格式
1. 想立住的善良类型
2. 受助对象
3. 当前代价
4. 具体动作
5. 边界设置
6. 这场戏同时推进的主线 / 关系
7. 后续回响

## 快速示例
### 输入
- 女主想立“温暖但不软弱”

### 输出
- 受助对象：一个与主线无关、临死前只想给家人留话的老人
- 代价：女主要为此多跑一趟、错过自己的安排
- 动作：先安抚老人，再替她传话，同时告诉对方自己会尽力但不乱许诺
- 边界：只帮传达和照看，不替其后人承担全部人生
- 回响：旁人第一次意识到女主不是嘴上心软，而是真的会伸手

## 注意事项
- **禁止**只用旁白说“大家都知道她很善良”
- **禁止**只对恋爱对象好，其他人都无感
- **禁止**为了善良让角色降智、反复吃亏
- **禁止**把无底线忍让当成善良
- **真正讨喜的善良，是有判断、有代价、也有温度。**
