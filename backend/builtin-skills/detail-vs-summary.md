先判断这一段的作用，再决定详写还是略写。**详写不是写得多，略写也不是偷懒；关键在于把笔墨留给真正能改动情绪、关系和局势的部分。**

## 适用情况
- 铺垫一写就拖
- 高潮场面不够重
- 不知道哪些片段该展开，哪些该一句带过
- 描写很多，但删掉又不影响剧情

## 输入
- 当前要写的段落或场景
- 这段在全文里的位置：铺垫 / 过渡 / 高潮 / 收尾
- 这段最想完成的效果
- 目前想写的细节清单

## 步骤
### 1. 先写这段存在的目的
用一句话写：
`这段是为了____。`

常见目的：
- 渲染气氛
- 立人物
- 推剧情
- 转情绪
- 埋伏笔
- 回收情感或冲突

### 2. 分清详写和略写
- **略写**：只交代主要事件、关键因果和结果
- **详写**：把场面、动作、语言、心理、气氛写到能让读者身临其境

### 3. 铺垫段用“详略结合”
铺垫时重点详写：
- 会被角色长期记住的片段
- 能改变关系温度的互动
- 明显能苏、能撩、能埋回收的场面

其他过程，能一句带过就一句带过。

### 4. 高潮段用“组合拳详写”
高潮优先补齐四类内容：
- **环境 / 服饰**：定气氛、立人物状态
- **动作 / 语言**：直接推进冲突
- **心理 / 回忆**：放大情绪重量
- **结果 / 余波**：让爆发落地

高潮不是只写一句“他很难过”，而是把现场打全。

### 5. 给每句描写标用途
写完后逐句标：
- 渲染
- 立人设
- 推剧情
- 转情绪
- 埋伏笔

标不上作用的句子，优先删。

### 6. 做删除测试
问自己：
- 删掉这段景物或外貌，剧情会不会一样
- 删掉这段心理，情绪还立不立得住
- 删掉这几句对白，关系有没有真正变化

如果删完几乎没影响，这部分就是冗余。

### 7. 最后按场景重新分配笔墨
- 铺垫：留关键镜头，压缩重复过程
- 过渡：让读者知道发生了什么即可
- 高潮：把最值钱的那口写满
- 收尾：写到情绪和后果落地就收

## 输出格式
1. 这段的作用
2. 该详写的点
3. 该略写的点
4. 详写时要补的四类内容
5. 建议删除的冗余
6. 改写思路

## 快速示例
### 输入
- 铺垫段：男女主几次日常相处
- 高潮段：男配抢亲、男主正面出手

### 输出
- 铺垫：只详写最能升温、最能埋回收的 1—2 场，其余略写
- 高潮：详写环境、动作、情绪、回忆与结果，不要一笔带过

## 注意事项
- **不要**铺垫和高潮写成一个力度
- **不要**每个场面都平均用力
- **不要**把能一句说清的过程写成大段注水
- **不要**把真正的爆点只写结果，不写过程
