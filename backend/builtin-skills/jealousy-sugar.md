把吃醋写成“在意变得可见”，不要写成莫名其妙发疯。**最甜的吃醋，不是角色亲口说“我吃醋了”，而是他 / 她已经在行为里抢回优先级。**

## 适用情况
- 想写吃醋糖，但总是一个模板反复用
- 吃醋场面太直白、太吵，反而不甜
- 修罗场热闹有了，CP 感没立住
- “万人迷”主角总像人见人爱，不像人见人醋

## 输入
- CP 设定与当前关系阶段
- 谁吃醋、为什么吃醋
- 触发对象：情敌 / 朋友 / 同事 / 师长 / 工作 / 责任 / 其他事物
- 想写的甜味：暗甜 / 酸甜 / 傲娇甜 / 修罗场甜

## 步骤
### 1. 先定这场吃醋到底在证明什么
用一句话写清：
`这场吃醋真正证明的是：TA开始在意____。`

常见证明点：
- 在意对方把注意力给了别人
- 在意自己不是第一顺位
- 在意对方和别人有了自己不知道的默契
- 在意某段关系、位置、称呼被别人占走

没有证明点，这场吃醋就只剩热闹。

### 2. 触发点不要只会“情敌出现”
可用触发点：
- 对方频繁向别人请教问题
- 对方和别人聊天太投入
- 对方把时间先给了工作、朋友、家人、任务
- 第三人抢先知道秘密、看见脆弱、得到偏爱
- 修罗场里别人先一步靠近

**吃醋不一定发乎爱情，也可以先发乎“我在你这里是不是特殊”。**

### 3. 按关系阶段控制浓度
#### 关系初期
重点写“不对劲”。
- 多问两句
- 语气拐弯
- 视线老往第三人那边扫
- 主动把话题接过来

#### 暧昧期
重点写“抢回优先级”。
- 主动接手问题
- 找借口把两人隔开
- 先给出更好的方案
- 表面若无其事，动作已经插进来了

#### 确认关系后
重点写“半明半暗的占有欲”。
- 半开玩笑地宣示位置
- 吃醋后要补回亲密或沟通
- 允许更明确，但别每次都炸场

### 4. 优先写隐晦吃醋
常用隐晦写法：
- 绕弯追问：`你刚刚和他聊什么？`
- 继续追问细节：`是哪种问题？`
- 顺势接手：`这种事以后来问我。`
- 不动声色换位置、换搭档、换话题
- 明明没说不高兴，反应却比平时多半拍

**越早期、越克制的人设，越适合隐晦写。**

### 5. 吃醋必须贴人设，不要一套走天下
- **温柔型**：不拦着吵，绕弯提问，最后把事情接过去
- **傲娇型**：嘴硬一句，东西先塞给对方，人先跑了
- **高冷型**：话少，但一句就改了优先级
- **腹黑型**：笑着截胡、安排、占位，表面体面，实际很会卡位
- **狼狗型**：脸臭、动作快、占有欲明显，但表达笨拙

同样是吃醋，**动作、台词、表情、节奏都要像这个角色本人。**

### 6. 让吃醋带来一个可见动作
不要只写心理活动，至少落一个动作：
- 把问题接过去
- 抢先送东西、送人、陪同
- 主动站到两人中间
- 在第三人面前显露熟悉度
- 临时改安排，把自己放回第一顺位

没有动作，吃醋很容易写空。

### 7. 同一种吃醋不要复读
逐条检查：
- 是不是每次都黑脸不说话
- 是不是每次都靠“你离他远点”
- 是不是每次都靠肢体拉走

改法：
- 换触发点
- 换表达方式
- 换后果
- 换人设口味

### 8. 修罗场和“万人迷”要写成人见人醋
如果主角有很多人围着：
- 不要只写“大家都喜欢 TA”
- 要写“大家都觉得自己在 TA 那里本来有特殊位置，结果被别人挤占了”

为不同角色设计不同醋法：
- 恋爱向：吃情敌的醋
- 友情向：吃“好姐妹 / 好兄弟被拐跑了”的醋
- 师生 / 主从 / 队友向：吃优先级被替代的醋

**“万人迷”写得好，往往不是人见人爱，而是人见人醋。**

### 9. 配角的醋也能点睛
不一定只有男主 / 女主能吃醋。
可用在：
- 女闺蜜酸男主抢走了自己姐妹
- 队友酸主角更信另一个搭档
- 师父酸弟子先去找别人请教

这类醋用一两笔就够，能放大主角的特殊性和场面活气。

### 10. 吃醋后要有余波
至少回收 1 项：
- 对方第一次意识到 TA 在意自己
- 两人关系更近一步
- 第三人看出了端倪
- 吃醋者暴露了新弱点 / 新萌点
- 下一次修罗场 stakes 更高

## 输出格式
1. 这场吃醋证明什么
2. 触发点
3. 人设口味
4. 外显动作
5. 台词 / 潜台词
6. 第三人的作用
7. 余波

## 快速示例
### 输入
- 设定：男主发现女主总向男配请教问题
- 需求：写隐晦吃醋

### 输出
1. 证明点：他开始在意自己是不是女主的第一顺位
2. 触发点：女主连续两次去找男配讨论问题
3. 人设口味：克制型
4. 外显动作：先问她聊了什么，再把问题接过去，当场解答
5. 台词 / 潜台词：`这种问题，你以后来问我就好。`
6. 第三人作用：只负责触发，不抢戏
7. 余波：女主后知后觉意识到他不只是热心

### 输入
- 设定：冬天校园，小狼狗男主在校门口等女主
- 需求：写傲娇吃醋 + 照顾

### 输出
1. 证明点：他嘴上不认，身体已经在抢位置
2. 触发点：看见别的男生和女主说笑走来
3. 人设口味：傲娇狼狗
4. 外显动作：把热豆浆硬塞进女主手里，只丢下一句“喝”
5. 台词 / 潜台词：嘴上凶，实际上在说“先拿我给的”
6. 第三人作用：衬托他的失衡
7. 余波：女主开始觉得他今天格外不对劲

## 注意事项
- **禁止**把吃醋写成失控霸凌，除非题材和人设本来就要承受这个后果
- **禁止**为了吃醋硬塞一个工具人情敌
- **禁止**只写黑脸，不写动作和后果
- **禁止**同一种醋法全文复读
- **越常见的桥段，越要写出角色自己的味道**
