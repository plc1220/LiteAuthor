先把打戏的布局搭好，再写每一轮出手和承受。**打戏不是招式报菜名，也不是一句“大战三百回合”；必须让读者看清谁先动手、为什么打、这一招危险在哪、打完局势怎么变。**

## 适用情况
- 比武、追杀、街头斗殴、群战、异能对轰、修仙斗法
- 打戏只剩动作流水账，看着平
- 只会写出招，不会写开打原因和打后余波
- 想让战斗更带感、更有爽点和压迫感

## 输入
- 冲突双方与立场
- 开打原因、导火索、输赢代价
- 场地、可用道具、旁观者
- 双方强点、弱点、武器/能力
- 这场打戏最终要把局势推到哪里

## 核心原则
1. **先有布局，后有描写。**
2. 每一轮都写清：谁主动、谁承受、有没有打中、局势如何变化。
3. **5D 是核心**：视觉、听觉打底；触觉、嗅觉、味觉挑关键点补刀。
4. 群战也要落回局部交锋，不要一团糊。
5. 打完必须改局势：伤势、站位、名声、关系、后续冲突至少变一样。

## 步骤

### 1. 先搭打戏布局
至少确认这些角色位：
- 主动发起方
- 被动接收方
- 双方同伴
- 旁观者 / 嘴炮 / 解说位
- 中立方（如果有）

旁观者的作用不是凑热闹，而是：
- 放大含金量
- 见证胜负
- 制造站队和情绪反馈
- 在关键处补一句“这下危险了”

### 2. 写开打原因和导火索
开打前先写三件事：
- 矛盾因为什么出现
- 谁和谁真正对上
- 导火索是什么

**导火索必须足够把冲突推到“非打不可”。**
如果只是互看不顺眼，读者很难真正站队。
优先让导火索命中：
- 尊严被踩
- 重要的人或物被威胁
- 退一步就会吃更大亏
- 规则被破坏到不能忍

### 3. 先排 3—5 个回合骨架
常用骨架：
1. 先手 / 试探
2. 一方压制
3. 对方拆招反压
4. 冲突升级 / 更狠导火索
5. 决胜 / 收尾

每一回合至少换一个变量：
- 速度
- 力量
- 范围
- 场地
- 伤势
- 围观者态度
- 情绪失控程度

### 4. 每回合按“前 / 中 / 后”写
#### A. 出手前
补 1—2 笔：
- 神态
- 站位
- 武器 / 道具
- 场景气氛
- 声音征兆

例：
- 指节捏紧
- 眼神发狠
- 玻璃瓶在掌心一转
- 风声、喘息、桌椅摩擦声先起

#### B. 出手中
写清四件事：
- 招式或动作怎么来
- 危险点是什么
- 承受方怎么感到危险
- 有没有拆招、闪避、硬吃、反击

优先写：
- 轨迹：劈、砸、扫、刺、撞、逼近
- 位置：头、肋、腕、膝、喉、腹
- 身体反应：手臂发麻、脚下一滑、呼吸一窒、虎口震疼
- 环境反应：桌翻、雪飞、墙裂、酒液四溅

#### C. 出手后
至少回收 2 项：
- 伤势
- 站位变化
- 武器状态
- 旁观者反应
- 下一轮更大的危险

### 5. 用 5D 提升临场感
不要每句都硬塞五感，**一段打戏抓 2—3 种最有效的就够了。**

#### 视觉
- 轨迹、残影、血、碎片、距离、站位

#### 听觉
- 破空声、撞击声、喘息声、骨裂声、玻璃炸裂声

#### 触觉
- 发麻、刺痛、灼痛、震得手臂发酸、冷风割脸、热浪扑面

#### 嗅觉
- 血腥味、焦糊味、酒气、尘土味、火药味

#### 味觉
- 喉头腥甜、铁锈味、苦涩、呛人的烟灰味

**视觉 + 听觉** 通常已经能立住；  
想再加重时，再补触觉或嗅觉；  
味觉只在受伤、血战、窒息等时点一下最有效。

### 6. 描写手段按优先级用
常用顺序：
1. 动作描写
2. 气氛渲染
3. 神态描写
4. 旁观者 / 嘴炮反应
5. 场景变化
6. 招式名称

**招式名不是核心。** 只有当招式会改变胜负、身份或读者期待时才重点写。

### 7. 不会写时，用最稳的新手模板
模板 1：
- 反派举起____砸/劈/扫向主角的____
- 主角侧身 / 下蹲 / 后撤半步避开
- 反派一击落空，砸中____
- 主角顺势反击，踹/砸/顶向对方的____

模板 2：
- 对手先压过来
- 主角硬接一下，代价是____
- 主角借这一下找到空当，换位 / 近身 / 缴械

模板不是羞耻，是骨架。骨架稳了，再往上加风格。

### 8. 打完别只写“赢了”
收尾至少回收：
- 胜负标志：步伐乱了、武器脱手、眼神慌了、身体跪下去
- 余波：众人沉默、尖叫、倒吸冷气、改口站队
- 代价：主角也受伤、耗尽、暴露底牌
- 后果：仇更深、关系变了、下一战更大

## 输出格式
1. 开打原因与导火索
2. 角色位与场地
3. 回合 1—5 骨架
4. 每回合的危险点 / 反击点
5. 每回合使用的 5D 感官
6. 胜负标志
7. 打完后的余波与后果

## 快速示例
### 输入
- 酒吧冲突，反派先抡酒瓶，主角反击

### 输出
- 导火索：反派不只想出气，还要伤到主角同伴
- 回合 1：反派抡瓶先手，酒瓶破空
- 回合 2：主角侧身闪开，酒瓶砸碎桌角
- 回合 3：反派抓碎瓶再扑，危险升级
- 回合 4：主角近身砸头，玻璃爆裂，血腥味和啤酒味一起冲上来
- 回合 5：反派昏倒，围观者噤声，主角手背也被划开

## 注意事项
- **禁止**只报招式名，不写危险感和结果
- **禁止**全程只有视觉，没有身体感受
- **禁止**双方像站桩互砍，没有导火索升级
- **禁止**打一场很久，局势却什么都没变
- 好打戏的关键不是写了多少招，而是**每一招都让读者知道：为什么危险，为什么爽，为什么这一战必须继续打下去。**
