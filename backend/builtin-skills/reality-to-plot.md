把现实素材先拆结构，再换题材外壳，做成能连载的网文梗。**灵感不是照抄现实，而是抓住现实里的爆点、反转、关系和情绪波峰。**

## 适用情况
- 缺脑洞，只看到热搜或新闻有感觉
- 有一张图、一句话、一个小见闻，但不知道怎么扩成故事
- 现实事件很抓马，想改成自己的题材
- 只会记素材，不会转成主角、冲突和主线

## 输入
- 素材来源：新闻 / 热点 / 图片 / 身边见闻
- 想写的频道或题材
- 想保留的情绪：爽 / 狗血 / 反转 / 治愈 / 轻喜
- 不想碰的元素

## 步骤
### 1. 先用一句话复述素材
只写：
- 谁和谁
- 出了什么事
- 为什么大家会围观

禁止一上来抄细节。

### 2. 拆出“最值钱的三样”
从下面选 3 项：
- 爆点
- 反转
- 身份差
- 利益冲突
- 舆论压力
- 旧账回收
- 连续作死的反派
- 主角翻盘时机

### 3. 给现实素材改成故事功能位
把现实人物改成角色功能，不照搬真人：
- 主角
- 对手 / 反派
- 推波助澜者
- 围观 / 舆论
- 关键证据或秘密
- 能把局势推到下一层的人

### 4. 找一句“底层冲突公式”
把素材压成一句可复用公式：
- 被压住的人夺回解释权
- 旧关系反噬当事人
- 舆论先审判，真相后翻盘
- 一次丑闻牵出更大的利益链

公式越清楚，越容易嫁接别的题材。

### 5. 套 1—2 个题材滤镜
常用滤镜：
- 穿书
- 种田
- 直播
- 校园
- 娱乐圈
- 豪门
- 末世
- 仙侠
- 悬疑
- 职场

**最多套 1—2 个。套太多就会散。**

### 6. 补主角“为什么是她 / 他”
必须补清：
- 主角现在缺什么
- 为什么会卷进这件事
- 她 / 他要赢回来什么
- 这件事会怎样改写命运

### 7. 把单个灵感扩成长期线
至少再长出 3 条线：
- 主线冲突
- 关系线 / 感情线
- 长期对手或更大世界线

如果长不出三条线，这还只是点子，不是能连载的故事。

### 8. 做“去真人化”处理
改掉：
- 可直接对号入座的职业、时间、名字、细节
- 现实人物的一比一对应关系
- 只靠影射成立的桥段

保留：
- 冲突结构
- 情绪起伏
- 爆点顺序
- 角色功能

## 输出格式
1. 素材一句话
2. 三个最值钱的爆点
3. 角色功能表
4. 底层冲突公式
5. 题材嫁接版 3 个
6. 最推荐版本
7. 20 章内主线
8. 长期线 3 条
9. 去真人化提醒

## 快速示例
### 输入
- 素材：一张大片向日葵花海的照片
- 想写：女频轻爽文

### 输出
- 爆点：花海、反差人生、重新开始
- 角色功能：失意女主、前未婚夫、围观流量入口
- 题材嫁接：穿书 + 种田 / 种田 + 直播 / 破产千金逆袭
- 推荐版本：女主穿书后家道中落，承包土地种向日葵；花海意外出圈，带出直播、事业和旧人回头线
- 长期线：种田经营、前任反噬、女主翻红

## 注意事项
- **不要**把热点原样搬进小说
- **不要**只记八卦，不提炼结构
- **不要**只有一个反转，没有长期剧情
- **先拆冲突，再换外壳**
