把一条主线事件继续长出转折、支线、人物关系变化和后续回收点，让故事变厚但不脱主线。**扩容不是往主线上乱挂事故；每新增一段剧情，都必须带来新的冲突、人物信息、伏笔或可兑现的后果。**

## 适用情况
- 这段只有一句话，写完就没了
- 中段总觉得没剧情可写
- 主线有目标，但过程太直、太薄
- 只会让主角做事，不会让周围的人和局势动起来
- 一扩写就注水、脱主线

## 输入
- 当前主线事件一句话
- 主角当前目标
- 当前阻力
- 这一段想落到的阶段结果
- 已出场人物 / 可用人物
- 可回收的设定、道具、地点、秘密（可选）

## 步骤
### 1. 先钉死主线
用一句话写：
`主角为了____，必须去做____。`

再补两句：
- 这段最后最想写到哪里
- 绝对不能偏离的主线是什么

### 2. 先写最朴素的直线版
先把不扩容时的版本写成 3 行：
1. 主角去做事
2. 事情顺利 / 不顺利
3. 主角得到结果

只有先看清这条直线，才知道该在哪里加分叉。

### 3. 从扩容杠杆里选 2—4 个
不要全上。每次只选最能服务这一段的几种。

#### A. 增加转折链
让一件事至少有 2—3 折：
- 外部环境卡主角
- 规则或市场卡主角
- 人为阻挠再补一刀

要求：每一折都要让主角改一次策略。

#### B. 上正反配角对照
同一件事里，安排：
- 支持主角的人
- 轻视、造谣、唱衰的人

对照能同时制造矛盾、站队和后续回收。

#### C. 写家人 / 同伴的立场冲突
不是所有阻力都来自反派。
可写：
- 出于关心而反对
- 出于利益而阻拦
- 先反对，后被说服
- 虽然不认同，但还是帮忙善后

这样能让主线更立体。

#### D. 做关系翻转
优先用这四种：
- 朋友变对手
- 中立者变盟友
- 亲近者真叛变
- 亲近者假叛变，实则钓人 / 布局

关系一翻，旧人物就能继续产出新剧情。

#### E. 增加竞争者或反派阻挠
不要只让主角和天斗、和地斗，也要让人来抢：
- 抢资源
- 抢机会
- 抢名声
- 抢解释权
- 抢主角手里的关键物件

人为障碍最容易做出推拉感。

#### F. 插入新人物或天降催化剂
新人物的作用不是凑热闹，而是：
- 带来新资源
- 带来新秘密
- 把主线接到下一段
- 提前带出更大世界 / 更大反派

新人物一上桌，就要想好后面还能怎么用。

#### G. 用支线穿插主线
让主角在推进主线时，顺手撞到一条副线：
- 一具枯骨
- 一件来历不明的物品
- 一个被欺负的小人物
- 一句不对劲的话

要求：这条副线以后能回收成资源、真相、站队或下一副本入口。

#### H. 给事件补秘密背景或旧案
问：
- 这件事表面上是什么
- 背后还藏着什么
- 为什么别人以前失败过
- 为什么大家默认这件事不能做

一旦补出秘密背景，普通事件就有了深度。

#### I. 人为提高难度
从这些里选 2—3 个加码：
- 工具不够
- 地点危险
- 时间太赶
- 情报不全
- 体力 / 身份 / 规则受限
- 旁人不断误解或拖后腿

难度不是只写险，而是让主角真的解决问题。

### 4. 每条扩容都写成四联表
每加一条，就写清：
`新增元素 → 当场带来的冲突 → 留下的伏笔 → 后续能回收成什么`

如果答不出后两项，这条多半只是凑字数。

### 5. 每个新人物至少留一个可开发点
常用可开发点：
- 身份没说透
- 立场不稳定
- 和某人有旧恩旧怨
- 手里握着主线资源
- 看似小人物，后面能提级使用

人物初登场时，先从四种基础位置里选一种：
1. 好人
2. 坏人
3. 被迫变坏的好人
4. 伪装成好人的坏人

### 6. 每个场景留一个无伤大雅的引子
好的引子不一定立刻回收，但以后能牵出剧情。
例如：
- 主角路过时看见一个不该出现在这里的人影
- 某人中途离开了一会儿，理由看似合理
- 某个配角说了一句当时没人在意的话

要求：
- 当下不影响阅读
- 回头能解释
- 既能展开成真问题，也能展开成误会澄清

### 7. 用为什么链往下挖
当不知道怎么扩时，一路追问：
- 主角为什么要做这件事
- 为什么非现在做
- 为什么别人不做 / 做不到
- 为什么有人反对
- 为什么过去会失败
- 为什么这次会不一样

每个为什么，都能再长出背景、人物和新冲突。

### 8. 检查新增剧情是不是还长在主线上
逐条问：
- 这条线是不是因主线事件自然长出
- 它有没有改写主角下一步
- 删掉它后，后文是不是会少一层冲突、伏笔或回收
- 它是不是只是热闹，却不影响主线推进

**如果只是换个地点再来一遍同规格冲突，宁可不要。**

### 9. 排回收顺序
把所有新增点分成三类：
- 近回收：2—5 章内就用
- 中回收：本卷中段用
- 远回收：下一卷 / 大后期再用

不要所有伏笔都压到最后，也不要刚埋就全揭。

## 输出格式
1. 当前主线一句话
2. 直线版剧情
3. 选用的扩容杠杆
4. 新增剧情四联表
5. 可回收人物 / 伏笔清单
6. 为什么链
7. 这一段的阶段结果
8. 后续可接的 3 条路
9. 注水 / 脱主线风险

## 快速示例
### 输入
- 主线：女主上山采药卖钱

### 输出
- 转折链：上山先救下一个陌生人；卖药时连着被几家药铺压价
- 正反配角：有人支持女主采药，有人趁机造谣
- 关系翻转：看似关系不错的同伴其实在背后使绊
- 伏笔：被救之人到底是谁；为什么最后有一家药铺肯收药；家人为何强烈反对上山
- 后续回收：被救之人的身份、药铺站队、幕后告密者、家里的旧事

## 注意事项
- **禁止**新增桥段只负责凑字数
- **禁止**开很多支线，却没有回收计划
- **禁止**每一折都和主线无关
- **禁止**靠临时新人物反复救场
- **好的扩容应该让读者觉得：故事是从主线自己长出来的，不是作者往里塞东西**
