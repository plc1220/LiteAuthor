围绕原著里最让人不平、惋惜或想替他翻身的角色，设计逆袭变量、阶段挑战、碰撞爽点和圆满回收。**同人逆袭不是随便给角色塞外挂，而是先吃透他原本为什么憋屈，再让变化后的优势、危机和成长一层层长出来。**

## 适用情况
- 想写红楼、历史、动漫、影视等同人的配角翻身
- 喜欢某个原著角色，但不知道他值不值得当主角
- 想写“我就想看他怎么逆袭”的爽感
- 想让读者既心疼角色，又愿意追他翻盘
- 不知道逆袭线该偏事业、感情还是双线

## 输入
- 原著名称
- 候选角色 1—3 个
- 角色原本的人设状态、处境、结局
- 读者最容易替他不平的点
- 想走的方向：事业 / 感情 / 双线
- 期待的结局氛围：爽快 / 救赎 / 圆满 / 带代价

## 步骤
### 1. 先挑有“翻身空间”的角色
优先选：
- 地位吃亏、待遇不公的人
- 结局缺憾、下场惨的人
- 有优点却没被善待的人
- 和核心角色关系密、容易产生碰撞的人
- 读者一听就会想“凭什么是他倒霉”的人

**只有喜欢、不带冲突的角色，不适合硬做逆袭主角。**

### 2. 梳理角色的原始盘
至少写清：
- 现在有什么
- 没有什么
- 被谁压着
- 最大的不公是什么
- 原著已经给过他的优势是什么
- 原著没来得及让他用出来的潜力是什么

### 3. 给角色一个“新变量”
常用变量：
- 提前知道真相
- 重来一次
- 多了一个盟友
- 获得关键资源
- 换了站队
- 在关键节点做出不同选择

**新变量只要一处够强就行，不要一口气全塞。**

### 4. 从新变量反推四件事
1. 逆袭后的第一优势是什么
2. 立刻会招来的新危机是什么
3. 中期必须跨过的挑战是什么
4. 最终要付的代价是什么

逆袭不是白拿好处，越早写出代价，后面的爽感越稳。

### 5. 定主线类型
#### A. 事业逆袭
重点看地位、资源、权力、战绩、家族位置、阵营位置怎么翻盘

#### B. 感情救赎
重点看关系、误会、站队、被珍惜、被选择怎么回收

#### C. 双线并进
先写清主次。  
**不要**嘴上说双线，实际两边都浅。

### 6. 设计三类必备爽点
- **翻身爽点**：第一次不再吃亏，第一次赢回来
- **碰撞爽点**：和原著重要人物重新交锋时，碰出新的关系火花
- **回收爽点**：原著里他失去的东西，这次怎么补回来

可直接写：
`原著里他最憋屈的是____，这次第一次翻身发生在____。`
`他和原著角色____碰撞时，最容易出的萌点/爽点是____。`

### 7. 让角色真的成长
每一阶段都写：
- 他学会了什么
- 失去了什么
- 对谁改观了
- 对自己有什么新认知

**从开头到结尾必须有所经历和成长。只虐别人、不改自己，逆袭线会空。**

### 8. 结局要兑现读者期待
检查：
- 读者点进来最想看什么被补偿
- 这份补偿是靠人物努力拿到，还是作者直接塞给他
- 通篇走下来，读者读完是“爽快”还是“只是看他报复完了”

最好让结局同时回收：
- 一份公道
- 一份成长
- 一份情感满足

### 9. 开篇直接落在原始痛点上
优先从这些地方开：
- 原著最不公的处境
- 原著命运拐点前后
- 原著第一次真正吃亏的地方
- 新变量刚刚生效的那一刻

## 输出格式
1. 候选角色比较
2. 最适合逆袭的主角
3. 角色原始盘
4. 新变量
5. 第一优势 / 新危机 / 中期挑战 / 最终代价
6. 主线类型
7. 三类爽点
8. 成长节点
9. 结局回收
10. 开篇场景

## 快速示例
### 输入
- 想写《红楼梦》里一个地位尴尬、家里不重视、身边全是麻烦的人物翻身

### 输出
- 原始盘：身份不低，但长期被轻视；家里没人真正替他说话；伴侣和亲族都可能拖后腿
- 新变量：提前看清家族走向
- 第一优势：比别人更早站队和自保
- 新危机：提前动作会先被当成异类或不孝
- 爽点：看他怎么一点点拿回家族话语权、怎么反压原著里轻视他的人
- 成长：不再只是会混日子，而是学会真正担责和取舍
- 结局：既有翻身，也有情感和家业上的回收

## 注意事项
- **不要**选一个本来就很圆满、没什么冲突的人硬写逆袭
- **不要**靠全员降智抬主角
- **不要**只顾打脸，不给角色成长和代价
- **不要**把原著里最值钱的委屈点一下子抹平，爽点会提前透支
- 好看的同人逆袭，本质是：**先让读者替他不值，再让读者一路看他把该拿回来的拿回来。**
