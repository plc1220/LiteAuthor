先让读者跟着主角活，再决定这一场给不给别的视角。**主角视角不是第一人称，也不是满篇心理独白，而是让场景中的信息、细节和情绪都先经过主角的所知、所感和所求。**

## 适用情况
- 开头像上帝旁白，主角像被作者摆布
- 明明是主角文，读者却代入不上
- 总写主角不该知道的信息
- 第三人称写着写着像监控录像
- 想增强苏爽文、甜文或剧情流的代入感

## 输入
- 题材类型
- 主角身份与当前处境
- 当前场景目标
- 主角此刻最想要 / 最怕失去的东西
- 读者此场需要知道 / 暂时不能知道的信息
- 当前难点：开头发飘、信息太多、视角乱切等

## 步骤

### 1. 先定这本书的主观度
按题材先选主观度：
- **苏爽文**：更主观，第三人称也要尽量贴近主角当下感受
- **甜文**：主客观相叠，既要代入，也要给关系留观察空间
- **剧情流**：仍优先跟着主角走，但心理不要一大块一大块解释
- **梗文 / 群像**：可以局部放宽，但每场戏仍要有主跟随对象

先写一句：
`这本书主要让读者跟着谁一起紧张、爽、疼、心动。`

### 2. 开头优先跟着主角进场
不要为了显得“像正剧”故意把主角藏起来。
除非结构有强理由，否则开篇优先让读者尽快知道：
- 主角是谁
- 主角现在处于什么局面
- 主角最先在乎什么

**先让读者抓住主角，再谈世界和群像。**

### 3. 每场戏先答主角四问
动笔前先写清：
- 他 / 她现在最想要什么
- 他 / 她现在最怕什么
- 他 / 她现在知道什么、不知道什么
- 他 / 她进入场景后第一眼会注意什么

这四问不清，细节就会飘，作者感就会重。

### 4. 让细节先经过主角过滤
不要把场景里所有信息平均写出来，只写会刺中主角的部分。
- 想自保的人先看危险和退路
- 想复仇的人先看破绽和站队
- 想表白的人先看距离、语气和回应
- 想查真相的人先看异常点

**同一场景，不同主角看到的世界应该不一样。**

### 5. 第三人称也要写出“贴脸感”
第三人称近距离写法推荐链条：
`外部刺激 → 主角判断 → 身体 / 情绪反应 → 下一步动作`

少写总结式旁白，多写当下感受：
- 视觉
- 听觉
- 触觉 / 身体感觉
- 气味 / 温度 / 痛感等即时感觉

可以不写“我”，但要让读者明显站在主角这边看。

### 6. 只写主角有资格知道的事
在主角视角里：
- 可以直写主角的判断、误会、偏见、欲望
- 不能直写别人的真实心理，除非切视角
- 不能把主角不知道的信息硬塞进来冒充伏笔

别人的心思，优先通过：
- 语气
- 停顿
- 动作
- 视线
- 站位
- 行为结果
来推。

### 7. 不要把主角写成剧情傀儡
高危写法：
- 主角没有合理动机，只是被剧情推着跑
- 为了埋伏笔，老写主角不该看到也不该关心的东西
- 明明该有反应，主角却像木头，只因作者要拖一段

先问：
- 这件事主角为什么在意
- 以他的身份和性格，第一反应应该是什么
- 如果他真不会这么做，是不是该改剧情

### 8. 也不要把第三人称写成碎碎念第一人称
贴近主角，不等于不停自我解释。
控制方法：
- 心理只写当下最必要的一笔
- 能用动作和场面展示的，不要全改成自白
- 一段里先让事情发生，再让主角判断，不要大段停下来讲道理

### 9. 用“四觉 + 一判断”改写平段落
把平铺直叙改成：
- 看见了什么
- 听见了什么
- 身上有什么感觉
- 环境有什么压迫或刺激
- 主角立刻怎么理解这件事

### 10. 题材校准补充
- 苏爽文：主角的感受和立场要足够接地气，读者才能代入爽点
- 甜文：主角视角要能看见对方的魅力，但别把对方写成挂件
- 剧情流：主角可以普通，但必须在关键场面里显出“不凡”，剧情也应尽量从他的选择里长出来

## 输出格式
1. 题材与主观度
2. 主角四问
3. 当前场景该写 / 不该写的信息
4. 主角会先注意的细节
5. 一段主角视角写法示例
6. 作者在场感风险点

## 快速示例
### 原句
塌了一半的隧道里，救援人员来回奔忙。有人惊呼伤者心脏都被刺穿了。

### 改写
微弱的灯光一晃一晃，湿冷顺着皮肤往骨头里钻。耳边脚步声乱成一片，有人扯着嗓子喊“这边还有活的”。下一秒，更近的一句砸下来——心脏都被刺穿了。意识发沉，唯一清楚的是：自己还没死。

### 改写说明
同样的信息，改成先经过主角的视觉、触觉、听觉和即时判断，代入感会更强。

## 注意事项
- **禁止**同一段无标记乱切别人的心理
- **禁止**把世界说明书硬塞进主角当下感受里
- **禁止**为了“客观”把主角写成监控摄像头
- **禁止**为了“主观”写成整页碎碎念
- 主角视角的核心不是“我”，而是：**这件事先落到主角身上，再落到读者心里**
