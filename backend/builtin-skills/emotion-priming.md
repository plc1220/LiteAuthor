先把目标情绪酝酿到位，再下笔写场景。**很多段落写不动人，不是因为不会技巧，而是情绪还没被点亮。先把情绪点亮，再把它翻成动作、细节和后果。**

## 适用情况
- 高潮、转折、告白、崩溃、离别、爆发写得发空
- 知道角色该难过、该燃、该心动，但下笔就是平
- 越想技巧越写不出感觉
- 想把情绪写得更细，而不是只会“很悲伤”“很激动”

## 输入
- 当前场景一句话
- 想打的主情绪与副情绪
- 角色当前处境
- 这场戏要完成的变化
- 作者更容易被什么触发：音乐 / 影视 / 动漫 / 小说片段 / 现实经历 / 环境氛围

## 核心原则
1. **先点亮情绪，再组织句子。**
2. 选真的能触发感觉的输入，不选“理论上应该有用”的输入。
3. 沉浸时先感受，**不要先拆技巧**。
4. 抓的是细微层次，不是一个大词压到底。
5. 情绪最终必须落到场景里，不能只停在结论句。

## 步骤
### 1. 先把情绪说具体
不要只写“难过、开心、燃”。
改写成更可执行的状态：
- 压抑到快爆
- 心软里带防备
- 想哭但不肯丢脸
- 高兴过头反而发虚
- 明明松了一口气，却更害怕失去

### 2. 选 1—3 种情绪输入载体
根据场景和作者习惯，给出最合适的输入来源：
- **音乐**：适合先拉整体情绪底色
- **影视 / 动漫片段**：适合捕捉动作、停顿、眼神、爆发节奏
- **小说片段**：适合提取语言层次和心绪递进
- **现实经历 / 生活环境**：适合抓最真实的身体反应和杂念

**优先选“真的能被打动”的，不优先选“名作”。**

### 3. 沉浸时先关掉学习脑
沉浸阶段不要先问：
- 这段镜头怎么切
- 这句台词怎么写
- 这个套路是不是高级

先只做一件事：**让自己完整进入那种情绪里。**

如果一次不够，就允许重复同一段载体，直到细微感受开始出现。

### 4. 记录“情绪反应单”
沉浸后，立刻抓下面 5 类信息：
1. 身体反应：喉咙发紧、手发麻、肩膀僵住、胃往下坠
2. 第一冲动：想骂、想逃、想笑、想装没事、想靠近又停住
3. 第一念头：一句自我安慰、自嘲、嘴硬或失控前的话
4. 相反情绪：高兴里带慌、愤怒里带委屈、悲伤里带麻木
5. 外界锚点：灯光、雨声、空调声、纸杯、门把手、手机震动

### 5. 把情绪翻成场景链条
按这个顺序整理：
`刺激 → 第一反应 → 外显动作 / 对白 → 周围反馈 → 结果`

情绪一旦不能改变动作、对话、站位或结果，就还没真正落地。

### 6. 立刻给这场戏安排 3—5 个可落笔细节
优先产出：
- 一个身体反应
- 一个想做又忍住的动作
- 一个能承载情绪的物件
- 一句最先冒出来的心里话或台词
- 一个会留下余波的结果

### 7. 控力度，别把情绪一次榨干
如果写出来过满、过猛、过直白，优先这么收：
- 少说一句总结
- 提前半拍停住
- 用动作代替自白
- 用细微波动代替持续嚎叫

**适当中止，往往比一路喊满更动人。**

### 8. 某种情绪点不亮时，换“能真触发”的入口
如果某种情绪总调不出来：
- 换一种载体，不要死守单一方法
- 换成相邻情绪先进入，再往目标情绪走
- 先找作者自己真正会快乐、愤怒、紧张、心酸的东西

**不要硬逼出一种假情绪。**

## 输出格式
1. 场景目标与主 / 副情绪
2. 推荐情绪输入载体
3. 沉浸时重点捕捉的 5 类反应
4. 可直接写进正文的细节 3—5 个
5. 情绪场景链条
6. 力度过猛时的收法
7. 一段示范写法

## 快速示例
### 输入
- 场景：主角在重大挫折前夜，表面平静，实际快撑不住
- 主情绪：压抑、不甘
- 副情绪：麻木、嘴硬

### 输出
- 推荐载体：低压、缓慢堆积的音乐；人物强忍失控的影视片段；自己某次明明很难受却还在装没事的记忆
- 捕捉点：下颌绷紧、手指反复掐纸杯、想解释又懒得解释、看见灯光就烦、脑子里只剩一句“再撑一下”
- 可用细节：纸杯被捏出折痕；回消息只回“没事”；说完转身时脚步明显快了；门关上后先笑一下再沉下去
- 落地方式：不要直接写“他很痛苦”，先写他怎么装、怎么忍、怎么在一个小动作里漏出来

## 注意事项
- **不要**一边酝酿情绪，一边急着分析技巧
- **不要**只记“很难过”“很开心”这种大词
- **不要**情绪很重，场景里却没有动作、物件和后果
- **不要**把所有情绪都写成哭、吼、发呆
- 真正有层次的情绪，不是词更大，而是**反应更细、动作更准、余波更真**。
