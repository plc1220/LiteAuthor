先选复盘模式，再找证据，不要笼统说“这文不行”。

## 适用情况
- 写到中途越写越歪，想及时纠偏
- 某段情绪一直压着，读者开始烦
- 完结后想总结哪里精彩、哪里疲软
- 感情线、剧情线、成长线不同步
- 支线很多，不知道哪些该删
- 想从单本文里提炼自己的写作体系

## 输入
- 章纲、章节简述或已写正文
- 当前想复盘的范围：近几章 / 某一卷 / 全文
- 作者自觉问题
- 读者反馈（可选）
- 若已有个人写作体系，先写出观察线；若没有，默认用四条线：剧情线、感情线、成长线、主题线

## 步骤

### 1. 先定观察线
优先写清本次复盘到底看哪几条线。

默认四条：
- **剧情线**：外部事件、目标、冲突、结果
- **感情线**：关系状态、节点变化、拉扯方式
- **成长线**：主角认知、能力、立场、选择变化
- **主题线**：作品想表达什么，靠哪些剧情表达

如果作者有自己的写作体系，允许替换，但**必须固定几条线反复观察**，不要今天看这个、明天看那个。

### 2. 选择复盘模式

#### A. 章纲复盘（写作中途用）
1. 建一张表，每章至少记录：
   - 章号
   - 这一章客观发生了什么
   - 推进了哪条线
   - 情绪是**压 / 扬 / 平**
   - 章尾钩子
   - 读者反馈（可选）
2. 连着看 5-10 章，重点查：
   - 某条线是否停了太久
   - 同一种互动是否在重复
   - 是否连续多章都在“压”而没有“扬”
   - 是否存在只热闹、不推进任何线的章节
3. 一旦发现失衡，立刻给出处理：
   - 修前文
   - 压缩重复桥段
   - 提前放一个推进节点
   - 补一个“扬”的回收场景
   - 把后续矛盾改成能真正推进关系/成长的矛盾

#### B. 全文复盘（完结后或写到大后期用）
1. 先把章节按大剧情归并成 3-8 个大段
2. 每个大段都总结四件事：
   - 这段大剧情是什么
   - 男女主关系推进到了哪一步
   - 主角或关键人物的成长高光在哪里
   - 主题是怎么被剧情表达的
3. 把这些放到一张表里，对照检查：
   - 大剧情和感情节点是否同步
   - 成长是否线性，还是反复横跳
   - 主题是否只在嘴上说，没有被剧情完成
   - 高潮是否都挤在前期/后期，导致中段发虚
4. 再把“作者自己觉得顺”和“读者反应最强”的段落单列出来，对比“作者也烦、读者也烦”的段落，看差别到底在哪

#### C. 部件拆解（问题说不清时用）
如果仅凭章纲/全文复盘还说不清问题，再额外拆六块：
1. 人物
2. 情节
3. 描写
4. 叙述方式
5. 结构（剧情 / 感情 / 成长 / 主题）
6. 节奏与信息量

### 3. 重点查五类高频问题

#### ① 情绪失衡
- 连续多章都在吵、误会、压抑、受气
- 情绪只会“压压压”，没有缓冲、回甜、回收
- 爽点、泪点、缓和点分布失衡

#### ② 线索不同步
- 大剧情很大，但感情线没动
- 感情拉扯很多，但人物没成长
- 主题想写 A，实际剧情一直在演 B

#### ③ 支线失效
- 配角支线很热闹，但对主角关系、成长、主题没有推动
- 男女主已经没有内部问题，却在外部副本里一起打怪打太久
- 某段剧情删掉后主线反而更清楚

#### ④ 角色关系踩读者情绪
检查某些关系设计是否挤占了题材读者最在意的“核心资源”，却没有做足补偿。

例如：
- 言情里，某个高存在感女配长期占走男角色关注，却没有被重新定位为别的关系价值
- 角色定位和读者默认期待冲突，导致“设定上没错，情绪上不讨喜”

这种情况要处理的不是“逻辑”，而是“情绪站位”。

#### ⑤ 冲突来源失效
- 前期冲突在主角之间，后期冲突全跑到主角之外
- 事件很大，但不能逼主角选择、成长、靠近或撕裂
- 冲突存在感很强，推进作用却很弱

### 4. 把问题变成可执行结论
每个问题都按下面格式落地：
- **证据**：哪几章 / 哪一段出现了什么
- **判断**：它具体拖慢了哪条线
- **处理**：删 / 并 / 改 / 提前 / 后置 / 重写
- **规律**：以后遇到同类问题，优先怎么判断

### 5. 输出下一本能直接用的规则
不要只总结“这次哪里写坏了”，还要再提炼成 3-5 条个人规则。

例如：
- 连续三章以上只压不扬，就必须回收一次
- 任何配角支线都要能解释“它推进了主角哪条线”
- 大剧情和感情节点不能长期脱节
- 如果一段冲突删掉后主线更清楚，就优先删

## 输出格式
1. 采用的复盘模式
2. 观察线
3. 章纲 / 全文复盘表
4. 主要问题列表
5. 最疲软段 vs 最顺段对比
6. 立即可执行的修改方案
7. 下本可复用的写作规则

## 快速示例
### 示例 1：章纲复盘
输入：近 6 章都是男女主吵架、误会、吃醋

输出：
- 感情线实质推进：0
- 情绪分布：连续 6 章都是“压”
- 问题：重复互动 + 情绪失衡
- 处理：第 7 章必须给出一次关系回收或信息澄清；删掉其中 2 章同义争吵

### 示例 2：全文复盘
输入：前期很好看，中后期读者说水

输出：
- 前期冲突来源：男女主之间的核心问题
- 后期冲突来源：大量与主角成长无关的配角事件
- 结论：后期把矛盾外包给支线，主角线停了
- 处理：压缩无效副本，把事件改成直接逼主角关系和成长

## 注意事项
- **禁止**只说“文笔不行”“节奏不行”，必须给证据
- **禁止**只看逻辑，不看题材读者的情绪反应
- **禁止**同时大改所有问题，先抓最拖后腿的 1-2 项
- 复盘的目的不是自责，而是形成下一次更稳定的写法
