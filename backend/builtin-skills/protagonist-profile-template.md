先把主角写成有入口、有层次、还能往上长的人。**主角至少要有三层：明面性格、暗面性格、转变性格；还要至少留一条明显上升空间。主角不怕强，怕的是开局就到天花板。**

## 适用情况
- 主角太完美、像模板，读者难代入
- 开局超神，后面只剩重复证明自己
- 有设定没活人感
- 前十章过去了，读者只记得设定不记得人
- 想写带感主角，又怕像玛丽苏 / 杰克苏

## 输入
- 题材与目标读者
- 主角身份、处境、能力
- 一句话卖点
- 开篇场景
- 主角后期会变成什么样

## 核心原则
1. **先有三层，再谈复杂。** 明面给第一印象，暗面给驱动力，转变给成长线。
2. **要有小缺点和生活习惯。** 没有毛边的人最像纸片。
3. 缺点用来增真实，**不能直接毁掉主角基本盘**。
4. 主角最好有一条脊梁、原则或想守住的东西。
5. 设定词必须落到场面里，不能只写在资料表里。
6. **能力、认知、关系至少留一条上升空间；别把三条都点满。**

## 步骤
### 1. 先写主角最值得看的点
填空：读者愿意继续看他 / 她，主要因为____。

### 2. 写三层性格
- 明面性格：读者一眼看到的样子，如冷静、张扬、嘴毒、温吞。
- 暗面性格：真正驱动选择的那层，如自卑、执拗、缺爱、控制欲、强烈羞耻感。
- 转变性格：后期要长出来的新层，如天真变有边界，冷漠变会在乎，冲动变会算代价。

句式：这个人表面上____，骨子里其实____，后来会因为____变得更____。

### 3. 选 3—5 个共鸣入口
优先从勇气、遭遇不公、有本事、善良、幽默感、身处险境、被真心在乎、肯努力、执着、逆袭潜力里选。

### 4. 补小缺点与习惯
- 小缺点：嘴硬、记仇、爱面子、轻微控制欲、容易钻牛角尖。
- 习惯：紧张时摸戒指、说谎前停顿、挑食、爱收集某样东西。

要求：**缺点最好无碍大原则，习惯最好能反复出现。**

### 5. 给主角留成长空间
从下面三条里，至少留两条“还没满”：
- **能力 / 资源**：还不会什么、还拿不到什么、还打不过谁、还缺什么资格
- **认知 / 性格**：还有什么盲点、旧伤、防御方式、不会表达、不会合作
- **关系 / 处境**：还没被认可、还没拿回话语权、还没得到真正重要的人 / 位置 / 阵营

检查三句：
- 开篇就把读者最想看的大问题解决了吗？
- 中期还有什么能明显上涨？
- 后期的爽点 / 成长，是“真的变强了”，还是只换地图继续碾压？

**开局最多让一条线很强，别三条全满。**

### 6. 写主角的脊梁
至少回答三句：
- 最不能接受什么
- 真到两难时最想守住什么
- 即使吃亏也不愿轻易放弃什么

### 7. 写关键旧事、长期影响和触发点
句式：因为过去____，所以现在一遇到____就会____。

### 8. 把形容词改成稳定反应
不要只写聪明、善良、果断，要改成：
- 遇事先怎么处理
- 对亲近者、陌生人、对手分别怎么说话
- 被戳中软肋时会怎么失控或收紧

### 9. 安排四类证明场面
1. 开场证明：前三章先吃到 2—3 个标签
2. 压力证明：真有事时这些标签还能站住
3. 两难证明：主角为底色付代价
4. 转变证明：后期再遇同类问题，做出和前期不同的选择

### 10. 排查太完美风险
逐条问：
- 是不是只有优点没有代价
- 缺点是不是只写着玩
- 习惯有没有真正进入生活
- 三层性格是不是三张皮
- 前十章能不能看出为什么值得跟
- **是不是入场就已经超神，后面只剩反复证明自己**
- **后续成长是不是只在加设定，没有改人物和关系**

## 输出格式
1. 主角最值得看的点
2. 明面性格 / 暗面性格 / 转变性格
3. 3—5 个共鸣入口
4. 小缺点
5. 习惯
6. 成长空间：能力 / 认知 / 关系
7. 脊梁 / 原则 / 信念
8. 关键旧事 / 影响 / 触发点
9. 4 个证明场面
10. 当前最容易翻车的点

## 注意事项
- **不要**只给主角贴高冷、聪明、强大就开写
- **不要**为了讨喜把人写得没脾气没瑕疵
- **不要**把“开局强”写成“开局没上升空间”
- **不要**只有设定，没有场景证明
- 主角先要解决的不是够不够复杂，而是**读者为什么愿意跟，以及后面还有什么值得看**。
