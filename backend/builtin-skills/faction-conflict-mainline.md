围绕“谁和谁真正对撞”搭全文。先判断主线靠背景矛盾还是势力矛盾，再把关键人物拉到面对面，让甜点、爽点和事业推进都从同一条主要矛盾里长出来。**人物好坏不等于站位；能支撑全文的，通常只有背景矛盾和势力矛盾。**

## 适用情况
- 男女主互动很多，但就是不带劲
- 反派、炮灰一个接一个，主线却不清楚
- 感情线和事业线像两篇文
- 有冲突，但都只是吵一下就过去
- 不知道该让谁和谁正面顶上

## 输入
- 一句话故事
- 想卖的主要体验：甜 / 爽 / 拉扯 / 升级 / 复仇
- 关键人物清单
- 当前阶段最想推进的主线
- 卡住的剧情段落（可选）

## 步骤

### 1. 先判断全文靠哪类矛盾撑
#### 背景矛盾
人物和恶劣环境、世界规则、时代处境对抗。
常见于：末世、无限流、生存文、行业逆袭文。

#### 势力矛盾
一个人或一群人的目标，和另一方的目标正面相撞。
常见于：言情拉扯、职场、宫斗、夺权、复仇、竞争文。

**普通人物矛盾往往只能撑一段；能撑全文的，通常得升级成背景矛盾或势力矛盾。**

### 2. 如果卖点是拉扯，优先让男女主站到主矛盾两边
尤其言情文，先别急着把对方当救场工具人。
先按这个句式想：
- 女主想要什么
- 男主眼下为什么偏不让她顺利拿到
- 男主想做什么
- 女主为什么偏不接受

**设计场面时，可以先把恋爱对象临时当成“当前场景的反派”来思考。**
不是写他坏，而是写他和主角的目标相反。

### 3. 先做“面对面”，不是让第三人替他们冲突
判断一个冲突是不是主矛盾，看这一条：
- 冲突是不是直接顶到关键人物脸上

反例：
- 经理刁难女主，男主只是路过开除经理  
  这更多是男主救场，不是男女主矛盾

正例：
- 男主虽然帮了女主，但转头把事件包装成公司宣传，让女主被迫当企业样板  
  这时，男主的处理方式本身就和女主正面对撞，矛盾才真正成立

### 4. 按“当前目标”给人物分势力，不按好坏分
对每个角色写三件事：
- 他在这一段帮谁达成目标
- 他在这一段阻止谁推进
- 他存在的功能是什么：加压 / 缓冲 / 见证 / 误导 / 推对撞

**站位看作用，不看道德标签。**
同一个人，上一段在男主方，下一段也可能转到女主方。

### 5. 让配角成为势力延伸，而不是独立乱窜
常见可用角色：
- 闺蜜、同事、下属、长辈、情敌、同桌、小弟、客户、队友

关键不在身份，而在功能：
- 他是不是把一方的意志、资源、误会、压力传到另一方身上
- 他是不是让主角们不得不更早面对面

### 6. 安排“摇摆角色”增加趣味
最好留 1—2 个站位会变化的人。
这类角色最适合：
- 一开始站 A 方
- 后来因为利益、情感、成长或误会，慢慢偏向 B 方
- 偶尔反复横跳，制造新波澜

这类角色能让关系网活起来，也能自然增加拉扯感。

### 7. 男女主暂时不能直接翻脸时，用势力矛盾托住
如果两人表面上还不能公开对立，就让他们的势力先撞上：
- 女主方的闺蜜 / 项目 / 旧友 / 立场，和男主方的公司 / 家族 / 团队 / 任务相撞
- 双方先在“替各自那一边做事”时碰撞，感情再从碰撞里长

### 8. 日常场面也要带矛盾
日常不等于没冲突。
小场面也能装势力矛盾，例如：
- 送吃的
- 传消息
- 选座位
- 分任务
- 一次补课
- 一次项目协作

只要这一场的行为背后代表某一方立场，日常也能很有拉扯。

### 9. 甜点、爽点和事业推进都从同一条矛盾里长
逐条检查当前事件：
- 这件事是谁的目标
- 它为什么让关键人物站到相反位置
- 第三人如何放大对撞
- 事件结束后，事业有没有推进，关系有没有变化，读者有没有吃到甜 / 爽

**最好同一件事同时推进两条以上。**

### 10. 每场戏都要留下“站位后果”
关键戏后至少回收 1—2 项：
- 谁更靠近谁
- 谁更防备谁
- 谁从中立变站队
- 哪一方损失了资源或面子
- 哪个配角以后会变成某一方延伸

没有后果的拉扯，很快就会塌成水戏。

## 输出格式
1. 一句话故事
2. 全文主矛盾类型：背景矛盾 / 势力矛盾
3. 当前阶段的主矛盾双方
4. 面对面设计
5. 角色站位表
6. 摇摆角色与变化条件
7. 当前事件如何同时推进甜 / 爽 / 事业
8. 站位后果
9. 跑偏风险

## 快速示例
### 示例 1：办公室线
- 反例：女主被上司欺负，男主来救场  
  只有救场，没有男女主对撞
- 正例：男主帮女主处理上司，却顺手把她包装成公司公关案例  
  女主觉得自己被利用，当场怼男主  
  这时才有真正的面对面矛盾

### 示例 2：校园线
- 男主的小弟原本代表男主方态度
- 女主给小弟补课后，小弟慢慢偏向女主方
- 小弟转站后，他和男主的互动就成了女主方与男主方的互动
- 这种“站位变化”能自然长出很多拉扯和趣味

## 注意事项
- **不要**把“谁是坏人”直接等同于“谁在哪一方”
- **不要**让炮灰独立刁难主角，却和主线无关
- **不要**只写第三人挑事，不写关键人物面对面
- **不要**把日常写成纯闲聊，忘了立场
- 真正好用的矛盾，不是吵得凶，而是**能一直往下长剧情**
