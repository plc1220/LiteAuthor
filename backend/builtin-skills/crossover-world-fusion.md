先定总规则，再让世界碰撞。**跨世界观同人最怕的不是角色多，而是规则没统一、冲突只剩串场。稳的写法是：前半段先让世界慢慢并轨，后半段让新规则开始反噬主角和关系，逼出真正的剧情与成长。**

## 适用情况
- 写综漫、综英美、综武侠、跨神话或多原作融合
- 人物都很带感，但放在一起后世界规则打架
- 只会把角色凑同框，不知道主线怎么长
- 前面搭设定很兴奋，后面开始失控
- 想写“强者跨到新世界”或“多个世界慢慢融合”

## 输入
- 参与融合的原著 2—4 个
- 每个世界的核心规则、力量体系、时代和阵营
- 主角是谁，原本站在哪个世界
- 融合方式：穿越 / 世界碰撞 / 时间线重合 / 血统 / 召唤 / 平行宇宙
- 想先卖世界变化，还是先卖人物冲突

## 步骤
### 1. 先写每个世界的“不能不管的硬规则”
每个世界至少列：
- 力量从哪来
- 谁掌权
- 最大禁忌是什么
- 角色最看重什么
- 这个世界最容易制造什么冲突

**不能只记角色名，不记世界运行方式。**

### 2. 选一个总融合轴
常用融合轴：
- 时间线并轨
- 地理拼接
- 同一神系 / 血统追根
- 重大事件引发世界重叠
- 穿越者 / 强者落地改变局面
- 平行世界互相渗透

一句话写：
`这些世界之所以能共存，是因为____。`

### 3. 统一“总规则”
至少回答：
- 力量体系怎么换算
- 不同世界的身份、机构、神明或官方如何互相承认
- 什么东西会被冲突放大
- 什么东西必须付代价才能跨过去

没有总规则，后面所有高光都像作者临时改判。

### 4. 决定先写哪一种开局
#### A. 强者落地型
一个强角色进入新世界，用他的视角观察和改动环境

#### B. 世界并轨型
多个世界先从局部异常开始，慢慢影响到大众秩序

#### C. 事件牵引型
用战争、灾变、比赛、召唤、神迹等大事件把世界拉到一起

### 5. 把结构分成前后两段
#### 前半段：世界搭建
- 从小范围变化写到大范围变化
- 先让读者看见规则怎么被打破
- 主角在旧秩序还没完全成形前，相对自由，能主动改变很多人和事

#### 后半段：规则反噬
- 当新世界开始稳定，机构、阵营、限制、舆论和旧账一起上来
- 主角不再能随便出手，必须在新规则里做选择
- 人与人之间的冲突、立场和因果开始集中爆发

**前半段卖新鲜感，后半段卖代价和成长。**

### 6. 冲突不要只写战力，要写观念和立场
每组主要冲突都写清：
- 他们为了什么起冲突
- 谁想保住旧秩序
- 谁想利用新秩序
- 谁最先在融合后失去利益
- 谁最可能和主角结盟或反目

### 7. 让世界变化逼出主角成长
写清三层变化：
- 世界怎么变
- 这种变化怎么改变主角生存环境
- 主角因此不得不放弃什么、学会什么、承担什么

**外部世界不改变主角，融合就只剩布景。**

### 8. 提前埋“因果回收”
把前半段主角做过的事、帮过的人、结过的盟留到后半段回收：
- 谁会在主角力有不逮时回头帮他
- 哪个早期善意会变成后期援手
- 哪个早期选择会反过来成为代价

### 9. 每个并入的世界都要承担功能
检查每个世界是否负责：
- 提供新冲突
- 改写旧关系
- 制造新规则
- 抬高主线 stakes

如果一个世界只负责彩蛋和客串，优先删掉或后置。

## 输出格式
1. 参与世界清单
2. 每个世界的硬规则
3. 总融合轴
4. 统一总规则
5. 开局类型
6. 前半段世界搭建节点
7. 后半段规则反噬节点
8. 主要阵营与立场冲突
9. 主角成长路径
10. 前期因果如何后期回收

## 快速示例
### 输入
- 想写一个原本来自高武世界的强者，进入规则更细密、机构更完整的新世界

### 输出
- 融合轴：强者穿越落地，引发两个体系接触
- 前半段：主角靠旧世界能力解决眼前危机，逐步带来连锁变化
- 后半段：新世界开始识别、监管并限制这套力量，主角不能再只凭个人强推
- 成长：从“我能改就改”变成“我必须考虑规则、同伴和后果”
- 回收：前半段被主角帮助过的人，在后半段成为他的关键援手

## 注意事项
- **不要**只把角色摆在一起同框，不解释规则为何能共存
- **不要**战力冲突全靠作者拍脑袋裁判
- **不要**前半段一直爽，后半段却没有新秩序和代价
- **不要**一味加世界，不增加真正冲突
- 好看的跨世界观同人，不是“人多”，而是**新世界一形成，所有人的命运和选择都被迫改写。**
