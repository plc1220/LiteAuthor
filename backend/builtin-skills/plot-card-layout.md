把脑子里觉得带感的片段，全拖到桌面上排。**故事写完觉得没那么有趣，往往不是文笔先出问题，而是有趣的情节卡太少、太散，或者篇幅比素材长。**

## 适用情况
- 脑子里想很精彩，写出来发淡
- 写到后面全是对话和过渡，自己都觉得无聊
- 不知道这个故事该写长还是写短
- 总是边写边加新点子，结果越写越乱

## 核心原则
1. **先数真有戏的桥段，再定篇幅。**
2. 情节卡不是「发生了一件事」，而是「发生后有什么变化」。
3. 卡不够，就加戏或缩篇；不是硬把空白用水文撑满。
4. 每章都该有小起伏、小信息或小回收，不要整章平推。
5. 新点子只有在增强全局时才加入，不是因为单独好玩。

## 情节卡最少要写四项
每张卡只写 1 个桥段，并填：
1. 发生什么
2. 为什么有趣
3. 这张卡改变了什么
4. 前置条件是什么

## 步骤
### 1. 把所有大小高潮都写成卡
包括：
- 大高潮
- 小反转
- 关键冲突
- 有记忆点的对话
- 关系推进点
- 爽点 / 虐点 / 悬念点

先不排顺序，先把能想到的全写出来。

### 2. 区分核心卡和调味卡
- 核心卡：删掉后主线会塌
- 调味卡：删掉后主线还在，但少了趣味和层次

核心卡太少，说明主线先天发虚。
调味卡太多，说明容易写散。

### 3. 先看卡量，再看篇幅
经验上：
- 五万字左右的中短篇，通常至少要有 10 张左右能带来变化的卡
- 长篇不是把同一张卡拖长，而是按阶段继续长新卡

这不是死数字，而是提醒：
**如果卡明显不够，优先考虑缩短，不要硬拖。**

### 4. 按因果和升级重排
排卡时不要只问「哪张先写最带感」，还要问：
- 这张卡为什么会发生
- 它会把主角推进哪一步
- 它如何顶出下一张卡

把顺序排成：
`启动 → 小起伏 → 加压 → 转折 → 高潮 → 余波 / 回收`

### 5. 检查前两章
如果前两章还没出现：
- 主角
- 主要矛盾 / 危机
- 至少一张能抓人的卡

读者很容易直接走人。
**普通作者不能指望读者耐心等很久。**

### 6. 把卡分配到章节
每章至少问自己：
- 这一章主打哪张卡
- 附带哪一个小起伏 / 小对话 / 小悬念
- 这一章结尾把读者推向哪张下一卡

做不到每章都大炸，但最好每章都有 1—2 个明显波动。

### 7. 对新点子做全局检查
突然想到更好的桥段时，先问：
- 它是让主线更精彩
- 还是只让某个片段更好看
- 它会不会打乱现有卡序
- 它是不是更适合留作番外、支线或下一本

如果只是一张漂亮但孤立的卡，别硬塞。

### 8. 最后做删卡测试
逐张问：
- 删掉这张卡，主角成长 / 关系 / 主线会少什么
- 它是推进，还是重复前一张卡
- 它有没有真的改变局势

重复卡、空转卡、只会说旧信息的卡，优先删或并。

## 输出格式
1. 全部情节卡列表
2. 核心卡 / 调味卡分类
3. 卡量与篇幅诊断
4. 因果排序后的卡序
5. 前两章必须前置的卡
6. 每章的波动分配
7. 应删 / 应并 / 可留作番外的卡

## 示例
### 输入
- 计划写一个五万字故事
- 目前只有几个好玩的对白和一个结局

### 输出
- 先补卡：把所有高潮、冲突、关键关系点各写成卡
- 诊断：只有 5 张有效卡，撑 5 万字偏虚
- 建议：要么补足 5 张以上新卡，要么把篇幅压短
- 分配：保证每章至少有 1 个小冲突或小回收，不要中间十几章都在过渡

## 注意事项
- **禁止**把脑补很爽当成正文一定有戏
- **禁止**卡不够还硬拖篇幅
- **禁止**只顾堆高潮，不排因果
- **禁止**因为喜欢某个片段就强行插入
- 情节卡真正的作用，是**把「感觉这故事挺有意思」变成一条条能落地的剧情节拍**。
