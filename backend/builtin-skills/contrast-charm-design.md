先给角色一个强烈的第一印象，再在关键时刻翻出第二面、第三面。**反差不是随便翻脸，而是让读者不断发现“原来他还有这一面”，每翻一次都更可爱、更帅、更带感。**

## 适用情况
- 角色强但不讨喜，或可爱但不够出圈
- 萌宠、搭档、配角缺少记忆点
- CP 火花弱，互动太平
- 角色只有单一标签，缺乏翻面感
- 想让强者更可爱，或让可爱角色在关键时刻更帅

## 输入
- 角色一句话设定
- 想强化的魅力方向
- 角色和主角的关系
- 代表能力、软肋或标志动作

## 步骤

### 1. 先定 A 面和 B 面
给角色写两个最核心的面：
- **A 面**：初见印象
- **B 面**：反转后的真实一面

常见搭配：
- 凶 / 萌
- 强 / 懒
- 高冷 / 笨拙
- 贪吃 / 可靠
- 嘴硬 / 护短

要求：
- A 面和 B 面必须能共存
- 两面最好由同一个核心性格长出来
- 不能今天高冷、明天逗比，完全没有内在逻辑

### 2. 第一层反差要在出场就兑现
不要只写设定说明，要用场面给读者看。

例如：
- 出场像被封印的大反派，解封后却是一只小猫
- 在普通人眼里是怪人，在另一个世界里却地位很高
- 平时像吉祥物，真到关键处才显出压场实力

### 3. 读者适应后，再翻第二层
第一层反差熟了以后，再补一次更强的翻面。

常见写法：
- 以为只是可爱，结果真身帅到压场
- 以为只是很强，结果有非常笨拙的一面
- 以为只是嘴硬，结果在关键时刻最可靠

**反差最好不是一次性，而是层层翻。**

### 4. 做“只对你特别”的对象差异
最容易出萌点和 CP 感的，不是“他对所有人都好”，而是：
- 对外人利落、冷硬、嫌麻烦
- 对主角嘴硬心软、总是破例
- 打别人很干脆，对主角却舍不得下重话

这种对象差异，会让反差从“角色设定”升级成“关系化学反应”。

### 5. 给反差设触发条件
反差不能乱跳，要有稳定触发器。

例如：
- 平时懒散，主角遇险时立刻认真
- 平时嘴上说不帮，最后总会出手
- 吵架后说绝不管了，听见主角有危险还是先护短

**有触发条件，反差才像人物规律，不像 OOC。**

### 6. 把反差落成“动作链”
不要只写结论，要写成可重复出现的动作模式。

常见动作链：
- 嘴上拒绝 → 行动帮忙
- 小体型卖萌 → 危机时真身救场
- 平时犯傻 → 关键时刻绝不掉链子
- 先凶后软 → 先帅后笨拙

这种“固定回路 + 轻微变化”，最容易让角色出圈。

### 7. 给关系再加一个钩子
如果想让反差不止可爱，还能快速拉近关系，就再加一个关系钩子：
- 两人共享同一个秘密
- 只有彼此看得见另一面
- 只有主角知道他真正强在哪里
- 只有这个角色会对主角露出反常的一面

**共享秘密 + 反差，会让关系升温非常快。**

### 8. 控制反差数量
通常 1—2 个主反差就够了。

如果同时塞进：
- 高冷
- 傲娇
- 沙雕
- 病娇
- 反派气场
- 萌宠感

很容易变成混沌，不是反差，是没定型。

### 9. 最后检查是否真的动人
逐条问：
- 读者能不能记住至少一个具体场面
- 这份反差是不是只对某些人、某些场景成立
- 它有没有推动关系或剧情
- 删掉这份反差，角色会不会立刻变平

## 输出格式
1. A 面 / B 面
2. 第一场出场反差
3. 第二场升级反差
4. 只对谁特别
5. 触发条件
6. 反差动作链
7. 会推进哪条关系或剧情
8. OOC 风险

## 快速示例
### 输入
- 角色：守护型大妖搭档

### 输出
- A 面：被封印时像危险反派
- B 面：解封后是贪吃小猫
- 升级反差：危机时露出真身，强大到压场；脱险后又因为小失误显出笨拙
- 对象差异：平时嫌主角爱多管闲事，真遇到危险却永远第一个冲上去
- 关系钩子：只有他和主角知道某个重要秘密

## 注意事项
- **禁止**只写“这个角色很反差”，却没有具体场面
- **禁止**今天一个人设，明天一个人设，没有触发逻辑
- **禁止**对所有人都同一种软化方式
- **禁止**让反差只负责可爱，不改关系、不推剧情
- 好反差不是乱，而是**层层翻面，越翻越真**。
