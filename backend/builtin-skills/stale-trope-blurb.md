把大众热梗拆成“基础梗 + 加码点 + 打脸场面 + 留白”，在熟悉套路里抬高读者期待值。**烂梗不可怕，可怕的是文案写得像别人那一百本。目标不是发明新题材，而是让读者觉得：这个组合我想看。**

## 适用情况
- 火葬场、替身、真假千金、白月光等热梗太常见
- 文案一眼看完没有点击欲
- 自己的文和同赛道作品区别不明显
- 不知道该把哪个狗血点放进文案，哪个留给正文
- 想让大众梗“再高半档”而不是硬编冷门设定

## 输入
- 频道 / 题材
- 基础热梗
- 主角受伤点 / 困局
- 可用附加元素（身份反转、真假千金、白月光、长辈 / 权势关系、职业反差等）
- 想要的核心爽感
- 旧文案（如有）

## 步骤
### 1. 先承认这就是大众梗
写一句：
`这篇文最基础的热梗是____。`

例如：火葬场、替身、真假千金、白月光、修罗场。

不要一上来追求“完全没见过”。大部分读者点进来，本来就是为了看熟悉的快感。

### 2. 先写读者默认能脑补的那一套
把套路链简写出来：
- 谁先受委屈
- 谁看错 / 伤害了谁
- 后面理论上会怎么回舔 / 追悔 / 打脸

这一步只是为了分清楚：
- **哪些必须写进文案**
- **哪些读者自己会脑补，不值得占字数**

例如火葬场文里，“男主后期后悔”通常不用细写，读者早知道。

### 3. 只选一个“期待值加码点”
从所有附加元素里只挑一个最能抬高期待值的：
- 大热里相对更新鲜的
- 能和基础梗直接咬合的
- 一眼能看出更想看的结果的

常见加码方向：
- **替身 + 真假千金**：身份翻盘更爽
- **火葬场 + 白月光亲自下场**：仇恨对象更具体
- **退婚 + 嫁给渣男长辈**：地位反转更直接
- **受气 + 家世翻盘**：打脸层级更高
- **大众梗 + 少见连接方式**：不是多一个梗，而是把两个梗真正拧在一起

**不要同时塞三四个加码点。文案里最值钱的是“这一刀扎在哪”。**

### 4. 把加码点落成一个具体场面
不要只写“后来她翻身了”，改成读者能立刻想象的名场面：
- 前未婚夫给她奉茶，喊“四婶”
- 白月光拿支票上门，结果被真千金身份反压
- 以为女主是替身，结果豪门寻亲宴上全场改口

**一句就够，但必须有画面。**

### 5. 让主角站在“有理位置”
大众爽文很吃立场。

检查主角是否站稳：
- 主角是不是被伤害的一方
- 主角有没有尊严底线
- 主角是不是在已知真相后还主动作践自己
- 反派的坏是不是明确、直给、容易共情

如果要写纯爽文：
- 可以让反派一坏到底
- 不必急着加复杂灰度
- 先保证读者情绪站在主角这边

### 6. 少写过程，多写最想看的结果预告
文案里优先写：
- 当前困局
- 关键加码点
- 最勾人的打脸预告

少写：
- 大段过程复盘
- 读者能自动脑补的追妻 / 回舔流程
- 为了圆逻辑塞进来的解释文字

记住：  
起因 / 过程 / 结果，**通常只放两个**。  
最稳的做法是：`受伤点 + 加码点 + 一句反转场面`

### 7. 做“同赛道辨识度测试”
把人名全去掉，再问：
- 这段文案像不像随手能找出十篇一样的？
- 读者记住的是题材词，还是那个特别的连接点？
- 最后一句能不能让人想看后续打脸？

如果还像模板公文，就再加强一项：
- 身份差
- 亲属 / 权力关系
- 外部对照组
- 更具体的打脸对象
- 更清晰的利益归属反转

### 8. 常用骨架
#### 骨架 1：火葬场 / 替身类
`她陪在他身边____，后来才知道自己不过是____。`
`____亲自上门，逼她让位。`
`她转身离开，本以为只是止损。`
`直到____，所有人才知道，原来真正被看错的人是____。`

#### 骨架 2：身份翻盘类
`所有人都以为____。`
`连主角自己也这样以为。`
`直到____出现，旧身份被当场推翻。`
`那个最看不起她的人，只能____。`

#### 骨架 3：关系地位反转类
`前一秒还是____。`
`下一秒，因为____，她成了____。`
`最该难堪的人，当场就得改口。`

## 输出格式
1. 基础热梗
2. 读者默认会脑补的部分
3. 期待值加码点
4. 主角站位检查
5. 最该写进文案的打脸场面
6. 文案 2 版
   - 直给版
   - 更抓马版
7. 该藏掉的过程
8. 辨识度风险

## 快速示例
### 输入
- 基础热梗：火葬场 + 替身
- 附加元素：白月光、真假千金
- 目标：让读者更想看女主打脸

### 输出
- 基础热梗：替身火葬场
- 默认脑补：男主后期会后悔回头
- 加码点：白月光其实占着女主的身份
- 打脸场面：白月光拿支票上门时，豪门寻亲的人先一步找到了女主
- 文案正文：先写女主发现自己只是替身，再写白月光耀武扬威，最后用“原来她才是豪门走丢的千金”收尾

## 注意事项
- **禁止**为了“新”硬塞不兼容的梗
- **禁止**把所有狗血点都堆上去，没有主次
- **禁止**让主角站不住脚，还指望读者代入她爽
- **禁止**把后期常规回舔流程写得太满
- 大众梗文案的核心不是“从没人写过”，而是**读者一看就知道熟，但还是更想点你这本**
