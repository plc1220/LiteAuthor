规划下一本书怎么换题材、换背景、换人物口味，但不把旧读者和旧优势一起换没。**转型最稳的做法不是大换血，而是保 1—2 样长项，只改一个主轴。**

## 适用情况
- 连着几本都像同一套换皮
- 自己写腻了，但又不敢一下换太大
- 想突破题材或风格，担心读者不买账
- 上一本和下一本之间，拿不准该换什么、不该换什么

## 输入
- 最近 2—5 本书的题材和成绩
- 自己最稳定的长项
- 已经写腻的固定套路
- 想尝试的新方向
- 风险承受度：求稳 / 可小冒险 / 可大尝试

## 步骤
### 1. 先盘点“自己真正稳定卖得动的东西”
不要只写题材名，要写：
- 节奏是快还是慢
- 擅长写什么关系模式
- 读者最认的爽点 / 甜点
- 最顺手的人设类型
- 最容易复读的桥段

### 2. 找出让自己疲劳的重复点
用具体句子写：
- 校园文总会写到什么
- 都市文总会落回什么
- 男主 / 女主总会变成什么模板
- 哪些桥段已经像换装游戏

越具体，转型越准。

### 3. 只选一个主轴先改
优先从这几个轴里选 1 个：
- 背景：现代 → 年代 / 古代 / 末世
- 主角身份：学生 → 打工人 / 贵族 / 修士
- 关系模式：死对头 → 先婚后爱 / 养成 / 搭档
- 叙事口味：甜 → 甜爽 / 狗血 / 轻悬疑
- 世界规模：日常 → 剧情流 / 群像 / 大世界

**第一步不要同时改三四个轴。**

### 4. 再选 1—2 个保持不动的长项
例如：
- 背景换了，但甜爽节奏不换
- 题材换了，但主角带感类型不换
- 人设换了，但“被偏爱”这口不换

这样读者有熟悉入口，作者也不至于彻底失手。

### 5. 做三级转型方案
- **微转型**：只换背景或职业外壳，核心口味不变
- **中转型**：换一个主轴，再微调一个辅助轴
- **大转型**：题材、口味、结构都大改，同时标出风险

### 6. 判断哪一版适合现在
- 求稳：优先微转型
- 想突破但不能断档：选中转型
- 不怕扑、就是想试：再考虑大转型

### 7. 给自己保留“写下去的新鲜感”
下一本至少要有 1—2 个让作者真感兴趣的新点：
- 新背景
- 新职业
- 新关系结构
- 新冲突机制

没有新鲜感，只会换皮继续腻。

### 8. 控制改动速度
如果仍需要稳定成绩：
- 不要一口气把口味、人设、结构、题材全换掉
- 先让读者接受半步，再走下一步
- 可以把转型拆成 2—3 本的阶梯

### 9. 最后做翻车排查
- 只是题材名变了，套路没变
- 一次改太多，旧优势全丢
- 明明想求稳，却选了大换血方案
- 新题材只新鲜 5 章，后面又写回老路
- 自己都不知道这次到底想突破什么

## 输出格式
1. 当前稳定长项
2. 已疲劳的重复点
3. 本次要改的主轴
4. 保持不动的长项
5. 微 / 中 / 大三版转型方案
6. 推荐版本
7. 下一本必须保留的新鲜点
8. 读者适应提醒
9. 翻车风险点

## 快速示例
### 输入
- 上一本：古穿今都市轻甜
- 想写新东西，但又不想完全换口味

### 输出
- 稳定长项：轻快节奏、女主事业反击、感情线偏甜爽
- 主轴改动：把背景换成“穿越九十年代种田”
- 保留长项：女主逆袭感、轻快节奏、偏甜爽的情绪回收
- 推荐：中转型。换背景，不一次把口味和结构全推翻

## 注意事项
- **不要**把转型理解成越猛越好
- **不要**连续重复自己已经写腻的桥段
- **不要**在需要稳定成绩时忽然完全换血
- **改变可以慢，但不能停在原地只换题材名**
