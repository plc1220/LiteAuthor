按信息量，而不是按“多少字一个高潮”排节奏。**节奏快慢的本质，不是字数，而是读者接收、消化并期待下一步的速度。**

## 适用情况
- 写了很多，读者却说节奏忽快忽慢
- 两个人突然就在一起了，前面铺垫不够
- 一个梗刚写出来就用完，后面没得写
- 中段总觉得平，只会靠新事件硬抬
- 总想问“多少字一个小高潮”

## 核心概念
- **有效信息**：能推进主线、关系、人设、局势或悬念的信息
- **无效信息**：删掉也不影响理解和期待的说明、寒暄、重复表达
- **信息级别**：同一梗在不同阶段能释放的不同强度信息
- **停刹**：一个梗写热以后，先切去别的线，再回头加热

## 核心原则
1. **节奏看信息量，不看固定字数。**
2. **同一个梗可以反复用，但每次都要多给一层新意义。**
3. 快，不是事多；慢，也不是细。关键看有没有新信息、有没有阶段变化。
4. 读者抱怨“太快”，通常是**质变来得太早**；抱怨“太慢”，通常是**无效信息太多**。

## 步骤

### 1. 先写清当前阶段和下一阶段
例如感情线：
- 有点互相喜欢
- 彼此确认喜欢
- 在一起
- 发糖稳定期
- HE

再写：
`角色进入下一阶段前，读者必须先看到哪些信息？`

**没有信息积累，阶段跳跃就会突兀。**

### 2. 给每个梗拆级，不要一次写死
按这个格式把一个梗拆成 3—5 级：
- 第 1 级：第一次露头
- 第 2 级：同一梗加码
- 第 3 级：局势或情绪翻面
- 第 4 级：关系阶段变化后，梗意义改变
- 第 5 级：后期回收，形成更强回响

### 3. 一个梗在不同阶段，意义不同
同一件事，因为情感和局势不同，会变成不同信息。

例如“借水杯”：
- 第 1 级：借到了，只是普通互动
- 第 2 级：不给，拉出更多互动后才借到
- 第 3 级：因为水杯翻出冷战、和好
- 第 4 级：男主开始珍视水杯，说明感情升级
- 第 5 级：后期把它当成执念回收，读者才会觉得扎心或甜

**写的还是水杯，但读者接收到的信息已经不是同一层。**

### 4. 学会“停刹”
一个梗写热以后，不要立刻写到头。

稳写法：
- 先把梗加热到读者开始在意
- 暂时切去别的线或别的梗
- 留一点余温
- 再回来继续加码

作用：
- 避免一个梗太早用尽
- 让读者保持追问
- 让多条线互相抬节奏

### 5. 每场戏至少释放一种新信息
逐场检查是否完成至少一项：
- 推主线
- 推关系
- 推人物
- 改局势
- 抛新疑问 / 回收旧疑问

如果一场戏只是重复上一场的同类情绪，没有新意义，就是在原地打转。

### 6. 快慢怎么判断
#### 节奏太快
常见信号：
- 角色刚有点好感，就突然确认关系
- 一个能力还没铺垫，就直接大开挂
- 读者还没吃到过程，结果先到了

解决：
- 补一到两级中间信息
- 让阶段变化先有小证据，再有大证据
- 用场景而不是旁白证明转变

#### 节奏太慢
常见信号：
- 同一情绪来回说
- 场景很多，但局势不变
- 读者早懂了，正文还在解释

解决：
- 删掉重复说明
- 提前给下一层信息
- 让同一梗立刻承担第二个功能

### 7. 允许“迷惑读者”，但要有回收
常用做法：
- 虐前先发糖
- 大剧情铺得很长时，用感情或小回报拉线
- 先给一个看似稳定的判断，再用新信息翻掉

但前提是：
- 迷惑不是拖延
- 后面必须回收
- 不能让读者觉得被骗得没有意义

## 输出格式
1. 当前阶段 / 下一阶段
2. 进入下一阶段前必须补齐的信息
3. 现有梗的 3—5 级拆分
4. 每一级对应的场景
5. 停刹点安排
6. 当前太快 / 太慢的具体原因
7. 应删的无效信息
8. 下一个最该释放的新信息

## 快速示例
### 输入
- 校园文，男女主已经互有好感，但作者不知道后面还能写什么

### 输出
- 当前阶段：互相喜欢但未确认
- 必补信息：嘴硬、试探、吃醋、护短、公开站队
- 梗拆级：
  - 第 1 级：借作业
  - 第 2 级：借作业时故意逗对方
  - 第 3 级：借作业引发误会和冷战
  - 第 4 级：公开护短后，作业成了双方心照不宣的暗号
  - 第 5 级：毕业时回收这个梗
- 停刹：借作业写热后，先切去竞赛线，再回到感情线升级

## 注意事项
- **不要**迷信固定字数的小高潮公式
- **不要**一个梗刚亮出来就立刻用到尽头
- **不要**同一信息换个说法重复很多遍
- **不要**让阶段质变发生在信息量不足的时候
- 节奏最稳的时候，是**同一梗能越来越热，信息一层层变重**。
