先守住主角不知道什么，再决定读者要知道到哪一步。**第一人称最值钱的不是全知，而是信息差。主角没看见的事，能变成悬念就别硬切上帝视角。**

## 适用情况
- 第一人称写到一半，主角离场后还有重要情节必须发生
- 主角不能知道真相，但读者需要感到后来一定出了事
- 想写残局被人收拾、有人潜入现场、有人暗中做局
- 不想用分割线强行切上帝视角
- 怕一句我不知道他露出怨毒的神色破坏代入

## 输入
- 当前主视角角色
- 主角离场后真实发生了什么
- 主角此刻能知道和不能知道什么
- 读者现在必须知道到哪一步
- 真相准备在何时揭开
- 这一段更想要悬念、清晰、反转还是压迫感

## 核心原则
1. **先决定要不要立刻说。** 很多必须写的事，其实可以后置。
2. **先给结果和痕迹，再给过程。** 这样最稳。
3. **主角不知道，不等于作者可以直接替主角看见。**
4. **能让读者猜到有人来过，往往比直接播全过程更有悬念。**
5. **番外是补充，不是偷懒。** 能在正文里靠痕迹和回收解决，就先别切番外。

## 步骤
### 1. 先定读者此刻要知道到哪一层
常见三层：
- 只知道有异常：现场变了，但不知道谁干的
- 知道有人介入：能判断有人来过或动过手脚，但身份未明
- 知道完整事实：读者现在就该知道过程，主角暂时不知道

如果悬念重要，优先前两层；如果紧张感靠读者比主角先知道，才选第三层。

### 2. 从五种稳妥方式里选一种
#### A. 回到现场看变化
让主角事后再回来，看到：
- 尸体被处理了
- 东西位置变了
- 门窗、脚印、血迹等出现异常
- 原本该消失的东西又出现了
- 原本该存在的东西消失了

写法重点：
`主角看到变化 → 判断哪里不对 → 推出离开后有人来过`

#### B. 通过别人转述
让陌生人、知情人、属下、路人、传言告诉主角：
- 某时某刻谁来过
- 现场后来如何如何
- 某人做了什么异乎寻常的动作

写法重点：
- 转述者要有来源
- 转述不必一次说满
- 可以故意保留误差，让主角先误判

#### C. 用道具和痕迹侧写
不直接写人来过，而是写：
- 某个独有物件流出现场
- 某个本应被毁掉的证据出现
- 某个东西被偷偷带走
- 现场被人清理、掩埋、篡改

这种写法最适合保持悬念。

#### D. 延后到真相揭晓
当下什么也不明说，只在前文埋细节：
- 某人知道得太多
- 某人反应不自然
- 某人看主角的眼神变了
- 某人对现场细节过分熟悉

等真相揭开时，再把离场后发生的事一次回收。

#### E. 番外或补叙
只有在下面两种情况再用：
- 正文视角无论如何都放不下这段完整过程
- 这段过程本身有独立观赏价值

**最好放在真相已揭或悬念已稳住之后。** 太早写番外，容易提前泄底。

### 3. 决定给过程还是给推断
先填一句：
`这一段最需要交付给读者的，不是完整过程，而是____。`

可填：
- 有人来过
- 现场被动过
- 主角的判断出了偏差
- 某人从此进入嫌疑名单
- 真相暂时还不能公开

### 4. 如果要后置真相，先埋 2—3 个可回收细节
优先埋：
- 反常知道的信息
- 不合时宜的镇定
- 对某件物品的异常关注
- 说漏嘴
- 眼神、语气、立场变化
- 对案发细节过熟

**前面不埋，后面就不是揭晓，而是天降答案。**

### 5. 把伪上帝视角改成可见证据
高危写法：
- 我转身离开，却不知道 A 在我身后露出了怨毒的神色
- 我不知道 B 其实回去清理了现场

更稳的改法：
- 让主角后来发现 A 对这件事的恨意异常深
- 让 A 在后文对现场细节知道得超出常理
- 让现场变化、证据缺失、传闻和行为反常去证明 B 来过

**不要替第一人称补一只看不见的摄像头。**

### 6. 检查这段会不会削弱悬念
逐条问：
- 现在就说满，会不会让后面无谜可猜
- 现在不说，会不会读者完全看不懂
- 主角是否有机会通过现有线索推断一部分
- 真相揭开时，前面的痕迹能不能一一对上

### 7. 最后安排回收节点
写清：
- 第一次让读者感到不对劲是在何处
- 第二次加重怀疑是在何处
- 真相彻底说破是在何处
- 说破后会改写谁的关系、嫌疑或局势

## 输出格式
1. 主角离场后真实发生的事
2. 读者此刻该知道到哪一层
3. 采用的揭示方式
4. 当前可见的结果或痕迹
5. 需要埋下的细节
6. 真相回收节点
7. 不能直接写的上帝视角句子
8. 更稳的替代写法

## 快速示例
### 输入
- 第一人称主角夜里血洗山庄后离开
- 实际情况：另一个人后来进入山庄，处理尸体并带走遗孤
- 主角暂时不能知道真相

### 输出
- 此刻只让读者知道：山庄后来被人动过
- 揭示方式：回到现场看变化 + 后文人物反常
- 当前痕迹：
  - 尸体被埋了
  - 本该消失的遗孤出现在别处
  - 某配角对现场细节知道得太清楚
- 真相回收：中后期揭晓该配角当夜折返山庄，前文异常全部对账

## 注意事项
- **不要**动不动就分割线切全知视角
- **不要**用我不知道他其实怎么样偷渡作者视角
- **不要**前面不给痕迹，后面突然报答案
- **不要**为了讲清楚，把悬念最值钱的地方先说掉
- 第一人称写主角没看见的事，关键不是怎么全写出来，而是**怎么只让读者知道该知道的那一部分**。
