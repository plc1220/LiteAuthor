先别急着追求「没人写过」，先找这本书和同题材作品真正不同的一刀。**原创性不是题材从未出现，而是同样写这个题材时，切口、关系、叙法或观点明显不同。**

## 适用情况
- 题材常见，担心撞梗
- 知道要写什么，但说不清自己这本书新在哪
- 想表达理念，一落笔就像大众正确或口号
- 同类作品太多，不知道该从哪里拧出个人版本

## 核心原则
1. 先回答「我要写什么」，再回答「别人通常怎么写」。
2. 差异必须落在**能持续产剧情**的地方，不是只改名词。
3. 原创性常见于四层：题材角度、人物关系、叙事方式、独特看法。
4. **理念最好呈现，不要直讲。** 真正有力的理念，是作者自己消化过的判断，不是现成口号。

## 步骤
### 1. 用一句话写清故事对象
`这本书要写____。`
只写题材核心，不要先写宏大口号。
例如：
- 一个老年痴呆患者的散乱回忆
- 一个三角关系里三个人互相误读的故事
- 一个家族内部反叛关系的崩裂与重组

### 2. 列同类常见写法
至少写 3 条：
- 这个题材别人常从什么年龄 / 身份 / 时间点写起
- 常见关系是什么
- 常见叙法是什么
- 常见观点是什么

### 3. 在四层里找差异
#### A. 题材角度差异
同样题材，换切口：
- 人家写青年失忆，我写老年失智
- 人家写大事件，我写余波中的小人物
- 人家写爱情，我写亲情、代际、同盟或背叛

#### B. 关系结构差异
同样关系，换关系张力：
- 顺写宠溺，改写成反叛、对手、交易、站队冲突
- 常规三角，改成互相利用、互相误判、立场互斥
- 常规师徒，改成继承与反噬并存

#### C. 叙事方式差异
同样故事，换讲法：
- 单一叙事改多叙事人
- 全知叙述改限知叙述
- 顺叙改成碎片回忆、层层揭密、互相矛盾的证词

#### D. 看法差异
别写「人要善良」「爱很伟大」这种人人都知道的话。
改成：
- 这个故事里，作者真正想提出的独特判断是什么
- 这个判断和大众常识哪里不一样
- 哪个情境最能证明这种看法

### 4. 做一轮「顺写 / 逆写」测试
连续写 3 个版本：
- 如果别人顺着写，我能不能反着写
- 如果别人写惯常答案，我能不能写代价更高的答案
- 如果别人写同一种关系，我能不能换成立场冲突或叙事冲突

### 5. 做「去差异测试」
把你找到的差异拿掉，再看一遍：
- 这本书会不会立刻变成普通同类文
- 这处差异能不能反复产戏，不是只支撑一个名场面
- 它会不会改变人物关系、剧情推进或阅读体验

如果拿掉以后几乎没变化，说明这个差异还不够值钱。

### 6. 选 1 个主差异 + 2 个辅助差异
- 主差异：这本书最该被记住的一刀
- 辅助差异：给主差异抬轿，不抢戏
**不要同时堆五六个新花样。**

### 7. 把理念落到剧情里
按这个顺序写：
- 谁做了什么选择
- 付出了什么代价
- 别人如何回应
- 关系和局势怎么变
让理念从人物与后果里长出来，不要作者站出来讲课。

## 输出格式
1. 故事对象一句话
2. 同类常见写法
3. 四层差异备选
4. 主差异 + 辅助差异
5. 去差异测试结果
6. 用什么场景呈现理念
7. 最容易写回套路的点

## 示例
### 输入
- 想写失忆题材

### 输出
1. 故事对象：一个老人记忆崩塌后的散乱回忆
2. 同类常见写法：青年失忆、爱情线回收、单线顺叙
3. 差异备选：
   - 题材角度：从老年失智切入
   - 关系结构：重点写父子关系而非恋爱关系
   - 叙事方式：用碎片回忆拼出真相
4. 主差异：老年失智 + 散乱回忆
5. 理念呈现：不直接说「人会被记忆定义」，而让人物在失去记忆后仍保留某些选择和情感惯性
6. 风险：后面如果又写回普通回忆杀恋爱戏，这个切口就浪费了

### 输入
- 想写三角恋

### 输出
- 常见写法：全知视角、谁更爱谁、误会拉扯
- 差异方案：用三个叙事人分头叙述，三个人的说法互相矛盾，让读者自己判断真相
- 结果：同样是三角恋，阅读体验会被叙事方式直接改写

## 注意事项
- **禁止**把原创性理解成题材名词换皮
- **禁止**只有设定差异，没有冲突差异
- **禁止**用大众正确冒充独特理念
- **禁止**差异只够撑一场戏，后面又回老路
- 原创性最值钱的地方，不是「别人没写过」，而是**同样的题材，只有这本书会这样写**。
