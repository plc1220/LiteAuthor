先让主角真的在做事，再安排别人看错他，最后用成果或掉马统一回收高光。**这类爽点的核心，不是恶毒配角主动上门找打，而是主角早就在主线里推进事情，读者等的是“真相揭开那一下”。**

## 适用情况
- 想写爽点，但不想全靠反派送脸
- 想让主角更主动，不像被剧情推着走
- 想写正剧 / 剧情流里的高光、掉马、身份揭晓、成就揭晓
- 打脸文写多了，觉得主角像工具人
- 想让主线更紧，而不是每次都靠配角找事

## 输入
- 主角正在暗中做的事
- 外界当前对主角的误判
- 前文可埋的钩子
- 揭晓时机
- 揭晓后想改写什么：认知、站队、资源、主线方向

## 核心原则
1. **主角先行动，反转后揭晓。**
2. **误判来自信息滞后，不来自全员弱智。**
3. **揭晓必须服务主线，不能只是炫一把。**
4. **最稳的高光，不是主角自己吹，而是有分量的人或结果替他证明。**

## 步骤

### 1. 先写清主角暗中在做什么
用一句话写：
`主角表面上像____，实际上一直在____。`

要求：
- 这件事必须和主线有关
- 不是临时加出来的外挂
- 主角真的花了行动、时间、代价在推进

常见类型：
- 暗中参加更高层级的比赛 / 任务 / 布局
- 偷偷调查、破案、攒证据
- 提前解决了一个大问题，只是别人还不知道
- 隐藏身份、资历、作品、战绩、人脉

### 2. 让外界误判，但误判要有依据
常见依据：
- 主角刻意低调
- 信息被遮住
- 主角表面成绩 / 身份太普通
- 真正的成果还没公布
- 高层资源 / 关系没曝光

**误判最好来源于“他们不知道”，不是“他们都是傻子”。**

### 3. 前面先埋钩子，不要等揭晓时硬甩答案
至少埋 2—3 个钩子：
- 不正常的时间去向
- 主角对关键问题异常熟悉
- 有人对主角态度微妙
- 某个高位人物提前注意到主角
- 主角说过一句当时没人听懂的话
- 某个结果出来时，主角反应过于平静

钩子的作用不是剧透，而是让读者回头一想：原来早就有迹可循。

### 4. 把爽点写成“主角行动线”而不是“配角找事线”
稳写法：
`主角主动做事 → 外界因为信息不足看错 → 关键时刻结果落地 / 身份揭开 → 高光自然爆出来`

不要倒着写成：
`配角突然跳出来 → 主角被迫证明自己 → 事情写完就散`

前者更稳，也更能带主线。

### 5. 选对揭晓载体
常用 5 种：
- **高位人物亲自上门**：教授、前辈、权威、领导、宗门长老、媒体
- **官方结果落地**：成绩、奖项、排名、任命、通告、审判结果
- **现场能力压场**：临时考核、公开演示、正式舞台、比赛现场
- **受益后果反证**：别人这时才知道，原来之前的事是主角做的
- **旧钩子集中回收**：多条线索在同一场景对上

选择标准：
- 谁最懂这件事的含金量
- 谁能立刻改站队或改资源分配
- 揭晓后能不能直接推动下一段主线

### 6. 揭晓后至少改三样东西
至少回收 3 项：
- 外界对主角的认知
- 主角的资源 / 机会 / 位置
- 某段关系或站队
- 下一阶段的主线目标
- 主角的含金量被更高层承认

如果揭晓完只是全场震惊，却不改任何东西，这个高光就偏空。

### 7. 把“反转爽”写得不降智
排查：
- 揭晓是不是前文真埋过
- 旁人看错主角是不是有现实依据
- 主角是不是一直在主动推进，而不是被动等认可
- 高位见证人是不是懂行，而不是单纯捧场
- 揭晓是不是和当前主线强相关

### 8. 想比打脸更稳，可以把重点放在“主角在做事”
如果想写得更像正剧 / 剧情流：
- 少写恶毒配角主动挑衅
- 多写主角在主线里的长期动作
- 让爽点来自“成果终于公开”
- 让围观者的震惊只是副产品，不是唯一目的

**这种写法的强点在于：主角高光更足，主线也更紧。**

## 输出格式
1. 主角表面身份 / 状态
2. 主角暗中行动线
3. 外界的误判依据
4. 前文埋下的钩子
5. 揭晓载体
6. 揭晓场景
7. 揭晓后改写的认知 / 资源 / 主线
8. 如果要再接下一轮，下一步怎么加码
9. 降智风险

## 快速示例
### 输入
- 校园正剧：同学都以为主角是学渣

### 输出
1. 表面状态：转校生、上课不积极、看起来不靠谱
2. 暗中行动线：主角一直在参加国际竞赛
3. 误判依据：竞赛信息没公开，主角也没张扬
4. 钩子：主角经常请假；对高难题反应太快；某位教授提前问过他的名字
5. 揭晓载体：教授带着满分答卷亲自到学校
6. 揭晓场景：全校师生在场，原本轻视主角的人也在
7. 回收：同学改观、老师改口、主角拿到更高平台机会
8. 下一步：由竞赛线接到更大的科研 / 升学主线
9. 风险：如果前文一点钩子都没有，教授突然上门会像天降设定

## 注意事项
- **禁止**主角什么都没做，只靠最后一句“其实他很牛”硬翻
- **禁止**误判拖太久，拖到读者都烦
- **禁止**高光揭晓和主线无关，只为震惊
- **禁止**所有人都失明，只有作者知道主角厉害
- **主动高光反转最爽的地方，不是别人找打，而是主角早就在做大事，终于轮到世界看见。**
