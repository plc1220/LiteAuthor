给甜文加冲突时，优先让麻烦从外面来，不要先把刀捅进两个人内部。**甜文最稳的写法，不是让情侣反复猜疑、误会、内耗，而是让他们一起面对外部问题，在处理问题的过程里变得更甜、更稳、更有默契。**

## 适用情况
- 担心一路发糖会腻，想加点波动
- 不知道甜文的反派和危机该怎么放
- 双向暗恋 / 恋爱文不想写成互相折磨
- 想让感情线更充实，但不想拆 CP
- 想加冲突，又怕读者对主角产生嫌隙

## 核心原则
1. **危机优先出现在两人外部，不优先出现在两人内部。**
2. 冲突的作用是加深关系，不是把关系写散。
3. 外部麻烦要让两个人站到同一边。
4. 即使有极品配角，也尽量让其停留在外围，不要让挑拨真正成功。
5. 危机后要有情绪回收，不能只有闹腾没有补偿。

## 可用的外部冲突来源
### 1. 外部施压
- 媒体失控
- 舆论风波
- 工作失误被放大
- 第三方制造麻烦

### 2. 事业线困难
- 项目翻车
- 任务升级
- 比赛 / 工作压力
- 资源争夺

### 3. 家人、朋友、上级带来的事
- 委托
- 求助
- 家庭矛盾
- 不得不接的差事

### 4. 旧伤和旧困扰浮现
- 过去的阴影重新冒头
- 某个长期没处理的心结被触发
- 让另一方参与安抚、抚平、陪伴

### 5. 轻量调剂项
- 宠物
- 小孩
- 临时照顾任务
- 日常突发状况

这类不一定是大危机，但很适合在甜文里制造新鲜感和互动空间。

## 步骤
### 1. 先定“这段糖不能伤到哪一步”
写清楚当前必须保住的东西：
- 信任不能碎
- 立场不能反
- 好感不能倒退太多
- 核心误会不能拖成长期内耗

### 2. 再选外部麻烦
问自己：
- 这件事是从哪里来的
- 它压的是处境、任务、名声，还是过去创伤
- 它会逼出两人哪种互动

### 3. 设计“同一边”的处理方式
把冲突写成：
`麻烦来了 → 两人共同面对 → 一方出手，一方接住 → 关系更近`

例如：
- 一方处理外部冲突
- 一方负责安抚、补位、善后
- 两人联手打脸外部麻烦

### 4. 如果必须放极品配角，把其功能控在外围
稳写法：
- 极品亲戚、青梅、同事、竞争者来搅局
- 两位主角不互相猜疑，而是一起看穿、一起应对
- 可以用一点逗趣描写，让气氛不至于太阴沉
- 紧接着给一记回击或打脸

危险写法：
- 极品一挑拨，情侣立刻互不信任
- 关系被拖成长期冷战
- 读者开始烦 CP 本身

### 5. 危机后立刻回收糖感
至少回收一项：
- 更信任对方
- 更清楚对方在意自己
- 暴露了新的脆弱面
- 确认了谁会在关键时刻站自己这边
- 关系进入下一阶段

### 6. 当“甜久了怕腻”时，不一定非要虐
可以改用这些补波动：
- 事业线升级
- 对外共同救场
- 家人朋友求助
- 宠物 / 小孩 / 临时任务
- 旧伤被触发后被另一方抚平

**不一定要先虐一把再飞快甜回来。** 很多时候，外部波动比内部撕裂更稳。

## 输出格式
1. 当前关系阶段
2. 这段必须保住的糖点
3. 外部冲突来源
4. 两人分别怎么应对
5. 极品配角是否存在、存在时的边界
6. 危机后的糖感回收
7. 下一步关系变化

## 快速示例
### 输入
- 甜文，担心后期太甜会腻

### 输出
- 必保糖点：两人信任不能碎
- 外部冲突：工作项目出问题，外界开始质疑
- 应对：一人去处理局面，一人帮忙善后并稳住情绪
- 回收：危机解决后，双方更确认彼此是第一顺位

### 输入
- 双向暗恋，想加反派

### 输出
- 外部人物：极品亲戚或朋友来搅局
- 正确写法：让两位主角一起面对这个人，边吐槽边回击
- 错误写法：让第三人轻易挑拨成功，情侣互相嫌弃十几章

## 注意事项
- **不要**把甜文冲突默认写成情侣互相不信任
- **不要**为了拖戏，让一句话能解释清的事硬拖很久
- **不要**让外部危机进场后，主角立刻掉好感
- **不要**危机过后不补糖，只留下疲惫
- 甜文的冲突，最稳的作用是**让两个人更像一队人，而不是更像两边人。**
