围绕读者当前深信不疑的认知，设计一次会把关系、身份或世界理解当场炸开的揭晓。**爆点不是单纯塞狗血，而是让“原来如此”与“怎么会这样”同时发生。**

## 适用情况
- 反转有了，但不震
- 身世、掉马、秘密揭晓像例行公事
- 想写狗血、悬疑、神展开，但桥段只剩信息说明
- 需要给一卷或一条关系线安一个真正能炸场的节点

## 输入
- 当前读者相信的版本
- 想打碎的认知
- 涉及的人物与关系
- 主角为什么必须在乎这件事
- 想要的爆炸感：禁忌 / 反差 / 秘密 / 骗局 / 世界观颠覆 / 身份落差
- 可接受尺度

## 核心原则
1. **先立“大家以为是什么”，再炸“其实是什么”。**
2. 爆点必须改写判断、关系、站队或目标，不能只是补资料。
3. 最稳的爆点通常同时带一层**泪点或爽点**。
4. 反差越大，前面的“正常认知”越要立得稳。
5. **没有代价的秘密，不够炸；没有人物立场的禁忌，只剩噱头。**

## 常见爆点源
- 隐藏身份
- 看似善良 / 普通的人，其实掌握巨大恶意或权力
- 对立双方其实有不可告人的关系
- 一直相信的历史、规则或真相，本身就是骗局
- 亲人、导师、恩人另有目的
- 未来的自己 / 过去的自己 / 身边人和自己存在更深层关系
- 主角深信不疑的价值判断被彻底打碎
- 平常触手可及的东西，其实是外界疯狂争夺的核心资源

## 步骤

### 1. 先写“基准认知”
一句话写清：
`现在所有人都以为____。`

这一步必须具体，不能只写“事情很正常”。

### 2. 再写“真正真相”
一句话写清：
`其实____。`

要求：
- 要能直接推翻上面的基准认知
- 要能影响至少一个核心人物的选择

### 3. 判断爆点属于哪一类
优先只选 1 个主类型，必要时混 1 个副类型：
- **禁忌型**：不该发生的关系、交易、欲望、立场
- **反差型**：表象与本质反着来
- **秘密型**：长期遮掩的身世、过去、真实动机
- **骗局型**：群体长期相信的东西其实是假的
- **身份型**：某人真实身份能改写权力和关系
- **世界观型**：主角和读者原本理解的规则不成立

### 4. 让爆点咬住具体人物
逐条写清：
- 谁会被这件事震到
- 谁会因此失去什么
- 谁会因此拿到主动权
- 主角为什么非得在场、非得在乎

**如果爆点只会改变设定，不会改变人，它就只是资料。**

### 5. 给爆点补“震感来源”
从下面至少选 2 项：
- 禁忌
- 反差
- 身份差
- 权力差
- 情感背叛
- 价值观坍塌
- 长期误会被推翻
- 甜蜜外表下的恶意
- 平静表面下的巨大骗局

### 6. 提前埋 2—3 个信号
可用信号：
- 不该知道却知道
- 反应过度或反应缺失
- 被刻意回避的话题
- 一句看似普通、回头才刺人的对白
- 反复出现的小物件
- 太合理以至于不对劲的解释

**埋信号时只点亮，不讲透。**

### 7. 选最值钱的揭晓时机
优先考虑：
- 信任刚建立时
- 关系即将更进一步时
- 主角已经按错误认知做出重大选择后
- 阶段高潮前一刻
- 小胜利刚到手、读者放松时

太早：炸不响。  
太晚：读者已经不在乎。  
最稳的是：**前面刚好积累到“不能不炸”的位置。**

### 8. 爆完以后必须改 3 件事
至少改动下面 3 项中的 2—3 项：
- 关系：谁还能信谁
- 目标：接下来要做的事是否变了
- 阵营：谁和谁站到一边 / 站到对面
- 情绪：爽、痛、后怕、愤怒、心疼是否被放大
- 世界理解：原本的规则是否不成立
- 主角身份或处境：是否被重新定义

### 9. 给爆点接一层泪点或爽点
爆点单独存在，常常只是一声“啊？”。

更稳的接法：
- **爆点接泪点**：揭晓后，读者终于知道某人的委屈、误会或悲剧从何而来
- **爆点接爽点**：揭晓后，主角终于能掉马、反击、翻盘、揭穿骗局
- **爆点接双重情绪**：先震，再心疼 / 再痛快

### 10. 反查是不是“空炸”
逐条检查：
- 这个真相是不是早就能猜到，而且没加码
- 揭晓后局势是不是几乎没变
- 除了“信息量大”，还有没有真正的情绪冲击
- 爆点是不是和主题、主线无关
- 这件事如果换到别的角色身上，效果会不会几乎一样

## 输出格式
1. 基准认知
2. 真正真相
3. 爆点类型
4. 震感来源
5. 关联人物与 stakes
6. 伏笔信号
7. 最佳揭晓时机
8. 揭晓场景
9. 爆后改写的关系 / 目标 / 阵营
10. 接续的爽点 / 泪点
11. 风险点

## 快速示例
### 输入
- 设定：主角一直以为照顾自己的前辈是救命恩人

### 输出
- 基准认知：前辈是保护者
- 真相：前辈其实一直在监视主角，并利用主角体内的东西布局
- 类型：秘密型 + 反差型
- 震感来源：信任反转、情感背叛、身份差
- 揭晓时机：主角刚准备彻底信任对方时
- 爆后变化：关系破裂、主角目标改写、旧线索全部重估
- 接续情绪：先心寒，再接主角反击和掉马爽点

## 注意事项
- **禁止**把百科设定当爆点
- **禁止**只追求震惊，不管人物代价
- **禁止**为了炸而炸，结果和主题脱节
- **禁止**没有前置认知就硬揭真相
- 爆点真正值钱的地方，不是“信息更大”，而是**它一炸，人物和读者再也回不到之前的认知**。
