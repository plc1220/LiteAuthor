让配角先带着轻视、怀疑、冷淡或偏见出场，再用主角的能力、修养、胆识或魅力把态度一点点掰过来。**真正好看的不是路人震惊，而是有分量的人改口、改站队、甚至主动来帮主角。**

## 适用情况
- 主角初次立威
- 团队、宗门、剧组、公司里让新人被接纳
- 写大佬改观、前辈真香、收小弟、收追随者
- 想用高位配角抬主角含金量
- 想把普通打脸写成“被主角征服”的魅力桥段

## 核心原则
1. **对象越有身份，态度反转越值钱。**
2. **负面态度必须有依据：信息差、地位差、立场差，不是无脑犯蠢。**
3. **主角要靠现场表现征服对方，不靠自吹和旁白认证。**
4. **态度变化要分层，不要一秒脑残粉。**
5. **反转后必须带来结果：维护、推荐、站队、传话、合作、追随。**

## 步骤

### 1. 先选“被征服”的对象
优先从有分量的人里选：
- 队长
- 前辈
- 学者
- 公爵 / 贵族
- 行业大佬
- 权威评委
- 老牌强者

如果只是随手一个炮灰，爽感会薄很多。

### 2. 先定他为什么看不上主角
常见原因：
- 主角地位低
- 年纪轻
- 出身差
- 名声差
- 初见寒酸
- 先前成绩不够
- 立场天然不合

**原因越合理，后面的改口越有分量。**

### 3. 选主角要展示的“征服点”
不要只想着打架。可用：
- 实力
- 财富
- 礼仪
- 学识
- 专业判断
- 临场胆识
- 担当
- 宽容
- 独特气质
- 人格魅力

最好选**正好击中对方价值观**的那一项。
例如：
- 学者更容易被见识征服
- 贵族更容易被风度和礼仪征服
- 队长更容易被实战能力征服
- 大佬更容易被判断力和气场征服

### 4. 把展示塞进现场事件
优先放在：
- 任务
- 比试
- 公开发言
- 处理危机
- 共同面对强敌
- 专业场合的判断和操作

**别让主角空口自夸，也别让别人替主角念履历。**

### 5. 按态度曲线写反转
常用曲线：
`轻视 / 冷淡 → 观望 → 松动 → 惊讶 → 认可 → 主动维护 / 合作 / 追随`

不同层级的人，转变速度可以不同：
- 亲友：本来就偏袒
- 普通同伴：先怀疑，再真香
- 最高位配角：前面克制，最后一锤定音

### 6. 多人见证时，别让所有人同一种反应
可至少放三类：
- 偏袒主角的人
- 不信主角的人
- 最懂含金量、最有资格点评的人

这样层次更丰富，也更能抬高主角。

### 7. 把“被征服”写成后果
态度反转后，至少回收 1—2 项：
- 替主角说话
- 公开认可
- 让出资源
- 主动结盟
- 成为追随者
- 帮主角拦下一次麻烦
- 把主角往更高层带

如果只是“他好厉害啊”，这场戏就白写了。

### 8. 进阶写法：征服链
可以先让 A 征服 B，再用 B 的分量抬 A。
例如：
- 主角先征服队长，再由队长带动全队改观
- 男主先征服业内大佬，再反过来被女主的特质征服
- 反派平时让所有人噤声，高潮里再被主角逼到失态或改口

这条链越往上，主角含金量越高。

### 9. 分清配角和反派
- **配角**：可以从讨厌主角转成欣赏、合作、追随
- **死敌反派**：更适合写成忌惮、失态、被压住，不必强行转粉

**别把你死我活的终局反派写成突然犯花痴式改观。**

## 输出格式
1. 被征服对象
2. 对方原本为什么看不上主角
3. 主角要展示的征服点
4. 现场事件
5. 态度曲线
6. 见证者反应
7. 反转后的实际后果
8. 还能如何继续加码

## 快速示例
### 输入
- 低等级新人第一次进狩猎小队

### 输出
1. 对象：队长、两名怀疑他的老队员
2. 负面态度：觉得他太年轻，扛不住实战
3. 征服点：实战判断和爆发力
4. 现场事件：与凶兽正面交手
5. 态度曲线：
   - 老队员：怀疑 → 震惊 → 真香
   - 队长：观望 → 点头 → 正式认可
6. 后果：小队接纳他，后续行动愿意把关键位置交给他

## 注意事项
- **禁止**让配角为了被征服而刻意降智
- **禁止**让主角一出场就全员脑残粉
- **禁止**只让低配炮灰改口，不给高位见证者
- **禁止**改口之后没有任何实际后果
- 好看的态度反转，本质是**主角用表现改写别人对他的判断**
