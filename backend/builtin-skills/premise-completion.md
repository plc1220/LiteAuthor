把只有开头、名场面或片段灵感的脑洞，补成能列大纲的完整故事。**只有一个很想写的场面，不等于已经有了一本书。最稳的补法是：灵感立人设，人设找成长线，成长线拉关系线，中间添剧情线，最后再压回起因—冲突—高潮—结果。**

## 适用情况
- 只有一个开头、一个画面、一个桥段
- 只知道最想写哪场戏，中后段和结尾发空
- 一做大纲就越加越多，越做越散
- 很喜欢原始脑洞，但展开后总不像最初那口味
- 不知道该继续想梗，还是硬做大纲

## 核心原则
1. **先补闭环，再列纲。**
2. **先问“这是谁的故事”，再问“发生什么”。**
3. 原始灵感负责点火；成长线和关系线负责把故事拉长。
4. 外部剧情线要不断逼人物变化，而不是单独平行存在。
5. 如果一句话里看不出高潮和结果，就还没到列纲阶段。

## 步骤
### 1. 锁定原始萌点或片段
先回答：
- 这条灵感最舍不得删的那一下是什么
- 它让人兴奋的，是爽、甜、虐、反差、悬念，还是别的
- 如果只保留一个场面，最想写的是哪场

先守住这口味，后面所有补强都围着它转。

### 2. 用片段反推主角
继续问：
- 什么样的人最适合出现在这场景里
- 这个人现在最缺什么、最怕什么、最执着什么
- 这个场面最容易戳中 TA 哪个弱点或欲望

这一步的目标是回答：**为什么偏偏是 TA 的故事。**

### 3. 补成长线
用一句话写：
`主角一开始____，最后要学会 / 放下 / 得到____。`

常见成长方向：
- 逃避 → 面对
- 自卑 → 建立自我价值
- 只顾自己 → 学会承担
- 冷硬防备 → 学会信任
- 执念太深 → 学会放下或转化

没有成长线，灵感很容易只够支撑几场名场面。

### 4. 补关系线
如果是 CP 文，继续写：
- 谁最能刺激主角变化
- 这个人为什么能卡住主角
- 两人的关系会经历哪 3—5 次变化

如果不是 CP 文，就用：
- 对手线
- 搭档线
- 亲情线
- 师徒线

关系线的作用不是单独发糖或吵架，而是**放大成长线**。

### 5. 补主线冲突
再写：
- 主角真正要解决的问题是什么
- 谁在拦，什么规则在拦
- 如果失败，会失去什么
- 这条外部主线如何持续逼人物选择

外部主线不是往灵感外面硬套事故，而是给成长线和关系线持续施压。

### 6. 找全书最值钱的高潮
直接写一句：
`这本书最后最该炸开的那场戏是____。`

常见高潮：
- 当众翻盘
- 关系摊牌
- 真相揭开
- 公开站队
- 旧账清算
- 最终选择

如果高潮还看不见，大纲很容易越做越散。

### 7. 补结果状态
问：
- 打完这场仗以后，主角变成什么样
- 关系停在什么状态
- 主角拿回或失去了什么
- 结尾最想给读者留什么感觉

结果不一定要很细，但必须知道大方向。

### 8. 把故事压成一句完整的话
用这个模板：
`因为____，主角不得不____；过程中，TA 在____的逼迫下，从____变成____；最终在____；结果____。`

这一步要同时能看见：
- 起因
- 主线冲突
- 成长线
- 高潮
- 结果

### 9. 检查能不能开始列大纲
只有下面四项都能答，才建议列纲：
1. 开头从哪里切
2. 中段主要靠什么推进
3. 高潮炸什么
4. 结局收成什么状态

缺两项以上，不要硬做细纲。

### 10. 一列纲就乱加时，用“三条删法”
新加内容只有满足下面至少一条，才留下：
- 能放大原始萌点
- 能推进主线冲突
- 能抬高最终高潮

“这个设定也很有趣”不等于“它该出现在这本书里”。

## 输出格式
1. 原始萌点 / 片段
2. 主角画像
3. 成长线
4. 关系线
5. 主线冲突
6. 最值钱的高潮
7. 结果状态
8. 一句话完整故事
9. 是否已到列纲阶段
10. 应删掉或后置的加料点

## 快速示例
### 输入
- 灵感：死对头被迫同住

### 输出
- 原始萌点：两个互相看不顺眼的人被迫共享生活空间，越相处越失衡
- 主角画像：表面很稳、其实极度需要掌控生活秩序的人
- 成长线：从只会控制一切，到学会接受失控和真正信任
- 关系线：互相嫌弃 → 被迫合作 → 发现彼此例外 → 关系失衡 → 公开站队
- 主线冲突：两人必须联手处理一件共同利益受威胁的大事
- 高潮：关键场合里，两人必须在外界面前选立场
- 结果：关系和外部问题一起翻盘
- 一句话完整故事：因为共同利益受威胁，一个过度掌控的主角不得不和死对头同住合作；过程中，TA 在相处摩擦和外部压力下，从只信自己变成学会信任他人；最终在公开对峙里完成站队；结果是关系和局势一起改写

## 注意事项
- **不要**把“我很想写这个场面”误当成“我已经有完整故事”
- **不要**在只有片段的情况下硬排几十章
- **不要**只有设定和事故，没有主角变化
- **不要**为了显得丰富，往里塞太多旁枝，把原始萌点冲淡
- 真正能列纲的梗，至少应该能说清：**为什么开始、主角会怎么变、跟谁纠缠、外面在打什么仗、最后炸哪一下。**
