先别急着重写，先把原文按“叙述 / 描写 / 对话”拆开看。**正文发平，往往不是句子不够漂亮，而是某一类表达压得太多，场景又没有目标、阻碍和变化。** 先找平在哪里，再决定该加矛盾、减说明，还是重排段落。

## 适用情况
- 章节像流水账，读完没感觉
- 叙述很多，戏感很弱
- 对话不少，但只是聊天
- 描写很多，却不推动任何事
- 自己觉得写得很满，读者却觉得淡

## 核心原则
1. **平铺直叙通常是结构问题，不只是文笔问题。**
2. **先诊断比例，再决定改法。**
3. **每一场戏至少要有：目标、阻碍、变化。**
4. **叙述、描写、对话都可以好看，也都可以很水，关键看有没有功能。**

## 步骤

### 1. 给原文做三色标记
把文本分成三类：
- **红色：对话**
- **蓝色：描写**（动作、环境、神态）
- **橙色：叙述**（背景、说明、总结、概括）

不用真的上色也可以，用标签标出来就行。

### 2. 看哪一种压得过多
常见问题：
- **橙色太多**：说明过重，像在讲大纲
- **蓝色太多**：画面很多，但没有事发生
- **红色太多**：角色一直说话，却没有目标和潜台词

先找主问题，再下刀。

### 3. 检查每一场戏有没有“目标—阻碍—变化”
逐场问：
- 这场戏里，主角想做什么
- 什么东西在拦
- 这场戏结束后，局势变了什么

如果三项缺一项，这场戏就容易平。

### 4. 按问题类型改
#### 橙色太多：说明重
改法：
- 把背景说明拆进动作、对话、现场反应
- 能后置的前情后置
- 先让角色做事，再补原因

#### 蓝色太多：描写重
改法：
- 每段描写都问“它服务了哪个冲突”
- 不服务冲突的景物，一句带过或删除
- 在描写里加入人物目标和外部压力

#### 红色太多：对话重
改法：
- 给每段对话加目的
- 补动作和环境，让话里有场面
- 增加潜台词、错位信息或立场冲突

### 5. 检查有没有“无功能段落”
给每段标功能：
- 推剧情
- 推关系
- 立人设
- 放大冲突
- 抛钩子
- 补必要信息

标不出来的段落，优先删、并或改写。

### 6. 给平的地方补一个变量
如果整场戏太顺，优先补一个变化：
- 新信息
- 新阻力
- 新选择
- 新误会
- 新站队
- 一个小目标失败

**不是硬加事故，而是让事情不再直线滑过去。**

### 7. 重跑一遍三色检查
改完后再看：
- 是否还是一大片同色
- 同一类内容是否连续太久
- 关键段落里有没有出现明确变化

## 输出格式
1. 三色比例判断
2. 主问题：叙述重 / 描写重 / 对话重
3. 每场戏的目标—阻碍—变化
4. 无功能段落清单
5. 需要补的变量
6. 调整后的段落顺序
7. 改写建议

## 快速示例
### 输入
一章 3000 字里大半都在讲女主背景和心理，正式冲突很少

### 输出
- 三色判断：橙色过多
- 主问题：叙述过重，像背景介绍
- 改法：把背景拆进当前事件；第一章直接切到关键矛盾；把“她为什么会这样想”改成现场反应和台词
- 结果：信息没少，但戏感会强很多

## 注意事项
- **不要**把平淡都归咎于“句子不够美”
- **不要**看到问题就只会加修辞
- **不要**让人物一直说、一直想、一直看，却没有任何局势变化
- **不要**把诊断做成机械配比，比例只是线索，不是死规则
- 平铺直叙真正要改的，是**没有目标、没有阻碍、没有变化的场面**
