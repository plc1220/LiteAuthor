先把“凉薄”写成稳定的价值排序，不要写成无来由的冷脸。**凉薄角色能让人信服，关键不在于冷，而在于她为什么这样想、这样选，以及别人为什么不认同。**

## 适用情况
- 想写本性凉薄、情感淡、利己理性型主角/女主/配角
- 角色容易被读者说成装酷、端着、没心肝
- 行为上很冷，但心理逻辑站不住
- 需要让读者理解角色，不一定赞同角色

## 输入
- 角色身份与处境
- 她最常见的“凉薄”行为 3—5 条
- 她最在乎的东西
- 与主角/家人/爱人/朋友的关系
- 当前要写的关键选择

## 核心原则
1. **凉薄不是没感情，而是价值排序不同。**
2. 每个冷淡选择都要有**自我合理化理由**。
3. 角色自己觉得自己是对的；冲突来自别人不这么看。
4. **不要靠作者旁白夸“她好冷好酷”**，要靠选择与后果立住。
5. 写出她的盲点，比反复写她冷漠更有力。

## 步骤
### 1. 先写她真正优先什么
用一句话写：
`在这类局面里，她永远先保____，再考虑____，最后才会顾及____。`

常见优先级：自保、利益、尊严、秩序、效率、旧伤、执念。

### 2. 给每个“凉薄行为”补自我理由
按这个格式逐条写：
- 她做了什么
- 她认为这样做为什么合理
- 她觉得不这么做会损失什么

句式：
`她不救/不哄/不退让，不是因为单纯坏，而是因为她认定____。`

### 3. 再写她没顾及到什么
每个行为都补一条：
- 她忽略了谁的感受
- 她低估了什么代价
- 她以为“别人也该懂”的地方是什么

**真正的凉薄感，常来自她的盲区，而不是恶语。**

### 4. 同一件事写两层视角
必须同时产出：
- **她自己的解释**
- **别人眼里的样子**

例如：
- 她觉得自己是在止损
- 别人觉得她太无情

这样角色才不会只有单层标签。

### 5. 给她一条稳定底线
凉薄角色最好仍有一条固定原则，例如：
- 不欺负弱者
- 不欠人情
- 不说空话
- 只对某类人例外
- 绝不牵连无关者

**有底线，角色是人；没底线，角色容易只剩工具感。**

### 6. 用选择写，不用口号写
优先安排能暴露凉薄的场面：
- 利益与感情冲突时先保哪边
- 别人求助时她给什么层级的回应
- 面对旧情、亲情、恩义时，她退到哪一步就不再退
- 她如何解释自己的冷淡

少写：
- “她天性凉薄”
- “她冷冷一笑”
- “她向来如此”

### 7. 给外部反应，不给作者裁定
让配角或环境给出真实后果：
- 有人寒心
- 有人误会
- 有人反而更依赖她的清醒
- 有人指出她遗漏的东西

**凉薄不是标签，而是会改变关系的处事方式。**

### 8. 防止写成装酷
出现以下情况就重写：
- 她只是说话冷，实际没有代价也没有选择
- 她每次都刚好最酷、最正确
- 所有人都被作者安排去夸她清醒
- 她的冷淡没有来路、没有盲点、没有后果

## 输出格式
1. 角色优先级
2. 3—5 个凉薄行为及其自我理由
3. 每个行为的盲点
4. 自我视角 / 他人视角对照
5. 稳定底线
6. 会改变关系的关键场面
7. 最容易写成装酷的风险

## 快速示例
### 输入
- 女主表面冷淡，不愿照顾亲近的人
- 目标：写出“凉薄但不装酷”

### 输出
- 优先级：先保自保与体面，再考虑感情
- 自我理由：她认为过度投入只会被拖累
- 盲点：她把别人的求助都看成索取，没看到其中的信任
- 他人视角：闺蜜觉得她薄情，敌人却觉得她难以操控
- 底线：她不落井下石，也不利用别人的真心
- 关键场面：别人受伤时她先处理局势，事后才补偿，而不是当场哭着表态

## 注意事项
- **禁止**把凉薄写成全天候面瘫
- **禁止**只给冷脸，不给理由
- **禁止**让角色永远正确、永远没后果
- **禁止**把“说话少”误当成“凉薄”
- 好的凉薄角色，读者可以不喜欢她，但会觉得**她这样想是成立的**。
