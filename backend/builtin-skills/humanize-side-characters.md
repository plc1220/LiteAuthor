先别写“一个小女孩”“一群灾民”“一百个死者”“某个被欺负的人”，先写一个具体的人。**身份标签只能告诉读者发生了什么，具体经历才会让读者在乎。** 想让读者心疼谁、恨谁、支持主角去救谁，先把相关人物写成人。

## 适用情况
- 主角救人、主持公道、复仇时，读者没感觉
- 想让反派显得可恨，却只有一句“他作恶多端”
- 想写路人、受害者、炮灰的可怜或可爱，但只剩标签
- 想让读者哭，却只写数字、身份和结果
- 想让读者支持主角出手，却只能靠旁人夸主角善良

## 核心原则
1. **信息越具体，情绪越强。**
2. **笔墨就是情感权重。** 谁被写得更具体，读者就更把谁当人。
3. 群体苦难、抽象恶行，最好先落到一个具体人物身上。
4. 先让读者在乎，再让主角出手。
5. 不用长篇小传，**3—8 句有效细节就够。**

## 输入
- 想调动的情绪：心疼 / 愤怒 / 厌恶 / 保护欲 / 敬佩 / 惋惜
- 需要具体化的人物或群体
- 反派或困境对他造成了什么
- 主角接下来要做什么
- 这段之后要接：救人、打脸、复仇、治愈还是虐点

## 步骤
### 1. 先定这段想让读者产生什么情绪
不要只写“想让这段感人”，先写清：
- 想让读者心疼谁
- 想让读者讨厌谁
- 想让读者希望主角做什么

如果这三点不清楚，具体化容易写散。

### 2. 选一个“情绪锚点人物”
如果场上是一群人，先挑一个最能承载情绪的代表。

优先选：
- 有明确关系的人：孩子、老人、兄妹、师徒、宠物主人
- 有简单愿望的人：想回家、想活下去、想见某人、想守住某样东西
- 眼下正在受害的人

不要一上来只写：
- 灾民
- 百姓
- 小女孩
- 某个受害者
- 死伤若干人

这些都是信息，不是情绪。

### 3. 给这个人补四块最值钱的信息
按这个顺序补：
1. **他是谁 / 和谁有关**
2. **他最简单的小愿望是什么**
3. **他眼下遇到了什么不公或危险**
4. **一个能记住的细节**

可直接套用：
`他本来只想____，却因为____变成了____。他手里/身上/嘴里还留着____。`

### 4. 把“结果”改成“过程中的一下”
不要只写：
- 她很可怜
- 他死了
- 她被欺负了
- 那里死了很多人

改写成读者能看到的一下：
- 他攥着给母亲买的药，不肯松手
- 她躲在桌下，反复念一个人的名字
- 他以为求饶有用，结果下一秒更惨
- 她抱着唯一剩下的东西，不知道该递给谁

**情绪常常不在结论里，而在那一下无力里。**

### 5. 想让读者恨反派，就让伤害落到具体人身上
不要只宣布反派坏：
- 杀人如麻
- 欺男霸女
- 祸害百姓

要写：
- 他具体怎么伤到这个人
- 他夺走了什么
- 他让这个人失去了谁、错过了什么、怕成了什么样

抽象恶行只会让读者“知道他坏”，具体伤害才会让读者“想他死”。

### 6. 想让主角的善良、保护欲或侠气成立，先别急着夸主角
低效写法：
- 主角救了一个陌生人
- CP 说：“你真善良。”

更稳的写法：
1. 先把受害者写得让读者在乎
2. 让读者自己希望主角出手
3. 再让主角出手解决

这样主角不是“被夸善良”，而是真正完成了读者期待。

### 7. 用“短链条”建立情绪，不要长篇外传
最常用的三段式：
`小愿望 / 小关系 → 不公或伤害 → 无力的一下`

例如：
- 想把药带回家 → 被恶霸抢走踩烂 → 还在泥里捡
- 盼父母回信 → 信没等到，灾难先到 → 只剩照片可抱
- 刚攒够学费 → 被人当众毁掉 → 还得强撑着说没事

**短、准、具体**，比一整段苦情履历更有效。

### 8. 主角出手时，要接住前面铺的情绪
如果前面铺的是：
- 无辜受害 → 主角的救场要给安全感
- 欺压不公 → 主角的反击要给公道感
- 可爱无助 → 主角的保护要给心疼回收
- 被羞辱被误解 → 主角的站队要给尊严回收

前文铺什么，后文就回收什么。  
**不要前面写得很惨，后面只让主角说两句漂亮话。**

### 9. 最后检查“是不是还能换成标签”
把这段里的具体人名和细节删掉，再看一遍。

如果删完以后，效果几乎没变，说明还不够具体。  
直到这个人不能再被“一群人”“一个孩子”“一个老人”替换掉，才算立住。

## 输出格式
1. 目标情绪
2. 情绪锚点人物
3. 这个人的小愿望 / 小关系
4. 当前不公或危险
5. 记忆细节
6. 无力的一下
7. 主角如何回收这段情绪
8. 读者会因此更讨厌 / 更心疼谁

## 快速示例
### 输入
- 目标：体现主角善良，让读者支持主角救人
- 原写法：主角路过，看见一个被欺负的小女孩，顺手把她救了。CP 夸主角善良。

### 输出
- 情绪锚点人物：替祖母抓药的小女孩
- 小愿望：天黑前把药带回家
- 当前危险：地痞抢走药包，故意踩进泥里，还笑她穷鬼
- 记忆细节：她跪在地上，一颗颗去捡散开的药丸，手都抖了，还怕捡不全
- 主角回收：主角出手打断欺凌，替她把药和钱都拿回来
- 效果：读者先替女孩着急，再为主角的出手叫好；“善良”不必靠别人夸，读者自己会认

## 注意事项
- **不要**把每个路人都展开成支线，情绪锚点够用就收。
- **不要**只写惨，不写这个人原本想要什么，读者会少一层共情。
- **不要**只写身份标签和数字，那是新闻，不是小说里的情绪。
- **不要**忘记回收。前面让读者心疼，后面就要给对应的救场、反击或余波。
- **笔墨会决定读者站队。** 想让读者在乎谁，就把最具体的细节给谁。
