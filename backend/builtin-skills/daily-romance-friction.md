写生活磨合，不靠失忆车祸，也能让感情线持续升温。围绕口味、习惯、审美、技能、社交方式和生活节奏，安排双方在小事上的分享、出糗、让步和逞强。**每一场磨合都必须同时做到两件事：暴露人设 + 让关系偏移半步。**

## 适用情况
- 甜文、慢热、先婚后爱、同居文缺少大矛盾
- 两人已经有基础牵绊，但互动太平
- 想写细水长流的感情增长
- 在一起前后都需要可持续的日常戏

## 输入
- 双方关系阶段
- 双方各自最喜欢 / 最擅长 / 最不适应的生活点
- 题材限制（现代、古言、校园、娱乐圈等）
- 这一场想写甜、尴尬、嘴硬心软还是小拉扯

## 步骤
### 1. 先找一个可见差异点
优先从这些地方选：
- 口味
- 作息
- 审美
- 宠物 / 洁癖 / 家务
- 社交习惯
- 消费观
- 兴趣爱好
- 生活技能

差异点越具体，场面越好写。不要只写“性格不合”。

### 2. 在四种磨合动作里选一种
#### A. 我把我最喜欢的分享给你
例：推荐食物、音乐、景点、电影、游戏。
重点不是一定合拍，而是“我第一反应想带你一起”。

#### B. 我把我最优秀的展现给你
例：做饭、办事、社交、照顾人、工作能力。
重点是展示时带一点想被看见、想被夸的心思。

#### C. 你最喜欢的，我也想去喜欢
例：怕狗的人陪你逛宠物店，不爱运动的人硬陪你爬山。
重点是“为了靠近你，主动跨过自己的不适区”。

#### D. 即使我不会，我也要在你面前会
例：不会修灯泡还硬装会修，不会做饭先偷偷查教程。
重点是逞强、护面子、想在对方面前可靠。

### 3. 给磨合安排一个小小失衡
不要一上来就完全顺利。优先加入：
- 口味不合
- 理解偏差
- 小狼狈
- 被看穿
- 逞强翻车
- 一方以为体贴，另一方实际不适应

**小吃瘪比纯发糖更能立住关系。**

### 4. 回收成“理解或偏爱”
这场戏结尾至少回收 1 项：
- 记住对方喜好
- 发现对方软肋
- 学会一种照顾方式
- 下次主动让步
- 嘴上嫌弃，行动补偿
- 从“我喜欢这个”变成“我想和你一起喜欢这个”

### 5. 让磨合连续升级
可按这个顺序排：
`分享 → 出糗 → 观察 → 让步 → 记住 → 反过来照顾`

磨合戏不是一次写完，而是让同类差异反复回响。

## 输出格式
1. 差异点
2. 本场磨合类型
3. 发起人及动机
4. 小冲突 / 小狼狈
5. 人设暴露点
6. 关系增量
7. 后续可回收的细节

## 快速示例
### 输入
- 男主肠胃弱，女主重口味
- 关系阶段：暧昧初期

### 输出
- 差异点：饮食偏好
- 类型：我把我最喜欢的分享给你
- 场面：女主兴冲冲带男主吃麻辣火锅，男主强撑吃完，夜里进医院
- 人设暴露：女主热情直接，男主嘴硬逞强
- 关系增量：女主第一次意识到“喜欢不是复制自己”，男主第一次愿意为了她硬扛不适
- 回收细节：以后女主会先问男主能不能吃辣，再悄悄给他点清汤

## 注意事项
- **禁止**把生活磨合写成纯流水账
- **禁止**只写甜，不写差异和代价
- **禁止**每次磨合都无后果
- 同一对 CP 的磨合，最好反复回收同一类差异，让“生活感”变成“关系感”
