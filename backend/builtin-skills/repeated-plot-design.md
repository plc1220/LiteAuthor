把会反复出现的桥段写出真实感和人物差异。**同类情节本身不可怕，可怕的是奇遇硬复读、人物反应像复制粘贴。**

## 适用情况
- 群像文、长篇里同类桥段会多次出现
- 担心“又是坐牢 / 比武 / 拜师 / 升级 / 任务 / 宴会”显得老套
- 想用同一类事件照出不同人物
- 作品处在某个制度或环境里，角色会经历相似流程

## 输入
- 要重复的桥段
- 这个桥段在世界里的常见原因或制度依据
- 涉及角色清单
- 每个角色的身份、处境、目标、底线
- 预计会出现几次

## 步骤
### 1. 先判断这是不是“能重复的桥段”
优先可重复：
- 日常流程
- 制度惩罚
- 社会潜规则
- 职业常态
- 修炼、考试、押送、审讯、入门等固定流程

高危重复：
- 掉崖遇高人
- 捡到秘笈
- 天降宝物
- 纯靠巧合的奇遇救场

**高危桥段不是不能用，但不适合机械复读。**

### 2. 写清它为什么常发生
用一句话说明：
- 谁会这么做
- 为什么长期存在
- 受什么规则约束
- 为什么主角群会反复碰上它

如果答不上来，这只是作者硬造的流程，不是真实桥段。

### 3. 固定“外部流程”
先写这类桥段的共用骨架：
`进入场景 → 压力出现 → 角色应对 → 代价 / 结果`

外部流程可以相似，但**戏肉必须不同**。

### 4. 每次重复至少改两项
从下面至少改 2 项：
- 参与人
- stakes
- 角色立场
- 应对策略
- 结果
- 余波
- 旁观者反应

如果只是同一流程换个名字，读者一定会觉得在复读。

### 5. 用同一组问题拷打不同人物
对每个角色都回答：
- 第一反应是什么
- 他最怕失去什么
- 他惯用什么办法：忍、打、骗、买通、借势、结盟、拖时间
- 他如何看待权力和规则
- 这次经历会让旁人怎样重新认识他

### 6. 差异一定要落到可见层
重点拉开：
- 台词口气
- 动作习惯
- 处理顺序
- 对羞辱、诱惑、威胁的态度
- 事后余味

**要做到闭眼只看反应，也能认出是谁。**

### 7. 把重复桥段当成人物照妖镜
每重复一次，都回答：
- 这次主要塑造了谁
- 主要显出他哪一面
- 与前一次相比，读者新认识到了什么

重复的价值不是“事情又来了一遍”，而是“同一炉火烧出不同人”。

### 8. 奇遇类桥段先做三选一
如果非写不可，先选一种处理：
- 删减次数
- 改成不同来源与不同代价
- 把重点从“遇到了”改成“遇到后不同人怎么处理”

### 9. 最后做“换名测试”
把角色名字遮住再读一遍。
如果三个人的反应互相都能替换，说明人物没有立住。

## 输出格式
1. 桥段真实基础
2. 共用外部流程
3. 每次出现的变化点
4. 角色 A / B / C 的反应对照
5. 这次重复主要塑造了谁
6. 应保留、删减还是改写

## 快速示例
### 输入
- 桥段：新到牢里，被差役勒索
- 人物：A 忍让老成；B 江湖硬汉；C 官场老油条

### 输出
- 共用流程：入狱 → 被勒索试探 → 角色回应 → 决定是否挨打 / 被照顾
- A：先低头摸清规则，再送钱求稳，事后只剩屈辱
- B：当场回顶，宁吃苦也不认怂，爽点强
- C：先一步送礼铺人情，把坏事变成机会，显出老练

## 注意事项
- **禁止**把低真实感奇遇机械复读
- **禁止**只换台词，不换策略和后果
- **禁止**同类桥段每次都靠同一个解法收场
- 重复桥段真正要重复的，不是事件表面，而是**世界规则；真正要变化的，是人物**
