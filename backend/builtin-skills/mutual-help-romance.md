写互帮互助感情戏时，让双方都先按自己的方式处理问题，再让对方补上自己解决不了的缺口。**重点不是“谁更强”，而是“你有你的解法，我有我的补位”。**这样既能写并肩感，也能避免一方长期当等救的挂件。

## 适用情况
- 双强、强强互补、成长型 CP
- 想写救场、共渡难关、并肩感
- 一方总在救另一方，关系失衡
- 需要在事件里自然升级感情

## 输入
- 当前危机 / 麻烦
- 双方各自的能力类型
- 谁先卷入问题
- 这场想突出尊重、心疼、默契还是偏爱

## 步骤
### 1. 先给双方各写一种“自己的解决办法”
不要让两个人只会同一种招。常见区分：
- 武力
- 智谋
- 资源
- 人脉
- 情绪安抚
- 善后能力
- 舆论 / 法律 / 社交处理

### 2. 让当事人先自救
遇事时，先让卷入麻烦的人自己反应、自己出手、自己扛一轮。
哪怕暂时解决不了，也要表现出：
- 有判断
- 有行动
- 有代价

**先自救，再被补位，比原地等救更带感。**

### 3. 再让另一方用不同方法补刀
补位不要重复前面的手段。优先写：
- 对方打完正面战，他来善后
- 对方扛过危机，他来追责或报复
- 对方解决表层问题，他来解决隐藏后果
- 对方保住结果，他来保住人

### 4. 帮助要落到“尊重对方能力”
最顺手的写法不是“我来替你做”，而是：
- 看见对方已经做了什么
- 承认对方的办法有效
- 只补对方暂时补不到的那一块

这样写出来是并肩，不是施舍。

### 5. 让互助赢来关系升级
一场互帮互助赢至少回收 1 项：
- 更信任
- 更欣赏
- 更理解对方的处事方式
- 发现对方不同于自己的强项
- 下次更愿意把后背交给对方

### 6. 控制“救场强度”
不要次次都救命。互帮互助也能落在小事上：
- 工作翻车后的补救
- 家庭危机里的站队
- 被误会后的澄清
- 舆论风波后的兜底
- 情绪崩溃时的稳住

## 输出格式
1. 当前问题
2. A 的解法
3. B 的解法
4. 谁先出手
5. 谁来补位
6. 事件后果
7. 关系增量

## 快速示例
### 输入
- 男主被混混围堵，女主武力强，男主擅长动用势力善后

### 输出
- A 的解法：男主先判断局势，准备先认打再事后报复
- B 的解法：女主路过，直接把混混揍散
- 补位：几天后男主动用势力把幕后的人一起收拾
- 关系增量：双方都看见了“对方不是等人救的废物，而是和自己一样有办法的人”

## 注意事项
- **禁止**把一方写成长期无能、只负责被救
- **禁止**让帮忙直接抢走对方高光和成长
- **禁止**两个人每次都用同一种解决办法
- 最好让双方的帮助既解决问题，也暴露价值观和性格差异
