围绕事业线先判断“全文到底靠什么矛盾支撑”：是世界、行业、身份落差构成的隐藏背景矛盾，还是明确敌对势力构成的反派矛盾。**这两种矛盾决定的不是一个桥段，而是整篇文的爽点来源、高潮位置和收尾方式。**

## 适用情况
- 升级文、娱乐圈、职场、竞赛文写着写着变平
- 不知道主线该盯世界规则，还是盯某个反派
- 打脸很多但不爽，感觉都在写小炮灰
- 大纲分裂，高潮不明确
- 想写事业 + 感情，但不知道哪条才是主线

## 输入
- 一句话题材
- 主角起点与核心能力 / 金手指
- 想让读者主要吃到什么：升级感 / 复仇 / 群体真香 / 感情互动
- 当前反派或世界阻力
- 预计篇幅

## 步骤

### 1. 先写主线承诺
写一句：
`主角通过____，最终达成____。`

再追问：
- 读者更想看主角不断爬高，还是狠狠干掉某个敌对势力
- 恋爱对象是主线矛盾，还是调味 / 并线

### 2. 区分两种事业线矛盾
#### A. 隐藏的背景矛盾
世界、行业、时代、身份落差本身就在和主角作对。

常见形态：
- 文化贫瘠、技术落后、行业标准低
- 穿书 / 重生后从极低起点重新拿回本事
- 社会评价与主角真实能力严重错位
- 世界天花板明确，可以一路升级

这类文的本质往往是：主角在对抗一个“大环境的缺口”。

#### B. 目标明确的反派矛盾
有清晰敌对方：
- 血海深仇
- 明确反派组织
- 竞对势力
- 一直阻拦主角达成目标的核心敌人

这类文的本质是：主角和一个势力不断对抗，直到解决它。

### 3. 用三个问题判断当前该选哪种
- 如果删掉具体反派，爽点还成立吗？
  - 还成立，多半是背景矛盾
- 如果没有世界 / 行业落差，这篇文还推得动吗？
  - 推得动，多半是明确反派矛盾
- 高潮更像“登上更高层级”，还是“击败某个势力”？
  - 前者偏背景矛盾
  - 后者偏反派矛盾

### 4. 如果是背景矛盾，重点写“对照”和“世界反馈”
优先设计：
- 主角的起点落差
- 行业 / 世界的低标准
- 主角每次出手后群体真香、认知改写、资源提升
- 世界天花板与换图节点

**不要把大量篇幅浪费在低级炮灰反复刁难上。** 背景矛盾的爽点，很多时候来自“整个环境被主角抬高了标准”。

### 5. 如果是明确反派矛盾，重点写“势力推进”和“终局收束”
优先设计：
- 反派势力结构
- 每一阶段怎么逼近核心敌人
- 反派如何影响主角现实利益
- 高潮如何正面解决敌对势力

**这种结构的结局通常是解决敌人，而不是无限拉长升级。**

### 6. 两种矛盾的节奏区别
- 背景矛盾：可长期拉升，高潮区间较分散，结局常由世界上限决定
- 反派矛盾：高潮区间更集中，结局常由敌我决战决定
- 如果中途换赛道，要明确是“换图”还是“升级反派层级”

### 7. 别让小炮灰遮住主线
反查当前打脸或挫折：
- 它是在放大世界落差，还是只是在让主角受气
- 它是在逼近核心反派，还是开了无关支线
- 读者期待的到底是看主角被刁难，还是看主角兑现能力

如果只是换着人名写刁难，多半是在消耗主线。

### 8. 事业线和感情线谁主谁副
- 如果事业线是主矛盾，感情线要服务事业推进或提供第二层拉扯
- 如果感情线是主矛盾，事业线更像放大环境
- **不要嘴上说双线并重，实际两条线互不相干。**

### 9. 最后确定结尾条件
- 背景矛盾文：主角到达世界上限 / 换图前阶段目标完成
- 反派矛盾文：核心敌对势力解决，主角完成立场或地位转换

结尾条件不清，大纲很容易越写越散。

## 输出格式
1. 一句话题材
2. 当前主线类型：背景矛盾 / 反派矛盾
3. 读者主要期待
4. 爽点主要来源
5. 高潮形式
6. 结尾条件
7. 当前桥段是否跑偏
8. 需要删弱的炮灰线
9. 事业线与感情线主次建议

## 快速示例
### 输入
- 影后穿成十八线，被群嘲，想写逆袭

### 输出
- 主线类型：背景矛盾
- 爽点来源：行业评价错位 + 群体真香
- 高潮形式：一轮轮作品、资源、口碑升级
- 风险：如果一直写剧组里某个小角色打压她，反而浪费了更大的背景矛盾

### 输入
- 主角背负血仇，要扳倒某组织

### 输出
- 主线类型：反派矛盾
- 爽点来源：不断逼近并解决敌对势力
- 高潮形式：最终正面对决
- 风险：如果中段全在写无关小炮灰，主线会散

## 注意事项
- **禁止**背景矛盾文全靠一个个路人炮灰撑篇幅
- **禁止**反派矛盾文迟迟不推进核心敌人
- **禁止**把感情线和事业线各写各的
- 确定事业线主要矛盾后，再决定爽点写法、高潮位置和篇幅长度
