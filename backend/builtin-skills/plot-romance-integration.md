围绕主线让人物、剧情、感情互相咬合。**剧情流里的感情线，不是停下主线专门谈恋爱，而是让事件刺激人物，人物选择推动剧情，剧情变化再反过来改变关系，形成“人物—剧情—感情”的黄金三角。最怕的不是一方暂时不在场，而是他一离场，故事也像把他忘了。**

## 适用情况
- 剧情线和感情线越写越像两本书
- 一推主线，另一位主角就像下线
- 读者总说另一位主角没存在感、想看糖
- 主角和配角玩得太热闹，CP 反而像工具人
- 长篇剧情流不知道怎么安排感情节奏

## 输入
- 题材与整体基调
- 主线目标与阶段高潮
- 两位主角的身份、能力、地点、任务与限制
- 当前关系阶段
- 是否存在长期分开、双线推进或戏份失衡

## 核心原则
1. **先判主次线。** 升级、悬疑、任务、冒险是主菜时，感情线要服务主线，不要空糖自转。
2. **人物、剧情、感情必须互相产出。** 事件刺激人物，人物推动剧情，剧情再改变关系。
3. 感情描写不等于亲亲我我。互信、补位、托底、站队，本身就是感情戏。
4. 一方可以暂时不在场，**但与他有关的目标、影响、后手、牵挂不能一起消失。**
5. **读者催 CP 往往是症状，不一定是药方。** 先查是不是剧情不够抓人、在场主角不够亮，再决定要不要加出场或发糖。
6. 每场关键戏最好同时推进至少两项：剧情变化、关系变化、人物立场。

## 步骤
### 1. 写清这本书真正卖什么
用一句话写：
`这本书最值钱的是____。`
再补：
- 主线负责什么
- 感情线负责什么

### 2. 从设定阶段就把两个人绑进主线
优先绑定：
- 技能互补
- 身份互补
- 阵营对立或合作
- 共同目标
- 共生危机
- 强绑定关系，如搭档、契约、婚约、共同行动身份

### 3. 分清两个人各自的剧情功能
分别写：
- A 推进什么
- B 推进什么
- 删掉 B，主线会少什么
如果一方除了被爱、被想起、最后救场之外没有别的功能，就要重做人设或任务分配。

### 4. 选并线方式
- 相辅相成型：一起办案、闯关、打怪，剧情推进本身就是感情推进
- 主线延伸式：主线不停，感情线插在阶段节点里
- 双线展开法：两边各自推进，但关键节点必须互相补完
- 交叉型：剧情高潮和关系高潮交替顶上来

### 5. 一方分开时，不让他真的失联
可用：
- 通讯、书信、视频、战报
- 信物、旧约定、未完成的事
- 某个物件、地点、任务触发想起对方
- 对方留下的布局、资源、麻烦
- 他人口中提到对方的新动作

### 6. 用人物魅力刷存在感
别只靠话多。优先回收：
- 言谈举止
- 做事风格
- 价值观、道德观、审美观
- 他给别人留下的规矩、影响和判断方式
做到“人不在场，风格还在场”。

### 7. 把糖打碎塞进剧情缝里
用小互动而不是整段停下来谈恋爱：
- 一个眼神
- 顺手擦汗、递水、护一下
- 忙里偷闲的几句通讯
- 睹物思人触发一小段回忆
- 危机里先确认对方有没有事
**最好前文有伏笔，避免牵强。**

### 8. 处理戏份失衡
如果配角比 CP 更像 CP，优先：
- 把关键情报、关键决策、关键代价分回另一位主角
- 让另一位主角和主线问题发生更直接的因果关系
- 重写两人的任务分工，别让一方长期只能等待

### 9. 别把读者吐槽当字面指令
例如“他很久没出场了”“别打怪了快谈恋爱”，先反查：
- 在场主角是不是不够亮
- 当前段落是不是爽度不够
- 这段剧情是不是缺少关系回收
找到根因，再决定是补人物、补节奏，还是补互动。

## 输出格式
1. 这本书真正卖什么
2. 主线 / 感情线各自职责
3. 两位主角与主线的绑定点
4. 两位主角的剧情功能分工
5. 选择的并线方式
6. 分开时的存在感回收方案
7. 可插入的细节糖
8. 当前戏份失衡点
9. 读者吐槽背后的真正问题
10. 一场关键戏如何同时推进剧情与感情

## 注意事项
- **不要**把剧情线和感情线长期分开跑
- **不要**用大段思念独白代替存在感
- **不要**让另一位主角除了恋爱什么都不干
- **不要**一见读者催 CP 就机械加戏
- 最稳的写法之一是：**先把两个人都绑进主线，再让剧情把感情一层层顶出来。**
