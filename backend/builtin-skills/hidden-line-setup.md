把必须存在、却不值得正面细写的内容改成暗线。**暗线不是没写，而是用侧面痕迹、时间安排、结果变化和人物缺席，把过程压进正文背后。**这样感情流不会被流水账拖慢，剧情也不会无凭无据。

## 适用情况
- 感情流、轻剧情文需要给人物行动依据
- 学习、事业、调查、准备过程全写太碎
- 不知道角色为什么会出现在某处、为什么心情突然变差
- 想预告大事将至，但不想提前掀牌
- 明明有剧情，却一写就变成“全过程直播”

## 核心原则
1. **明线写当下最值钱的戏，暗线写必要过程。**
2. **暗线一定要能回收，不能只停在作者脑补。**
3. **侧面描写看结果和余波，不追全过程。**
4. 一个暗线至少承担一种功能：解释出场、支撑情绪、铺垫转折、预告大事。

## 输入
- 当前明线是什么
- 哪条内容不适合正面细写
- 这条暗线之后要解释什么、推动什么
- 当前题材与节奏（感情流 / 校园 / 悬疑 / 日常等）

## 步骤
### 1. 先分清明线和暗线
分别用一句话写清：
- 明线：这一段读者正在看的主戏是什么
- 暗线：哪条事必须存在，但不想正面展开

例如：
- 明线：男女主关系升温
- 暗线：女主正在为考试冲刺

### 2. 给每条暗线写“回收问题”
先回答：
`这条暗线以后要解释什么？`

常见回收：
- 为什么角色会在这个时间、这个地点出现
- 为什么最近不回消息、不参加聚会
- 为什么情绪低落、身体变差、反应异常
- 为什么某个转折来得不是凭空天降

### 3. 选 3—5 个侧面证据
不要直接写过程，优先写痕迹：
- 外貌变化：黑眼圈、消瘦、狼狈、旧伤
- 作息变化：晚归、缺席、熬夜、早起
- 行为变化：推掉聚会、少说话、频繁看时间
- 结果变化：成绩波动、工作失误、东西准备齐全
- 旁人反馈：朋友随口提一句、老师点名、同事抱怨
- 环境证据：书桌凌乱、资料堆积、手机未读很多

### 4. 把证据分散到场景里
暗线不要集中说明一大段，优先用这些方式带出：
- 一笔带过的外貌和状态
- 对话里顺手提到
- 配角一句无心的观察
- 场景里的物件和痕迹
- 角色因为暗线而错过、迟到、分心

**原则：让读者感到“事情一直在发生”，而不是“作者突然想起来补设定”。**

### 5. 大事将至时，提高信号密度
如果这条暗线后面要爆：
- 前期：轻轻露一次
- 中段：再露一次，并让角色付出一点小代价
- 爆发前：信号明显一点，但不要直接说破

这样转折会像水下早有暗流，而不是凭空掉下来。

### 6. 回收时把线接上
当暗线转明线时，至少做到一项：
- 让读者明白前面的痕迹都指向这里
- 让角色行动更有根据
- 让关系变化更顺
- 让后续冲突显得“早有苗头”

### 7. 控制隐和明的比例
- 太隐：读者会觉得莫名其妙
- 太明：就失去暗线的节奏价值

稳妥标准：**第一次读能感到不对，回收时能想起前面早埋过。**

## 输出格式
1. 当前明线
2. 需要处理的暗线
3. 暗线以后要回答的问题
4. 可用的侧面证据 3—5 个
5. 每个证据适合放在哪场戏
6. 爆发前信号如何加重
7. 暗线转明线时怎么回收
8. 太隐 / 太明的风险点

## 快速示例
### 输入
- 题材：校园感情流
- 明线：男女主暧昧升温
- 暗线：女主的真实目标是考上清华

### 输出
- 回收问题：为什么女主最近很少出现、为什么情绪突然低落、为什么会在考试后失态
- 侧面证据：黑眼圈、拒绝同学聚会、书包里一直带着错题本、某次考试从第一掉到第二
- 分散方式：约会前打哈欠、朋友说她最近根本约不出来、男主在她家门口看见她房间灯总是亮到很晚
- 回收节点：成绩波动和升学压力正式爆开，前面的所有迹象都有了解释

## 注意事项
- **不要**把暗线写成作者自己知道、正文完全看不见的设定
- **不要**一写暗线就把全过程详写出来
- **不要**让暗线只负责解释，不负责推动后续
- **不要**每次都靠旁白总结“她最近很努力”
- 好用的暗线，不是藏得深，而是：**平时像余波，回收时像必然。**
