写破镜重圆时，先把“为什么会碎”和“凭什么还能圆”同时设计出来。**破镜重圆最怕两头都轻：分开理由不够狠，重逢拉扯也不够真。**稳写法是：让当年的“破”看起来足够致命，却又留有可回收空间；让现在的“圆”不是突然原谅，而是在被迫重逢和不断碰撞里，看见旧伤、旧爱和新选择。

## 适用情况
- 写旧情人重逢、分手多年复合、离婚后回头、真香回头
- 不知道该从初恋、分手还是重逢写起
- 分手理由不够信，或一写就变成不可原谅
- 重逢后只有回忆，没有新的拉扯和张力
- 想写误会型重圆，却总觉得“圆不回来”

## 核心原则
1. **先定“破”，再定“圆”。**
2. **破镜理由要像石锤，但必须可回收。**
3. **重逢必须有抽不开身的绑定。**
4. **表面敌对，底下仍有默契、旧习惯和余情未了。**
5. 真正让两人复合的，往往不是一个误会真相，而是：**旧刺被拔掉后，他们在现在仍然选择彼此。**

## 输入
- CP 模式（旧情人 / 离婚 / 分手 / 火葬场）
- 当年分开的直接原因
- 真正的底层原因（误会 / 不成熟 / 立场冲突 / 现实阻隔）
- 现在重逢的处境
- 想卖的口味（酸涩 / 真香 / 破防 / 成熟感 / 追妻火葬场）

## 步骤
### 1. 先判断“破镜”能不能圆
优先选择这些分手原因：
- 误会看起来像石锤
- 年轻时不成熟、太骄傲、不会表达
- 现实压力、立场冲突、身份问题
- 一方做了伤人的选择，但仍有可理解前因和代价

谨慎使用：
- 触底线的背叛
- 杀亲灭门式血海深仇
- 无法补回的长期恶意伤害

**如果当年的“破”已经让读者觉得不该原谅，就不要硬写成甜圆。**

### 2. 把“石锤”和“真相”拆开
破镜好看，往往在于：
- 当时看，真的足够分手
- 后来回头看，事情并不完全是表面那样

先分别写清：
- 当年角色以为发生了什么
- 读者最初会怎么判断
- 后续真正揭开的真相是什么
- 真相揭开后，哪些伤害仍然存在，不能一笔抹掉

### 3. 先定“第二次感情”的起点
大多数破镜重圆，正片应该从这里开始：
- 重逢当下
- 即将被迫合作的前夜
- 两人已经不能再装不认识的节点

**不要**默认从第一次相遇开始顺叙。除非第一次恋爱本身就是全书最大卖点，否则很容易把主体写歪。

### 4. 给重逢安排“抽不开身的绑定”
让两人不能见一面就翻脸离开。

常用绑定：
- 同项目、同剧组、同案子、同任务
- 家庭事务、孩子、合同、继承、债务
- 被迫同住、同队、同公司、同门
- 必须共同面对的外部危机

**没有绑定，重逢就容易只剩回忆杀。**

### 5. 重逢阶段写“仇人态度 + 余情未了”
表层可以是：
- 冷淡
- 阴阳怪气
- 针锋相对
- 互相揭旧账

底层必须留住：
- 太熟了，所以一眼就知道对方下一步会怎么做
- 明明在怼，关键时刻还是会护
- 嘴上不想见，身体和选择先露馅
- 争吵里有旧伤，也有旧默契

**不要**写成真的恨到骨髓里，不然“圆”会很假。

### 6. 给两人各埋一根“旧刺”
破镜重圆里，阻碍通常不只一个误会，而是两层：
1. 当年的那根刺
2. 现在谁都拉不下面子的自尊

可用问题：
- 我以为你不爱我
- 我以为你背叛我
- 我其实知道你有苦衷，但还是过不去
- 明明还爱，却不想先低头

### 7. 让现在时线单独成立
现在的关系线必须有：
- 新的吸引
- 新的抗拒
- 新的冲突
- 新的选择

也就是说，读者不只是靠回忆嗑他们，还要看到：
- 他们现在为什么又被彼此吸引
- 他们现在有没有变得比从前更会爱人
- 他们这次凭什么能成

### 8. 插叙只回收“会改变现在”的信息
过去的回忆至少完成一项：
- 揭示当年分开的关键原因
- 推翻读者或角色的旧判断
- 让现在的关系更近一步或更痛一步
- 展示人物成长后的差异

如果一段回忆既不改当前关系，也不补关键认知，就删。

### 9. 安排“双向低头”的节点
破镜重圆的高光，常常不是一个人突然想开，而是：
- 两边都还在爱
- 两边都在嘴硬
- 两边都准备自己低头
- 却发现对方也在低头

这种节点前，先写足：
- 拉扯
- 误判
- 不舍
- 一次关键的主动选择

### 10. 最后检查“圆”的凭据
逐条确认：
- 旧刺是否真的被处理，不是装没发生
- 现在的两个人是否比当年多了新选择和新能力
- 重逢后的相处里，是否写出了新的信任基础
- 复合是不是建立在“现在仍然互相选择”上

## 输出格式
1. 破镜类型
2. 当年的石锤表象
3. 真正的底层原因
4. 第二次感情的起点
5. 被迫重逢的绑定方式
6. 当前阶段的“仇人态度 + 余情未了”表现
7. 两人的旧刺和面子问题
8. 3 个现在时转折点
9. 插叙要回收的信息
10. 最终复合的凭据

## 快速示例
### 输入
- 旧情人分手多年后重逢
- 女主当年以为男主出轨
- 现在两人要一起负责同一个项目

### 输出
- 石锤表象：当年所有证据都指向男主劈腿
- 底层原因：信息被刻意误导，外加两人年轻时都太骄傲，谁也不肯解释
- 起点：从项目合作重逢写起，而不是从学生时代初遇写起
- 绑定：项目不结束，两人谁也走不开
- 当前关系：表面公事公办，私下却总能接住对方的话和习惯
- 旧刺：女主过不去“被背叛”的羞辱感；男主过不去“你从没信过我”的伤
- 转折点：一次争吵、一次救场、一次真相回收
- 复合凭据：真相只是拔刺，真正让他们复合的是这次两人都学会了不再逃避

## 注意事项
- **不要**把分手理由写得太轻，像小学生闹别扭
- **不要**把分手理由写得太重，重到根本不该圆
- **不要**让重逢后只有回忆，没有现在时主线
- **不要**真相一揭开就立刻和好，旧伤和面子都要处理
- 好看的破镜重圆，不是“误会解除就自动复合”，而是：**当年的刺拔掉以后，他们在现在仍然想走回彼此身边。**
