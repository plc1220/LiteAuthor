围绕“爱情必须经受考验”的命题，按“相遇—相爱—定情—分离—追寻—重逢”生成感情主线。重点写清两人为什么相爱、为何被迫分开、谁主动追回、阻碍如何升级，以及重逢后感情为什么比从前更牢。

## 适用情况
- 想写真爱经受考验的爱情主线
- 两个角色已经相爱，但中段发空，不知道该怎样拆开又如何合回去
- 想把追寻、营救、重聚写得更有层次
- 想让重逢不只是见面，而是关系升级

## 输入
- 恋人双方的人设、缺口、魅力点
- 两人为何偏偏选中彼此
- 定情方式：承诺 / 信物 / 婚约 / 生死与共 / 共同目标
- 分离原因：反派、战争、误判、现实阻碍、疾病、伤痛等
- 主动追寻的一方是谁
- 阻碍势力是谁或是什么
- 结局倾向：HE / BE / 开放式

## 步骤
### 1. 先立住“为什么是这两个人”
分别写清：
- 各自 1 个让读者易代入的点
- 各自 1 个独特而有趣的点
- 各自 1 个让对方无法替代的吸引点

**不要只写“很爱”“很配”。要写成：换成别人，这段感情就不成立。**

### 2. 安排相遇与定情
用 1—3 场关键戏完成：
- 相遇
- 建立关系
- 深度相爱
- 某种形式的定情

定情最好留下可回收凭据，例如信物、誓言、约定、共同秘密、共同选择。

### 3. 提前埋一处不和谐
即使两人深深相爱，也保留一处会在后文放大的差异：
- 性格差
- 身份差
- 立场差
- 创伤反应差
- 表达方式差

这不是为了拆 CP，而是为了让后续考验更真实。

### 4. 设计分离事件
分离必须足够重，能真正威胁关系。
优先使用：
- 外部强敌
- 重大意外
- 身份暴露
- 战争 / 家族 / 阵营阻隔
- 疾病、伤痛、失踪、被囚

**不要用轻飘误会硬拆。** 分离后，至少有一方必须主动寻找、营救或试图重新结合。

### 5. 明确谁是主动追寻方
写清主动方：
- 为什么必须由 TA 先行动
- TA 要跨过哪些现实阻碍
- 被动方为什么暂时只能等待、躲避或受困

双方主动/被动不受性别限制，关键是行动链谁来推。

### 6. 用“三次法则”安排追寻过程
按三次尝试排：
1. **第一次失败**：确认阻碍真实存在
2. **第二次失败**：代价更重，感情更痛
3. **第三次关键尝试**：逼出最终选择或真正的解决办法

每次失败后都要新增一种东西：新情报、新伤口、新决心或新代价。

### 7. 把情绪落到场景里
围绕这些情绪安排具体戏：
- 恐惧
- 厌恶
- 吸引
- 失望
- 重新接纳
- 圆满

**不要空喊深情。** 少写“我爱你”，多写“我为你做了什么、失去了什么、仍然怎么选”。

### 8. 设计高潮重逢
高潮至少同时完成三件事：
- 主动方找到打败反派或穿越阻碍的方法
- 恋人重新站到一起
- 感情被重新定义得更牢、更重、更成熟

重逢不是回到原点，而是比原来多一层共同命运、共同选择或共同责任。

### 9. 检查是否写成空泛煽情
逐条检查：
- 感情是不是只靠台词，不靠行动
- 分离是不是只为拖时间
- 追寻失败后有没有新变化
- 重逢后关系有没有升级，而不是恢复出厂设置

## 输出格式
1. 核心爱情命题
2. 双方不可替代的吸引点
3. 相遇与定情节点
4. 分离原因与主要阻碍
5. 主动追寻方及行动路线
6. 三次尝试设计
7. 情绪层次表
8. 高潮重逢场面
9. 重逢后的关系升级
10. 翻车风险

## 快速示例
### 输入
- 落难少主 × 民间女医
- 定情方式：共患难后互换玉佩
- 分离原因：少主被敌国掳走

### 输出
1. 核心命题：只有经受生死与立场考验的爱情才站得住
2. 吸引点：女医看中少主的担当，少主看中女医在乱局中的清醒与善意
3. 定情：逃亡途中女医放弃安全退路救他，两人以玉佩为信
4. 分离：少主被押往敌都，女医被迫留在边城
5. 主动追寻方：女医先行动，借行医身份潜入敌国
6. 三次尝试：潜入失败 → 交换人质失败 → 借敌国内乱成功救出
7. 高潮：两人联手扳倒幕后黑手
8. 关系升级：从私人深情升级为共同命运

## 注意事项
- **禁止**用轻飘误会强拆强圆
- **禁止**把追寻过程写成重复闯关
- **禁止**只写感伤，不写行动与代价
- **禁止**重逢后直接回到从前，好像什么都没发生
- 真正打动人的不是“他们终于见面了”，而是**他们为了再次站到彼此身边，已经被考验成了更能相爱的两个人**。
