把一个带情绪冲击的细节当成种子，往前追前因，往后追后果，再给它配人物、关系和时代背景，长出完整故事。**最能生发故事的细节，往往本身就像一个高潮、终点或余波。**

## 适用情况
- 先想到一个画面、一个细节，却不知道怎么展开
- 细节很有味道，但故事接不上
- 想写短篇、单元故事、人物前史
- 想从一枚物件、一道伤痕、一处遗痕长出情节

## 核心原则
1. 最适合生发故事的细节，通常自带**高潮感**。
2. 好细节会自动逼出追问：是谁留下的？为什么会这样？这之前发生了什么？这之后谁被改变了？
3. 先别急着解释，把细节当成故事的爆点、终点或余波来看。
4. 好细节不是装饰，它能反推人物性格、身世、境遇、关系和时代。

## 步骤
### 1. 选“有故事味”的细节
优先选：
- 带伤痕、遗痕、残留的物件或场景
- 一个不合常理的状态变化
- 一个过分鲜明的动作或后果

差的种子是：只是好看，没有追问。

### 2. 先写这枚细节里藏着什么情绪
填空：
- 这枚细节最强的情绪是____
- 它像故事里的____（高潮 / 终点 / 转折 / 余波）

### 3. 往前追五步
依次回答：
1. 这是谁留下的
2. 当时发生了什么
3. 他为什么会走到这一步
4. 他最放不下的人或事是什么
5. 若没有这次事件，他原本会过什么生活

### 4. 往后追三步
继续回答：
- 这枚细节改变了谁
- 它会被谁看见、误解、纪念或利用
- 它会引出什么新的关系、冲突或寻找

### 5. 给细节配人
至少补三项：
- 性格
- 身世 / 处境
- 与另一人的关系

同一枚细节，放在普通人、将军、逃犯、恋人身上，故事完全不同。

### 6. 给细节配背景
问：
- 什么时代、环境最能放大它
- 这是个人命运，还是时代压上来的痕迹

### 7. 拉出故事线
常用走法：
- **细节 → 追问 → 人物 → 前因 → 转折 → 细节回响**
- **细节 → 见证人 → 误解 / 寻找 → 真相揭开**

### 8. 检查是不是只剩气氛
如果删掉这枚细节，人物关系、情节方向都不受影响，说明它还只是摆设。

## 输出格式
1. 核心细节
2. 细节里的情绪
3. 前因五步
4. 后果三步
5. 关键人物设定
6. 背景与时代
7. 一句话故事
8. 可直接开写的开头

## 快速示例
### 输入
- 城墙上留下一个旧箭坑

### 输出
- 情绪：终点感、遗恨
- 前因：死者是谁，为谁而战，是否被命运或自己人背刺
- 后果：远方是否有人在同一时刻失去什么；多年后谁看见这个箭坑并追查
- 一句话故事：一个本可回家的人，死在离胜利最近的一刻，而箭来自自己一方

## 注意事项
- **不要**同一篇里同时抓太多等重细节
- **不要**只有意象，没有人物
- **不要**把解释一次说完，先让追问带着读者走
- **不要**把细节当成纯装饰，它必须能反推剧情
