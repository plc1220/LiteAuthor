把感情线拆成阶段 + 横杠 + 打点 + 错位。阶段决定能做什么，横杠是共同经历，打点是情感改变的证据，错位是拉扯来源。**不要只写感情加深了，要写清从什么变成什么。**

## 适用情况
- 感情线只会直推，不会分阶段
- 从熟悉到心动像突然跳级
- 两人互动很多，但读者感受不到关系变化
- 想写双暗恋、慢热，却总是发空

## 常用阶段
- **陌生**：警惕、排斥、客气、绕开
- **熟悉**：能正常来往、会帮忙、会站在对方立场想一层，**但还不是心动**
- **暗恋**：目光跟着走、评价改口、会吃醋、注意力被夺走
- **告白前主动**：开始试探、争取独处、想把关系往前推
- **刚在一起**：关系确定了，但对新身份、亲密动作、外界目光不适应
- **稳定期**：默契自然，外人面前也不再束手束脚

## 写法步骤
1. 先把每一大段写成 `阶段 → 阶段`，例如陌生→熟悉、熟悉→暗恋。
2. 给中间那条横杠安排**共同经历**：一起工作、一次误会澄清、一次危机共担、连续日常相处。只问一句：`这件事之后，对对方的评价具体改了什么？`
3. 每次转段都做**前后对照打点**。最稳的写法是同一刺激，不同反应。
   - 早期：看到别人靠近 B，A 只觉得 B 挺有魅力
   - 暗恋期：再看到同样场面，A 会烦、会酸、会挑刺
4. 每次转段至少翻一项认知：不靠谱→靠得住；普通同事→特别的人；花心浪子→其实有原则。
5. 允许**阶段错位**：A 已暗恋，B 还在熟悉；A 想告白，B 刚意识到心动。错位本身就是矛盾来源。一段关系通常以**更慢的那个人完成转变**为阶段结束。
6. 涉及感情推进的场景，优先写我眼里的你变成了什么人。能开 POV 就写印象变化；想保苏感就用行为和旁观者反应代替直白心理。

## 输出格式
1. A / B 当前阶段
2. 目标阶段
3. 共同经历
4. 前后对照打点
5. 是否错位
6. 本段结束后的关系余波

## 注意事项
- **禁止**把熟悉直接写成心动
- **禁止**没有共同经历，只靠作者宣布关系变了
- **禁止**两人永远完全同步
- **禁止**刚确认关系就直接写成老夫老妻
