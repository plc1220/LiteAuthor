把事业线当成主角从 0 级到满级的过程来设计。**事业不等同职业，只要贯穿全文、逼着主角长期行动的目标，都算事业线。**

## 适用情况
- 想写升级、逆袭、创业、复仇、寻药、求学、修炼等长期目标
- 只有几个桥段，没有完整事业线
- 不知道起点、中点、终点怎么排
- 事业线写着写着跑散，或者中段突然没目标

## 输入
- 题材与一句话核心目标
- 主角当前状态
- 终点想让主角达到什么
- 预计篇幅
- 是否需要兼顾感情线

## 步骤
### 1. 先定终点
用一句话写：
`主角最终通过____，达到____。`

终点必须是可验证的变化：拿到药、成为盟主、发表论文、拿下公司、复仇成功、洗清冤屈。

### 2. 把终点翻成“0级→满级”
列 3—5 个能被读者看见的升级刻度。
- 身份 / 地位
- 资源 / 能力
- 身体 / 处境
- 社会评价
- 能解决的问题级别

如果开局就是满级，改用：
- **满级→0级→再满级**
- **跨行业 / 换赛道，从 0 重新开始**

### 3. 设计起点：1 个欲望 + 1 个缺陷
每个阶段最好只保留 **1 个欲望 + 1 个缺陷**，最多两组。
- 欲望：主角主动想得到什么
- 缺陷：主角现在缺什么、卡在哪里

**欲望和缺陷必须有逻辑关系。**
- 好例：想活下去 / 快死了
- 好例：想赚大钱 / 现在极度缺钱
- 差例：想赚钱 / 但核心缺陷是“没人夸我”

### 4. 写出“谁能满足欲望，谁能解决缺陷”
分别写 2 个答案：
- 谁 / 什么能满足欲望
- 谁 / 什么能解决缺陷

两者可以相同，也可以不同。
如果想顺手带感情线，优先让其中至少一个答案和恋爱对象有关。

### 5. 设计中点：先满足旧需求，再长出新需求
**满足起点需求，通常不是终点，而是中点。**
中点常见写法：
- 旧目标达成，但关系出了新问题
- 旧缺陷解决了，却出现副作用
- 拿到了关键资源，却付出了更大代价
- 原事业完成，开启新事业

### 6. 判断要不要转换事业
常见三种：
1. 完成旧事业后，开启新事业
2. 放弃旧事业，改做更高一级的新事业
3. 同时推进两条事业线

第三种最难控，**同时进行的事业线最好不要超过 2 条**。

### 7. 每一阶段都按“目标→障碍→结果→新目标”推进
把篇幅切成几个阶段，每阶段只做一件事：
- 提出目标
- 遇到障碍
- 完成或放弃
- 引出新目标

小目标必须和总目标有递进关系，不要平级乱堆。

### 8. 用“虽然……但是……”拉长剧情
当旧目标暂时快完成时，继续追问：
- 虽然拿到了药，**但是**药有副作用
- 虽然有了论文，**但是**还差实验结果
- 虽然快当上盟主，**但是**还没有能服众的战绩

注意：**不要和阶段性小目标杠太久。** 小目标的急迫度不能长期压过总目标。

### 9. 控制数量，防止跑题
出现以下情况就要重排：
- 一个阶段塞了 3 个以上欲望
- 一个缺陷衍生出 4、5 个平级重点
- 主角天天忙小目标，几乎不再提总目标
- 中点后像换了一本书

## 输出格式
1. 事业线一句话定义
2. 0级状态 / 满级状态
3. 起点的欲望与缺陷
4. 满足欲望者 / 解决缺陷者
5. 中点变化
6. 终点回收
7. 阶段目标表（目标 / 障碍 / 结果 / 新目标）
8. 是否转换事业
9. 跑题风险

## 示例
### 输入
- 男主要寻药续命，20 万字，兼顾言情

### 输出
- 事业线：寻药活命
- 0级：病弱濒死，没人会为他破例
- 满级：拿到药并有能力掌握自己命运
- 起点欲望：拿到救命药
- 起点缺陷：快死了，且没有资源
- 满足欲望者：神医
- 解决缺陷者：能说动神医的女主
- 中点：拿到了药，但药有副作用；同时男主和女主决裂
- 后半段新目标：既要解决副作用，也要修复关系

## 注意事项
- **先有欲望，再写缺陷卡住欲望**，通常比“因为自卑所以行动”更有主动感
- **每阶段只抓重点**，重点太多就没有重点
- 事业线最怕“目标不升级、障碍不升级、代价不升级”
