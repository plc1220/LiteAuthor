把书名、文案和题材承诺当成和读者签的合同，先决定读者是为哪一口点进来，再把所有线索按主次重排。**一篇网文通常只该有 1 条主线 + 最多 2 条复线。多线不是不能写，而是每条线都必须服务同一承诺，且切线前要先兑现一个点。**

## 适用情况
- 设定很多，写着写着像大杂烩
- 题材承诺和正文重心不一致
- 家长里短、感情线、事业线、配角升级线互相抢戏
- 每条线都在吊胃口，但很少真正爽到、虐到或解决到
- 读者反馈“货不对板”“四不像”“越看越累”

## 输入
- 书名、文案、题材词
- 主角是谁，读者为什么要跟 TA
- 现有全部线索列表
- 目标读者最想看的体验
- 预计篇幅

## 步骤

### 1. 先写题材承诺
用读者语言，不用作者术语。

例如：
- 重生学霸：重生改命、学习升级、成绩打脸、竞赛逆袭
- 娱乐圈逆袭：出圈、涨粉、拿资源、公开认可
- 斗极品：受气 → 反击 → 翻盘 → 改处境

**书名和文案答应了什么，正文就要长期供应什么。**

### 2. 做“随机组合预期测试”
把书名与候选线索两两或三三组合，写一句“读者会期待什么”。

例如：
- 《重生学霸》+ 赚钱线：期待学业升级顺手改善处境
- 《重生学霸》+ 斗极品：期待成绩和认知优势反击亲戚
- 《重生学霸》+ 男主选秀长线：大概率违和，除非它直接服务女主学霸主线

如果某条线一组合就像另一本书，**删、弱化、后置，或干脆另开一本。**

### 3. 给每条线标四件事
逐条写清：
- 这是谁的线：主角 / 配角 / 群像
- 提供什么体验：爽、甜、虐、悬念、主题
- 会不会让读者因为它点进来
- 删掉后，正文是更清楚还是更空

### 4. 只保留 1 条主线 + 最多 2 条复线
判断标准：
- **主线**：最能兑现题材承诺、最能长期留住读者的线
- **复线**：能放大主线，或提供兼容的第二、第三种快感
- **删除 / 合并 / 后置**：不能放大主线、只会分走注意力的线

稳妥写法：
- 主线：学习升级 / 事业升级 / 复仇翻盘
- 复线 1：赚钱改善处境
- 复线 2：斗反派 / 斗极品 / 关键关系辅助

### 5. 检查冲突对象是否适配题材
- **爽文、升级文**：优先用清晰、可反击、可见证、能回收打脸的对立面
- **优质文、现实向**：可以用灰度亲情、复杂人性、说不清道不明的关系
- 如果卖点是“打脸逆袭”，就不要长期用“难以痛快解决的家庭琐碎矛盾”当主驱动
- 如果主打“大女主升级”，就不要让配角拥有比主角更完整的升级过程

### 6. 为每条保留线写“起点 → 卡点 → 回收 → 余波”
每条线都要回答：
- 一开始缺什么
- 中途被什么卡住
- 靠什么推进或翻盘
- 最后拿到什么结果
- 这个结果怎样反过来抬高主线

**只有过程，没有回收，就是吊胃口。**

### 7. 切线前先兑现一个点
正确切法：
- 主线刚完成一个小升级 / 小打脸 / 小回收
- 再切去复线补另一种推进
- 很快切回主线继续拉高

错误切法：
- 学习线刚要出成绩 → 切亲戚线
- 亲戚线刚要打脸 → 切感情线
- 感情线刚要表态 → 切赚钱线

这样会让读者一直“快爽到时被拖走”，最后变成**半爽不爽**。

### 8. 控制篇幅占比和主角 ownership
- 主线长期占最大篇幅
- 主角的推进频率必须高于配角
- 连续 3—5 章不推进主线，就要警惕跑偏
- 如果读者更清楚配角在干嘛，而不清楚主角主线到哪了，说明失衡
- 配角复线如果比主角更带感，要么改成服务主角，要么单独立项

### 9. 做一次“赛道一致性”检查
问自己：
- 结构是在写爽文、优质文，还是脑洞文
- 冲突对象和节奏是否与赛道一致
- 人设是需要脸谱鲜明，还是少数角色立体深入
- 文笔是该快、狠、准，还是该细腻、层层加深

**不要结构在写爽文，冲突却在写现实灰度家庭；也不要书名卖升级流，正文长期在写配角人生。**

### 10. 最后做“货不对板”测试
逐条问：
- 如果把书名改成另一题材，会不会更贴正文
- 读者为 A 进来，正文是否长期在写 B
- 文案最亮的卖点，正文是否真占最多资源
- 读者最在乎的那条线，是否拥有最多回收节点

有两项以上答“是”，就要重排主次。

## 输出格式
1. 题材承诺
2. 主线
3. 复线 1
4. 复线 2
5. 应删 / 应合并 / 应后置的线
6. 每条保留线的回收节点
7. 篇幅分配建议
8. 冲突对象适配度
9. 赛道一致性问题
10. 货不对板风险

## 快速示例
### 输入
- 题材：《重生学霸》
- 现有内容：学习逆袭、投资赚钱、斗亲戚、男主选秀、校园八卦

### 输出
- 主线：学渣逆袭学霸，竞赛升级，成绩打脸
- 复线 1：投资赚钱，改善家庭处境
- 复线 2：斗极品亲戚，扩大反击爽感
- 删除 / 大幅弱化：男主选秀全过程、与主线无关的校园八卦
- 原因：不承接“学霸”承诺，还会抢走主角升级篇幅和高潮节点

## 注意事项
- **禁止**让复线比主线更像主线
- **禁止**每条线都只吊胃口，不真正回收
- **禁止**用一种题材引流，用另一种内容占篇幅
- **禁止**主角文写成配角个人升级合集
- 线索多不是问题，**没有主次、没有兑现、没有题材对齐**才是问题
