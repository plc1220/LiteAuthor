按弹簧思路安排情节：**基准点 → 压抑值 → 爆发点 → 愉悦连锁。** 爽文不是一直爽，也不是一直惨；重点是让读者先感到“被按下去”，再看到它连续弹回来。

## 适用情况
- 故事情节平，主角很苏但剧情没起伏
- 中间明明留了低谷，读起来却没有危机感
- 情节总是几百字就带过，爆发不够
- 压抑写多了变憋屈，爆发写少了又不够爽
- 不知道一段爽感该铺多久、爆多久

## 核心原则
1. **基准点要清楚，压抑才有参照。**
2. 压抑必须递进，不能同一种委屈原样复读。
3. 爆发不要只打一拳就结束，最好有连续回合。
4. **爆发后要留一段“愉悦连锁反应”，把爽感吃满。**
5. 最大压抑和最大爆发，距离不要太远。

## 一个好用的十段参考
如果把一段情节粗分成十个单位，可以先这样排：
- **基准点 2 段**
- **压抑值 3 段**
- **爆发点 4 段**
- **愉悦连锁 1 段**

这不是死公式，但很适合检查：
- 前面是不是铺得太少，后面爆得太空
- 是不是压太久却没给回报
- 是不是刚爽一下就立刻切走

## 步骤
### 1. 先写基准点
基准点要交代清楚：
- 主角现在站在什么位置
- 主角本来有什么 / 应得什么
- 主角想守住什么 / 拿到什么
- 当前局面看起来有多稳或多顺

基准点的作用，是让读者知道“等下压下去的到底是什么”。

### 2. 再写压抑值
压抑阶段至少安排 3 次递进，每次都要加重不同的东西：
- 待遇更差
- 误解更深
- 机会被夺
- 局势更险
- 支持更少
- 名声更坏

**不要只是同一个人反复来找茬。**

### 3. 把爆发拆成连续 4 回合
爆发不一定是打架，也可以是翻盘、揭穿、关系反转、局势倒转。

推荐拆法：
1. **第一下回弹**：主角不再被动，先顶住或先反打
2. **第二下扩大**：主角把优势滚大，对方开始失衡
3. **第三下确认**：旁人、局势、规则开始站到主角这边
4. **第四下兑现**：主角拿回东西，对手吃到代价

这样比“憋很久，最后一句话翻盘”更有爽感。

### 4. 最后留一段“愉悦连锁”
爆发后不要立刻切走，要给一段后续反应：
- 旁人改口
- 反派破防
- 主角拿回评价权
- 关系升温
- 新资源到手
- 下一轮更大的期待被打开

**这一段是让读者把爽感真正吞下去的。**

### 5. 用情绪词检查每一段
给每一段都标一句：
- 这一段要让读者感到什么
- 这一段结束后最想看什么

例如：
- 基准：安心 / 期待
- 压抑：憋 / 急 / 替主角不平
- 爆发：解气 / 惊艳 / 痛快
- 连锁：舒坦 / 真香 / 还想继续看

### 6. 反查两种常见失衡
#### 压抑过量
- 一直憋屈
- 主角长期没尊严
- 爆发来得太晚

#### 爆发过轻
- 只靠旁白说“他赢了”
- 只让反派心里后悔一下
- 没有旁人见证
- 没有拿回任何实际东西

## 输出格式
1. 这段情节的基准点
2. 三个压抑节点
3. 四个爆发回合
4. 一段愉悦连锁
5. 读者情绪变化
6. 当前失衡点
7. 调整建议

## 快速示例
### 输入
- 主角起点很高，但剧情总写不出危机感

### 输出
- 基准点：主角站在高处，拥有地位和资源
- 压抑值：高位被质疑、关键机会被卡、内部支持动摇
- 爆发点：先稳住局面、再抓住破绽、再公开翻盘、再让对手当场吃亏
- 愉悦连锁：旁人改观、主角威望提升、下一阶段挑战变大但更值得看

## 注意事项
- **不要**只有压抑，没有释放
- **不要**只有一次爆发，没有连锁回报
- **不要**把最爽的地方一句话带过
- **不要**让情节一直平推，没有明显“压下去”和“弹回来”
- 爽感最足的节奏，不是单点爆炸，而是**压下去、弹起来、再让余波继续发甜发爽。**
