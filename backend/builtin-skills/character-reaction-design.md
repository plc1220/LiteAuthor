先确定这场戏要写的不是生气、难过或善良本身，而是**这个人为什么会这样选**。读者通常不是反对角色做某件事，而是反对角色**没理由地**做这件事。

## 适用情况
- 卡在这个角色此刻到底会怎么做
- 某个关键行为一写就像 OOC
- 想安排救人、原谅、翻脸、黑化，但转不过去
- 同一类情绪总写成一个模子

## 输入
- 场景事件
- 角色当前人设底色
- 角色最关键的旧事或执念
- 当前关系阶段
- 这场戏最想让读者感到什么

## 核心原则
1. **先找心理根子，再选动作台词。**
2. 同一刺激，不同性格的人反应必须不同。
3. **看似违和的行为，要补桥。**
4. 短挣扎比长解释更有力。
5. 反应写完后，关系或局势最好有变化。

## 步骤
### 1. 先写场景目的
填三句：
- 这场戏后，读者更应该觉得这个人____
- 另一角色要因此产生____
- 关系要从____变到____

### 2. 写角色此刻的三件事
- 最在乎什么
- 最怕失去什么
- 眼下最想做到什么

### 3. 找触发点
问清：
- 这件事戳中了什么旧伤、底线或执念
- 为什么偏偏这里会失衡
- 如果不行动，角色会失去什么

### 4. 先写最符合旧人设的第一反应
这一步很重要。哪怕后面要反转，也先写本能反应：
- 先躲
- 先冷处理
- 先顶回去
- 先不信
- 先保自己或先保某个人

### 5. 如果要让角色做看似不像他/她的事，用四步补桥
1. **本能抗拒**：先表现原本不会这么做
2. **新刺激**：出现一条足以改变选择的信息或冲击
3. **合理动机**：给角色一个过得了自己那关的理由
4. **行动余波**：做完以后关系、情绪或判断发生变化

常用句式：
本来想____，但因为____被击中，所以决定____。这不是变成另一个人，而是原有人设在新刺激下露出了另一面。

### 6. 一次列 3 条反应路线
- 体面压怒型
- 脆弱失衡型
- 当场反打型
- 冷处理型
- 反差翻面型

然后选最服务当前剧情的那条，不是只选最顺手的那条。

### 7. 把反应落到可见行为
每条路线都要写清：
- 表层情绪
- 可见动作
- 台词或沉默方式
- 暴露出的内里
- 后续作用

### 8. 只补一小笔心理或过往把它钉实
可以是：一句过往回响、一个身体反应、一句别人说过的话、一个提前埋过的习惯。

### 9. 做 OOC 排查
逐条问：
- 这次行为有没有足够重的触发点
- 换个角色，这条反应还成立吗
- 这是角色的选择，还是作者为了桥段硬推
- 反应后有没有留下余波

## 快速示例
### 冷情角色救了害过自己的人，怎么写才不白莲
- 本能反应：她第一时间并不想救
- 新刺激：她发现对方腹中有无辜者，或这件事会波及她真正想保护的人
- 合理动机：她不是原谅对方，而是**不愿让无辜者死在眼前**，也不愿让自己越过那条底线
- 行动余波：救人后仍保持距离，甚至重新划清边界

读者往往不是反对救人，而是反对**没有挣扎、没有理由、没有后果**的救人。

## 输出格式
1. 场景真正想表现什么
2. 角色此刻最在乎 / 最怕 / 最想做到什么
3. 触发点
4. 本能第一反应
5. 动机补桥四步
6. 反应路线 A / B / C
7. 最优路线与理由
8. 3—7 句可直接使用的场景写法
9. OOC 风险点

## 注意事项
- **不要**只在哭哪种、骂哪种、摔不摔东西上打转
- **不要**让违和行为直接发生，中间没有桥
- **不要**用长篇解释替代行动和余波
- 好反应的标准不是花哨，而是**像这个人、服务这场戏、能带出后续**。
