先把主线立住，再给它长枝叶。**想象力不是往文里乱扔脑洞，而是让脑洞替主题、情节和人物工作。**

## 适用情况
- 点子很多，一落笔就前后不搭
- 灵机一动不断冒出来，容易偏题
- 设定很华丽，但故事空
- 写现实题材时容易变成流水账
- 写幻想题材时只有世界观，没有像样的情节

## 核心原则
1. **所有桥段都要服务主题。**
2. 再精彩的点子，只要冲淡主线，就要忍痛割爱。
3. 现实题材的想象力，常藏在生活细节的加工和独特视角里，不一定靠离奇事件。
4. 幻想题材越依赖想象力，越需要情节上的严谨构思。
5. **大设定不等于有想象力，细节往往比设定更能见功夫。**

## 步骤
### 1. 先写清这篇文到底想表达什么
用一句话写：
`这篇文章真正要写的是____。`
不清楚这一句，后面所有点子都容易乱飞。

### 2. 做一次「剥壳测试」
把奇幻、穿越、异能、行业外壳先拿掉，再看：
- 这到底是成长故事、关系故事，还是求生、复仇、改命故事
- 没了设定外衣，核心冲突还站不站得住

如果剥壳后只剩空架子，说明主线还没立住。

### 3. 从一个点子往四周辐射
有一个好点子时，不要立刻上文。
先写：
- 它最直接会引发什么
- 接下来可能出现哪些局面
- 哪些人物会被卷进来
- 哪个结果最适合主线

### 4. 把候选桥段写成卡
把可能的场面、反转、支线都先写成卡。
再排列这些卡，寻找：
- 哪张和哪张有因果
- 哪张只是漂亮但孤立
- 哪张能成为主线，哪张只适合番外

### 5. 用四问筛掉跑偏点子
每个新点子都问：
1. 它是不是在服务主题
2. 它有没有让冲突更强
3. 它会不会破坏前后因果
4. 它有没有让人物更鲜明，而不是只让设定更花哨

四问里有两问答不上，先别用。

### 6. 只选一条主线
试验结果可以有很多种，但正文只能选一条主线来写。
其余可能性：
- 留作番外
- 变成备用方案
- 留到下一本

**什么都想写，最容易什么都写不清。**

## 现实题材怎么用想象力
### 1. 从生活细节里二次加工
别先追求惊天大事件。
优先找：
- 日常里不寻常的一句废话
- 一个有味道的小习惯
- 一个能暴露关系的小动作
- 一件小事里隐藏的失衡和荒诞

### 2. 换视角，流水账就会变
同一件事，换个视角可能立刻有趣：
- 当事人视角
- 旁观者视角
- 非常规观察者视角
- 某种被忽略的边缘视角

现实文的创意，很多时候不是编怪事，而是**找到别人没这样看的角度**。

## 幻想题材怎么用想象力
### 1. 先把情节主干钉牢
越是大世界观，越不能拿「设定新奇」代替剧情推进。

### 2. 把想象力落到细节
优先写：
- 物件
- 习惯
- 规则代价
- 日常场景
- 小生物、小装置、小禁忌

读者往往是被这些细节说服，而不是被一段宏大设定说明说服。

### 3. 让细节反复起作用
好的细节不是只惊艳一次，而是能反复参与剧情、氛围和人物关系。

## 新点子是否加入正文
如果想到更好的新桥段，最后再问一遍：
- 它对整个故事的作用有多大
- 是让情节更精彩，还是只是让某一段更花
- 它能不能自然接进现有框架
- 加进去后会不会让主次失衡

## 输出格式
1. 主题一句话
2. 剥壳后的核心故事
3. 候选点子卡
4. 主线与番外的划分
5. 现实题材可用的细节 / 视角
6. 幻想题材可用的细节 / 规则
7. 应删掉的跑偏点子
8. 最终保留的主线方案

## 示例
### 输入
- 想象力很多，但写着写着总跑偏

### 输出
- 主题：先写清真正想说什么
- 剥壳后：核心其实是一个人的成长 / 一段关系 / 一次求生
- 候选卡：把灵机一动的各种桥段先写卡，不直接入文
- 处理：只留最能服务主线的一条，其他存档
- 细节提醒：幻想文别只堆世界观，优先补能落进场面的细节；现实文别只写日常，先找独特视角和被加工过的生活细节

## 注意事项
- **禁止**为了想象力而想象力
- **禁止**因为点子好玩，就不管主线
- **禁止**把设定厚度误当成剧情厚度
- **禁止**一口气把所有可能性都写进正文
- 想象力真正有用的时候，是**读者觉得这个世界很新、这条线很顺、这些细节很妙，而且故事没有散**。
