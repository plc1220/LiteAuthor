先把这场日常戏里“本来很平”的动作找出来，再往里塞一个能改变局势的小矛盾。**日常戏不是不能平静，而是不能什么都不发生。读者要看的不是吃饭、上课、散步本身，而是这些动作里长出的冲突、拉扯和变化。**

## 适用情况
- 校园、恋爱、宅斗、种田、职场日常写出来像流水账
- 主角一直在互动，但场面不带感
- 情节不算差，却总是干巴巴
- 只会写“聊天、吃饭、上课、做事”，不会让这些场景有戏

## 输入
- 当前日常场景
- 参与角色与关系
- 这场戏本来想达成的结果
- 想给读者的情绪：心动、拉扯、紧张、爽、尴尬、好笑等

## 步骤
### 1. 先写出最平版
用一句话写：
`谁在什么日常场景里做什么。`
如果这句话听起来只是流程，没有变化，这场戏就还没立起来。

### 2. 给场景加一个“现在就会撞上的矛盾”
优先从这些里选 1 个：
- 人和人不对付
- 双方目标相反
- 外部挑衅/打断
- 信息差或误会
- 立场差
- 身份差
- 竞争关系
- 想隐藏的东西快暴露
- 一件小事引发面子、资源、位置之争

**不要一口气加很多，只加最能让这场戏动起来的那一个。**

### 3. 冲突要长在日常动作上
不是先写日常，再硬插事故；而是让矛盾直接附着在动作里。
例如：
- 写作业 → 死对头坐成对角线，还共享了一副耳机
- 吃饭 → 有人挑衅、羞辱、泼饭、抢位置
- 考试 → 被人抄袭、连累、公开点名
- 散步 → 必须演关系、躲某人、套消息
- 家务/经营 → 分工冲突、资源不够、旁人掺和

### 4. 优先选“能暴露关系”的矛盾
好日常戏的冲突，不只让事情变热闹，还要让关系露出来：
- 谁更在乎输赢
- 谁嘴硬心软
- 谁更主动
- 谁受不得激
- 谁会顺水推舟
- 谁在外人面前护短/拆台

### 5. 冲突不一定大，但必须有过程
至少写出：
- 谁先出招
- 谁接招
- 局势怎么变
- 谁最后拿到一点主动权/吃了一点亏

**核心看点是矛盾里的处理过程，不是“有矛盾”这四个字。**

### 6. 给这场戏留一个变化
结尾至少留下 1 项：
- 关系推进半步
- 新误会
- 新好感
- 新仇
- 新把柄
- 新期待
- 主角被更多人看见
- 下一场戏自然被顶出来

如果一场日常戏结束后，什么都没变，那它大概率就是水。

### 7. 感情向和剧情向分别这么用
#### 感情向
优先让冲突暴露化学反应：
- 互怼
- 试探
- 故意逗
- 被迫靠近
- 表面不对付，实际很在意

#### 剧情向
优先让冲突顶出下一步情节：
- 人际矛盾
- 资源争夺
- 名声变化
- 误会扩散
- 事件升级

### 8. 检查是不是“为了有戏乱加事”
重做信号：
- 冲突和人物性格没关系
- 换任何角色都能发生
- 只是突然来一个路人找茬
- 写完很热闹，但对关系和主线没影响

## 输出格式
1. 最平版场景
2. 这场戏的核心矛盾
3. 矛盾怎么长在日常动作上
4. 角色各自的处理方式
5. 过程中的关系/立场暴露点
6. 结尾留下的变化
7. 下一场戏怎么被顶出来

## 快速示例
### 输入
- 两个学霸在教室写作业
- 想写校园感情线的火花

### 输出
- 平版：两个人各写各的作业
- 核心矛盾：两人互看不顺眼，但又被迫共享了耳机和错误拿到的校服
- 过程：
  - 一个故意念英文情诗刺激对方
  - 另一个表面冷着脸，实际被逼出反应
  - 周围同学起哄，气氛被放大
- 结尾变化：关系没缓和，但读者已经看到两人之间的火花和失衡

## 注意事项
- **禁止**把日常等同于无事发生
- **禁止**只靠旁白说“他们关系不一般”，不让场景自己证明
- **禁止**为了制造冲突强塞无关事故
- **禁止**一场戏里什么都想写，最后反而稀释重点
- 好看的日常戏，不是事大，而是**小事里有矛盾、有处理、有余味。**
