保留原梗最值钱的情绪引擎，用性转和同步重构把难写版本改成更顺手的新版本。**性转不是只把“她”改成“他”或反过来，而是连同社会评价、情绪外化、事业赛道、关系结构和读者期待一起重做。**

## 适用情况
- 一个角色很带感，但放在当前主位上会压垮另一主角
- 原版男女主厚度失衡，CP 一边厚一边薄
- 原版是甜爽双线、多线并跑或强设定版，新手扛不住
- 舍不得丢掉脑洞核心，想换一种更好写的版本
- 想测试一个梗为什么总写不顺、卖点不准

## 核心原则
1. **先找不动点，再翻可变项。**
2. **保住的是伤口、欲望、爽点和关系驱动，不是表面标签。**
3. **性转后要同步改“别人怎么看 TA”和“TA 怎么表现自己”。**
4. **如果真正问题不是性别，而是高低位、叙事位置或受众预期，也要一起翻。**
5. **不要只做字面性转，不改剧情后果。**

## 步骤
### 1. 先写原版一句话
用一句话写清：
`谁因为____，在____关系里卷入____故事。`

然后补一句：
`原版最值钱的不是设定词，而是____。`

### 2. 诊断原版卡点
重点查 5 类问题：
- **厚度失衡**：一方设定太满，另一方只像工具人
- **表达不顺**：某些情绪、行为、台词放在当前版本里很别扭
- **赛道不对**：明明想写感情文，实际主看点跑到竞争、斗智或事业去了
- **难度过高**：要同时控甜、控爽、控群像，新手容易全崩
- **读者站位不稳**：该心疼谁、该追什么，不够清楚

### 3. 抽出“不动点”
把原梗里真正不能丢的东西单独写出来：
- 核心伤口
- 核心欲望
- 核心爽点 / 甜点
- 关键关系张力
- 最想保住的名场面

示例问法：
- 这梗最值钱的是“嫉妒”本身，还是“长期被压着后拼命证明自己”？
- 最值钱的是“豪门真假千金”，还是“被更耀眼的人长期盖住”？
- 最值钱的是“病娇”，还是“缺乏安全感又别扭地要爱”？

**先把本体抽出来，再决定外壳怎么换。**

### 4. 决定谁性转、谁保持不动
可选方式：
- 只性转主角
- 只性转官配
- 关键对照组一起调整
- 只做“假性转测试”：不真改性别，只改社会评价和叙事位置

判断标准：
- 谁更适合放在“被逐步了解”的位置
- 谁更适合当主视角
- 谁一旦换性别 / 位置，整本书的题材承诺会更清楚

### 5. 同步重写五项
#### A. 社会评价词
同一种内核，换个版本后，外界给 TA 的命名和读者接受度会变。

例如：
- 嫉妒 → 不服输 / 吃醋 / 胜负欲强
- 脆弱敏感 → 逞强 / 绷着 / 控制欲强
- 娇气任性 → 面子重 / 别扭 / 要强
- 细腻多思 → 心思重 / 观察力强 / 记仇

**这些不是固定对应，只是提醒：性转后不能机械复读原词，要改成新版本里更自然的外化方式。**

#### B. 情绪外化方式
问：
- 原版是怎么发作的
- 性转后更自然的发作方式是什么
- 哪些情绪适合内化，哪些适合动作化、事件化

例如：
- 原版私下胃痛、失眠、自我消耗
- 新版本可能更适合写成硬扛、死磕、吃醋、别扭、控制细节

#### C. 事业线 / 赛道
问：
- 原版这条事业线还适不适合新版本
- 换个赛道后，角色的执念会不会更亮
- 这条赛道能不能顺手带出更多剧情

例如：
- 原版拼不过台前风头
- 性转后改成走更能体现死磕和含金量的科研、查案、修炼、职业线

#### D. 关系结构与主视角
问：
- 复杂人设放在谁身上更有利于写关系
- 谁来负责“掀套娃”
- 这段关系是双向互掀，还是一方解密另一方

很多头重脚轻的感情文，换法不是“把两人都写更复杂”，而是：
- 把更复杂的人设放到**被逐步了解**的一方
- 让另一方承担视角、行动和解密功能
- 把感情线改写成“随着了解不断改观”

#### E. 开篇钩子
性转后，开篇也要重做。
重点检查：
- 第一眼冲突还成不成立
- 哪种反感点 / 吸引点更适合新版本
- 开篇更该卖甜感、谜感、爽感还是人物反差

### 6. 重新判断结构：双线还是单线
性转后不要默认沿用旧结构。

可选：
- **双线甜爽**：感情线 + 打脸 / 竞争线并跑
- **单线程解密感情**：主看两人关系，事业和主线作为解密载体
- **主剧情 + 感情副线**：把关系嵌进更强的职业、悬疑或冒险线

新手优先判断：  
**这次改完，是不是能把主卖点说成一句人话。**

### 7. 重写前三章
至少重写 4 件事：
1. 新版本第一眼最抓人的人设点
2. 前 3 章最早的关系火花
3. 第一条明确主线
4. 第一个让读者“原来改完更顺了”的场面

### 8. 至少出两个版本
不要只出一个。
- **稳版**：最容易写顺、最贴题材
- **更敢写版**：更强钩子、更强反差、更偏卖点

比较时只看：
- 哪个版本主看点更清楚
- 哪个版本男女主更平衡
- 哪个版本前 10 章更容易留人
- 哪个版本自己更想写

### 9. 风险检查
出现下面情况说明还没改到位：
- 只是换了称谓，冲突还是老样子
- 核心伤口被改没了，只剩噱头
- 性转后人物说话做事更别扭
- 另一主角还是很薄
- 题材承诺更乱，不是更清楚

## 输出格式
1. 原版一句话
2. 原版卡点
3. 不动点
4. 谁性转、谁不动
5. 社会评价词 / 情绪外化 / 事业线 / 关系结构 / 开篇钩子的改法
6. 新版本一句话
7. 前三章节拍
8. 稳版 vs 更敢写版
9. 风险点
10. 最终推荐版

## 快速示例
### 原版
- 复杂病娇千金女主 × 温柔暖男男主
- 问题：女主层次太多，主线真正好看的是她和对照组斗智斗勇，男主显得很薄

### 性转后
- 把复杂“病娇 + 要强 + 缺安全感”的人设挪到男主身上
- 女主改成主视角和解密者，通过工作、查案或相处逐步了解他
- 原版直说的“嫉妒”改成更自然的“不服输、别扭吃醋、死磕”
- 事业线改成更能体现死磕和含金量的专业赛道
- 结构改成“单线程感情 + 轻悬疑 / 职业线”

效果：
- 复杂人设被放到了更适合被一层层掀开的那一方
- 女主不再只负责被爱，而是负责观察、误判、改观、靠近
- 故事更容易写成“随着解密不断升温”的剧情感情线

## 注意事项
- **禁止**把性转理解成“换个性别就自动新鲜”
- **禁止**一边性转，一边保留所有原版社会反馈和行为逻辑不变
- **禁止**为了性转硬套刻板印象
- **禁止**改完以后主看点反而更模糊
- 性转改梗真正有用的地方，不是猎奇，而是**帮作者重新找到这个故事最顺、最好卖、最有火花的版本**。
