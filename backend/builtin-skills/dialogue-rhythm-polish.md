把一场对话拆成**话外之意、对象差别、动作衔接和情绪节拍**再重写。**好对话不是把事情解释得一清二楚，而是让读者一边听见表层台词，一边看见人物关系、立场和情绪在变化。**

## 适用情况
- 对话像说明书，角色总在替作者解释剧情
- 两个人一直说话，却单调、发干、没层次
- 高情绪场面一写就喊口号、哭诉、直白表白
- 想写有趣、反讽、耐品的台词，却总写得很硬
- 对话与对话、对话与描写之间衔接僵硬

## 输入
- 场景目标
- 说话双方及关系
- 说话对象是谁
- 当前情绪强度
- 必须明说的信息 / 适合暗说的信息
- 想保留的气质：克制 / 冷硬 / 幽默 / 温柔 / 失控 / 哲理感等

## 核心原则
1. **相信读者，不要把话说满。**
2. 有趣的对话常来自**反讽、反差、半截话和留白**。
3. **情绪越重，台词往往越该收着写。**
4. 同一个人，对不同对象，说法必须不同。
5. 对话里不只是人在说话，还要有人在动、在忍、在想。
6. 动作描写可以控制阅读速度：动作多一点，节拍慢一点；动作少一点，推进快一点。
7. 台词除了传递信息，也可以顺手暴露人物的价值观和基本信念。

## 步骤
### 1. 先写清这场对话到底要改变什么
用 3 句话写清：
1. 开头谁占上风
2. 中段哪里变调
3. 结尾谁拿到主动权 / 谁露出破绽 / 关系怎么变

如果这三句写不出来，这场对话大概率还只是“有人在说话”，不是“戏在推进”。

### 2. 分开写每个人的表层目的和里层目的
分别写：
- 表层在说什么
- 真正想得到什么
- 哪些不能直说

**有里层目的，对话才会有味。**

### 3. 先删掉读者本来就能懂的解释
逐句检查：
- 这句是不是在反复解释前文已经知道的事
- 这句是不是作者担心读者不懂，硬塞的说明
- 这句能不能改成停顿、动作、半截话或一个更准确的词

### 4. 选这场要用的“含蓄工具”
从下面选 1—2 个，不要全开：
- **反讽**：说出来的话和本意有反差
- **半截话**：故意不说满，留读者补全
- **平常话**：越关键时越说日常话，反而更有劲
- **玩笑 / 轻口气**：在重场面里举重若轻
- **沉默**：关键时刻不回答，也是一种回答

### 5. 高情绪场面优先“收着写”
尤其是：
- 生离死别
- 告白 / 分手
- 崩溃 / 决裂
- 真相揭开前后

常用处理：
- 少喊大道理，多说平常话
- 少一口气宣泄，多留卡住、咽回去、改口
- 必要时让一句最轻的话承担最重的情绪
- 极端情况下，可以让人物来不及说完

### 6. 让角色因“对象不同”而换说法
同一个人面对：
- 爱人
- 仇人
- 长辈
- 下属
- 陌生人
- 危机中的亲近之人

说话方式都该不同。**身份是人物的外壳，对象是台词的开关。**
但也要保住人物最核心的信念和底色；如果信念变了，必须让转变有因果。

### 7. 在对话中插入动作和心理动作
两个人别一直“静着说”。可插：
- 肢体动作：靠近、后退、抬手、放下杯子、绕开、停住
- 神态变化：移开视线、盯住、笑一下、脸色变了
- 心理动作：想反驳、忍住、误会、被刺中、临时改口

**一句台词后，不一定非要接另一句台词。** 让反应参与进来。

### 8. 决定衔接是“慢接”还是“快切”
- **慢接**：台词前后加动作、姿态、环境，适合拉气氛、加层次
- **快切**：少动作、重对白，适合加速、逼问、抢话、对冲

别全篇只用一种接法。

### 9. 最后检查每一轮对话是否真的有变化
至少命中一项：
- 局势变了
- 情绪变了
- 关系变了
- 读者多知道了一点
- 某个人的立场更清楚了
- 后续冲突被打开了

如果什么都没变，这轮对话就是白说。

## 输出格式
1. 场景目标
2. 角色表层目的 / 里层目的
3. 应删的解释性台词
4. 本场使用的含蓄工具
5. 动作与心理动作插笔点
6. 节奏分拍表
7. 改写后的对话
8. 这场对话推进了什么

## 快速示例
### 输入
- 场景：生死关头道别
- 问题：原稿一直在哭喊表白，显得太满

### 输出
- 删掉大段“我爱你、你不能死、我舍不得你”式直白宣泄
- 改成：人物先照常交代一件很小的事，比如“天冷了，记得把窗关上”
- 再让话在最该重的时候卡住、停住或被打断
- 用动作和停顿把真正的情绪顶出来

## 注意事项
- **禁止**所有角色都说一个腔调
- **禁止**把聪明台词当成好对话的全部
- **禁止**每句后面都跟一大段解释
- **禁止**高情绪场面只靠喊和哭撑力度
- **必须**让台词贴人物、贴对象、贴当下局势，不要为了“金句感”硬写。
