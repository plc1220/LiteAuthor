先按故事类型选结构，再打点和连线。**结构不是先写满桥段，而是先钉住主干变化点，再决定中间怎么递进。**

## 适用情况
- 大纲只有设定，没有起伏
- 中段不知道该放什么情节
- 升级流、种田、宅斗、单元剧、双线文想先搭骨架
- 感情主线或事业主线缺少明确转折

## 步骤

### 1. 先判断故事属于哪一类
从下面选最接近的一类：
1. **成长线**：主角有明确成长指标，如等级、地位、财富、势力、名望
2. **单元线**：有一条大悬念或大目标，途中不断解决小谜题或阶段任务
3. **三转折长篇**：不靠数值升级，主要靠重大转折推动
4. **平行线**：两条或多条线并行，靠交叉点和互相照亮推进

如果拿不准，就先问：读者到底在看“主角一路变强”，还是“主线一步步揭开”，还是“多条线如何汇合”。

### 2. 成长线先打起点和终点
先分别写清：
- 起点：身份、关系、资源、等级、心理状态
- 终点：身份、关系、资源、等级、心理状态

**先有起点和终点，再填中间情节点。**

中间情节点分大点和小点：
- 大点：升阶、翻盘、换地图、关系重大变化、阵营变化
- 小点：试探、试错、小胜、小败、埋线、预热

排列原则：
- **不要一直小小小，也不要一直大大大**
- 节奏像一条起伏的波浪线，整体越往后越高
- 每个点至少改一个变量：实力、关系、处境、目标、认知

### 3. 单元线先画主线，再在线上打单元点
先写一句：
`这条主线最终要揭开的真相 / 完成的目标是____。`

再排每个单元：
- 这一单元解决什么小问题
- 它给主线提供什么新线索、新代价或新误判
- 主线是如何一步步递进的

关键规则：
- **核心设定和最终揭秘，必须在开端或前半部开始铺垫**
- 最后一个单元要承担关键揭秘或总爆发
- 全文最高潮要和全文终点对应，不能像额外加的一场大戏

### 4. 三转折长篇按“四阶段 + 三个点”搭
#### 阶段1：开篇铺垫
交代人物、冲突、预期、压力。

#### 点1：重大转折1
主角被迫入局，或拿到外挂，或背上更大的压力。

#### 阶段2：局势抬高
把前面得到的优势、机会、关系玩起来，让读者看到“爽”或“希望”。

#### 点2：重大转折2
规则变了、外挂坏了、秘密暴露了、局势突然恶化了。

#### 阶段3：局势压低
主角被打回去，问题变重，危机逼近。

#### 点3：重大转折3
大危机正面砸下，人物跌到低谷，必须做真正选择。

#### 阶段4：收尾
完成任务、回收成长、结算关系和代价。

### 5. 平行线先定交叉点，再排各自推进
先分别写每条线的：
- 起点
- 阶段目标
- 终点

再先钉住交叉点：
- 两条线在哪里交叉
- 为什么必须在这里交叉
- 交叉后哪条信息、关系或局势被改写

**交叉点要提前打好。**
不要等写到一半再临时让两条线碰头。
如果交叉后什么都没变，这不叫结构点，只是两场戏挨在一起。

### 6. 感情线也要打点
恋爱主线不是“没有结构”，而是点打在关系状态变化上，例如：
- 互相利用 → 决定暂时站一起
- 一方先动心 → 另一方还没跟上
- 一方终于确定关系 → 另一方因为旧伤后撤
- 真相揭开 → 两人交心
- 正式站队 → 关系进入新阶段

### 7. 最后做总检查
逐点检查：
- 这个点之前和之后，到底变了什么
- 这个点是大点还是小点
- 大小点是否错落
- 下一个点是不是由上一个点逼出来的
- 删掉这个点，主干是否会塌

如果删掉后主线几乎不受影响，这个点多半只是热闹，不是结构点。

## 输出格式
1. 故事类型
2. 主线一句话
3. 起点 / 终点
4. 关键结构点列表
5. 每个点改变的变量
6. 大小点分布
7. 波峰波谷说明
8. 高风险空段

## 快速示例
### 输入
- 宅斗成长文
- 女主从庶女一路爬到能掌局的人

### 输出
- 类型：成长线
- 起点：身份低、资源少、依附嫡母、身边只有少数可用之人
- 终点：地位稳、关系网成形、能自己布局
- 大点：被卷入寿礼风波 → 第一次漂亮反制 → 站稳内宅位置 → 更大权力斗争
- 小点：争取下人、试探嫡母、处理姐妹关系、回收前史线
- 波形：前期小胜，中期有一次大挫，后期连续抬高并总爆发

## 注意事项
- **不要**先堆桥段，再回头硬找结构
- **不要**只有起点，没有明确终点
- **不要**每个节点都同等重要
- **不要**把“发生了很多事”当成“结构完整”
- 先打点，再连线，再补细节
