先定主角、反派、目标，再把高潮写成三对抑扬。**高潮不是把冲突拉长，而是让同一个目标在主角和反派之间反复翻面，直到局势被彻底改写。**

## 适用情况
- 一个阶段 / 副本 / 事件需要写出 8k—1w 左右的大场面
- 高潮只有一拳，写不出一波三折
- 打脸、争夺、对峙、比赛、夺宝、公开翻盘写着发平
- 想让剧情高潮之后自然接感情高潮

## 输入
- 当前事件 / 副本
- 主角
- 反派或敌对阵营
- 双方共同争夺的目标 / 资源
- 双方的手段、限制、stakes
- 场景地点、在场人物、见证者
- 是否有多个反派 / cp 双主角

## 核心原则
1. **高潮至少有三个基本要素：主角、反派、目标。**
2. **争夺本质是零和资源争夺。**
3. **结构用三对抑扬：抑1扬1 + 抑2扬2 + 抑3扬3。**
4. **每一折都要让反派失去一点、主角得到一点。**
5. **反派动机要明确且带人品缺陷；主角应对要主动、积极。**
6. **惩罚反派不能靠意外。**

## 步骤
### 1. 先写清这场高潮在抢什么
目标可为：
- 荣誉 / 奖项 / 名额
- 资源 / 位置 / 资格
- 宝物 / 线索 / 生存机会
- 某个人 / 某段关系 / 舆论解释权

如果谁拿都一样，就不够激烈。

### 2. 排出抑扬骨架
#### 抑
展现反派视角或反派占上风的场景，包含：
- 反派想坑主角什么
- 动机是什么
- 计划如何落地
- 反派为何觉得自己稳赢

#### 扬
展现主角视角或主角翻盘的场景，包含：
- 主角察觉危险后的反应
- 主角如何用能力 / 优势 / 准备化险为夷
- 主角如何直接或间接打回去

#### 扬+
可选：反派受挫、围观嘲笑、局势改写、下一折伏笔

### 3. 把它写成三对
#### 抑1
反派先手，主角第一次被压

#### 扬1
主角扛住第一击，拿到第一层主动权

#### 抑2
反派恼羞成怒，换更狠的手法或更高位资源

#### 扬2
主角换方法回击，局势进一步倒向主角

#### 抑3
反派上最终杀招，stakes 最大化

#### 扬3
主角亮出最重的底牌，彻底改写结果

### 4. 每一折至少升级一项
- 对手更强
- 规则更严
- 见证层级更高
- 失败代价更大
- 主角必须换招
- 揭开的真相更多

### 5. 抑里写什么
- 反派的动机
- 反派的手段
- 主角将要失去什么
- 反派的得意和轻视

### 6. 扬里写什么
1. 主角被攻击时的感受
2. 主角如何积极应对
3. 主角如何利用自身优势翻盘
4. 主角如何把力打回反派身上

#### 主角反应的底线
- 不写成长期胆怯、软弱、只会受气
- 可以吃亏，但要主动迎战
- 反击手段要光明正大或至少符合主角底线

### 7. 每回合都写资源流向
- 反派失去什么：名声、位置、帮手、信心、规则优势
- 主角得到什么：认可、资源、话语权、盟友、下一轮筹码

没有得失变化，高潮就只是热闹。

### 8. 反派惩罚一定要来自主角
- 主角亲手反打
- 主角借规则、证据、见证人反打
- 受到主角影响的人替主角补刀

不要用天降意外替主角出拳。

### 9. 多反派和 cp 的排法
#### 多反派
按层级出场：
- 小反派先上
- 大反派压轴

#### cp 双主角
可用理想排法：
- 扬1 展示 A
- 扬2 展示 B
- 扬3 A + B 合力收尾

也可以一场 A 压轴，下一场 B 压轴，避免每次同一人吃满高光。

### 10. 想让剧情高潮后接感情高潮
在大场景前先埋感情矛盾。
大场景解决后，不要立刻散场，接：
- 承认错误
- 公开站队
- 后怕照料
- 关系更进一步

这样剧情峰值能自然转成关系峰值。

## 输出格式
1. 目标 / 资源
2. 主角 / 反派 / 见证者
3. stakes
4. 抑1 / 扬1
5. 抑2 / 扬2
6. 抑3 / 扬3
7. 每折的资源得失
8. 最终奖励与余波
9. 如有 cp：谁在第几折压轴
10. 如要接感情高潮：前置矛盾与后续回收

## 快速示例
### 输入
- 娱乐圈文：主角和反派争夺年度最佳新人

### 输出
- 目标：最佳新人奖和随之而来的顶级资源
- 抑1：反派先放黑料，想让主角失去评选资格
- 扬1：主角拿出现场实力和关键证明，保住资格
- 抑2：反派勾连评委和资方，想在最终舞台压死主角
- 扬2：主角换打法，在公开舞台上直接打出超预期表现，赢得高位见证人
- 抑3：反派发动最后一轮舆论与规则攻击
- 扬3：主角拿出前文埋好的证据与人脉，反派丢奖又翻车，主角拿奖并抬升到更高平台

## 注意事项
- **禁止**只有主角高光，没有反派真正出手
- **禁止**三折只是同一招重复三遍
- **禁止**高潮打完，局势什么都没变
- **禁止**反派作恶很重，最后却靠天灾人祸草草退场
- **一波三折的重点，不是字数更长，而是局势三次翻面，且每次翻面都更值钱。**
