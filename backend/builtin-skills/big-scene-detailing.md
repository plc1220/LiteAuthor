先定这场“大”到底大在什么：繁华、庄肃、混乱、压迫、悲壮，还是凯旋。**大场面不是把人、车、旗、建筑全报一遍，而是用少量代表性细节，让读者一下看到规模、气氛和人味。**

## 适用情况
- 想写市井繁华、宫廷盛典、凯旋仪仗、战争场面
- 场面明明很大，写出来却像流水账
- 只会堆名词，不会让场面动起来
- 细节很多，但没有主题

## 输入
- 场景类型
- 这场最核心的主题词
- 视角人物
- 这场对剧情的作用
- 想让读者记住的一个画面

## 核心原则
1. **先定主题，再选细节。**
2. 大场面要同时有：**环境骨架 + 人群流动 + 一个局部镜头**。
3. 细节要能代表整体，不要平均铺满。
4. **人是场面的证据**。没有人的动作，再华丽也容易空。
5. 视角要稳定，别一会儿俯瞰，一会儿钻进十个路人的脑子里。

## 步骤
### 1. 用一句话钉死场面主题
写成：
`这场大场面要让读者第一感觉是____。`

例如：兴旺、庄严、末世繁华、肃杀、混乱、悲壮。

### 2. 先搭环境骨架
只选 2—3 笔大的东西定场：
- 天空 / 光线 / 天气
- 城门 / 河流 / 宫殿 / 山岭 / 营帐
- 旗帜 / 车马 / 船只 / 城市轮廓

这一步只负责让读者看见“场子有多大”。

### 3. 再选 3—5 个代表性细节
优先从下面挑：
- **声音**：叫卖、号角、马嘶、金鼓、笙歌、呼喊
- **气味**：食物香、烟火味、血腥味、尘土味
- **群像动作**：招呼买卖、探身围观、列队肃立、奔逃推搡、扯帆操浆
- **关键物件**：灯火、旌旗、兵器、酒器、桥梁、篝火
- **边角人情**：一个摊主、一个旅人、一个小兵、一个旁观者

**至少有一个细节要带“人的反应”。**

### 4. 安排一个“局部小镜头”
大场面最稳的写法之一，是在宏观中插一个微观镜头：
- 一个摊主如何招呼客人
- 一名士兵如何系紧甲带
- 一群人怎样探身看桥下的船
- 宴会中某个旁观者怎样偷看主位

这个小镜头能把“整体热闹/紧张/庄严”落到真实可见的行为上。

### 5. 按类型换细节组合
#### 写繁华
- 环境亮、层次多、流动性强
- 多用叫卖、香味、往来、围观、货物、桥路水道
- 重点是“人怎么活、怎么忙、怎么被吸引”

#### 写庄肃或凯旋
- 环境整齐、秩序感强
- 多用仪仗、辇车、服饰、站位、队列、乐声
- 重点是“等级与秩序”

#### 写战前
- 环境压低、准备感强
- 多用乌云、篝火、兵刃、低声命令、紧张备战
- 重点是“蓄压”

#### 写战中
- 不要报菜名式写兵种
- 重点抓冲撞、失衡、呼喊、救援、误伤、互相看不见彼此的混乱
- 重点是“人性与动作”

#### 写战后
- 把场面从响变静
- 多用残火、断旗、空地、伤兵、失语、余味
- 重点是“后果”

### 6. 让细节带剧情，不只带气氛
每个细节都问：
- 它是不是服务当前主题
- 它会不会影响角色判断或下一步行动
- 它是不是能顺手带出社会风貌、阶层关系或战争状态

答不上来就删。

### 7. 控制镜头顺序
推荐顺序：
`大环境 → 人群流动 → 局部小镜头 → 视角人物反应/下一步动作`

不要一直停在环境说明里不让人物动。

## 输出格式
1. 场面主题
2. 环境骨架 2—3 笔
3. 代表性细节 3—5 个
4. 局部小镜头
5. 视角人物反应
6. 这场对剧情的作用
7. 应删的堆砌性细节

## 快速示例
### 输入
- 场景：都城繁华
- 目标：写出“热闹而兴旺”

### 输出
- 环境骨架：晴空、城门、河道、桥梁
- 代表性细节：货船满河、食物香味、桥上叫卖、驼队进城
- 局部镜头：外地客站在桥头左右为难，两个摊主同时伸手招呼
- 作用：用一小段人群反应，把整座城写活

### 输入
- 场景：大战前夜
- 目标：写出“压抑待发”

### 输出
- 环境骨架：低云、山岭、营火
- 代表性细节：士兵磨刀、战马喷气、火光照甲、低声传令
- 局部镜头：一个年轻士兵系第三次护腕，还是没系稳
- 作用：用小兵的紧张，把大战前的压力落地

## 注意事项
- **禁止**把大场面写成名词清单
- **禁止**只有建筑和摆设，没有人群动作
- **禁止**每个角落都平分笔墨
- **禁止**只顾热闹，不让场面服务剧情
- 好的大场面，不是写得多，而是**一两组细节就让读者觉得这地方真的活着、真的要出事**。
