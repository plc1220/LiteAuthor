把故事分成两层来搭：**大模板**决定读者最后要吃到哪类补偿，**情节骨架**决定故事一路靠什么推进。先搭底盘，再选发动机，故事更稳。

## 适用情况
- 有题材、有设定，但主线发散
- 只想到名场面，不知道用什么骨架串起来
- 想判断该写探寻、追逐、复仇、推理还是成长
- 中段容易发空，不知道该靠什么持续推进
- 想保住大众接受度，又不想只会堆设定

## 输入
- 主角开局处境
- 主角最后最想得到什么
- 读者最该吃到的情绪补偿
- 题材 / 时代 / 世界观
- 篇幅与主打：动作 / 人物 / 感情 / 悬念

## 步骤
### 1. 先写读者最想看主角得到什么
不要先写题材，先写结果：
- 被看见
- 被选择
- 身份翻身
- 洗清冤屈
- 受苦后得到报偿
- 真相被揭开
- 爱情被夺回
- 完成自我成长

### 2. 先选一个大模板
#### A. 灰姑娘型
核心快感：低位者被看见，进入更高舞台并被确认价值。  
常见变体：逆袭、选秀、职场、校园、升级。

#### B. 双身份型
核心快感：隐藏身份、伪装、马甲、信息差，靠揭穿时刻制造戏剧性。  
常见变体：马甲、卧底、变身、隐藏大佬。

#### C. 受难得偿型
核心快感：无辜者受压、受冤、受苦，最后得到昭雪、补偿或更高回报。  
常见变体：复仇、虐后回收、沉冤昭雪、流放逆袭。

**主模板只保一个，副模板最多一个。**

### 3. 再选一个情节骨架
按主推进方式挑一类：

#### 行动 / 事件驱动
- **探寻**：主角寻找某个东西、答案或归属；表层在找目标，里层在找自己。归来时人通常已变。
- **探险**：旅程本身就是乐趣；重场地、事件和新鲜感，主角不一定非要大变。
- **追逐**：先定规则、赌注和起跑点；靠逆转、意外、空间限制维持紧张。
- **解救**：反派夺走重要之人/物，主角一路追到敌方地盘救回；重点是强反派和连续障碍。
- **逃跑**：被困、试逃失败、条件变化、最终脱身；乐趣在计划总赶不上变化。
- **复仇**：先让读者目睹罪行并绑定情感，再写正规途径失效、主角亲自动手、复仇付出代价。
- **推理**：核心是一道难解谜题；线索摆在明处，但不该被轻易看穿。
- **对手戏**：势均力敌的两人争夺同一目标；双方各强一面，靠权力曲线的来回拉扯推进。
- **落魄之人**：弱者被压到谷底，再靠勇气、智慧或坚持反扑；先让读者同情，再让其逆转。

#### 人物 / 内心驱动
- **诱惑**：主角被引去做不明智、错误或不道德的事；重点是代价、后果和内心清算。
- **变形记**：诅咒、异化、兽化、非人化；重点是关系演变与解除条件。
- **转变**：截取一段会改写人生的时期；重自我反思和“我到底变成了谁”。
- **成长**：年轻角色的信念被现实打碎又重建；重点是逐步长大，不要一夜成熟。
- **爱情故事**：爱情受阻，最终重获或痛失；重点是阻碍是否真的卡住这段关系，角色是否足够有魅力。

### 4. 用四格骨架把选中的类型落地
无论选哪类，都先写四格：
1. **布局**：主角原本处于什么状态，导火索是什么
2. **过渡 / 上升**：障碍、逆转、试错如何连续发生
3. **高潮**：最终对决 / 真相揭示 / 最终选择是什么
4. **余波**：问题如何解决，人物或世界发生了什么改变

### 5. 补该骨架的必备零件
根据类型重点检查：
- 探寻 / 转变 / 成长 / 诱惑：**必须有主角变化**
- 探险：重点是一路的精彩事件和地点，不必强求大变化
- 追逐 / 解救 / 逃跑：**必须有时间、空间或规则限制**
- 复仇：**必须先立罪行与情感动机**
- 推理：**谜底要在明眼处**
- 对手戏 / 落魄之人：**必须有权力曲线逆转**
- 爱情故事：**阻碍必须真实，不是硬拖**

### 6. 用三次法则稳节奏
常用稳写法：
- 三次尝试：前两次受挫，第三次突破
- 三角关系：除重大对抗场面外，单场核心人物尽量别太多，三角关系最稳
- 三段升级：前段立题，中段加压，后段回收

### 7. 把模板翻成当前题材版本
保留骨架，不照抄外壳。
例如：
- 复仇骨架可以写成修仙、职场、宅斗、娱乐圈
- 推理骨架可以写成古言断案、校园悬疑、无限流副本
- 成长骨架可以写成少宗主历练、青春校园、末世少年团

### 8. 最后做“去皮测试”
把设定皮肤拿掉，只看骨架：
- 去掉世界观后，故事还成立吗
- 高潮是不是兑现了前面答应读者的那口补偿
- 这套骨架能不能连续产出 50 章 / 100 章 / 完整篇幅

## 输出格式
1. 读者最想吃到的结果
2. 主模板 / 副模板
3. 选中的情节骨架
4. 四格骨架
5. 该骨架的必备零件
6. 三次法则安排
7. 当前题材改写版
8. 结局兑现清单
9. 容易跑偏的地方

## 快速示例
### 输入
- 修仙题材
- 主角家族被灭，想查真相并报仇
- 希望有升级和反转

### 输出
- 主模板：受难得偿型
- 副模板：双身份型
- 情节骨架：复仇 + 推理
- 四格骨架：
  - 布局：灭门惨案发生，主角侥幸活下
  - 过渡：查线索、混入敌营、数次误判与反转
  - 高潮：真凶现身，主角在公开真相与立即复仇之间选择
  - 余波：血仇得报，但宗门旧秩序也被撕开
- 必备零件：先立罪行、线索明藏、复仇代价
- 三次法则：三次接近真相，前两次都抓错人，第三次才逼出幕后黑手

## 注意事项
- **不要**同时选太多主模板
- **不要**只有题材外壳，没有底层快感
- **不要**中途换骨架却不回收前面承诺
- **不要**把成长、爱情、复仇、推理全塞成平均用力
- 模板的作用不是限制脑洞，而是**先把故事底盘和推进引擎搭稳**
