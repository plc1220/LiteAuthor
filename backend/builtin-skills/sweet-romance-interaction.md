按关系阶段设计能让读者真正吃到甜味的对手戏。**甜不是把情话和肢体接触一次性砸满，而是一方有行动，另一方有回应，在分寸里一点点递进。**

## 适用情况
- CP 有设定，但互动不甜
- 撒糖总是过油、发腻、像模板
- 只会靠台词发糖，不会写动作和反应
- 暧昧期、确定关系期、和好戏缺细节

## 输入
- 双方人设与当前关系阶段
- 这场戏想完成的目标：试探 / 安慰 / 拉扯 / 和好 / 确认心意
- 当前场景
- 这场戏之前两人的关系状态

## 核心原则
1. **一方有动作，另一方必须有回应。**
2. **甜感的核心不是肉麻，而是温柔、偏爱、例外和分寸。**
3. **糖要循序渐进，不要一上来糖精倾倒。**
4. **小动作往往比大告白更甜。**
5. **霸道可以有，伤害和违背意愿不行。**

## 步骤
### 1. 先定这场戏属于哪个阶段
常见阶段：
- 互相试探
- 暗戳戳偏心
- 关系失衡
- 明确心意前后
- 和好补偿

不同阶段，糖的浓度不同。暧昧期重在半含半露，确认关系后才适合更直给。

### 2. 先写“谁先动，怎么动”
不要直接想情话，先写：
- 谁先靠近
- 为什么要靠近
- 是用语言、动作、照顾、护短，还是故意逗人来推进

稳的互动一定有主动态。

### 3. 给这场糖找一个真正的甜点
优先从下面选 1—2 个：
- **温柔**：细心、耐心、接住情绪
- **偏爱**：对别人一样，对 TA 不一样
- **例外**：别人不行，TA 可以
- **克制**：明明想更进一步，却只做到刚刚好
- **包容**：接住对方的小脾气、小狼狈
- **保护**：替对方挡一层风雨
- **纵容**：嘴上嫌，行动上照顾

### 4. 用“小动作 + 反应”代替空口发糖
把互动写成这个结构：
`动作 → 触感/细节 → 对方反应 → 关系变化`

例如：
- 他替她拂掉头发上的雨珠 → 指尖停了一瞬 → 她下意识屏住呼吸 → 两人都开始不自然
- 她嘴硬不肯说疼 → 他把药放到她手边没多问 → 她嘴上还顶着，动作却慢了下来

### 5. 优先写有分寸的肢体接触
如果题材和关系阶段允许，优先用小而准的接触：
- 扶一下
- 挡一下
- 碰一下耳朵、手背、后颈
- 替对方整理衣领、擦泪、拂发

**越是克制的小动作，越容易显得甜。**

### 6. 给互动加一点“只有他们懂”的暗号
可用：
- 只有彼此明白的小动作
- 别人听不懂的半句台词
- 当众很普通，私下很暧昧的举动
- 某个反复出现、只属于两人的习惯

这种私密感会让糖更有记忆点。

### 7. 用反差制造额外甜度
常见有效反差：
- 对别人冷，对 TA 例外
- 前期嘴硬，后期真香
- 平时强势，关键时刻小心翼翼
- 看着不解风情，实际处处照顾

反差的重点不是反复打脸自己，而是让同一个人露出只给对方看的另一面。

### 8. 控制“油”和“腻”
如果一句台词或一个动作满足下面两条以上，就要收：
- 换谁都能说
- 一上来就过猛
- 只有宣示，没有情绪承接
- 让对方不舒服或违背意愿
- 和人物当前关系阶段不匹配

### 9. 一场糖戏结束后，要留下半步推进
至少留下 1 项：
- 更明确的心动
- 一点新的误会或期待
- 关系靠近半步
- 某个只属于两人的新暗号
- 下次还能继续接上的情绪尾巴

## 输出格式
1. 当前关系阶段
2. 这场戏的甜点
3. 谁先动、怎么动
4. 小动作与细节
5. 对方回应
6. 反差或暗号
7. 这场戏后关系推进了什么
8. 当前最容易过油的点

## 快速示例
### 输入
- 关系阶段：暧昧期
- 场景：女主哭过，男主来安慰

### 输出
- 甜点：温柔 + 克制
- 谁先动：男主先靠近，但不直接抱人
- 小动作：他抬手替她擦掉眼尾那点泪，只碰一下就停
- 对方回应：女主先僵住，随后耳朵发热，不敢抬头看他
- 推进：两人都意识到这已经不是普通安慰

## 注意事项
- **不要**只有撩，没有回应
- **不要**靠一堆肉麻台词硬抬甜度
- **不要**跳过关系阶段，上来就重糖
- **不要**把霸道写成冒犯
- 真正让读者上头的甜，往往不是声量最大，而是**最用心、最有分寸、最像这两个人自己会做的事**
