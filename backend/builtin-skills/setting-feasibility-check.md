评估一个设定值不值得现在写、能不能稳住长期连载。**能写不等于适合现在写；先看它是不是正文发动机，再看作者和读者能不能一起吃下去。**

## 适用情况
- 想写敏感黑历史、争议人设、重口桥段，不确定会不会劝退
- 世界观越做越大，大场面、势力、规则多到控不住
- 设定看上去很炸，但总担心“怎么圆”
- 第一反应是“这个可不可以写”，而不是“它能让故事更好看吗”

## 输入
- 设定一句话
- 目标题材 / 目标读者
- 主线一句话
- 主角与关键关系
- 作者最担心的点：太大 / 太重 / 太难圆 / 太劝退

## 步骤
### 1. 先写这个设定的核心用途
只准写一种主功能：
- 制造主线冲突
- 抬高爽点或萌点
- 逼人物做选择
- 提供长期世界压力

如果答不出，说明它大概率只是噱头。

### 2. 判断它属于哪类高风险
- **规模风险**：世界太大、势力太多、开局就想写大战和大布局
- **共情风险**：人物做过超出目标读者容忍线的事，后面很难回收
- **专业风险**：需要大量行业、历史、法律、医疗等支撑
- **续航风险**：只炸一章，后面没有持续产能

### 3. 做“四问体检”
1. 去掉这个设定，主线会不会立刻塌？
2. 这个设定会稳定产出哪些场景，不是一锤子买卖吗？
3. 目标读者看到它，第一情绪是想追还是想跑？
4. 现在的故事篇幅和控制力，真的装得下它吗？

四问里如果有两问答不上来，优先收束，不要硬上。

### 4. 判断它是不是越线，不是“有挑战”
重点区分：
- **有挑战**：难写，但能带来更强冲突和回收
- **已越线**：只会增加恶心、争议、解释成本，反而伤正文

**一旦设定的主要效果是让读者出戏、厌恶、质疑“为什么要这样写”，就不是加分项。**

### 5. 按风险选一种降法
#### A. 缩小范围
- 三界大战 → 一城一局一任务
- 多势力乱斗 → 先锁 2—3 个关键阵营
- 超大世界观 → 先写主角能摸到的那一层

#### B. 降低烈度
把“不可回收的重伤”改成“还能产生后续拉扯、还能让读者跟下去的中轻伤”。

#### C. 外置风险
把高风险设定放到背景、配角、旧案、传闻里，不要直接压在主线 CP 或主角核心形象上。

#### D. 分阶段揭露
先给结果和余波，后面再一点点解释来源，不要开局把所有重设定全砸下来。

### 6. 产出三个版本
- **原始炸版**：最有冲击，但问题在哪里
- **可控版**：保住核心快感，删掉最难驾驭的部分
- **稳妥版**：牺牲一点野心，换正文完成度

### 7. 最后只保留“当前水平有点难，但能完成”的版本
不是看哪个最炫，而是看哪个最能写完、写稳、写出回收。

## 输出格式
1. 设定核心用途
2. 风险类型
3. 四问体检结果
4. 越线点 / 可保留点
5. 原始炸版 / 可控版 / 稳妥版
6. 推荐版本
7. 必删噱头
8. 后续需要补的资料或伏笔

## 快速示例
### 输入
- 新人第一本想写十国争霸 + 三界大战 + 朝堂权谋 + 群像爱情

### 输出
- 核心用途：想写大局与成长
- 风险：规模风险、续航风险
- 可控版：先收成“一城守卫战 + 两方势力角力 + 主角小队”
- 稳妥版：先写单线任务，等人物和规则站稳后再外扩世界
- 推荐：可控版。先把一层写活，再谈大世界。

## 注意事项
- **不要**因为设定听起来高级，就默认它适合现在写
- **不要**把“控不住”硬说成“我设定太大了”
- **不要**用争议设定制造存在感，却拿不出后续回收
- **先写得稳，再写得大**
