先按感情阶段切时间，再决定年份怎么跳。青梅竹马好写的不是“从小到大都写”，而是“哪些年真正改了关系”。

## 适用情况
- 故事从童年写到成年，不知道该从哪里切入
- 想写两小无猜，但节奏总是慢得发散
- 担心时间跨度大，写成流水账
- 不确定童年该详写还是一句带过

## 核心原则
1. **切点跟着感情走，不跟着日历走。**
2. **不改关系的年份，直接跳。**
3. 童年重点写“特殊感”和“只属于彼此的记忆”，不是硬写早恋。
4. 真正该放慢脚步的，通常是“第一次把对方当异性看”的阶段。
5. 如果主故事重点在后期，前面只保留必要节点，不要把所有成长过程全摊开写。

## 步骤
### 1. 先定主故事真正发生在哪一段
先回答：
- 这本书真正最想卖的是童年萌、少年开窍，还是成年恋爱？
- 两人的核心矛盾和最带感的变化，发生在什么时候？

如果真正想写的是高中、大学或成年后的关系变化，就不要把幼年写成几十章前传。

### 2. 选四到五个必写节点
青梅竹马最稳的节点通常是：
1. **第一次见面**：不用写得很重，但要让读者记住“从这里开始”
2. **小时候的约定**：比如一句玩笑婚约、一个专属承诺、一次站队
3. **最好朋友阶段**：小学或初中，重点写“和别人不一样”
4. **第一次意识到对方是异性**：关系真正开始变味
5. **确认关系或第一次挑明**：把前面铺垫回收

### 3. 给每个节点写清关系变量
每个节点至少回答：
- 这一段之前是什么关系
- 这一段之后变成什么关系
- 读者以后会记住什么细节

示例：
- 初见：陌生人 → 记住彼此
- 约定：普通玩伴 → 关系里多了专属感
- 最好朋友阶段：同学 → 最特殊的人
- 异性意识阶段：习以为常 → 心动和别扭

### 4. 小时候重点写“萌”，不是写“恋爱”
在最好朋友阶段，优先写：
- 彼此和别人不一样的亲近感
- 一些以后能回收的共同记忆
- 小事件、小站队、小偏爱
- “朋友以上，恋人未满”的雏形

这段可以稍微多写一点，但重点是关系基础，不是提早把恋爱桥段全部用完。

### 5. 真正放慢的是“开窍期”
从第一次意识到对方是异性开始，时间线就可以放慢。
重点写：
- 熟悉里突然出现的新鲜感
- 以前习以为常的动作、气味、外貌，现在开始不一样
- 谁先心动，谁先失衡
- 双方怎么从“最好的朋友”拐到“喜欢的人”

**这一段通常才是青梅竹马最值钱的正文区。**

### 6. 用时间跳跃处理空白年份
出现下面情况时，直接跳：
- 这一年没有新的关系变化
- 只有日常相处，没有新冲突或新记忆点
- 只是年龄增长，没有情感升级
- 只是练级、读书、长大，没有直接影响后文

跳法可用：
- 一句话带过
- 一段蒙太奇
- 用回忆补关键细节
- 用一个旧物、旧约定把几年连起来

### 7. 给前期埋“回忆杀”材料
前期最好至少埋 2—3 个后面能回收的点：
- 小时候的承诺
- 某个只有两人懂的称呼
- 一起经历过的小意外
- 某件礼物、道具、地方

后期回收时，这些小点会让时间跨度变成感情积累，而不是流水账。

### 8. 检查切入点有没有离主戏太远
如果从幼年开始写，却离真正要写的矛盾和高光很远，容易节奏难走。
除非本来就是：
- 养成系
- 慢节奏两小无猜
- 故事核心就是“一起长大”

否则更稳的写法通常是：
- 从主戏阶段切入
- 童年只用回忆、插叙和关键节点补齐

## 输出格式
1. 主故事重点阶段
2. 必写节点时间线
3. 每个节点的关系变量
4. 哪些年份直接跳过
5. 前期要埋的回忆杀细节
6. 详写重点
7. 节奏风险点

## 快速示例
### 输入
- 想写两人从幼儿园认识，到高中互相开窍，大学正式在一起

### 输出
1. 主故事重点：高中开窍到大学确定关系
2. 必写节点：
   - 幼儿园初见
   - 小时候一句“长大后嫁给你/娶你”的玩笑约定
   - 初中阶段最好的朋友和小偏爱
   - 高中第一次意识到对方是异性
   - 大学关系挑明
3. 跳过内容：没有新变化的成长年份直接带过
4. 详写重点：高中开始的别扭、吃醋、重新看见对方

## 注意事项
- **不要**把每一年都平均用力
- **不要**童年写太满，后面真正开窍反而没空间
- **不要**只写“他们一起长大”，不写关系到底怎么变
- **不要**把所有时间都花在重复功能的日常上
- 青梅竹马最稳的节奏，不是“从小写到大”，而是**用几个关键节点让读者相信：他们真的一起长大，也真的一步步变得不一样。**
