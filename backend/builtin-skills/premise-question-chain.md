先把模糊主梗压成一句中心句，再围绕这句话连续追问。**先放量，不设限；追问到够多，再删减。**

## 适用情况
- 只有一句设定或一句文案，扩不成故事
- 有题材壳子，但人物、冲突、世界都发空
- 想到了开头，却不知道中段和后续怎么长
- 脑子里只有类似“女主穿书成女配复仇”这种概括句

## 输入
- 一句话主梗
- 题材与基调
- 已确定不能改的设定
- 当前最想强化的点：爽点 / 冲突 / 感情 / 悬念 / 反转

## 步骤

### 1. 先写一句中心句
用最短的话概括故事骨架：
`谁因为什么，进入什么处境，想达成什么。`

例如：
- 女主穿书成女配复仇
- 男主重生后要夺回宗门
- 小透明进娱乐圈翻身

**中心句先求清楚，不求华丽。**

### 2. 第一轮追问基础骨架
至少追问这 8 类：
- 她 / 他是谁
- 为什么会进入这个处境
- 这件事是怎么发生的
- 主角真正想要什么
- 谁或什么在拦他
- 成功会得到什么
- 失败会失去什么
- 这件事最特别的地方是什么

把每个答案写成短句，不要急着展开成正文。

### 3. 第二轮对答案继续追问
对上一轮每个有戏的答案，再往下问：
- 为什么偏偏是这样
- 如果不是这样，会怎样
- 谁会因此受益，谁会因此受伤
- 更糟的版本是什么
- 更带感的版本是什么
- 这个答案会逼出什么场面
- 它会让主角和谁发生正面冲突

**关键不是问得漂亮，而是问到能长情节。**

### 4. 让问题形成“问题树”
把追问分成 4 组：
- 人物：身份、动机、短板、秘密
- 关系：谁爱谁、谁恨谁、谁互相利用
- 设定：世界规则、资源、限制、代价
- 事件：开局冲突、阶段危机、高潮、结局

优先保留那些能同时影响两组以上的问题。
例如“为什么会穿书”如果同时牵动身份秘密、主线反派和结局回收，就继续深挖。

### 5. 放量时先别审判
先允许自己写出 20—50 条问答，不要第 3 条就开始嫌俗、嫌不合理、嫌不好圆。
此阶段目标只有一个：**把可能性摊开。**

可以用下面的快问快答模板：
- 为什么？
- 为什么一定是现在？
- 为什么一定是他？
- 为什么别人做不到？
- 为什么这会更糟？
- 为什么这会更爽？
- 为什么这能逼出下一场戏？

### 6. 把答案改写成“能落地的材料”
把空话改成可写内容：

- 空话：女主很惨  
  改写：她必须嫁给仇家换全家活命
- 空话：男主很强  
  改写：他能赢，但每赢一次都会暴露身份
- 空话：反派很坏  
  改写：反派掌握主角最不能见光的证据

**答案必须能变成设定、动作、冲突、选择或后果。**

### 7. 做第一次删减
逐条问：
- 能不能制造持续矛盾
- 能不能长出具体场面
- 能不能和同类文拉开差异
- 能不能影响后续至少一段剧情
- 删掉它后，故事会不会更清楚

有 3 条答不上来的，先删。

### 8. 把保留下来的答案拼成粗纲
按顺序整理成：
1. 开局：主角被什么推进故事
2. 第一阶段：最先要解决什么
3. 第二阶段：局势怎么变大
4. 转折：主角失去什么 / 发现什么
5. 高潮：最大矛盾怎么正面对撞
6. 结局：主角最终拿到什么，付出什么

### 9. 最后再精修逻辑
等大方向出来后，再集中处理：
- 哪些答案互相冲突
- 哪些设定还缺前置解释
- 哪些桥段有趣但脱主线
- 哪些问题还能继续深挖成支线或反转

## 输出格式
1. 一句话中心句
2. 第一轮问题清单
3. 第二轮问题树
4. 保留答案
5. 删除答案与删除理由
6. 由问答整理出的粗纲
7. 还可以继续追问的 3—5 个关键点

## 快速示例
### 输入
- 女主穿书成女配复仇

### 输出
- 中心句：女主穿进一本宫斗文，成了注定替人背锅的女配，想活下来并反咬仇人
- 第一轮追问：
  - 为什么会穿书：因为原身死前求她替自己改命
  - 为什么一定要复仇：不复仇就会按原剧情被灭门
  - 最大阻力是谁：表面是皇后，实际是男主背后的旧案
- 第二轮追问：
  - 原身为什么选中她：她和原身长得一样，且知道后续剧情
  - 为什么旧案会牵到男主：男主母族就是当年陷害原身家的受益者
  - 更带感的版本：男主一开始以为女主在勾引自己，后来才发现她在查自己家
- 粗纲：
  1. 穿书保命
  2. 借宅斗试探敌人
  3. 查出旧案牵连男主
  4. 与男主从互防到同谋
  5. 正面翻案复仇

## 注意事项
- **不要**只追问世界观，不追问人物目标和冲突
- **不要**把所有答案都留下，删减本身就是创作
- **不要**追问完还停在设定层，必须落到情节
- **不要**一开始就追求完美闭环，先长出来，再修
