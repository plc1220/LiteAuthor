先让文名承担“让读者一眼知道这是什么文”的工作，再考虑气质。**新手文名最怕玄、绕、装，不怕直白。**

## 适用情况
- 新书不会起名
- 文名看着有感觉，但一眼看不出题材和卖点
- 想提高首批点击
- 已有几个标题候选，拿不准选哪一个

## 核心原则
1. 文名的第一任务是**帮助对味读者点进来**，不是展示作者审美。
2. 对新作者来说，**先吸睛，再逼格**。
3. 文名至少要让读者快速知道以下之一：题材、核心梗、主角身份关系、主打情绪。
4. 核心信息尽量前置，越前面越值钱。
5. 简洁优先；太抽象、太文艺、太像散文题目，都会损失点击。

## 四种常用取法
### 1. 一目了然
直接点题材或类型。

适合：重生、穿越、系统、末世、娱乐圈、宫斗、先婚后爱等强类型文。

公式：
- `题材词 + 核心对象`
- `题材词 + 目标/处境`

例：
- 《重生后我不当贤妻了》
- 《末世囤货指南》
- 《系统逼我当反派》

### 2. 破梗
标题里直接抛出最抓人的设定反差或冲突。

公式：
- `被谁/和谁/在什么处境里 + 怎么办/怎么活`
- `所有人都以为……，其实……`

例：
- 《和死对头成亲后我掉马了》
- 《全宗门都以为我是废物》
- 《被前夫的白月光看上了怎么办》

### 3. 简单粗暴
把主角职业、身份、关系、处境直接摆出来。

适合：职业文、古言、种田、都市、群像。

公式：
- `身份/职业 + 关系`
- `谁家的 + 什么人`

例：
- 《将军家的小医娘》
- 《娱乐圈第一经纪人》
- 《侯府养女翻身记》

### 4. 卖萌
用通知体、问句体、长标题、反差口吻吸引点击。

前提：**正文调性也允许卖萌。**

例式：
- 《国家一级捉鬼师上岗通知》
- 《和前男友穿进同一本书怎么办》

## 步骤
### 1. 先列 4 项材料
- 题材词
- 主角身份
- 核心梗或异常点
- 主打爽点或萌点

### 2. 决定这本书最该先让读者知道什么
四选一：
- 先知道题材
- 先知道梗
- 先知道主角
- 先知道关系冲突

### 3. 先出 5—10 个直白候选
不要第一轮就追求高级。

优先保证：
- 读者一眼能读懂
- 核心信息在前半截
- 不需要解释才能懂

### 4. 做“三秒测试”
把标题给没看过文案的人看 3 秒，检查：
- 能不能猜出题材
- 能不能猜出大概冲突
- 能不能说出“这文可能在写什么”

三项里至少中两项，才算稳。

### 5. 做“筛选测试”
好文名不是吸引所有人，而是吸引对味的人。

检查：
- 喜欢这类题材的人会不会更想点
- 不喜欢这类题材的人能不能被及时筛掉
- 标题是不是把错误读者骗进来

### 6. 最后再看气质
如果直白版和文艺版都可用，优先选**更能带来点击**的那一个。
尤其新手，别和大神比“逼格命名”。

## 快速模板
- `重生之____`
- `穿成____后我____`
- `和____成婚后____`
- `全____都以为我____`
- `____家的____`
- `____通知`
- `被____看上了怎么办`

## 输出格式
1. 题材词
2. 主角身份/关系
3. 核心梗
4. 取名方向
5. 候选文名 5—10 个
6. 最优文名
7. 选择理由
8. 高风险备选及问题

## 快速示例
### 输入
古言宅斗，女主被迫嫁人后想和离，男主后知后觉真香。

### 输出
- 题材词：先婚后爱 / 古言宅斗
- 核心梗：她想和离，他却不肯放
- 取名方向：破梗 + 关系冲突
- 候选：
  - 《和离后夫君真香了》
  - 《被迫出嫁后我只想和离》
  - 《侯爷不肯放妻书》
- 最优文名：《被迫出嫁后我只想和离》
- 理由：题材、处境、冲突都清楚，筛选效率高

## 注意事项
- **不要**只写风月、山河、旧梦、长欢这类空泛词，除非作者已有稳定读者盘
- **不要**堆专有名词和冷僻字
- **不要**标题看不出题材，文案还要重新解释一遍
- **不要**正文轻松直白，文名却故作高冷
- 文名真正值钱的地方，是**让对的人先点进来**。
