先别急着写“这个主角很复杂、很带感、很深刻”，先把抽象情结翻成具体设定、具体事件、具体判准。**不能用情节表现出来的情结，暂时不要写。**

## 适用情况
- 想写非常复杂、极端、危险的主角
- 脑子里全是感觉和气质，落不到具体剧情
- 设定词很唬人，但不知道怎么证明给读者看
- 担心主角写崩、写空，或者说服不了读者
- 不确定自己的笔力和当前方案能不能接住这个人设

## 输入
- 一句话主角设定
- 这名主角最吸引人的点
- 当前想到的核心机制 / 世界规则
- 预计题材与基调
- 已想到的关键剧情（可选）

## 核心原则
1. **抽象词必须落到具体场面。** “正义”“邪恶”“疯”“偏执”“救赎”都不是剧情。
2. **价值观必须经得起不同类型事件测试。** 不是喊一句口号就算立住。
3. **核心概念必须有边界。** 读者得知道什么算，什么不算。
4. 极端动机、扭曲追求、自毁倾向都必须有来路。
5. **文风先定，再写情节。** 轻苏、外轻内重、沉郁正剧，落笔方法完全不同。
6. 只要有关键一项答不出来，就先别开写，继续补。

## 步骤
### 1. 先把设定词翻成可演的定义
把主角设定里最唬人的 2—4 个词拆开。

按这个格式写：
- `设定词：____`
- `它在文中具体表现为：____`
- `读者第一次看见它的场景是：____`

例如：
- “超正义三观”不能只写成嘴上厌恶邪恶
- 要落到：遇到哪类事会站出来、会救谁、会放弃什么、会原谅什么

### 2. 用“三类事件”测试主角的判断标准
至少列出 3 种不同性质的事件：
- 利益冲突型
- 道德灰区型
- 私人感情型

对每类都写：
1. 主角会怎么处理
2. 为什么这样处理
3. 这能证明主角哪条底层判断

**如果同一种设定换 3 个事件就反应乱套，说明人设还没立稳。**

### 3. 把核心机制和胜负条件讲清楚
凡是涉及“被正义击败”“完成审判”“满足条件才能跳世界”这类设定，都必须写清：
- 谁来判定
- 按什么判定
- 什么算成立
- 什么算失败
- 读者能不能一眼理解这套规则

**边界不清，后面所有高潮都会像作者强行判赢判输。**

### 4. 追溯极端动机的来路
如果主角有自毁、受虐、扭曲崇拜、强烈自厌等倾向，补这条链：
`过去经历 → 形成的错误认知 → 现在的执念 / 防御方式 → 关键触发点`

要求至少写清：
- 这不是突然发病
- 这种倾向平时怎么表现
- 被戳中时会怎么升级
- 后期会不会变化，代价是什么

### 5. 先写 3 个“证明场景”
不要先写人物小传，先写 3 场戏：
1. **开场证明**：读者第一次知道这个人不一样
2. **压力证明**：高压下，这个人设还能成立
3. **反证证明**：出现会让读者质疑他的场面，再用行为把人设钉实

如果 3 场戏写不出来，说明现在只有情结，没有故事。

### 6. 选文风，不准含糊
从下面选一个主风格：
- **苏爽平面**：强调带感、效率、强刺激
- **外轻内重**：表面好读，内里逐步变深
- **沉郁正剧**：强调压迫、代价、心理重量

再写：
- 为什么选这个风格
- 这个风格下，主角最该怎么出场
- 至少 2—3 部能立刻想到的参考作品

**连参照都举不出来，通常说明脑内风格还是糊的。**

### 7. 做“读者说服力”检查
逐条问：
- 这个主角最容易让读者不信的点是什么
- 需要用哪场戏去证明
- 需要哪些伏笔提前垫
- 哪些解释不能靠旁白，必须靠选择和后果来证明

### 8. 做“当前能不能写”结论
分三档判断：
- **可以开写**：核心设定、判断标准、关键场景、文风都清楚
- **先补再写**：有情结，但缺规则、场景或动机链
- **暂缓挑战**：大部分内容还停在抽象词层面

## 输出格式
1. 一句话主角设定
2. 最危险的 2—4 个设定词及其具体定义
3. 三类事件测试表
4. 核心机制 / 胜负条件
5. 极端动机的成因链
6. 3 个证明场景
7. 文风选择与参考作品
8. 读者最可能不信的点
9. 当前结论：开写 / 补充 / 暂缓

## 快速示例
### 输入
- 主角极度推崇正义，却认定自己是世界最大的恶

### 输出
- 设定词拆解：
  - “推崇正义”＝会主动帮助弱者、厌恶滥杀、愿意为善行付代价
  - “世界最大的恶”＝他的存在会持续放大恶意、吸引恶人或制造灾难
- 三类事件测试：
  - 利益冲突：面对能救一人但会放走恶人的局面，主角怎么选
  - 灰区事件：面对做过恶但仍有善念的人，主角怎么判
  - 私人感情：面对欣赏的善人要杀自己，主角如何回应
- 核心机制：什么才叫“被正义击败”，由谁认定，是否可钻规则空子
- 成因链：为什么他会把自己的惨烈下场，当成正义伟大的证明
- 结论：如果这些都答得出，可以写；答不出，就先别急着上

## 注意事项
- **不要**只有一句很酷的人设口号，没有具体情节
- **不要**用大段解释代替场景证明
- **不要**边界含糊，高潮时再临时改规则
- **不要**把“复杂”写成“混乱”，“深刻”写成“说不清”
- **不要**因为舍不得这个设定，就硬写一个自己暂时接不住的主角
- 真正能写的复杂主角，不是概念多，而是：**每个抽象词都能被具体情节证明。**
