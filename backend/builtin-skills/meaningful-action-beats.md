把空泛的“点头、耸肩、站起、走开、看着对方”改成带态度、姿势、情绪和关系信息的动作。**动作不是陪衬对白的摆设；好动作会让读者听出语气，看出立场，感到人物当下的心情。** 适合对白发干、动作词重复、人物像木头时使用。

## 适用情况
- 对话场景里只会写点头、耸肩、看着、走过去
- 人物明明在说重要的话，场面却没有戏
- 动作很多，但每个动作都像可有可无
- 需要在少心理描写的段落里让动作替人物说话

## 输入
- 原场景或对白
- 这段最重要的功能：传递信息 / 制造张力 / 安抚 / 恐吓 / 试探 / 体现关系
- 参与人物的情绪与关系
- 当前 POV（可选）

## 核心原则
1. **动作要表达态度，不只表达“做了什么”。**
2. 姿势、力度、速度、距离、接触物，都会改变语气。
3. 如果这段主要在传递事实，动作别塞太满，优先放在开头或结尾。
4. 情绪一变，动作也要跟着变。
5. **一个有内容的动作，胜过一串零碎空动作。**

## 步骤
### 1. 先圈出空动作
高危词：
- 点头
- 耸肩
- 看着
- 站起来
- 坐下
- 走过去
- 微笑
- 叹气
- 打呵欠

这些词不是不能用，而是默认信息量太低。

### 2. 给动作补“它想表达什么”
每个动作至少回答一个问题：
- 这个人现在是在压人、退让、安抚、烦躁还是心虚
- 这个动作暴露了什么情绪
- 这个动作改变了人物之间的距离或气氛没有

### 3. 用四种方式给动作加内容
从下列里选 1—3 项，不要全堆：
- **姿势**：前倾、后仰、僵住、塌肩、撑桌、倚门
- **力度 / 速度**：狠狠、慢慢、顿了一下、猛地、信步、拖着
- **接触物**：攥杯沿、塞进抽屉、拍肩、按住门框
- **距离变化**：逼近、绕开、停在半步外、隔着桌子

### 4. 让动作和对白互补
动作最好补足对白没有明说的那一层：
- 嘴上说别担心，动作却在用力合上抽屉 → 烦躁
- 嘴上挖苦，动作却先递伞 → 口是心非
- 嘴上轻描淡写，动作却明显慢下来 → 心虚或在意

### 5. 传递信息时，动作别把重点埋掉
如果这段主要是交代线索、计划、规则：
- 动作放开头定气氛
- 中间让信息清晰落地
- 结尾再补一笔动作收情绪

**不要**一句信息配一个动作，把重点打散。

### 6. 情绪变化时，动作要换
同一段里如果情绪从轻松变成紧张、从自信变成慌乱，动作也要同步变：
- 起初信步、拍肩
- 后来攥紧、停顿、避开视线

### 7. 低心理版本里，让动作多承担一点
如果这篇文心理描写不多，就更要让动作带出：
- 态度
- 情绪
- 关系强弱
- 说话语气

## 输出格式
1. 原段落里的空动作
2. 每个动作该表达的态度
3. 改写后的动作清单
4. 最终场景改写
5. 信息是否被动作干扰

## 快速示例
### 原写
- 霍金斯看着中尉。
- 中尉耸了耸肩，站起来，向门口走去。

### 改写
- 霍金斯身子微微前倾，手指点着桌上的证物。
- 中尉笑着站起身，拍了拍他的肩，把门口让成一条出去的路。

### 改写说明
- 前一个动作带出烦躁和压迫感
- 后一个动作带出自信与安抚
- 不改对白，也能看出两人的不同状态

## 注意事项
- **禁止**为了“有动作”而塞满无关动作
- **禁止**所有人物都共用一套动作模板
- **禁止**重要信息段落里每句都插动作，导致读者看花
- **禁止**只写动作名词，不写动作带出的态度
- 目标不是动作更多，而是**每个动作都更值钱**
