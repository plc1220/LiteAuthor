先用一句话写清故事，再把感情线改成有起伏的“葫芦”形。**纯感情文最怕写成一根绳：从开头直接拴到结尾，中间只有同行、相处、聊天，没有真正改变关系的波折。**

## 适用情况
- 感情文写出来像流水账，只有相处没有故事
- 男女主很快互相喜欢，但后面不知道写什么
- 明明有很多事件，关系推进却不成立
- 不知道哪些地方该详写，哪些地方该一笔带过
- 想写纯感情主线，不想靠大量副线硬撑篇幅

## 输入
- 一句话故事梗概
- 男女主当前关系起点
- 最终关系结果：在一起 / 错过 / 分开 / 复合 / HE / BE
- 当前想到的主要事件
- 预计篇幅

## 先判断：这是不是一个完整故事
把故事压成一句话，再检查落脚点。

### 1. 看一句话的重点落在哪
对比：
- `男女主历经阻碍，最后走到一起。` → 有明确结果，能形成完整故事
- `男女主一起经历了很多事。` → 只有经历，没有结果，容易写成流水账

### 2. 检查中间事件和结局有没有因果
逐条问：
- 这些事件有没有真正改变两人的关系
- 把这些事件换成“吃饭、逛街、看电影”，结局还会不会照样成立
- 如果答案是“会”，说明事件只是陪跑，不是故事重心

**感情文里的事件，必须推动“更靠近 / 更疏远 / 更误解 / 更确认”，而不是单纯证明两个人待在一起。**

## 葫芦结构怎么搭
把感情主线拆成 6 段：

1. **相识 / 入局**
2. **第一次升温**（小口）
3. **第一次冲突或误会**（收窄）
4. **重新靠近，感情升到更高点**
5. **大冲突**（大肚子）
6. **解决或不解决冲突，落结局**

可写成：
`相识 → 升温 → 小冲突 → 更深升温 → 大冲突 → 结局`

### 为什么叫“葫芦”
因为它不是一路平推，而是：
- 开口小：快速建立关系基础
- 中间先放大，再收一下：先给甜头，再制造阻碍
- 大肚子最大：把最重的情绪和冲突留到后半段

**纯感情文最值钱的部分，不是相识，也不是结局本身，而是“感情升到最高时被迫面对的大冲突”。**

## 步骤

### 1. 先定结局承诺
先写一句：
`这段感情最后要落到____。`

例如：
- 最后在一起
- 最后错过
- 最后复合
- 最后确认彼此相爱，但立场无法统一

没有这个落点，前面的波折就容易散。

### 2. 再定“小口”写什么
小口只做两件事：
- 让读者知道两个人为什么会互相吸引
- 让后面的冲突有感情基础

这里不必写太多日常。**只要把“为什么是这两个人”立住即可。**

### 3. 安排第一次小冲突
第一次冲突的作用不是拆散，而是让关系第一次出现阻力。

常见来源：
- 误会
- 性格差异
- 身份差异
- 外部阻碍
- 价值观分歧

要求：
- 冲突能解释为什么他们不能轻易在一起
- 冲突不能一上来就把牌打空

### 4. 让关系升到更高点
在大冲突之前，必须先让读者看到：
- 他们已经比最初更在意彼此
- 他们的关系已经值得失去
- 他们以为自己快要得到结果

**没有这一步，大冲突就不够疼，也不够值钱。**

### 5. 把最大笔墨留给“大肚子”
大冲突必须同时满足两点：
- 真正威胁最终结局
- 逼两个人做出选择

可用：
- 家庭 / 阵营 / 利益对立
- 核心秘密暴露
- 旧伤被翻出
- 重大误判
- 生死、前途、立场不能两全

### 6. 按重要性分配笔墨
优先详写：
- 第一次真正心动或关系突破
- 小冲突爆发时的具体变化
- 感情升到最高点的确认场面
- 大冲突的爆发、拉扯、选择、后果

可以简写：
- 重复功能的日常相处
- 不改变关系的并肩行动
- 只是陪衬气氛的琐事

**详写关系变化，略写关系不变的过程。**

### 7. 检查是不是又写成“绳子”了
出现以下情况，说明结构发平：
- 从相识一路相处到结局，中间几乎没有真正转向
- 冲突只是为了拖时间，没有改关系
- 结局要兑现“在一起”，中间事件却都和“在一起”无关
- 最大篇幅花在日常、路程、闲聊，不花在转折点

## 输出格式
1. 一句话故事
2. 结局承诺
3. 相识 / 小口
4. 第一次升温
5. 第一次冲突
6. 更高点
7. 大冲突
8. 结局落点
9. 详写段落
10. 可删或可简写段落

## 快速示例
### 输入
- 一句话：男女主一起经历很多困难

### 输出
- 问题：这句话只有“经历”，没有“结果”，容易写成同行流水账
- 改写：男女主在门第差距和误会中几次靠近又错过，最终决定站到一起
- 结构：
  1. 相识：因利益合作认识
  2. 第一次升温：共同扛过一次危机
  3. 第一次冲突：发现彼此立场并不一致
  4. 更高点：误会暂时解除，双方第一次明确动心
  5. 大冲突：家族与前途逼他们站到对立面
  6. 结局：共同做出选择，付出代价后在一起
- 详写重点：第一次明确动心、大冲突爆发、最终选择
- 简写重点：重复的合作过程、只起陪衬作用的日常片段

## 注意事项
- **禁止**把“在一起”当前提，再用一串无因果的事件往后拖
- **禁止**只有甜，没有阻力；或只有阻力，没有关系升级
- **禁止**把最大篇幅浪费在不会改变关系的相处上
- **禁止**让冲突只负责拖字数，不负责改局势
- 纯感情文的核心不是“他们做过多少事”，而是**这些事怎样一步步把他们推到结局那里**
