把“主角越来越强 / 越来越红 / 越来越有地位”拆成可见的升级线、奖励节点和获取路线。**升级要明确，奖励要兑现；还要让读者看见主角为了拿到奖励具体做了什么。**

## 适用情况
- 娱乐圈、修炼、竞赛、职场、权谋等事业线写不出持续上升感
- 爽文前期有卖点，后期升级开始发虚
- 只会写“人气涨了”“实力变强了”，但读者感受不到分量
- 每次高潮结束后没有拿到真正奖励
- 大纲里知道要拿哪些资源，却排不清获取路线

## 输入
- 主角起点
- 题材与世界规则
- 读者最想看到的回收
- 可用的升级维度
- 同期对照角色或行业平均线
- 可选：已经想到的奖励 / 资源清单

## 步骤
### 1. 先写起点和读者承诺
用两句写清：
- 主角一开始处在什么低位 / 困局 / 被轻视的位置
- 读者点进来最想看他拿回什么

例如：
- 被全网黑的花瓶 → 读者想看她靠实力和美貌翻盘
- 外门废材 → 读者想看他一路升级踩过门槛

### 2. 列出 3—5 个必须兑现的奖励
奖励不只是一句“变强了”，要写成可见结果：
- 资源
- 头衔
- 排名
- 奖项
- 阵营地位
- 公开认可
- 曾经看不起主角的人反过来争取他

### 3. 设计 2—4 条升级线
不要只用一条干巴巴的数字线。常见组合：
- 人气线
- 资源线
- 能力线
- 地位线
- 奖项线
- 势力线
- 感情 / 站队线

每条线都写：
`当前等级 → 下一节点 → 达成标准`

### 4. 每个奖励前都先放一个大高潮
按这个顺序设计：
`阻碍出现 → 主角顶住或翻盘 → 奖励落袋`

**奖励不是系统弹窗，是高潮的兑现。**

### 5. 奖励要和世界规则挂钩
问自己：
- 这个奖励在这个行业 / 世界里为什么值钱
- 拿到它后，主角具体多了什么权限、名声、机会
- 同级别人通常要花多大代价才拿到

### 6. 一定要有对照组
至少设计 1 个：
- 同期竞争者
- 行业平均水平
- 普通人晋升难度
- 主角前一阶段的自己

**如果人人都能轻松拿到这些节点，主角就不爽了。**

### 7. 升级不要只升一种东西
每阶段至少同时改变 2 项：
- 个人能力
- 外部评价
- 可接触到的舞台
- 可争取到的资源
- 能左右局势的话语权

### 8. 用“游戏化路线图”排获取顺序
把主角当成玩家来排，但**奖励只是路上的回报，不是故事本身**。

先写四样：
- 主角当前能活动的地图 / 场域
- 每个场域能拿到什么类型的奖励
- 主角要做哪些动作，才能解锁下一层奖励
- 哪些奖励是普通回报，哪些是关键节点

可把奖励粗分为：
- **R**：小回报，证明在前进
- **SR**：阶段性资源 / 名望 / 关系推进
- **SSR**：分卷级大奖励，会改写下一阶段局势

然后给每张“奖励卡”写：
`奖励内容 → 解锁条件 → 前置动作 → 拿到后的局势变化`

### 9. 控制奖励密度
- 不要走两步就来一张 SSR
- 小回报可以密，大回报必须贵
- 越大的奖励，前置条件越要清楚，兑现越要有场面

### 10. 奖励落地后立刻写余波
每拿到一次奖励，都补 2—3 个后果：
- 谁开始重新评估主角
- 谁因此被压住或被吸引
- 主角下一阶段能去哪里
- 新敌人或新机会是什么

### 11. 最后检查是不是“假升级”
出现以下情况就重做：
- 只有数字，没有场面
- 只有赢了，没有拿到实质性东西
- 奖励轻到和前期困难不成正比
- 升级节点太密，走两步就拿大奖
- 世界里没有普通难度作为参照
- 奖励像清单式打卡，和故事结局没关系

## 输出格式
1. 主角起点
2. 读者承诺
3. 必兑奖励 3—5 个
4. 升级线 2—4 条
5. 各节点的达成条件
6. 奖励卡路线图（R / SR / SSR 可选）
7. 各节点前的高潮
8. 对照组设计
9. 奖励余波
10. 升级过快或过虚的风险点

## 快速示例
### 输入
- 题材：娱乐圈逆袭
- 起点：全网黑花瓶

### 输出
- 奖励：洗白、拿到高奢资源、作品爆红、奖项提名、顶流主动合作
- 升级线：粉丝数、商业资源、影视资源、奖项认可
- 路线图：
  - R：一次舞台出圈
  - SR：拿下护肤品代言
  - SR：综艺导师邀约
  - SSR：蓝血资源合作
- 每张奖励卡都写清：要先练什么、拿什么作品、过哪场舆论战、赢过谁
- 对照组：同阶段女配辛苦很久也只能摸到边缘资源，用来凸显节点难度

## 注意事项
- **禁止**只用一句“她红了 / 他变强了”带过升级
- **禁止**让奖励脱离世界规则
- **禁止**节点只升数值，不改局势
- **禁止**把奖励写成清单式打卡，忘了故事本身
- 升级文的快感，来自“每次赢都真的有收获，而且收获是一步步挣来的”
