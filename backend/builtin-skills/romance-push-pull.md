推拉不是反复作，也不是一直不长嘴。**推拉 = 靠近 + 阻力 + 后撤 + 余温。** 每一回合都要让两人更在意对方一点，但又暂时不能彻底在一起。

## 适用情况
- 男女主有火花，但互动一写就直线发糖
- 两人总在吵架，却看不出关系真的在变
- 暧昧期想写上头感、失衡感、嘴硬心软
- 想让救场、误会、分享喜悦、拌嘴都服务感情推进
- 需要一边推进主线，一边让 CP 持续有张力

## 输入
- 双方当前关系阶段
- 双方明面态度与真实在意程度
- 双方掌握的信息差/秘密
- 当前外部阻力：身份、立场、规则、任务、误会
- 这一段想把关系推进到哪里

## 步骤
### 1. 先定当前是谁在失衡
每个阶段先写清：
- 谁先心动但不承认
- 谁更敢靠近
- 谁掌握更多秘密
- 谁更怕关系失控
- 谁表面强势，谁情绪上更被牵动

推拉好看，往往不是“谁追谁”，而是**谁先失衡**。

### 2. 每一回合都要有“靠近点”和“后撤点”
用一句话各写一个：
- `这回合他们因为什么靠近`
- `这回合他们又因为什么退开`

常用**靠近点**：
- 救场
- 分享好消息/小愿望
- 主动来找
- 一起走夜路、吃饭、旅行、躲雨
- 帮对方兜底
- 吃醋或偏心
- 看见对方脆弱的一面

常用**后撤点**：
- 身份没说
- 规则不允许
- 误会没解
- 怕失去控制
- 嘴硬、自尊、立场
- 关系一旦挑明会付出更大代价

### 3. 让“嘴上退，行动上进”
最稳的推拉，不是光靠台词，而是制造反差：
- 嘴上说路过，实际上专门来找
- 说别再联系，实际随时在等
- 说不相信对方，实际第一时间去救
- 说只是顺手，实际记住了对方说过的小事

**行动比台词更能卖心动。**

### 4. 每回合至少改一项关系变量
可改的变量包括：
- 称呼
- 联系方式/见面频率
- 信任度
- 共享秘密
- 身体距离
- 吃醋资格
- 求助习惯
- 对未来的默认想象

如果一场戏只是更暧昧一点，却没有任何变量变化，中段很快会干。

### 5. 用“小甜 + 小翻车 + 小补偿”维持上头感
常见波形：
- 先一起开心一下
- 立刻被身份、误会、外部麻烦打断
- 吵一句、退半步
- 然后再用一个动作补回来

这种写法比一直甜或一直虐都更耐看。

### 6. 让外部事件服务推拉，不要和感情分家
把这些事件都当推拉工具：
- 女主遇困，男主救场
- 男主有秘密，女主试探
- 一方分享喜悦，另一方装不在意
- 吵架后分开，危急时想到的还是对方
- 第三人出现，逼出吃醋、在意、站队

**事件不是背景板，要顺手改关系。**

### 7. 保证每次后撤都留“余温”
后撤之后，至少留一种东西：
- 对方一句没说透的话
- 一个没否认的称呼
- 一次及时赶到
- 一个偷偷准备好的补偿
- 一个只有两人知道的小秘密
- 一场未说完的误会

没有余温，后撤就只是断线。

### 8. 危机时用“本能选择”确认感情
高压时最容易看出真在意：
- 第一时间想到谁
- 冒险时先护谁
- 明明生气，出事时还是赶来
- 嘴上说别管我，行动却已经在对方身边

危机是推拉里的“深回合”，别只拿来热闹，要拿来确认失衡。

### 9. 推拉要越推越深，不要原地重复
推荐递进：
1. 记住对方
2. 会主动找对方
3. 会被对方情绪影响
4. 会为对方破例
5. 会把对方放进未来选择
6. 会因为对方可能离开而失衡

### 10. 反查“假推拉”
出现下面情况就重做：
- 只是重复拌嘴，没有关系增量
- 一方长期羞辱另一方，读者只觉得消耗
- 误会全靠不长嘴硬拖
- 每次都用同一种吃醋/救场桥段
- 只会后撤，不会补偿
- 全程谁都不动心，只剩作者说他们有火花

## 输出格式
1. 当前关系阶段
2. 谁先失衡
3. 这一回合的靠近点 / 后撤点
4. 嘴上和行动的反差
5. 改动的关系变量
6. 小甜 / 小翻车 / 小补偿
7. 余温
8. 下一回合能怎么加码

## 快速示例
### 输入
- 设定：强势男主 × 倔强女主
- 需求：写暧昧期上头感

### 输出
1. 当前阶段：互相记住，但都嘴硬
2. 谁先失衡：男主先被牵动，但不承认
3. 靠近点：女主想分享找到工作/拿到机会的喜悦，于是找男主
4. 后撤点：男主不肯承认自己一直在关注她，故意说得很冷
5. 反差：嘴上嫌麻烦，实际早就为“她随时会来找自己”做好准备
6. 变量变化：从“只在危机时见面”推进到“开心时也会先想到对方”
7. 余温：两人虽然吵散了，但下次遇险时女主第一个想到的还是男主

## 注意事项
- **不要**把推拉写成纯误会文学
- **不要**一边推拉，一边让关系完全原地踏步
- **不要**只给吵架，不给补偿和余温
- **不要**让一方长期高位碾压，另一方只能受着
- 好看的推拉，不是“他们总在折腾”，而是**每次靠近都更近一点，每次退开都比上一次更舍不得。**
