把背景设定当成外部压力和剧情推手。系统、重生、职业、身份、家族恩怨、世界规则都应该逼角色行动、逼关系变化、逼冲突升级。**只会说明世界观的设定，不算有效设定。**

## 适用情况
- 人设冲突不够，剧情推不动
- 互动写不长，篇幅撑不起
- 想写剧情流，但关系线偏弱
- 总想靠“再加一个设定”救文，结果越加越乱

## 输入
- 题材
- 主角核心矛盾
- 现有背景设定
- 当前卡点（可选）

## 步骤

### 1. 先分清两层
必须先区分：
- **核心人设 / 关系矛盾**：全文发动机
- **背景设定**：给发动机加压力的外力

例如：
- “被迫倒追高冷对象”可能是核心矛盾
- “不追就会死的系统”是背景压力

### 2. 给背景设定只安排 1-2 个主功能
常见功能只有三类：
1. 推动感情或互动
2. 验证人物品质与选择
3. 提供苏爽语境和资源舞台

不要一个设定同时承担所有功能，容易发散。

### 3. 问四个关键问题
把每个设定都过一遍：
- 角色如果什么都不做，会失去什么？
- 这个设定怎样反复逼角色出手？
- 它会逼两人合作、对抗，还是互相利用？
- 它能连续升级，还是只够用一次？

### 4. 把设定写成“压力链”
不要只写背景，要写它怎么逼出剧情：
`外部规则 / 事件 → 角色必须行动 → 关系或局势变化 → 新压力再来`

能循环的设定，才适合长篇。

### 5. 用背景设定补短板，但别喧宾夺主
- 互动拉不长：适当增加外部压力层
- 剧情太散：减少无关设定层
- 感情线弱：让背景直接制造共同任务、利益绑定、站队压力
- 人物不立：让背景专门去考验人物底线

### 6. 判断设定是不是摆设
出现以下任一情况，就重做：
- 去掉它，剧情几乎不变
- 它只负责解释，不负责制造后果
- 它只在开头惊艳一下，后面再也没用
- 它压掉了人物自身的魅力和冲突

### 7. 用设定控制篇幅
当单靠互动拉不长时，可以加外部压力；当读者开始疲劳时，就减掉旁枝设定。

**背景设定可以扩篇幅，但不能靠堆设定硬拖。**

## 输出格式
1. 核心人设 / 关系矛盾
2. 背景设定清单
3. 每个设定的主功能
4. 压力链设计
5. 能反复触发的剧情节点
6. 需要删掉的摆设型设定
7. 篇幅控制建议

## 快速示例
### 输入
- 题材：穿书恋爱文
- 当前设定：系统强迫女主攻略男主
- 问题：互动写两章就空了

### 输出
- 核心矛盾：女主必须接近男主，但男主本人极难亲近
- 背景压力：不攻略就受罚
- 主功能：推动互动
- 压力链：系统任务升级 → 女主必须一次次接近 → 男主起疑或误会 → 关系变化 → 新任务逼近
- 优化建议：别只写系统发布任务，要让任务结果真正改变关系和局势
