先把大设定拆成“会真正上场的设定”和“只做背景板的设定”，再做一张能在大纲期和连载期反复翻看的世界观表。**大设定不是比谁想得多，而是谁能收得住、排得开、用得上。**

## 适用情况
- 世界观很大，脑子里有很多种族、势力、等级、道具、地图
- 开始写时信息太杂，自己都容易忘
- 设定越写越多，大纲开始失控
- 写到一半常忘掉专有名词、规则和前文设定
- 大型玄幻、修真、星际、架空、群像题材需要长期对照表

## 输入
- 一句话题材
- 主角主线 / 核心冲突
- 目前想到的全部设定
- 预计篇幅
- 当前最乱的部分

## 核心原则
1. **先服务主线，再服务世界厚度。**
2. **背景板不用按前台规格细讲。**
3. **设定最好落成表格，方便大纲和正文随时对照。**
4. **设定只有能逼人行动，才算有效。**

## 步骤
### 1. 先写这本书真正写什么
用三句话钉死：
- 主角最核心要做的事
- 这本书的主打亮点
- 去掉 70% 设定后，仍然必须存在的剧情是什么

剩下还不可删的，才是前台设定。

### 2. 先做一张世界观总表
在动大纲前，先列最基础的分类。按题材自定义，但通常至少有：
- 种族 / 身份
- 阵营 / 势力 / 宗派
- 地图 / 区域 / 重要地点
- 等级 / 境界 / 职业链
- 规则 / 禁忌 / 代价
- 资源 / 货币 / 稀缺物
- 武器 / 道具 / 药品 / 技能
- 生物 / 灵兽 / 机械 / 怪物
- 时间线 / 历史节点 / 大事件
- 重要专有名词

**写文时想到的新东西，也继续补回这张表。**

### 3. 把设定分三层
- **前台设定**：首卷或当前阶段一定要上场
- **中台设定**：后面会用，现在只留名字或钩子
- **背景板设定**：只增厚世界，不推动当前剧情

### 4. 每个前台设定都过“五问”
1. 它解决了什么叙事问题
2. 它制造了什么冲突或选择
3. 它会被谁真正用到
4. 第一次出现时，读者能不能立刻感到后果
5. 删掉它，主线会不会明显变弱

答不上来，就降级或删除。

### 5. 把设定从“名词”改成“结果表”
不要只记名字，还要记：
- 谁压谁
- 谁和谁对立
- 规则怎么触发
- 违规后果是什么
- 资源谁掌握
- 主角怎么拿到
- 这个地点最容易发生什么戏

### 6. 首卷先收范围
优先只保：
- 1 个主舞台
- 1 条主系统
- 1 组核心势力关系

不要第一卷就把全地图、全门派、全历史一起开。

### 7. 排信息释放顺序
把设定分成三类：
- 现在必须知道
- 先提名，后解释
- 真用到再说

### 8. 写大纲时边对照表，边补表
大纲推进到哪一步，就补哪类设定：
- 需要新门派，就回表里补门派关系
- 需要新道具，就回表里补来源、限制和名称
- 需要新地点，就回表里补地图位置和作用

**让表格服务剧情，不是让剧情给表格打工。**

### 9. 写正文时把生僻信息交给表，不要交给记忆
专有名词、武器名、药品名、节庆、官职、货币、境界称呼等，统一记在表里。
忘了就翻，避免：
- 前后叫法不一致
- 规则互相打架
- 作者自己写着写着忘设定

### 10. 连载中失控时的补救
1. 停止继续加新系统
2. 找回当前最重要的舞台和冲突
3. 其余设定降成背景板或传闻
4. 优先修表，再修文
5. 把前面抛出的规则漏洞先回收掉

## 输出格式
1. 一句话主线
2. 世界观总表分类
3. 前台 / 中台 / 背景板设定清单
4. 首卷必须保留的 3—5 个设定
5. 每个前台设定的“五问”答案
6. 信息释放顺序
7. 需要补回表格的专有名词清单
8. 当前最大的设定风险

## 快速示例
### 输入
- 题材：大门派修真文

### 输出
- 世界观总表：
  - 种族：人 / 妖 / 灵
  - 阵营：正道 / 魔道 / 中立
  - 宗派：青云、天音、鬼门……
  - 境界：练气 / 筑基 / 金丹……
  - 武器：本命法器 / 神器 / 炼器
  - 灵兽：按级别分类
- 首卷只保留：一个门派、当前境界链、门派内部竞争
- 其他洲、远古秘闻、完整神器谱先后置

## 注意事项
- **不要**把“设定多”误当成“故事大”
- **不要**所有分类都在首卷详细展开
- **不要**只记名词，不记后果和用途
- **不要**靠脑内硬记整套大设定
- 大设定真正值钱的不是宏大，而是**能持续产出情节，还不打架**
