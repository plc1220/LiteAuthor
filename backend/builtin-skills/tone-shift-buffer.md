先确定这次文风变化是不是故事自然长出来的。**文风可以变，但不能像换频道。读者能接受“越来越沉”“越来越冷”“越来越狠”，接受不了“上一章还是这本书，下一章像另一本书”。**

## 适用情况
- 前期轻松，后期变沉重 / 黑暗 / 虐
- 连载中主角遭遇升级，前后口吻差太大
- 角色被重大事件触发后，章节气质突然变了
- 读者说前后像两篇文
- 想保留情节转折后的重量，又怕整体风格散掉

## 输入
- 前文主风格一句话
- 后文目标风格一句话
- 导致转型的关键事件
- 前文已写章节或片段
- 可承担缓冲作用的配角 / 关系线

## 核心原则
1. **先让情节合理，再让语言变化。**
2. **文风变重，不等于把原来的写法全部推翻。**
3. 一次只变几项：情绪浓度、玩笑密度、句式节奏、角色声线，不要同时大换血。
4. 用过渡场景消化转型，不要硬切。
5. 稳定的配角和稳定的叙述习惯，是最好的缓冲带。

## 步骤
### 1. 写清旧风格、新风格和共同底色
按下面格式先写三句：
- 前文主味：____
- 后文主味：____
- 全文共同底色：____

共同底色可以是：利落、克制、俏皮、冷感、机敏、抒情但不腻。  
**后面再怎么变，共同底色都尽量保住。**

### 2. 写转型因果链
填空：
`因为____发生，主角 / 关系 / 局势对____的看法变了，所以文章从____转向____。`

写不出来，说明这次文风变化还没有剧情根据。

### 3. 回收前文的“语言锚点”
把前文多看几遍，列出 2—3 个继续保留的东西：
- 常用句式和句长
- 常用词汇或口头感
- 叙述距离：贴人物还是稍远
- 反复出现的意象
- 轻松时常见的节奏感

**保留锚点，不是复制旧文，而是让读者还认得这是同一篇文。**

### 4. 决定什么变、什么不变
做一张简表：

| 项目 | 前文 | 后文 | 是否保留一部分 |
|---|---|---|---|
| 情绪浓度 | 轻 | 重 | 是 |
| 玩笑密度 | 高 | 低 | 是 |
| 句式节奏 | 短促 | 可稍压沉 | 是 |
| 配角声线 | 活 | 稳 | 是 |
| 场面类型 | 热闹 | 收束 / 对峙 | 否 |

原则：**降低某项，不等于清零。**  
比如后文变沉重，玩笑可以少，但不必完全消失；否则会像突然换作者。

### 5. 安排 2—3 个缓冲场景
每个缓冲场景至少做两件事：
- 交代重大事件的后果
- 让人物消化情绪
- 保留旧风格的一点尾音
- 先露出新风格的一层
- 让稳定配角出场，帮文章过桥

常用缓冲场景：
- 事后收拾残局
- 老关系在新处境下第一次对话
- 熟悉人物用旧语气出场，但场面意义已经不同
- 小范围冲突，先试出新气压

### 6. 有意识回收前文常用的词和句式
找到前文你最常用、最顺手的：
- 轻松时常用的短句
- 固定的叙述转折
- 角色之间熟悉的说话节奏

后文适度回收，**不要完全改成另一套语言系统**。

### 7. 加大“相对稳定人物”的缓冲作用
选 1—2 个受主线冲击较小的人物：
- 让他们在后文继续保有原来的说话习惯
- 让他们和主角互动，形成新旧状态的对照
- 让他们承担部分情绪缓冲

但注意：**不要让配角硬插科打诨，破坏已经成立的情绪重量。**

### 8. 画一张章节气氛曲线
给接下来 5—8 章标 1—5 级气氛强度：
- 1：轻松
- 2：轻快但有暗流
- 3：压住笑意，明显转沉
- 4：紧张、对峙、失衡
- 5：沉重、高压、爆发

如果曲线是 `2 → 5 → 1 → 5`，大概率会乱。  
更稳的是 `2 → 3 → 4 → 4 → 5`，或在高压中插入一小口缓气，但不跳频道。

### 9. 最后复查三件事
1. 如果去掉角色名，读者还认得是同一篇文吗？
2. 这次变重 / 变冷，是剧情逼出来的，还是作者忽然改口吻？
3. 前文最有辨识度的表达习惯，后文还剩多少？

## 输出格式
1. 前文主风格 / 后文目标风格 / 共同底色
2. 转型因果链
3. 需要保留的语言锚点
4. 变化项与保留项简表
5. 2—3 个缓冲场景
6. 可承担缓冲的配角安排
7. 章节气氛曲线
8. 最容易突兀的风险点

## 快速示例
### 输入
- 前文：轻松江湖群像
- 后文：主角家破后转入复仇
- 问题：怕前后像两本书

### 输出
- 共同底色：利落、带一点讥诮
- 保留锚点：短句推进、人物互呛的说话节奏
- 缓冲场景：
  1. 事后收尸与安置，第一次把玩笑压低
  2. 老搭档按原来口气说话，但主角已经接不住
  3. 主角独处时开始出现更冷的判断句
- 气氛曲线：2 → 3 → 4 → 4 → 5
- 结果：文风变沉了，但读者仍觉得是同一篇文自然长出来的

## 注意事项
- **不要**上一章插科打诨，下一章突然全员哲学独白
- **不要**为了“变沉重”把所有角色都写成同一种冷脸
- **不要**情节还没到，先把文风硬拗过去
- **不要**把缓冲写成拖沓
- 好的文风转型，不是前后完全一样，而是**前后不同，但能看出是同一条路走过去的**。
