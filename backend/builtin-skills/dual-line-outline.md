先别按时间顺序把事件一格格排满，先把**人物会把故事推向哪里**想清楚。再把事业线和感情线分开列、互相对照、最后合并。**双线大纲不是两篇文拼在一起，而是同一批人物在两条线里彼此抬轿。**

## 适用情况
- 人设有了，大纲还是空
- 按时间线列细纲，写出来像流水账
- 甜文、爽文、甜爽文想兼顾感情和剧情
- 感情线和事业线各写各的，互相不抬轿
- 已经想到很多梗，但不知道该往哪条线里放

## 输入
- 主角 / 主 CP 的核心人设
- 题材卖点
- 主视角人物的事业目标
- 主角关系目标
- 已经想到的零散桥段（可选）

## 步骤
### 1. 先从人设里挖“剧情发动机”
对主角和关键角色，至少写这 5 项：
- 他 / 她害怕什么
- 会为了什么成长
- 最重要的东西是什么
- 当前最大动力是什么
- 别人现在怎么看他 / 她

这 5 项不要停在抽象词上，要立刻往剧情上翻译。

例如：
- 害怕家庭压力 → 可长出“被家人强行带走”的剧情
- 想证明自己 → 可长出“主动争关键比赛 / 任务”的剧情
- 别人都把她当花瓶 → 可长出“被质疑后正面打脸”的剧情

**先从人设发散事件，再决定事件顺序。**

### 2. 先用剧情立住人设，再让人设反推后续剧情
开篇和前期不要急着讲完整人设，先用几个有效场景让读者记住：
- 这个人遇事怎么反应
- 这个人最明显的优点 / 弱点是什么
- 这对关系现在是什么味道

等读者脑子里有了稳定印象，再继续问：
- 这样的人后面会被什么击中
- 这样的人会为谁改选择
- 这样的人最不肯放手的是什么

**前期是剧情立人，后期是人设推剧情。**

### 3. 把两条线分开列阶段
先只写阶段，不急着写细节。

推荐写法：
- **事业线阶段 1—2—3—4**
- **感情线阶段 1—2—3—4**

每个阶段只写一句：
- 现在卡在哪
- 这一阶段要解决什么
- 结束时局面会怎么变

例如：
- 事业线 1：重组 / 起步 / 证明自己
- 事业线 2：第一次真正卡关
- 事业线 3：赢得认可 / 遭遇更大对手
- 事业线 4：关键胜负 / 身份跃迁

- 感情线 1：初识 / 误判 / 吸引
- 感情线 2：合作 / 试探 / 心动
- 感情线 3：秘密加压 / 关系失衡
- 感情线 4：坦白 / 站队 / 确认关系

### 4. 把已想到的桥段分门别类
把所有灵感剧情按三类归档：
- **只属于事业线**：比赛、升级、任务、查案、夺权、打脸、成长
- **只属于感情线**：试探、误会、暧昧、照顾、秘密施压、关系突破
- **两者皆有**：同一件事既推进事业，又推进感情

**最值钱的是“两者皆有”的桥段。**

例如：
- 一场比赛既让主角翻身，又让 CP 第一次真正默契配合
- 一次家庭危机既压事业线，又逼感情线站队
- 一次打脸既立主角能力，又让对方彻底改观

### 5. 甜文多填互动，爽文多填打脸，甜爽文刻意合并
按题材调整重心：
- **甜文**：多放互动、照顾、试探、靠近、关系变化
- **爽文**：多放冲突、反击、升级、站队、结果回收
- **甜爽文**：优先保留“互动 + 打脸”“关系推进 + 事业推进”同时成立的事件

如果一个事件只能甜，不能推主线，就要控制比例。
如果一个事件只能爽，和关系完全无关，也要看是不是偏离卖点。

### 6. 排顺序时，按“读感”排，不按日历排
不要因为事情本来是这样发生的，就机械按天排。
优先按这几个标准调整顺序：
- 哪个事件更适合立人设
- 哪个事件更适合做阶段转折
- 哪个事件放在前面更能形成期待
- 哪个事件必须并到一起，效果才会更强

**时间线是骨架，不是枷锁。**

### 7. 逻辑缺口，用设定补；不是先堆设定再求剧情
如果排完后发现某个转折站不住，再补：
- 规则
- 身份
- 家庭背景
- 职业约束
- 资源差
- 旧事

不要一开始就把设定铺满。
**设定的作用是给剧情补逻辑，不是替剧情干活。**

### 8. 做删减：不立人设、不被人设推动的，删掉
每个场景、配角、剧情点都检查：
- 它是在**立主角人设**吗
- 它是在被主角人设**自然推动**吗
- 它有没有推进事业线 / 感情线 / 两者之一
- 它符不符合这本书真正的卖点

如果四问都答不上来，删掉。

尤其检查：
- 所有配角的出场是否必要
- 每个场景是否和主角密切相关
- 主视角人物身上是否真的挂着事业线
- 有没有为了换副本而换副本

### 9. 最后看“卖什么”
用一句话回答：
`这本书本质上在卖什么？`

再反查：
- 双线里哪一部分最能持续供应这口体验
- 哪些桥段虽然好写，但不属于这本书的卖点

**不符合卖点的部分，哪怕单看不错，也该删。**

## 输出格式
1. 主角 / 关键角色的 5 项剧情发动机
2. 前期用于立人设的 3—5 个场景
3. 事业线阶段 1—4
4. 感情线阶段 1—4
5. 事业线事件
6. 感情线事件
7. 两者皆有的事件
8. 事件顺序调整理由
9. 需要补的设定
10. 应删的场景 / 配角 / 支线
11. 全书核心卖点一句话

## 快速示例
### 输入
- 电竞甜爽文
- 女主事业低谷，男主空降战队

### 输出
- 剧情发动机：女主怕被家里逼退圈；想重回巅峰；别人以为她只是花瓶。男主的动力是靠近女主，但他不说。
- 事业线：战队重组 → 资格赛 → 第一次真正磨合失败 → 翻盘晋级
- 感情线：初识 → 配合变默契 → 男主秘密加压 → 公开站队
- 两者皆有事件：线上赛双排打出完美配合；女主被家人带走，男主去找人；女主被黑时男主公开护人
- 删除项：与主线无关的配角日常、不能立人设的闲聊、只是为了换地图的副本

## 注意事项
- **不要**按时间顺序把情节塞满，就以为那叫大纲
- **不要**让感情线和事业线长期互不相干
- **不要**把设定当剧情
- **不要**保留既不立人设、也不推剧情、还不符合卖点的桥段
- 双线大纲最稳的写法是：**先有人设发动机，再有两条线，再让好桥段往中间合并**
