先还原角色的骨架，再安排新剧情。**同人里的“像不像这个人”，不在于复述名台词，而在于：换了处境后，他依然会以“像他”的方式判断、选择、表达和付代价。**

## 适用情况
- 写同人时角色容易 OOC
- 想写 AU、分歧线、救赎线，但怕原著人物崩坏
- 角色台词像在说作者的话
- 新剧情推进顺了，人却不像原著
- 明明很熟原著，落笔还是不稳

## 输入
- 原著名称与角色
- 角色在原著中的关键经历
- 角色的核心欲望、底线、恐惧、关系
- 本文和原著的分歧点
- 计划新增的剧情、CP 或世界变化

## 步骤
### 1. 先写角色的“硬骨架”
至少列 7 项：
- 出身和位置
- 最重要的过去经历
- 最想得到什么
- 最怕失去什么
- 绝不轻易踩的底线
- 习惯性的表达方式
- 与 2—4 个关键角色的关系模式

### 2. 区分“不可动”和“可动”
#### 不可动
- 核心欲望
- 价值排序
- 触发点
- 做人底线
- 对关键关系的基本态度

#### 可动
- 所处环境
- 知道的信息
- 手里资源
- 对某件事的理解深度
- 经历新事件后的成长方向

**改设定可以，别把骨架一起改没。**

### 3. 明确这篇文到底改了什么条件
用一句话写：
`和原著相比，这次真正改变的是____。`

只改三类条件：
- 信息差
- 资源差
- 时机差

再问：
`如果角色多知道这一点 / 多拥有这一点 / 提前遇到这件事，他会怎么重新选择？`

### 4. 用“判断 → 选择 → 后果”写角色逻辑
每个关键桥段都按这个顺序写：
1. 他先在意什么
2. 他怎么理解眼前局面
3. 他会选哪条路，为什么
4. 这个选择会付什么代价

**不是作者想让他做什么，而是这个人到了这里，不做别的才更像他。**

### 5. 尽量用有限视角，不用上帝视角替角色说话
- 从主角或贴近角色的视角看世界
- 先写角色看到、误会、在意和判断的东西
- 少用全知旁白替角色总结三观
- 少夹带作者自己的社会批评和“正确答案”

同人更重要的是把人物写活，不是借人物发表议论文。

### 6. 抓住角色的“手感”
至少固定 4 类细节：
- 说话节奏：直说 / 绕弯 / 冷淡 / 戏谑 / 克制
- 情绪外化：发火、尴尬、心软时怎么表现
- 关系距离：面对亲近者、陌生人、上位者、敌人各怎么说
- 习惯动作或称呼方式

### 7. 如果是 AU 或颠覆设定，保留两个锚点
即使世界完全变了，也至少保住 2 项：
- 欲望 + 底线
- 关系模式 + 反应方式
- 价值排序 + 说话手感

这样改得再大，角色也还有原著辨识度。

### 8. 用 3 个问题排查 OOC
- 删掉名字和设定，只看行为，读者还能认出这是他吗？
- 把这段选择换给另一个角色，是否也完全成立？如果成立，说明写得不够像这个人。
- 这次变化是被新条件逼出来的，还是作者硬按出来的？

## 输出格式
1. 角色硬骨架
2. 不可动 / 可动清单
3. 本文改动条件
4. 关键桥段的判断 → 选择 → 后果链
5. 说话与情绪手感
6. AU 仍保留的两个锚点
7. OOC 风险点
8. 需要修正的桥段

## 快速示例
### 输入
- 想写一个原著里嘴硬心软的冷脸角色，放进“提前知道真相”的分歧线

### 输出
- 硬骨架：护短、爱面子、不肯示弱，真正怕的是重要的人出事
- 改动条件：他提前知道某次背叛
- 判断：先不信，再自己查证
- 选择：不会立刻痛哭摊牌，更可能冷着脸提前布置
- 后果：表面更难接近，实际保护动作更早出现
- 结论：剧情变了，但人没变成另一个模板

## 注意事项
- **不要**靠复读名台词冒充还原
- **不要**为了推进 CP 或爽点，让角色突然失去原有判断力
- **不要**一边写“忠于原著”，一边把关键底色全换掉
- **不要**让人物变成作者发言器
- 同人人物还原最稳的标准是：**处境可以不同，选择逻辑不能乱。**
