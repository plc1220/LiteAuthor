按篇幅和卖点提前排大高潮，把长连载写成一段段能冲起来的高峰。**大高潮不是突然放大招，而是前面一段时间持续抬压、层层加码后的集中兑现。**

## 适用情况
- 50 万字左右的爽文 / 剧情流不知道多久该来一次大高潮
- 中段发平，读者只觉得在走流程
- 只会写一个大爆点，不会安排长线起伏
- 上架 / 入 V / 分卷节点不会卡

## 输入
- 总字数或分卷字数
- 主打类型（感情流 / 剧情流 / 升级流 / 复仇流）
- 每一阶段想兑现的奖励
- 当前已有的重要事件

## 步骤
### 1. 先决定“大高潮到底爽在哪”
每个大高潮只能主打一口：
- 感情确认 / 关系大变
- 虐渣打脸 / 公开翻盘
- 升级夺冠 / 资源到手
- 真相揭露 / 立场翻转

### 2. 先按篇幅排高峰位置
可用这个基准线：
- **40—80 万字爽文**：大致每 **8—12 万字** 来一波大高潮
- 每波大高潮从“进入事件”到“收尾落袋”，常见跨度 **2—3.5 万字**
- 两个大高潮之间，要有若干小高潮和阶段性回报

这不是死规定，但**长篇太久不起峰，读者就会掉线**。

### 3. 关键收费点怎么卡
- **感情流**：优先卡男女主关系变化、暧昧失控、表白前夜、关系翻车
- **剧情流 / 爽文**：优先卡虐渣打脸、真相爆开、主角出手前一秒

### 4. 每波大高潮内部用“一波三折”
常见排法：
1. **第一折**：主角被轻视或遇到第一层阻碍
2. **第二折**：更强对手、更高规则或更大代价压上来
3. **第三折**：真正的强敌或克制点出现，局势看似最糟
4. **顶点**：主角翻盘、爆底牌、拿奖励
5. **余波**：站队变化、关系变化、下一波钩子

### 5. 每一折都要升级，不要换皮重复
至少升级 1—2 项：
- 对手更强
- 围观层级更高
- 失败代价更大
- 主角方法被逼着改变
- 读者对主角的期待更高

### 6. 大高潮后必须真兑奖
每波高潮至少回收 2 项：
- 资源 / 奖励到手
- 地位提升
- 感情推进
- 新敌人或新阶段开启
- 旧判断被推翻

没有兑奖，高潮就会发空。

### 7. 平淡期不要只灌水
当文章进入平淡期、离完结还远时，可用这些方法提气：
- 提前抛出新悬念
- 制造一次有后果的损失或代价
- 让配角承担后果，放大危险
- 用误导或错判把读者重新拉进猜测

**不是为了虐而虐，而是让局势重新变重。**

## 输出格式
1. 总篇幅与分段
2. 每波大高潮的位置
3. 每波大高潮的爽核
4. 每波的“一波三折”
5. 高潮后的奖励兑现
6. 关键收费点推荐怎么卡
7. 中段平淡期的提气方案

## 快速示例
### 输入
- 50 万字剧情爽文
- 主线：主角参加晋级赛一路成名

### 输出
- 10 万字左右第一波大高潮：首次公开立威
- 20 万字左右第二波大高潮：挑战成名对手
- 30 万字左右第三波大高潮：对上克制主角的天才人物
- 每波内部：
  1. 被看轻
  2. 干掉小有名气的对手
  3. 对上更强更克制的强敌
  4. 主角亮出更高层底牌
  5. 余波：成名 / 结仇 / 下一战钩子

## 注意事项
- **不要**把全部王炸挤在前 10 万字
- **不要**只写过程，不写奖励落袋
- **不要**每一波都只是“来了个更强的人”
- 长连载的关键，不是一直冲，而是**冲得有间隔、有蓄力、有兑现**
