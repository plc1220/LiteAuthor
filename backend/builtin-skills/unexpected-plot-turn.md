先把当前最平的直线剧情写出来，再设计一个会打断路线、制造落差、逼角色做选择、最后把故事推回主线的意外。**意外不是乱翻桌，而是让故事偏离无聊路线后，以更高代价或更高回报回到主目标。**

## 适用情况
- 剧情按部就班，像流水账
- 明明有主线目标，中间过程却很闷
- 铺垫太长，作者自己都等不到最想写的部分
- 想写更有爆点、更酸爽的故事
- 卡文时总觉得“后面还有很多铺垫要走”

## 输入
- 当前主线目标
- 原本准备怎么推进
- 这一段最平的 1—3 个环节
- 在场角色与各自动机
- 可用伏笔、世界规则或外部压力

## 核心原则
1. **意外必须先打断固定路线。**
2. **打断后必须立刻造成情绪落差和新问题。**
3. **意外不能只热闹，必须逼角色做选择。**
4. **最后要回到主线，而且比原路线更高能。**
5. 意外最好来自角色动机、既有规则、旧伏笔或真实代价，少用纯作者硬掰。

## 步骤
### 1. 先写“最平版”
用一句话写清原路线：
`主角想____，准备通过____，最后达到____。`

如果这句话一看就知道会顺顺地发生什么，说明这里适合加意外。

### 2. 找出最无聊的确定性
检查哪一点最像流程：
- 一切都按计划来
- 角色只是在等结果
- 没有新的损失和风险
- 没有人会做出出格反应
- 读者已经能猜到每一步

把最无聊的那一点圈出来。

### 3. 选“打断什么”
优先从这些里选 1 项下手：
- 钱 / 资源没了
- 关键人不见了
- 时间突然不够
- 身份暴露 / 被误判
- 原计划被第三方搅乱
- 本来稳的关系突然失衡
- 以为到手的东西被截胡

**一次先打断一项，别一口气全炸。**

### 4. 给意外找来源
优先来源：
- 某个角色按自己的性格做了事
- 之前埋过的伏笔回咬
- 世界规则 / 社会规则突然生效
- 对手提前一步出手
- 主角自己一个小选择引发连锁反应

最弱的来源：
- 纯天降巧合
- 所有人突然降智
- 只为了拖字数的事故

### 5. 让意外先把局面打坏
好用的链条是：
`原计划 → 突发打断 → 立刻损失 / 错愕 → 被迫改道`

常见损失：
- 失去资源
- 失去面子
- 失去机会
- 被迫背锅
- 关系恶化
- 主角原本稳住的东西突然不稳了

### 6. 再埋一个不起眼的新机会
意外之后，不要只给坏消息。补一个看似不重要的小东西：
- 顺手拿到的线索
- 被迫认识的新人物
- 临时做出的选择
- 某个旧物 / 小奖券 / 小承诺
- 某句话 / 某次误会留下的后手

这个小点后面要能翻成更大的回收。

### 7. 用“先坏后好”抬爆点
酸爽常见节奏：
1. 读者以为会平推
2. 突然出事
3. 主角懵 / 急 / 受挫
4. 读者开始担心后续
5. 小伏笔翻面
6. 主线回归，而且结果比原计划更大 / 更狠 / 更失控

**不是所有意外都要变好，但至少要让局势真的升级。**

### 8. 保证人物不是木偶
逐个问：
- 这件意外里，谁最想趁机做什么
- 谁会因此高兴、恐惧、补刀、救场
- 每个人的反应是否贴合前文性格

如果意外发生后，所有人只会按作者安排站桩，戏还是不活。

### 9. 检查能不能回到主线
意外再炸，也要回答三句：
- 它把主线暂时拐到哪去了
- 它最后怎么把主角推回原目标
- 回来时多了什么：更高 stakes、更强关系、更大损失，还是更好的回报

**回不来，是翻车；回得更高，是升级。**

### 10. 做“可追更”排查
一场有效意外至少要留下 2 项：
- 新问题
- 新选择
- 新关系状态
- 新机会
- 新敌意 / 新联盟

如果只是热闹一下就回到原样，这个意外就偏空。

## 输出格式
1. 原本的平版路线
2. 最无聊的确定性
3. 打断点
4. 意外来源
5. 立刻损失 / 情绪反应
6. 被迫改道后的新路线
7. 暗埋的小机会
8. 回归主线的方式
9. 升级后的结果
10. 降智风险

## 快速示例
### 输入
- 主线：主角想存够钱，一年后去旅行

### 输出
- 平版路线：每个月攒钱 → 一年后旅行
- 打断点：工作突然没了
- 意外来源：公司裁员 / 上司翻脸
- 立刻损失：主角失去收入，旅行计划泡汤
- 被迫改道：先解决生存问题
- 暗埋小机会：主角失魂落魄时随手买的一张小奖券
- 回归主线：奖券中奖，主角不仅能旅行，还获得更大选择权
- 升级结果：主线目标没丢，反而从“去旅行”升级成“彻底改写人生”

## 注意事项
- **不要**把意外写成纯随机灾难
- **不要**只打断，不回收
- **不要**每隔一章就硬翻桌，容易失控
- **不要**为了刺激让角色突然不按人设行动
- 意外最值钱的地方，不是“吓读者一跳”，而是**打断无聊路线后，让故事变得更好看**
