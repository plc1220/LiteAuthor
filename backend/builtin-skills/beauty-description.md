先定读者第一眼要收到什么印象，再按视线移动去写。**外貌描写最稳的顺序是：从远及近、从大到小、从外到内、从最突出的印象到次要细节。** 不要一上来五官报菜名，也不要把互相打架的气质词硬塞在同一层。

## 适用情况
- 人物初见写成模板词堆砌
- 一段外貌里同时塞进太多矛盾气质，读者想象不出来
- 想写清冷、病弱、锋利、娇美等复杂感觉，却越写越乱
- 需要给主角或重要配角写有层次的出场外貌

## 输入
- 角色身份、年龄、时代、当前场景
- 观察者是谁，与角色什么关系
- 角色最先给人的整体印象
- 2-3 个必须写到的突出细节
- 想让读者最后留下的情绪或判断

## 步骤
### 1. 先定“第一眼印象”
用一句话写清远看最抓人的东西：
- 高瘦 / 压迫感 / 明艳 / 病弱 / 学生气 / 贵气 / 狼狈 / 干净
**先有整体，再有局部。**

### 2. 按观察路线排顺序
优先按这条线：
1. 远看轮廓、站姿、身形
2. 走近看穿着、体态、动作
3. 对视后写眼神、表情
4. 最后落到嘴唇、手指、痣、伤痕、习惯动作等微小细节

### 3. 每一层只写服务当前印象的细节
不要把头发、眼睛、鼻子、嘴巴按器官顺序点名。
每补一个细节，都问：
- 它强化了什么印象？
- 观察者为什么此刻才注意到它？
- 它会引出什么感受？

### 4. 复杂气质用“递进”，不要用“并列打架”
人可以复杂，但外貌感受必须有路径。
- 可递进：病弱 → 倔强；温柔 → 疏离；漂亮 → 危险
- 易打架：娇柔 + 刀锋般冷厉，如果要并存，必须写出先后和转折
**不要为了显得特别，硬把互相拆台的形容词并排摆上去。**

### 5. 先写观察，再写判断
多写：
- 先看到什么
- 近了又发现什么
- 最后为什么改了印象
少写作者直接下结论。
让读者跟着观察者一起形成判断。

### 6. 只抓 2-3 个最能记住的点
初见时优先保留：
- 一个整体印象
- 一个中层细节
- 一个最小但最扎人的记忆点
比起一整页五官，读者更容易记住“一双太冷的眼”和“病白的脸色”。

### 7. 高颜值也别只会说“好看”
如果角色设定是美人或帅哥，除了正面描写，再补 1-2 个自然反应：
- 别人多看一眼
- 对话时短暂停顿
- 不自觉放轻语气
- 误判对方性格或身份
**侧面反应要像生活反应，不要全世界一起发呆。**

### 8. 首次出场别写尽，后文再补
第一次只打最强的一枪。
后续在新场景里补：
- 笑起来时才看得出的变化
- 情绪上来才露出的锋利或脆弱
- 关系拉近后观察者才注意到的微小细节
这样外貌才会越写越活，不会一段写完就死。

### 9. 最后做一致性检查
- 去掉“漂亮、清冷、娇柔、锋利”这些结论词后，画面还站得住吗？
- 同一段里有没有互相打脸的形容？
- 读者最后记住的是一个清晰的人，还是一串形容词？

## 常用模板
### 模板A：初见外貌
远看整体轮廓 → 走近补身形或穿着 → 对视写眼神 → 最后给一处最小但最扎人的细节

### 模板B：反差外貌
先给表层印象 → 近看发现不一样的地方 → 用一个细节解释这份反差为什么成立

## 快速示例
### 输入
- 角色：高挑但消瘦的女生
- 目标：写出“先冷，再惹人心疼”

### 输出
远远看去，她高挑得有些过分，薄薄立在风里，像一截快被吹折的竹。走近了才发现她并不是单薄得难看，肩线和腰身都收得利落，只是瘦得过分，衣裙压不住那股清冷。再近些，对上那双眼睛，冷意先一步逼过来，像是不许任何人靠近。可她脸色又白得近乎病态，嘴唇抿得很紧，像把所有不舒服都咽了回去，于是那点生硬的冷，反倒衬出一股让人想护着的倔强。

## 注意事项
- **禁止**一上来并列三个互相打架的气质词
- **禁止**按器官顺序机械报菜名
- **禁止**所有细节都在第一句交代完
- **优先**写观察路线，而不是作者说明书
