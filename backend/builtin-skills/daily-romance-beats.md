把感情流日常写成 情绪有起伏、关系有增量 的小场面，不要只记两个人做了什么。**感情流的波澜不一定靠虐，更多时候靠心理落差、意外惊喜、被看见、被接住。**

## 适用情况
- 吃饭、散步、探病、聊天、送东西写得像流水账
- 同样是甜，自己的互动就是不带劲
- 每章都有同框，但关系像原地踏步
- 不知道一章该把情点放哪里

## 输入
- 当前关系阶段
- 这场互动的日常壳：吃饭 / 送礼 / 同路 / 探病 / 并肩办事
- 这场想推进的情感变化
- 双方人设、身份限制和当前心结

## 步骤
### 1. 先定这场的起点和终点
写清：
- 开始时，谁对谁是什么判断和情绪
- 结束时，谁要发生什么细微变化

没有变化，这场戏就很容易变成记账。

### 2. 设计一个 原以为 与 实际得到 的落差
常用落差：
- 以为只是例行公事，结果被特殊对待
- 以为对方不懂自己，结果被悄悄记住
- 以为今天要受委屈，结果被护住了面子
- 以为只能忍着，结果被雪中送炭
- 以为关系还停在原地，结果被推进半步

**甜点往往长在落差里，不长在平铺直叙里。**

### 3. 一章只抓一个最值钱的情点
如果这章重点是 被看见，就重点写这个。
如果重点是 小吃醋，就让别的内容让路。

宾客、景色、流程、闲聊都可以有，但不能和本章重点抢力气。

### 4. 用五步写法排这场日常
旧状态 → 触发细节 → 情绪翻面 → 双方回响 → 关系余波

可直接套用：
- 旧状态：他以为这只是普通应酬
- 触发细节：对方点的全是自己喜欢却从没说出口的菜
- 情绪翻面：他从意外到被击中
- 双方回响：对视、试探、嘴硬、接话
- 关系余波：第一次意识到对方一直在观察自己

### 5. 甜要贴人设和身份，不要换谁都能用
同样是 准备一桌菜：
- 对普通人，可能只是体贴
- 对一个从不能公开偏好、长期压抑自己的人，就是被理解、被照顾、被保全面子

先问：**这份照顾为什么对这个人特别值钱？**

### 6. 不一定要虐，但一定要有波澜
可用的轻波澜：
- 欲扬先抑
- 小误判后被纠正
- 明面克制、暗里偏爱
- 轻微吃醋、试探、让步
- 被人群包围时仍然被精准照顾

### 7. 每场戏至少推进 1 项
- 好感增加
- 信任增加
- 边界改变
- 默契增加
- 对对方有了新判断
- 埋下下一次更近一步的理由

如果推进 0 项，这场戏再甜也容易发平。

## 输出格式
1. 当前关系阶段
2. 本场起点情绪 / 终点情绪
3. 核心落差
4. 五步场面设计
5. 本章最值钱的情点
6. 需要略写的次要内容
7. 本场关系增量

## 快速示例
### 平写
- 两个人一起吃饭，一个给另一个夹菜、剥虾，对方开心，结束。

### 改写方向
- 先设定其中一人平时不能表现偏好，也从没说过自己爱吃什么。
- 让另一人精准点到他的口味和忌口，而且照顾到了他的身份和体面。
- 重点写 从没被这样对待过 到 被看见 的心理翻面。
- 宴席来了谁、景色多美、流程多完整，都略写。

## 注意事项
- **禁止**只记互动流程，不写情绪变化
- **禁止**同一章平均用力，什么都写一点
- **禁止**甜戏结束后关系毫无变化
- **禁止**把所有波澜都理解成虐
- 感情流日常真正好看，是因为**这一场结束后，他们对彼此不一样了**。
