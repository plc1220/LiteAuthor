用一个背景轻、分歧明的小案件或小风波，让多名角色在同一场戏里各自表态，顺手写出立场、关系和团队生态。重点不是把案子写大，而是借案子把人写活。

## 适用情况
- 配角很多，读者记不住
- 职场、官场、校园、宗门、团队文需要快速立群像
- 主线大案之间需要过渡，但不想写成流水账
- 想让二线人物进场，却不想给每人单开支线

## 输入
- 当前群像名单
- 想让每个角色暴露的点
- 可用的小案件或小争议
- 主角本场要被推向什么位置

## 核心原则
1. **小案件要小在背景，不小在分歧。**
2. **每个角色只暴露一两个最值钱的点**，别一场戏全写完。
3. **发言必须带立场、经验或利益**，不能像同一个作者轮流发言。
4. **主角必须被卷进去**，否则只是配角聊天。
5. **场后要有余波**：改观、结盟、记恨、重新定位，至少留一样。

## 步骤
### 1. 选一个“解释成本低、站队空间大”的小案子
优先选择：
- 规则边界模糊
- 道德和现实会打架
- 不用长篇交代前因后果
- 一听就明白争议点

### 2. 先给每个角色分“这场该露什么”
从下面里每人只拿 1—2 项：
- 价值观
- 职业习惯
- 说话方式
- 胆量
- 洞察力
- 地位
- 和主角的关系
- 会不会和稀泥 / 会不会站出来

### 3. 给每人一句代表性表态
不要长篇雄辩。每人只要一句最像他的判断，外加一个动作或反应即可。

常见分工：
- 权威派：先定规则框架
- 现实派：先算代价
- 理想派：先讲原则
- 油滑派：先躲责任
- 起哄派：先看热闹

### 4. 安排表态顺序
推荐顺序：
1. 高位角色先定调
2. 两三种立场碰撞
3. 主角被逼表态或做选择
4. 关键配角用一句最能代表他的发言抢出记忆点

### 5. 让“小案子”反咬主线
本场结束前，至少接回主线一项：
- 主角的判断被推动
- 某配角正式站到主角这边或对面
- 团队关系变化
- 新行动被引出
- 读者更清楚这个群体的生态

### 6. 控制篇幅
小案件的作用是“亮人”，不是“自己长成一条大支线”。
如果案情解释比人物表态还多，说明写偏了。

### 7. 做群像检查
写完后逐个回答：
- 读者能不能用一句话说出这个人本场的立场？
- 这句立场和他的人设有没有绑定？
- 删掉这个人的发言，场面会不会少一种角度？

## 输出格式
1. 小案件 / 小争议一句话
2. 参与人物清单
3. 每人要暴露的点
4. 表态顺序
5. 主角被推到的处境
6. 场后余波
7. 哪些发言太像作者、需要改掉

## 示例
### 输入
- 题材：职场 / 司法群像
- 需求：让办公室里几名二线人物迅速立住
- 小案件：要不要追究酒驾司机的副驾驶

### 输出
- 争议点：副驾驶没有亲自开车，但是否该承担责任
- 群像暴露：
  - 领导：重效率，倾向少惹麻烦
  - 理想派同事：强调原则和示范性
  - 现实派同事：强调实务边界和证据难度
  - 男主：两边都能理解，暴露他当下的摇摆
  - 女主：一句犀利反问，把男主从被围攻的位置里救出来，顺手立住业务能力和气场
- 余波：读者不仅知道这个小案子的争议，更记住了办公室每个人各自怎么想、怎么说话、会不会站人

## 注意事项
- **不要**为了写群像，给小案件补一整套完整流程
- **不要**让所有角色都说“正确答案”
- **不要**让配角轮流替作者讲道理
- **不要**让主角完全置身事外
