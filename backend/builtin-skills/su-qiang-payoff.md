先把这段戏的爽感拆成“主角到底强在哪、怎么让读者看见、反派怎么一步步被压住”。**苏强不是一句标签，而是能力差 + 反击节奏 + 多方反应的组合。**

## 适用情况
- 女主/男主设定很强，写出来却差一口气
- 打脸回合有结果，但体感不够爽
- 只会写“别人很弱，主角赢了”，场面单薄
- 反击写得太憋屈或太平
- 想写主角轻松压制、从容翻盘、强势出场

## 输入
- 主角这段要突出的苏强点
- 主角比对手强在哪
- 对手是谁，想压主角什么
- 在场见证者有哪些
- 这段想给读者的主要体感：从容、狠、爽、轻蔑、反差等

## 步骤
### 1. 先钉死这段到底爽在哪
用一句话写：
`这段不是单纯让主角赢，而是让读者看到主角在____上压过对手。`
常见落点：
- 实力碾压
- 头脑更快
- 情绪更稳
- 嘴上不输、手上更狠
- 被挑衅后当场打回去
- 明明随手就能挡，还留有余裕

### 2. 把“苏强”拆成可见能力差
不要只写“她很强”“他很苏”，要写清：
- 主角具体会什么
- 对手具体差在哪
- 这场冲突里，主角凭什么能轻松挡、反压、反制

差：
- 女主很厉害，所以所有人都怕她

好：
- 女主身法更快、判断更稳、出手更准，因此对手连碰都碰不到

### 3. 先定矛盾中心，再安排侧面
这类场景里通常有两层人：
- **矛盾中心**：主角 vs 对手
- **侧面见证**：围观者、小弟、朋友、同伴、群众

中心负责打，侧面负责放大体感。**围观者不是凑热闹，而是用来证明局势变化。**

### 4. 至少从 5 个层面一起展现
同一段苏强，优先同时调动：
1. **行动**：主角怎么挡、怎么反打、怎么碾压
2. **语言**：主角说话是否稳、冷、嘲、淡定
3. **主角反应**：是否从容、嫌烦、轻松、游刃有余
4. **对手反应**：震惊、恼羞、逞强、破防、恐惧
5. **他人反应**：围观起哄、改站队、敬畏、看傻

**只写动作，爽感会单薄；多层一起上，体感才会立起来。**

### 5. 反击优先写成“不憋屈递进”
欲扬先抑不是唯一方法。很多苏强场景更稳的节奏是：
- 对手先试探/先挑衅
- 主角不憋屈地挡住或小反击
- 对手不服，继续加码
- 主角彻底打脸，拿回主动权

这种写法比“先吃很大亏，最后一下反杀”更适合强主角。

### 6. 用三回合把爽点抬起来
- **回合 1：试探**
  - 对手出招
  - 主角轻松化解
  - 读者先意识到：主角没那么好惹
- **回合 2：加码**
  - 对手恼羞成怒，升级手段
  - 围观者、同伴、环境开始站队或起哄
  - 局势变热
- **回合 3：彻底打脸**
  - 主角用实力/判断/气场狠狠干回去
  - 对手丢面子、丢主动权、丢气势
  - 见证者完成“这人真的强”的确认

### 7. 笔墨重点放在真正解气的位置
如果这场戏的重点是后半段彻底打脸，后半段就要更细：
- 动作怎么成立
- 对手怎么破防
- 他人怎么反应
- 主角怎么收尾

**不要前面铺很长，真正最爽的地方一笔带过。**

### 8. 细节要贴人设，不要乱用通用词
反派和主角的细节描写都要贴合人设：
- 暴躁型反派：更适合铁青、咬牙、失控、硬撑
- 爱面子型反派：更适合挂不住、强撑气势、当众下不来台
- 从容型主角：表情少、话少、动作干脆
- 轻佻型主角：可以带笑、带玩味、边打边嘲

**不要为了“有细节”随便塞不贴人设的反应。**

### 9. 粗暴词慎用，别伤体感
如果不是题材和人物必须，少用过量脏话、粗口堆气势。很多时候，**真正让人觉得强的不是骂得多脏，而是主角越淡定，对手越狼狈。**

### 10. 收尾要留下局势变化
结尾至少回收 1—2 项：
- 对手服软或更怕了
- 围观者态度反转
- 主角威望上升
- 后续更大冲突被引出来
- 这场戏成为别人重新认识主角的节点

## 输出格式
1. 这段要突出的苏强点
2. 主角和对手的能力差
3. 矛盾中心/侧面见证者
4. 五层展现清单
5. 三回合递进表
6. 重点详写段
7. 人设贴合细节
8. 收尾后的局势变化

## 快速示例
### 输入
- 女主被地痞围堵
- 想写“女主看着纤弱，实际又强又稳”

### 输出
- 苏强点：不是苦战取胜，而是被围住时依然从容
- 能力差：对手只会蛮力，女主速度、判断和爆发都高一截
- 五层展现：
  - 动作：轻松闪避、突然反击
  - 语言：不急不怒，反问和嘲讽都很轻
  - 主角反应：嫌烦、像在赶时间
  - 对手反应：从轻视到恼羞，再到发怵
  - 围观者反应：起哄 → 震惊 → 敬畏
- 三回合：
  1. 对手挑衅，女主轻松躲开
  2. 对手叫人围攻，女主仍有余裕
  3. 女主直接狠狠干翻，对手当众服软

## 注意事项
- **禁止**只写“主角赢了”，不写怎么赢、别人怎么看见
- **禁止**让强主角长期憋屈却不给当场回击
- **禁止**所有人反应一个样，只会“震惊”
- **禁止**真正最爽的段落写得过快、过虚
- 苏强场景最值钱的，不是“主角很强”的结论，而是**读者亲眼看到她/他怎么强、怎么把场子压住。**
