把话收着说，把情绪放进对话、神态、第三者视角、场景气味和事后痕迹里。**暧昧最怕太用力；亲密最怕只剩刺激。**

## 适用情况
- 想写互有情愫却未挑明的关系
- 暧昧戏一写就太直白
- 想含蓄写亲密场面，不想把每个动作正面写尽
- 亲密戏总和人物、情节脱节
- 想让读者自己品出来，而不是角色直接说出来

## 核心原则
1. 暧昧是中间状态：似是而非、欲语还休、爱恨两难。
2. **忌讳把话说得太实在。** 先让读者感觉到，再让角色承认。
3. 不只靠两人正面对话，也可借第三者的眼光、言语、误解、旁观来成立关系。
4. 气氛是放大器，不是独立风景。香气、寂静、雨声、封闭空间，都要服务人物关系。
5. 亲密描写必须至少承担一项功能：揭示人物、推进情节、改写关系、呼应主题。否则删。
6. 很多场面可以用**事后痕迹**代替正面描写，让读者自己补完。

## 步骤
### 1. 先写不能明说的东西
填空：
- A 此刻真正想要的是____
- A / B 为什么不能明说____

### 2. 选暧昧载体
从下面选 2—3 个：
- 对话里的半句话、反话、试探
- 神态、停顿、视线、呼吸变化
- 第三者的观察、误会或插话
- 气氛细节：香、雨、夜、狭小空间、突然的安静
- 事后痕迹：衣褶、抓痕、物件移位、次日反应

### 3. 写“说一半，留一半”
常用做法：
- 关键话不说满
- 嘴上岔开，动作却承认
- 让第三者先看出来
- 直接写结果或余波，不把中间过程全部摊开

### 4. 用场景给情绪加压
不要空写气氛，必须回答：
- 这股气氛有没有逼得人物更靠近
- 它有没有让谁更失控、谁更退缩
- 删掉这层气氛后，关系张力会不会明显变弱

### 5. 写亲密场面时先做功能检查
逐条勾选至少一项：
- 暴露人物更隐秘的一面
- 让两人关系真正变化
- 引出后续误会、承诺、代价或冲突
- 回应主题

若全都没有，删。

### 6. 能不正写时，优先“背面写”
可用：
- 次日清晨的疲惫、伤痕、凌乱
- 某个小物件的损坏、遗留、错位
- 一句轻描淡写却信息量很大的回味

让读者通过余波倒推出昨夜发生了什么。

### 7. 最后查分寸
- 读者能不能看懂两人有情
- 这场有没有具体证据
- 有没有太满、太硬、太像作者在推
- 亲密部分删掉后，故事是否照样成立；若成立，多半写多了

## 输出格式
1. 这场不能明说的感情 / 欲望
2. 选用的载体
3. 暧昧线索
4. 气氛细节
5. 可用的事后痕迹
6. 这场推进了什么
7. 需要删掉的直白表达

## 快速示例
### 输入
- 两人互有情意，但谁都不愿先挑明

### 输出
- 不能明说的东西：A 已经把 B 看得比别人重
- 载体：对话试探 + 第三者误会 + 雨夜室内气氛
- 暧昧线索：话说半截，明明要走又停下
- 事后痕迹：第二天对方袖口留下一道抓痕，谁都不解释
- 推进：关系从“心照不宣”升级到“不能再假装没有”

## 注意事项
- **不要**把暧昧写成读者看不懂
- **不要**为刺激而写亲密戏
- **不要**每次都正面写满，中间留白常常更有劲
- **不要**让气氛脱离人物动作独立存在
- 好看的感情戏，不是作者告诉读者他们很爱，而是**读者自己从细节和余波里品出来。**
